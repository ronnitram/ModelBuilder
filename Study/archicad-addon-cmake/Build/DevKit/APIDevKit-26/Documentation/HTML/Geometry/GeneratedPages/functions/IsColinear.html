Geometry::IsColinear Function

Returns true whether the direction of the two vectors is same or reverse.

template <class NumberType>
bool IsColinear (
    const Geometry::Vector2< NumberType > & v1,
    const Geometry::Vector2< NumberType > & v2
);

Template parameters

NumberType
The type of the number is set by the developer.

Parameters

v1
[in] The first vector.
v2
[in] The second vector.

Return Value

Returns true whether the direction of the two vectors is same or reverse. Otherwise false.

Description

This method returns true whether the direction of the two vectors is same or reverse.
