Geometry::TMBCalcRottran

Initializes a transformation matrix of a rotation around a vector.

     void TMBCalcRottran (
      COORD3*   t,
      double    alpha,
      BASTRAN*  rottran
    );

Parameters

t
Coordinates of the direction vector of the rotation axis.
alpha
Angle of rotation.
rottran
Transformation matrix.
