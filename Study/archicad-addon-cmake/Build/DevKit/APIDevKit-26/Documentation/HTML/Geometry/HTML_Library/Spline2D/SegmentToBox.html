Geometry::SegmentToBox

Calculates the minimal bounding box of a spline segment between [a..b]. a and b describes only this segment. 0 <= a <= b <= 1.

     void SegmentToBox (
      Box2DData*   boundbox,
      double           x0,
      double           y0,
      double           x1,
      double           y1,
      double           x2,
      double           y2,
      double           x3,
      double           y3,
      double           a,
      double           b
    );

Parameters

boundbox
Pointer to the calculated bounding box.
x0
x coefficient of the constant term.
y0
y coefficient of the constant term.
x1
x coefficient of the linear term.
y1
y coefficient of the linear term
x2
x coefficient of the quadratic term.
y2
y coefficient of the quadratic term.
x3
x coefficient of the cubic term.
y3
y coefficient of the cubic term.
a
Starting parameter of the boxed spline segment. For the whole segment set it to zero.
b
Ending parameter of the boxed spline segment. For the whole segment set it to one.

Requirements

Version: 1.0 or later
Header: Spline2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
