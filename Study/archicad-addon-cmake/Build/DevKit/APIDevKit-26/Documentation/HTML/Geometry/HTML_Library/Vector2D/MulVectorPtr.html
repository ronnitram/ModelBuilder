Geometry::MulVectorPtr

Scales a vector by a factor.

    Vector* MulVectorPtr (
      Vector*  v,
      double   r
    );

Parameters

v
Pointer to the vector to scale. It is changed by the operation.
r
The scaling factor.

Return Values

Popinter to the result of the scaling. It equals v.

Requirements

Version: 1.0 or later
Header: Point2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
