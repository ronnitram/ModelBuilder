Geometry::TM2dNoRot

Initializes a 2D transofrmation matrix with scaling, translation and no rotation.

     void TM2dNoRot (
      double    xs,
      double    ys,
      double    xd,
      double    yd,
      TRANMAT*  tran
    );

Parameters

xs
Pointer to the x coordinate of the scaling.
ys
Pointer to the y coordinate of the scaling.
xd
Pointer to the x coordinate of the translation.
yd
Pointer to the y coordinate of the translation.
tran
Pointer to the transformation matrix.

Remarks

The transformation matrix is equvivalent to the next two GDL command:

        MUL2  xs, ys
        ADD2  xt, yt

Requirements

Version: 1.0 or later
Header: TM.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
