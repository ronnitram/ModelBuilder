Geometry::operator * Function

Returns the result of the scalar (or vectorial) product.

template <class NumberType>
Geometry::Vector2< NumberType > operator * (
    const NumberType &  num,
    const Geometry::Vector2< NumberType > & v2
);

This method returns the result of the vectorial product.

template <class NumberType>
NumberType operator * (
    const Geometry::Vector2< NumberType > & v1,
    const Geometry::Vector2< NumberType > & v2
);

Template parameters

NumberType
The type of the number is set by the developer.

Parameters

num
[in] The number parameter of the scalar product.
v2
[in] The vector.
v1
[in] The vector.

Return Value

Returns the result of the scalar product.

Returns the result of the vectorial product.
