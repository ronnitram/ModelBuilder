Geometry::SplineSegNormTangPoints

Determines those points of a spline segment where the tangent of the spline is either perpendicular or parallel to the line connecting the actual point to a base point. Only one point is cosen for each pixel.

     long SplineSegNormTangPoints (
      Coord**       coords,
      DirType**     dirs,
      long          segment,
      const Coord*  baseC,
      double**      hSect,
      double        xPixelSize,
      bool          normal
    );

Parameters

coords
Coordinates of the spline.
dirs
Directions of the spline.
segment
Index of the spline segment.
baseC
The base point.
hSect
Array for the points of the spline.
xPixelSize
The size of a pixel.
normal
If it is true, those points are determined where the tangent of the spline is perpendicular to the line between the abse point and the actual point. If it is false, the points wiht parallel tangent are chosen.

Return Values

Number of the spline point where the tangent are perpendicular or parallel, based on the value of normal.

Requirements

Version: 1.0 or later
Header: Spline2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
