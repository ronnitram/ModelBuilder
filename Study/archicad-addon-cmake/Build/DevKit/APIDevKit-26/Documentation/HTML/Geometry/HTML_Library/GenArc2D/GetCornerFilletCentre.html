Geometry::GetCornerFilletCentre

Gives the centre of the circle with a given radius which is tangent to two curves - straight line or arc.

     bool GetCornerFilletCentre (
      Coord*  cprev,
      Coord*  c0,
      Coord*  cnext,
      long    indf,
      Coord*  oprev,
      double  fprev,
      long    inds,
      Coord*  onext,
      double  fnext,
      bool    d1,
      bool    d2,
      double  disp,
      bool    dflag,
      Coord*  newc
    );

Parameters

cprev
Beginning point of the first circle.
c0
End point of the first and beginning point of the second circle.
cnext
End point of the second circle.
indf
Index of the first circle.
oprev
First center point.
fprev
First angle.
inds
Index of the second circle.
onext
Second center point.
fnext
Second angle.
d1
First offset flag.
d2
Second offset flag.
disp
Displacement.
dflag
Displacement direction flag.
newc
Coordinates of the newly calculated point.

Return Values

true
The two boxes are intersecting, i.e. boundBox may enclose one or more coordinates.
false
Otherwise.

Requirements

Version: 1.0 or later
Header: GenArc2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
