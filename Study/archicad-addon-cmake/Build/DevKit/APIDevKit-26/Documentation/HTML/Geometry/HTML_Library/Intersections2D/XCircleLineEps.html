Geometry::XCircleLineEps

Calculates the intersection points (stored in a sector structure) of a circle and a sector.

     char XCircleLineEps (
      const Circle*  c,
      const Sector*  s,
      Sector*        xs,
      double         eps
    );

Parameters

c
Pointer to the circle.
s
Pointer to the sector representing the line.
xs
Pointer to the sector with the intersection points as end points.
eps
Threshold value for the calculation.

Return Values

The right end bit is set to one, if the first point of xs is a valid intersection point.
The bit next to the right end bit is set to one, if the second point of xs is a valid intersection point.

Requirements

Version: 1.0 or later
Header: Circle2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
