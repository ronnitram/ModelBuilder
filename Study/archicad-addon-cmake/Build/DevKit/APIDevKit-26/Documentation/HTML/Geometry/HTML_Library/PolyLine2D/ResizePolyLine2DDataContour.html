ResizePolyLine2DDataContour

Enlarges or shrinks a contour of a polyline. The result will be regularized.

     GSErr ResizePolyLine2DDataContour (
      const Polygon2DData&     poly,
      long                     contourIdx,
      double                   offset,
      VAArray<Polygon2DData>*  resultPolys
    );

Parameters

poly
Polyline that the enlarged/shrunk contour belongs to.
contourIdx
Index of the enlarged/shrunk contour.
offset
Extent of the resize.
resultPolys
Pointer to the array of result polylines.

Return Values

Returns NoError, if operation is completed successfully.

Remarks

Contour is enlarged when offset is positive, and shrunk when offset is negative.

Requirements

Version: 1.0 or later
Header: Polygon2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
