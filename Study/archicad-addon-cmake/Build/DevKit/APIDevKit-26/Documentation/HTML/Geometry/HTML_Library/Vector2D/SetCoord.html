Geometry::SetCoord

Creates a Coord stucture from coordinates in a Cartesian system.

    Coord SetCoord (
      double  x,
      double  y
    );

Parameters

x
The x coordinate used in the initialization.
y
The y coordinate used in the initialization.

Return Values

The created Coord structure.

Remarks

The difference between SetCoord and SetCoordPtr is that SetCoord gives back a newly created Coord (it could take longer time, if the compiler is not makes function-ending optimization), while SetCoordPtr initializes the Coord given in the parameter list.

Requirements

Version: 1.0 or later
Header: Point2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
