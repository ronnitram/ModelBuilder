PGRelPolyPosExt

Relative position of two polygons.

    typedef enum PGRelPolyPosExt {
        UnkownStatus=-1,
        PolysOverlap=0,
        Poly1CoversPoly2=1,
        Poly2CoversPoly1=2,
        PolysEqual=3,
        PolysDisjunct=4,
        PolysDisjunct_Node=5,
        PolysDisjunct_Edge=6
    } PGRelPolyPosExt;
Enumerator Value Meaning
UnkownStatus Unknown
PolysOverlap The two polygons have a real intersection.
Poly1CoversPoly2 Poly1 entirely covers poly2.
Poly2CoversPoly1 Poly2 entirely covers poly1.
PolysEqual The two polygons equal.
PolysDisjunct The two polygons have no intersection.
PolysDisjunct_Node The polygons have only overlapping vertices.
PolysDisjunct_Edge The polygons have only overlapping edges.
