PGPOLYEXT

Describes a callback polygon.

    struct PGPOLYEXT {
      PGPOLYSTATUSEXT   status;
      long              nEdges;
      Box2DData         boundBox;
      double            epsilon;
      double            angleEpsilon;
      double            minVertexDist;
      GSSize            sizeOfData;
      GSPtr             data;
      GSSize            sizeOfVertexData;
      GSSize            sizeOfInhEdgeData;
      GSSize            sizeOfUniqEdgeData;
      GSPtr             defaultVertexData;
      GSPtr             defaultInhEdgeData;
      GSPtr             defaultUniqEdgeData;
      GetPGEdgeProcExt  getEdge;
      NewPGEdgeProcExt  newEdge;
      GSTime            modiTime;
   };

Members

status
Status information.
 
nEdges
The number of edges in all contours. The item numbering runs from one to "nEdges".
 
boundBox
The minimal bounding box of the polygon.
 
epsilon
The threshold to decide from two reals to be equal or not.
 
angleEpsilon
The threshold to decide from two angles to be equal or not.
 
minVertexDist
The minimal distance which must be between two vertices.
 
sizeOfData
Size of the attached data in bytes.
 
data
Attached data.
 
sizeOfVertexData
Size of one vertex data.
 
sizeOfInhEdgeData
Size of one inheriting edge data.
 
sizeOfUniqEdgeData
Size of one unique edge data.
 
defaultVertexData
The default of attached vertex data.
 
defaultInhEdgeData
The default of attached inheriting data.
 
defaultUniqEdgeData
The default of attached unique data.
 
getEdge
The user-defined callback function which can give back an edge with random indexing.
 
newEdge
The user-defined callback function which can receive an edge.
 
modiTime
The last modification date of the polygon. It is used in internal caching. If you do not want to use it, simply set this field to zero.
 

Requirements

Version: 1.0 or later
Header: Polygon2DOperations.h
