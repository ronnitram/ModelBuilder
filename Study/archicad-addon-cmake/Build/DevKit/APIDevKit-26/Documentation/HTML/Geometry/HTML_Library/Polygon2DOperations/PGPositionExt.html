PGPositionExt

Describes the relative position of a polygon and a point or sector.

    typedef enum PGPositionExt {
        PGInsidePoly=0,
        PGOutsidePoly=1,
        PGOnPolyEdge=2
    } PGPositionExt;
Enumerator Value Meaning
PGInsidePoly Entirely inside.
PGOutsidePoly Entirely outside.
PGOnPolyEdg On edge or across edge.

Requirements

Version: 1.0 or later
Header: Polygon2DOperations.h
