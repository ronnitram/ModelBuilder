Geometry::MidAngle

Gives the bisector direction of two directions.

    double MidAngle (
      double  alfa,
      double  beta
    );

Parameters

alfa
First direction.
beta
Second direction.

Return Values

The bisector direction.

Remarks

The first and second directions are not interchangeable!

Requirements

Version: 1.0 or later
Header: AngleData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
