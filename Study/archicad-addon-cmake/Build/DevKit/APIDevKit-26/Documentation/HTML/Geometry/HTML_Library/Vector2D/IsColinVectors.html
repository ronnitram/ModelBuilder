Geometry::IsColinVectors

Tests wether two vectors are in one line. Their directions can be either equal or opposite to accept them as colinear.

    bool IsColinVectors (
      const Vector&  v1,
      const Vector&  v2
    );

Parameters

v1
The first vector.
v2
The second vector.

Return Values

true
The two vectors are colinear, or at least one vector has zero length.
false
The two vectors are not colinear.

Requirements

Version: 1.0 or later
Header: Point2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
