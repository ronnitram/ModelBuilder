Geometry::ComputeEllipseCoord

Calculates the coordinates of a point that has a given perimeter angle relative to an ellipse.

     void ComputeEllipseCoord (
      const GenArc*  pArc,
      double         fi,
      Coord*         pt
    );

Parameters

pArc
The arc representing the ellipse.
fi
The preimeter angle of the point.
pt
Calculated coordinates of the point.

Requirements

Version: 1.0 or later
Header: Ellipse2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
