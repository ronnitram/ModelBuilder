Geometry::SplineLen

Gives the length of a spline piece between [a..b] global parameters.

     double SplineLen (
      Coord**    coords,
      DirType**  dirs,
      double     a,
      double     b
    );

Parameters

coords
Coordinates of the spline.
dirs
Directions of the spline.
a
Global spline parameter of the first point.
b
Global spline parameter of the second point.

Return Values

The length of the spline between the two points.

Remarks

Global spline parameter = segment index + spline parameter in the given segment.

Requirements

Version: 1.0 or later
Header: Spline2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
