Geometry::XSplineGenArc

Calculates the intersection of a spline and a GenArc.

     long XSplineGenArc (
      long      nCoords
      Coord*    pCoords,
      DirType*  pDirs,
      bool           closed,
      const GenArc*  arcT,
      double**       hSect
    );

     long XSplineGenArc (
      Coord**        coords,
      DirType**      dirs,
      bool           closed,
      const GenArc*  arcT,
      double**       hSect
    );

Parameters

nCoords
Number of Coodinates (at pointer version).
pCoords / coords
Control points of the Bezier quad points of the spline.
pDirs / dirs
Directions of the Bezier quad points of the spline.
closed
If true, spline is closed.
arcT
Pointer to the generic arc.
hSect
Coordinate values of the intersection points.

Return Values

Number of the intersection points.

Requirements

Version: 1.0 or later
Header: Spline2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
