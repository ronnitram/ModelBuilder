PolyArcRec

Describes the arc information of a curved polygon edge.

    struct PolyArcRec {
      long        begIndex;
      long        endIndex;
      double      arcAngle;
    };

Members

begIndex
Starting vertex index of the edge.
 
endIndex
Destination vertex index of the edge.
 
arcAngle
The view angle (deflection) of the arc. The 0.0 value means straight edge, and not permitted here.
 

Requirements

Version: 1.0 or later
Header: Polygon2DData.h
