GetPGEdgeProcExt callback

User-defined callback function which can give back an edge with random indexing.

    typedef PGEDGEEXT (*GetPGEdgeProcExt) (
      const PGPOLYEXT*  polygon,
      long              ind
    );

Parameters

polygon
The polygon which edge is queried. The function is the member of this polygon structure.
ind
The index of the queried edge.

Return Values

@@@_Return_values_@@@

Remarks

You can write this callback function so if it is called you make a PGEDGEEXT structure and give it back if and algorhytm needs it.
