Geometry::ArcGetOrigo

Gives the centre of a circle arc. The angle is the viewangle of the two endpoints and its negative value means negative orientation.

     bool ArcGetOrigo (
      const Coord*  begC,
      const Coord*  endC,
      double        angle,
      Coord*        origo
    );

Parameters

begC
Coordinates of the beginning of the arc.
endC
Coordinates of the end of the arc.
angle
View angle of the arc from its center.
origo
Coordintaes of the center of the arc.

Return Values

Returns true if the operations performed without any problems. false if distance of begC and endC or angle is near zero.

Requirements

Version: 1.0 or later
Header: GenArc2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
