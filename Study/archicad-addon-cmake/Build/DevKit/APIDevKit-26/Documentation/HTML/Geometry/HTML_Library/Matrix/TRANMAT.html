TRANMAT

The homogeneous transformation matrix used in Geometry.

    typedef struct TRANMAT {
      short    status;
      short    dummy1;
      short    dummy2;
      short    dummy3;
      double   tmx[12];
    } TRANMAT;
Constant Description
TR_IDENT Identical transformation. (Ones in the diagonal.)
TR_DET_1 The determinant of the upper left 3x3 matrix is one.
TR_SINGULAR The determinant of the upper left 3x3 matrix is zero.
TR_MIRROR The transformation described by the matrix changes the orientation of the elements.
TR_TRANSL_ONLY The transformation described by the matrix is translation only.
TR_SCALE_ONLY The transformation described by the matrix is a scale only.
TR_ROTATE_ONLY The transformation described by the matrix is a rotation only.
TR_TRANSL_INCL The transformation described by the matrix includes translation.
TR_SCALE_INCL The transformation described by the matrix includes scaling.
TR_ROTATE_INCL The transformation described by the matrix includes rotation.
TR_UNDECOMPOSED Used by Archicad internally. (BG.c)

Members

status
Various statuses.
 
dummy1, dummy2, dummy3
Dummy fields for right aligning.
 
tmx[12]
The matrix members in three rows and four columns.
 

Remarks

It differs from the usual transformation matrices in that its size is not 4x4, but 3x4. The missing row is supposed to be {0, 0, 0, 1}.
