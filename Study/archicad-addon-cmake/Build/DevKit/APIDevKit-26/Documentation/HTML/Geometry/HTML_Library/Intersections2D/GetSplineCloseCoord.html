Geometry::GetSplineCloseCoord

Calculates the coordinates and parameter of the spline point that is closest to a given point.

     double GetSplineCloseCoord (
      Coord**       coords,
      DirType**     dirs,
      long          segment,
      const Coord*  cc,
      Coord*        ec
    );

Parameters

coords
Control points of the Bezier quad points of the spline.
dirs
Directions of the Bezier quad points of the spline.
segment
Number of spline segments.
cc
Coordinates of the given point.
ec
Coordinates of the spline point.

Return Values

Parameter of the spline point.

Requirements

Version: 1.0 or later
Header: Spline2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
