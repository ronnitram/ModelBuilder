Geometry::BezierToCubic

Converts the Bezier control points of a spline to the equvivalent cubic representation of a spline.

     void BezierToCubic (
      Coord*   pco0,
      Coord*   pco1,
      Coord*   pco2,
      Coord*   pco3,
      double*  x0,
      double*  y0,
      double*  x1,
      double*  y1,
      double*  x2,
      double*  y2,
      double*  x3,
      double*  y3
    );

Parameters

pco0
Pointer to the coordinates fo the first Bezier point.
pco1
Pointer to the coordinates fo the second Bezier point.
pco2
Pointer to the coordinates fo the third Bezier point.
pco3
Pointer to the coordinates fo the fourth Bezier point.
x0
Pointer to a coefficient of the equation for the cubic representation of the spline.
y0
Pointer to a coefficient of the equation for the cubic representation of the spline.
x1
Pointer to a coefficient of the equation for the cubic representation of the spline.
y1
Pointer to a coefficient of the equation for the cubic representation of the spline.
x2
Pointer to a coefficient of the equation for the cubic representation of the spline.
y2
Pointer to a coefficient of the equation for the cubic representation of the spline.
x3
Pointer to a coefficient of the equation for the cubic representation of the spline.
y3
Pointer to the coefficient of the equation for the cubic representation of the spline.

Remarks

The spline coordinates for any given parameter t between 0 and 1 are
x(t) = x = x0 + (x1 + (x2 + x3 * t) * t) * t
y(t) = x = y0 + (y1 + (y2 + y3 * t) * t) * t
