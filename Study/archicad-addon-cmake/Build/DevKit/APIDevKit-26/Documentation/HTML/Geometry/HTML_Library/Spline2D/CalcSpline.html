Geometry::CalcSpline

Gives the all directions of all segments of a spline. (Converts a Natural spline to a Bezier spline represented by Bezier directions.)

     bool CalcSpline (
      Coord**    coords,
      DirType**  dirs,
      long       coordCount,
      bool       isClosed
    );

Parameters

coords
Points of the spline. Should contain coordCount elements. This is the input of the function.
dirs
Output directions. Should contain coordCount elements. In case of an open spline, its first and last element must be initialized.
coordCount
The count of the coordinates of the spline.
isClosed
If true, the spline is closed in which case the last point connects to the first one. If false, the spline is open.

Return Values

true if the operation is completed successfully, false otherwise.

Remarks

In case of an open spline, the first and last elements of dirs are input, too - they determine the starting and ending direction of the spline.

Requirements
