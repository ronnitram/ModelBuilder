Geometry::ComputeRotFi

Calculates the deflection (signed angle) of rotating one direction to another. The directions are represented by the starting and end points of their direction vectors.

    double ComputeRotFi (
      const Coord*  o,
      const Coord*  c1,
      const Coord*  c2,
      bool          negArc
    );

Parameters

o
Common starting point of the two direction vectors.
c1
End point of the direction vector where the rotation starts.
c2
End point of the direction vector where the rotation ends.
negArc
If it is true, the rotation is clockwise, else counter clockwise.

Return Values

The deflection.

Requirements

Version: 1.0 or later
Header: Point2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
