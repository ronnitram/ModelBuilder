Geometry::EllipseTangentCircle

Aproximates an ellipse between points of [f1, f2] with circle arcs. The maximal distance between the ellipse and the points of the arcs will be less than the given distance.

     bool EllipseTangentCircle (
      const GenArc*  pEll,
      const Coord*   f1,
      const Coord*   f2,
      Coord*         origo,
      double         arcDiff,
      bool*          exist
    );

Parameters

pEll
Arc representing the ellipse.
f1
Begginning coordinates of the approximated ellipse part.
f2
End coordinates of the approximated ellipse part.
origo
Center of the approximating arc.
arcDiff
Maximal allowed distance between the points of the approximating arc and the ellipse.
exist
true if both circles determined by f1, f2, tangent of ellipse in f1 and f1, f2, tangent of ellipse in f2 exist, false otherwise.

Return Values

true if approximating arc exists, false otherwise.

Requirements
