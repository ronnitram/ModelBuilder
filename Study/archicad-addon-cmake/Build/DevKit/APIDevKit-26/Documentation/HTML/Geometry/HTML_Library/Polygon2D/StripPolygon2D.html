StripPolygon2D

Intersects a polygon with a strip (infinitely long ribbon). The result will be regularized.

     bool StripPolygon2D (
      HalfPlane2D*             cutline1,
      HalfPlane2D*             cutline2,
      const Polygon2DData*     poly,
      VAArray<Polygon2DData>*  respoly,
      long*                    wrcode
    );

Parameters

cutline1
First border line of the ribbon.
cutline2
Second border line of the ribbon.
poly
Pointer to the polygon to intersect.
respoly
Pointer to the array of the result polygons.
wrcode
Warning error code. There are 8 codes. The first one is that, "Not enough memory to cut a polygon". The rest of them are "Internal errors".

Return Values

true
The ribbon and the polygon are disjunct or the operation is not completed successfully.
false
The ribbon and the polygon are intersecting.

Requirements

Version: 1.0 or later
Header: Polygon2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
