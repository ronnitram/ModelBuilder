Geometry::EmptyBox3D

Decides if the box is empty, i.e. whether along at least one coordinate axis the maximal coordinate is not greater than the minimal one. The threshold value is SmallEps.

     bool EmptyBox3D (
      const Box3DType*  pCBox
    );

Parameters

pCBox
Pointer to the box.

Return Values

True
Box is empty.
False
Box is not empty.

Requirements

Version: 1.0 or later
Header: Box3DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
