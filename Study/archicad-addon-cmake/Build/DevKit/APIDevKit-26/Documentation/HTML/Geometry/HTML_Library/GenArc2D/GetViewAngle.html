Geometry::GetViewAngle

Calculates the view angle of an arc from its center, beginning and end points.

     void GetViewAngle (
      Coord*   corig,
      Coord*   cbeg,
      Coord*   cend,
      double   negOrientFlag,
      long     arcFlag,
      double*  newfi
    );

Parameters

corig
Coordinates of the center of the arc.
cbeg
Coordinates of the beginning point of the arc.
cend
Coordinates of the end point of the arc.
negOrientFlag
Signs the orientation of the arc. If the orientation is negative, it is smaller than 0.0.
arcFlag
It equals -1 when the arc is straight. In this case the function does nothing.
newfi
The calcuated view angle.

Return Values

Requirements

Version: 1.0 or later
Header: GenArc2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
