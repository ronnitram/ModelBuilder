PGEDGESTATUSEXT

PGEDGESTATUSEXT

The statuses of a PGEDGEEXT. It is the part of it.

    struct PGEDGESTATUSEXT {
      long             isArc             : 1;
      long             isArcRevoriented  : 1;
      long             isBegHole         : 1;
      long             isEndContour      : 1;
      long             isEndPolygon      : 1;
      long             isSplitLine       : 1;
    };

Members

isArc
True if this edge is an arc.
 
isArcRevoriented
True if the arc of this edge is negative oriented.
 
isBegHole
True if this edge is the beginnig of a hole. (Contour other than the first.)
 
isEndContour
True if this edge is the last in a contour. (Including the first.) Must be set even if it is an end of the polygon too.
 
isEndPolygon
True if this edge is the last in the polygon.
 
isSplitLine
True if this line is a splitter line. It's created when a plane splits a polygon on an other plane.
