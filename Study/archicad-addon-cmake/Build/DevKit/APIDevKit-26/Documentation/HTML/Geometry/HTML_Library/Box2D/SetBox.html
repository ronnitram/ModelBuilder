Geometry::SetBox

Initializes a box with its lower and higher coordinates.

     void SetBox (
      Box2DData* abox,
      double           xMin,
      double           yMin,
      double           xMax,
      double           yMax
    );

Parameters

abox
Pointer to the box.
xMin
Lower horizontal coordinate.
yMin
Lower vertical coordinate.
xMax
Higher horizontal coordinate.
yMax
Higher vertical coordinate.

Return Values

No values are returned.

Requirements

Version: 1.0 or later
Header: Box2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
