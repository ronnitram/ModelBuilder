F_Sector

Sector represented by float numbers. Obsolete structure, use it for conversions from old files only.

    typedef struct struct {
      F_Coord  c1;
      F_Coord  c2;
    } F_Sector;

Members

c1
Coordinates of the beginning point of the sector.
 
c2
