Geometry::UnitVectorPtr3D

Initializes a 3D unit vector that is parallel to a given 3D vector.

     double UnitVectorPtr3D (
      const Vector3D*  v,
      Vector3D*        w
    );

Parameters

v
Pointer to the parallel vector.
w
Pointer to the vector to be initialized.

Return Values

Length of the original vector.

Requirements

Version: 1.0 or later
Header: Point3DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
