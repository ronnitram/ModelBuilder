F_BoxType

Obsolete structure for a rectangle box with float coordinates.

    typedef struct F_BoxType {
      float  xMin;
      float  yMin;
      float  xMax;
      float  yMax;
    } F_BoxType;

Members

xMin
Minimal x value of the rectangle.
 
yMin
Minimal y value of the rectangle.
 
xMax
Maximal x value of the rectangle.
