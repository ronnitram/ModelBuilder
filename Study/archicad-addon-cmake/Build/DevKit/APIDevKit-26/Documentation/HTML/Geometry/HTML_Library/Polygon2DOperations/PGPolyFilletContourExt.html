Geometry::PGPolyFilletContourExt

Makes a fillet on all vertexes of the given contour of a polygon. The result will be regularized.

     GSErr PGPolyFilletContourExt (
      const PGPOLYEXT*  polygon,
      long              filletedContourIdx,
      double            filletRadius,
      PGRESPOLYEXT*     newPolygons
    );

Parameters

polygon
The polygon.
filletedContourIdx
Index of the contour to be filleted. If contourIdx is 0 then all contours will be filleted.
filletRadius
Radius of the fillet.
newPolygons
Pointer to the result polygons.

Return Values

Returns NoError, if operation is completed successfully.

Remarks

The function won't change those vertexes which couldn't be filleted with filletRadius value.

Requirements
