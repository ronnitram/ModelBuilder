Geometry::NormFi02Pi

Normalizes a number into the [0..2*PI) interval with given radEps gravity to k*PI/2 (k=0,1,2,3).

    double NormFi02Pi (
      double  angle
          double  radEps
    );

Parameters

angle
The number to be normalized.
angle
The threshold used both for normalizing and gravity.

Return Values

The normalized number.

Remarks

An important example: If the incoming number is small negative but within the radEps range it will be 0.0, otherwise it will be a near 2*PI value.

Requirements
