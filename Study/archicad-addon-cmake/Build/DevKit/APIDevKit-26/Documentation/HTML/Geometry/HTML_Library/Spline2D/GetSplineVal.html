Geometry::GetSplineVal

Gives the point of spline determined by its parameter.

     Coord GetSplineVal (
      long      nCoords
      Coord*    pCoords,
      DirType*  pDirs,
      double     t
    );

     Coord GetSplineVal (
      Coord**    coords,
      DirType**  dirs,
      double     t
    );

Parameters

nCoords
Number of Coodinates (at pointer version).
pCoords / coords
Coordinates of the spline.
pDirs / dirs
Directions of the spline.
t
Parameter of the point.

Return Values

The coordinates of the point.

Requirements

Version: 1.0 or later
Header: Spline2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
