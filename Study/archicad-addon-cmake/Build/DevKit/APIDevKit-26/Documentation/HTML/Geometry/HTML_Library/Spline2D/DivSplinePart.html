Geometry::DivSplinePart

Approximates an spline with given length sectors.

     long DivSplinePart (
      Coord**       coords,
      DirType**     dirs,
      bool          closed,
      bool          forw,
      const Coord&  startCo,
      double        partLength,
      double**      hSect
    );

Parameters

coords
Coordinates of the spline.
dirs
Directions of the spline.
closed
If the spline is closed, it must be true.
forw
If false, the approximation process goes from the end of the spline to its beginning.
startCo
Beginning coordinates of the first approximating sector.
partLength
Length of the approximating sectors.
hCo
Coordinates of the approximating sectors. The i-th element is the beginning coordinates of the i-th and end coordinates of the (i-1)-th sector, except the first and the last, which are the beginning of the first sector and the end of the last sector, respectively.

Return Values

Number of the approximating sectors.

Requirements

Version: 1.0 or later
Header: Spline2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
