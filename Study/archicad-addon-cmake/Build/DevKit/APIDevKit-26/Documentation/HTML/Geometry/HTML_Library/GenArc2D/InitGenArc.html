Geometry::InitGenArc

Initializes a GenArc structure to a zero circle.

     void InitGenArc (
      GenArc*  arc
    );

Parameters

arc
The arc to be initialized.

Remarks

All parameters of the arc are set to zero except the ratio that is set to 1.

Requirements

Version: 1.0 or later
Header: GenArc2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
