Geometry::DivEllipseDiff_Arc

Approximates an ellipse with circle arcs. All arc points are not farther from the ellipse than a given value.

     long DivEllipseDiff_Arc (
      const GenArc*  pEll,
      bool           forw,
      double         diff,
      Coord**        hCo
    );

Parameters

pEll
The arc representing the ellipse to approximate.
forw
If false, direction of ellipse is reversed.
diff
Maximal allowed distance of arc points from ellipse.
hCo
Beginning coordinates of the approximating arcs. End coordinates of a given arc are the beginning coordinates of the next arc.

Return Values

Number of the approximating arcs.

Requirements

Version: 1.0 or later
Header: Ellipse2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
