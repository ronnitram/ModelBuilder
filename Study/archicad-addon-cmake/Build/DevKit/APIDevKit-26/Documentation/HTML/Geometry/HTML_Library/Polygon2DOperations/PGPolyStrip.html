Geometry::PGPolyStrip

Intersects a polygon with a strip (unlimited ribbon). The result will be regularized.

     bool PGPolyStrip (
      HalfPlane2D*      cutline1,
      HalfPlane2D*      cutline2,
      const PGPOLYEXT*  polygon,
      PGRESPOLYEXT*     newPolygons,
      long*             wrcode
    );

Parameters

cutline1
The one half plane.
cutline2
The other one half plane.
polygon
The polygon which is intersected.
newPolygons
It's a callback function, which is called for all pieces of the result polygon.
wrcode
Warning error code. There are 8 codes. The first one is that, "Not enough memory to cut a polygon". The rest of them are "Internal errors".

Return Values

true
The strip and the polygon are disjunct.
false
The strip and the polygon are intersecting.

Remarks

The segment of the half planes must not be empty.

Requirements
