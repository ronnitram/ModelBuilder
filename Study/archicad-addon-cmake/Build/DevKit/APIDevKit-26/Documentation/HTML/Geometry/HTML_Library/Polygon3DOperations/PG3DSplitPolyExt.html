Geometry::PG3DSplitPolyExt

Splits a 3D polygon with a plane.

     GSErr PG3DSplitPolyExt (
      PG3DPOLYEXT*    poly,
      const PlaneEq&  plane,
      PG3DPOLYEXT*    pFront,
      DESC*           frontEdges,
      PG3DPOLYEXT*    pBack,
      DESC*           backEdges,
      short*          splitRes
    );

Parameters

poly
Pointer to the polygon.
plane
Defining equation of the splitting plane.
pFront
Pointer to the split polygon that is in front of the splitting plane.
frontEdges
Edges in front of the split plane.
pBack
Pointer to the split polygon that is behind the splitting plane.
backEdges
Edges behind the split plane.
splitRes
Information on the relative position of the polygon and the plane. See the description of enumerated type PGPoly3DPosition.

Return Values

Returns NoError, if the operation is completed successfully.

Requirements

Version: 1.0 or later
Header: Polygon3DOperations.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
