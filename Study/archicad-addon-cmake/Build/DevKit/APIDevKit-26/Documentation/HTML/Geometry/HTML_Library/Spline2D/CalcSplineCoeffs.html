Geometry::CalcSplineCoeffs

Gives the all coefficients of all segments of a spline. (Converts a Natural spline to a Bezier spline represented by cubic equations.)

     void CalcSplineCoeffs (
      bool      opened,
      bool      cyclic,
      long      n,
      double**  x0,
      double**  y0,
      double**  x1,
      double**  y1,
      double**  x2,
      double**  y2,
      double**  x3,
      double**  y3,
      double    xtf,
      double    ytf,
      double    xtl,
      double    ytl
    );

Parameters

opened
true if the spline is opened.
cyclic
true if the spline is cyclic.
n
Number of the nodes.
x0
x coordinates of nodes.
y0
y coordinates of nodes.
x1
x coefficients of the linear terms.
y1
y coefficients of the linear terms
x2
x coefficients of the quadratic terms.
y2
y coefficients of the quadratic terms.
x3
x coefficients of the cubic terms.
y3
y coefficients of the cubic terms.
xtf
Tangential vector at the first node.
ytf
Tangential vector at the first node.
xtl
Tangential vector at the last node.
ytl
Tangential vector at the last node.

Requirements

Version: 1.0 or later
Header: Spline2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
