Geometry::TMDecomposeMat2

Decomposes the given complex transformation matrix to 3 transformation matrixes (left rotation, scaling (along x and y axis) and right rotation).

     GSErr TMDecomposeMat2 (
      TRANMAT*  tran,
      double*   c1,
      double*   c2,
      double*   alpha,
      double*   beta
    );

Parameters

tran
The given complex transformation matrix.
c1
The value of the scaling along the x axis.
c2
The value of the scaling along the y axis.
alpha
The value of the rotation to the left.
beta
The value of the rotation to the right.

Return Values

Returns NoError, if the operation has been completed successfully.

Requirements

Version: 1.0 or later
Header: TM.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
