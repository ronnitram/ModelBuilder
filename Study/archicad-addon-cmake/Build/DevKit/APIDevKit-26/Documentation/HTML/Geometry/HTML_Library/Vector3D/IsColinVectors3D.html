Geometry::IsColinVectors3D

Decides if two 3D vectors are colinear.

     bool IsColinVectors3D (
      Vector3D  v1,
      Vector3D  v2
    );

Parameters

v1
First vector.
v2
Second vector.

Return Values

true
The two vectors are colinear.
false
The two vectors are not colinear.

Remarks

If any vectors are zero length, then true is returned.

Requirements

Version: 1.0 or later
Header: Point3DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
