Geometry::FindArc

Finds an arc starting from a given vertex in an array of arcs. Used in polygon operations.

     long FindArc (
      const PolyArcRec*  polyArcs,
      long               nArcs,
      long               vertexIdx
    );

Parameters

myPolyArcsPtr
Array of PolyArcRec structures.
nArcs
Number of elements in the array above.
vertexIdx
Index of the vertex the arc starts from.

Return Values

1
The arc that starts from the vertex characterized by vertexIdx has been found.
-1
No arc has been found that starts from the vertex characterized by vertexIdx.

Requirements

Version: 1.0 or later
Header: Polygon2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
