Geometry::BezierPointsH

Gives the four Bezier control points of a spline segment with the beginning and end index is given. The specification of the end index is necessary because of the existence of circular splines.

     void BezierPointsH (
      Coord**    coords,
      DirType**  dirs,
      long       ibeg,
      long       iend,
      Coord*     p0,
      Coord*     p1,
      Coord*     p2,
      Coord*     p3
    );

Parameters

coords
Coordinates of the spline points (handle).
dirs
Directions belonging to the spline points (handle).
ibeg
Beginning index of the coordinates and directions of the spline segment in coords and dirs.
iend
End index of the coordinates and directions of the spline segment in coords and dirs.
p0
Pointer to the first element in the Bezier quad point, equals the ibeg-th element in coords, *p0 = (*coords)[ibeg].
p1
Pointer to the second element in the Bezier quad point, calculated from the ibeg-th elements in coords and dirs.
p2
Pointer to the third element in the Bezier quad point, equals the iend-th element in coords, *p2 = (*coords)[iend].
p3
Pointer to the second element in the Bezier quad point, calculated from the iend-th elements in coords and dirs.

Remarks

Usually iend equals ibeg + 1, however, in case of last quad point of circular splines it is not true.

Requirements

Header: Spline2DData.h
