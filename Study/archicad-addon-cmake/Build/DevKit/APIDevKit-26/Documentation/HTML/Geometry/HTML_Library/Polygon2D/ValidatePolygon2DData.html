ValidatePolygon2DData

Does two thing:

  • calculates the polygon bounding box: the minimal box which fully contains the polygon
  • orients all contours according the Polygon2DData specification: first is positive, the rest are negative. Optionally, it is possible to specify a vertex index and an edge index of the polygon before the orientation. The function calculates the index of the same vertex or edge in the correctly oriented polygon

You should call it before any other polygon operations, if the polygon is modified by hand by you.

     GSErr ValidatePolygon2DData (
      Polygon2DData*  poly,
      long*           vertIdx = NULL,
      long*           edgeIdx = NULL
    );

Parameters

poly
Polygon to be validated.
vertIdx=NULL
Pointer to the index of a vertex in the polygon before orientation. After the function execution the index of the same vertex in the oriented polygon is pointed by vertIdx.
edgeIdx=NULL
Pointer to the index of an edge in the polygon before orientation. After the function execution the index of the same edge in the oriented polygon is pointed by edgeIdx.

Return Values

Returns NoError, if operation is completed successfully.

Requirements

Version: 1.0 or later
Header: Polygon2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
