Geometry::TDInit

Initializes the Delaunay triangulizer with the polygon and the ridges of polygon.

     bool TDInit (
      double*  polyx,
      double*  polyy,
      double*  polyz,
      short*   mask,
      long     ncoord,
      double*  meshLevelx,
      double*  meshLevely,
      double*  meshLevelz,
      long     numMeshLevelPts,
      long*    subLineEnds,
      long     numSubLineEnds
    );

Parameters

polyx
x coordinates of all contours of the polygon being triangularized. Must have ncoord elements.
polyy
y coordinates of all contours of the polygon being triangularized. Must have ncoord elements.
polyz
z coordinates of all contours of the polygon being triangularized. Must have ncoord elements.
mask
It' a value assigned to the coordinates. This value is -1 at the last coordinates of the contours, otherwise 0. Must have ncoord elements.
ncoord
Sum of the number of the coordinates of all contours of the polygon being triangularized.
meshLevelx
x coordinates of all inner points and ridges of the polygon being triangularized. Must have numMeshLevelPts elements.
meshLevely
y coordinates of all inner points and ridges of the polygon being triangularized. Must have numMeshLevelPts elements.
meshLevelz
z coordinates of all inner points and ridges of the polygon being triangularized. Must have numMeshLevelPts elements.
numMeshLevelPts
Number of all inner points and ridges of the polygon being triangularized.
subLineEnds
Array of indices of the ends of the ridges. A sole point is a special - one element - ridge. Must have numSubLineEnds element.
numSubLineEnds
Number of separate ridges.

Return Values

true
Initialization is successful.
false
Initialization failed.

Requirements

Version: 1.0 or later
Header: TD.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
