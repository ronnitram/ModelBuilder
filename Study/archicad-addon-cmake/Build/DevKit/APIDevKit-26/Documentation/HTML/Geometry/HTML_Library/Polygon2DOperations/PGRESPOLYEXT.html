PGRESPOLYEXT

Structure to hold a set of polygons.

    struct PGRESPOLYEXT {
      GSPtr             polygonsData;
      NewPGPolyProcExt  newPolygon;
    };

Members

polygonsData
A pointer which you can use to refer to the stored polygons. (Eg. a pointer to an array.)
 
newPolygon
User-defined callback function, which can store a polygon.
 

Requirements

Version: 1.0 or later
Header: Polygon2DOperations.h
