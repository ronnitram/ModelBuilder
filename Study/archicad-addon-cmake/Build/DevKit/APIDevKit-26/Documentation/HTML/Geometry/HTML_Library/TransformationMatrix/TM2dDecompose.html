Geometry::TM2dDecompose

Decompose the 2D part of a transformation into translation, rotation and scaling parts.

     void TM2dDecompose (
      const TRANMAT*  tran,
      double*         angle,
      double*         xs,
      double*         ys,
      double*         xd,
      double*         yd
    );

Parameters

tran
Pointer to the transformation matrix.
angle
Pointer to the angle of the rotation part.
xs
Pointer to the x coordinate of the scaling part.
ys
Pointer to the y coordinate of the scaling part.
xd
Pointer to the x coordinate of the translation part.
yd
Pointer to the y coordinate of the translation part.

Remarks

Result identical to:

                ROT2  angle
                MUL2  xs, ys
                ADD2  xd, yd

Requirements

Version: 1.0 or later
Header: TM.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
