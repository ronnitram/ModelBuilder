Geometry::PGPolyClip

Intersects a polygon and a Box2DData. (Clips a polygon to a Box2DData.) The result will be regularized.

     bool PGPolyClip (
      Box2DData*          clip,
      const PGPOLYEXT*  polygon,
      PGRESPOLYEXT*     newPolygons,
      long*             wrcode
    );

Parameters

clip
Clipping rectangle to cut by.
polygon
Polygon to cut.
newPolygons
Pointer to result polygons.
wrcode
Warning error code. There are 8 codes. The first one is that, "Not enough memory to cut a polygon". The rest of them are "Internal errors".

Return Values

true
The polygon is completely outside of the clipping rectangle (they are disjunct).
false
Otherwise.
