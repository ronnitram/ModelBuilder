Geometry::FindAngArc

Given three points. They are considered the beginning and end points of an arc and a point on the line that is tangential to the arc in its beginning point. The function decides if the arc determined by the three points above is a real arc or a segment. If it is a real arc, then it calculates the center of the arc and an arbitrary point on the line that is tangential to the arc in its end point.

     bool FindAngArc (
      const Coord&  orientC,
      const Coord&  begC,
      const Coord&  endC,
      bool*         pIsArc,
      Coord*        pOrigC,
      Coord*        pOrientC,
      bool*         pnegArc
    );

Parameters

orientC
Point on the line tangential to the arc in its beginning point.
begC
Beginning point of the arc.
endC
End point of the arc.
pIsArc
true if the arc is a real one, false otherwise.
pOrigC
Center of the arc.
pOrientC
Point on the line tangential to the arc in its beginning point.
pnegArc
@@@_parameter_description_@@@

Return Values

false
No arc exists that is determined by the three given points.
true
Otherwise.
