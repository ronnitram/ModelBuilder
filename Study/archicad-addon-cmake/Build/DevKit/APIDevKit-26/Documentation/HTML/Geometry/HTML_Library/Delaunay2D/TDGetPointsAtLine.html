Geometry::TDGetPointsAtLine

Calculates the points where an arced line intersects the triangles of a mesh.

     bool TDGetPointsAtLine (
      double*        pointsx,
      double*        pointsy,
      double*        pointsz,
      long           numPts,
      TDOEDGE*       edges,
      long           numEdges,
      TDOTRIANGLES*  triangles,
      long           numTriangles,
      const COORD2&  ptArcBeg,
      const COORD2&  ptArcEnd,
      double         angArc,
      bool           onlyForced,
      double**       linex,
      double**       liney,
      double**       linez,
      double**       lineAngle,
      long*          numLines
    );

Parameters

pointsx
Description of the mesh: the x coordinates of mesh points.
pointsy
Description of the mesh: the y coordinates of mesh points.
pointsz
Description of the mesh: the z coordinates of mesh points.
numPts
Description of the mesh: number of points.
edges
Description of the mesh: the edges.
numEdges
Description of the mesh: number of edges.
triangles
Description of the mesh: the triangles.
numTriangles
Description of the mesh: number of triangles.
ptArcBeg
Beginning of the input arc.
ptArcEnd
End point of the input arc.
angArc
Angle of the input arc
onlyForced
If True, only forced edges are taken into account.
linex
The x coordintaes of the output lines.
liney
The y coordintaes of the output lines.
linez
The z coordintaes of the output lines.
lineAngle
The angles of the output lines.
numLines
Number of output lines.

Return Values

True
Operation is successful.
False
Otherwise.

Requirements

Version: 1.0 or later
Header: TD.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
