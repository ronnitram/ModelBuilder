Geometry::TMTranCode

Codes for transformations.

    enum TMTranCode {
        TM_ADD_X,
        TM_ADD_Y,
        TM_ADD_Z,
        TM_ROT_X,
        TM_ROT_Y,
        TM_ROT_Z,
        TM_SCA_X,
        TM_SCA_Y,
        TM_SCA_Z
    };
Enumerator Value Meaning
TM_ADD_X Shifting along the X axis.
TM_ADD_Y Shifting along the Y axis.
TM_ADD_Z Shifting along the Z axis.
TM_ROT_X Rotating around the X axis.
TM_ROT_Y Rotating around the Y axis.
TM_ROT_Z Rotating around the Z axis.
TM_SCA_X Scaling along the X axis.
TM_SCA_Y Scaling along the Y axis.
TM_SCA_Z Scaling along the Z axis.

Requirements

Version: 1.0 or later
Header: TM.h
