Geometry::ValidGenArcPoint

Decides if a point is a valid point of a generic arc. The arc is treated as closed at its endpoint with threshold value of MiddleEps6.

     bool ValidGenArcPoint (
      const GenArc*  a,
      const Coord*   c
    );

Parameters

a
Pointer to the arc.
c
Pointer to the coordinates of the point.

Return Values

true
The point is a valid point of the arc, i.e. it is located on it.
false
The point is not located on the arc.

Requirements

Version: 1.0 or later
Header: GenArc2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
