Geometry::TMRotation_Z

Initializes a homogeneous transformation matrix for a rotation transformation around the Z axis, and sets its status corresponding.

     void TMRotation_Z (
      TRANMAT*  tran,
      double    angle
    );

Parameters

tran
The initialized matrix. Can be NULL.
angle
The angle of the rotation.

Requirements

Version: 1.0 or later
Header: TM.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
