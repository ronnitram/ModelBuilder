Geometry::DirType

Describes a direction in a Bezier-type spline. In a control point there are two symmetric handles. The length of both handles and the direction angle of the first one are stored in this structure. Any Bezier quad point can be resotred from two control points and two DirTypes.

    struct DirType {
      double   lenPrev;
      double   lenNext;
      double   dirAng;
    };

Members

lenPrev
Length of the first handle.
 
lenNext
Length of the second hadle.
 
dirAng
Direction angle of the first handle.
 

Requirements

Version: 1.0 or later
Header: Spline2DData.h
