Geometry::IsNotNear Function

Returns whether the first point (parameter Nr. 1) is not equal to the second point (parameter Nr. 2).

template <class NumberType>
bool IsNotNear (
    const Geometry::Point2< NumberType > & p1,
    const Geometry::Point2< NumberType > & p2
);

Template parameters

NumberType
The type of the number is set by the developer.

Parameters

p1
[in] The first point.
p2
[in] The second point.

Return Value

Returns true if the first point is not the same as the second. Otherwise false.

Description

This method returns true if the first point is not the same as the second.
