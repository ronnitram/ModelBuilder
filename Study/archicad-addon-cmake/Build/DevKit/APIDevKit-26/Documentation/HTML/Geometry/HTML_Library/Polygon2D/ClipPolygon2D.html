ClipPolygon2D

Clips a polygon to a rectangle, i.e. calculates the intersection of a polygon and a rectangle. The result polygons will be regularized.

     bool ClipPolygon2D (
      Box2DData*         clip,
      const Polygon2DData*     poly,
      VAArray<Polygon2DData>*    respoly,
      long*                    wrcode
    );

Parameters

clip
Pointer to the Box2DData structure representing the clipping rectangle
poly
Pointer to the polygon.
respoly
Pointer to the array of result polygons.
wrcode
Warning error code. There are 8 codes. The first one is that, "Not enough memory to cut a polygon". The rest of them are "Internal errors".

Return Values

true
The polygon and the clipping rectangle are not intersecting, or the operation is not completed successfully.
false
Otherwise.
