Geometry::Ellipse

Structure to describe an ellipse.

    struct Ellipse {
      Coord    origC;
      double   a;
      double   ratio;
      double   angle;
    };

Members

origC
The center of the ellipse.
 
a
The size of main axis
 
ratio
The ratio of the secondary axis to the main axis. It equals 1 in case of circles.
 
angle
The direction angle of the main axis.
 

Requirements

Version: 1.0 or later
Header: Ellipse2DData.h
