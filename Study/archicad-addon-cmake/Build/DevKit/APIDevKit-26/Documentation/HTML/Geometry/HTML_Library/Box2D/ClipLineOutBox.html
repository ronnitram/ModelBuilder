Geometry::ClipLineOutBox

Gives the parts of a line that are outside a box.

     bool ClipLineOutBox (
      Box2DData* clip,
      Coord*              pBeg,
      Coord*              pEnd,
      long*            npart
    );

Parameters

clip
Clipping box.
pBeg
A two-dimensional Coord array. The 0-th element should contain the input starting point.
pEnd
A two-dimensional Coord array. The 0-th element should contain the input ending point.
npart
Number of the cut parts.

Return Values

True
Sector is completely inside the box.
False
Sector is partially or completely outside the box.

Requirements

Version: 1.0 or later
Header: Box2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
