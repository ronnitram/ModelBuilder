Geometry::BezierPoints

Gives the four Bezier control points of a spline segment with the beginning and end index is given. The specification of the end index is necessary because of the existence of circular splines.

     void BezierPoints (
      const Coord*    coords,
      const DirType*  dirs,
      long            ibeg,
      long            iend,
      Coord*          bezp
    );

Parameters

coords
Coordinates of the spline points.
dirs
Directions belonging to the spline points.
ibeg
Beginning index of the coordinates and directions of the spline segment in coords and dirs.
iend
End index of the coordinates and directions of the spline segment in coords and dirs.
bezp
Array of the four coordinates of the Bezier quad point.

Remarks

Usually iend equals ibeg + 1, however, in case of the last quad point of circular splines it is not true.

Requirements

Version: 1.0 or later
Header: Spline2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
