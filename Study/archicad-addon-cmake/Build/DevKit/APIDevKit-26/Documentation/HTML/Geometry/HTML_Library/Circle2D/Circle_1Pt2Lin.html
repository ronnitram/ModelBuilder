Geometry::Circle_1Pt2Lin

Calculates the center of the the circle determined by a point and two lines. The lines are tangents of the circle, the point is located on it. The reference point helps to determine the palne-part, if necessary.

     void Circle_1Pt2Lin (
      const Coord&   c,
      const Sector&  s1,
      const Sector&  s2,
      const Coord&   refCo,
      Coord*         origo,
      bool*          exist
    );

Parameters

c
Coordintes of the point.
s1
Sector representing the first line.
s2
Sector representing the second line.
refCo
Coordinates of the reference point.
origo
Pointer to the coordinates of the center of the circle.
exist
It is false when no circle appropriately determined by the lines and the point exists, true otherwise.

Return Values

No values are returned.

Requirements

Version: 1.0 or later
Header: Circle2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
