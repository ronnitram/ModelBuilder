PG3DEDGEEXT

Structure to describe a 3D segment. It is the same as the 2D edge in addition with a "z" coordinate for the starting vertex of the segment.

    struct PG3DEDGEEXT {
      PGEDGEEXT    d2;
      double       zz;
    };

Members

d2
2D segment for the common data fields of the 3D and 2D segments.
 
zz
The z coordinate for the starting vertex of the segment.
 

Requirements

Version: 1.0 or later
Header: Polygon3DOperations.h
