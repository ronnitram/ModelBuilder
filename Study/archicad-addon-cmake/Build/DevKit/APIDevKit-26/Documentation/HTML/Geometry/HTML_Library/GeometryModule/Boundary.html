Boundary

Geometry::Boundary

All boundaries of set of points can be open or closed. You can use this enum to sign the appropriate case.

    enum Boundary {
        Boundary_Opened,
        Boundary_Closed,
        Boundary_Mixed,
        Boundary_Unknown
    };
Enumerator Value Meaning
Boundary_Opened Sign an open set of points.
Boundary_Closed Sign a closed set of points.
Boundary_Mixed Sign a mixed boundary set of points.
Boundary_Unknown Sign an unknown boundary set of points.
