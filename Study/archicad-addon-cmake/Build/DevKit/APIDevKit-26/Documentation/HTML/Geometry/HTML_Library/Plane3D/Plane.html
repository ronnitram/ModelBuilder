Plane

Defines a plane with its three non-degenerated points.

    typedef struct struct {
      Coord3D  c1;
      Coord3D  c2;
      Coord3D  c3;
    } Plane;

Members

c1
Coordinates of the first defining point.
 
c2
Coordinates of the second defining point.
 
c3
Coordinates of the third defining point.
