Geometry::SetVectorPtr3D

Initializes a 3D vector or point with the given coordinates.

     Vector3D* SetVectorPtr3D (
      Vector3D*  w,
      double     x,
      double     y,
      double     z
    );

Parameters

w
Pointer to vector to initialize.
x
x coordinate of the vector.
y
y coordinate of the vector.
z
z coordinate of the vector.

Return Values

Pointer to the initialized vector.

Requirements

Version: 1.0 or later
Header: Point3DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
