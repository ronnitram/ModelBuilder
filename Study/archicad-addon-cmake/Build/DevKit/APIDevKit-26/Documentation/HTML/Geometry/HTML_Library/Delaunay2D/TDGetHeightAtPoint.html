Geometry::TDGetHeightAtPoint

Calculates the height of a triangulated mesh at a given location.

     short TDGetHeightAtPoint (
      double*        pointsx,
      double*        pointsy,
      double*        pointsz,
      long           numPts,
      TDOEDGE*       edges,
      long           numEdges,
      TDOTRIANGLES*  triangles,
      long           numTriangles,
      const COORD2&  ptLoc,
      double*        height
    );

Parameters

pointsx
Description of the mesh: the x coordinates of mesh points.
pointsy
Description of the mesh: the x coordinates of mesh points.
pointsz
Description of the mesh: the x coordinates of mesh points.
numPts
Description of the mesh: number of points.
edges
Description of the mesh: the edges.
numEdges
Description of the mesh: number of edges.
triangles
Description of the mesh: the triangles.
numTriangles
Description of the mesh: number of triangles.
ptLoc
The location where the mesh height is measured.
height
The calculated height.

Return Values

Index of the triangle that contains the location where the mesh height is measured.

Requirements

Version: 1.0 or later
Header: TD.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
