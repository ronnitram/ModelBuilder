HalfPlane2DData

Describes a half plane in 2D.

    struct HalfPlane2D {
      double       ca;
      double       cb;
      double       cc;
      GSSize       sizeOfInhEdgeInfo;
      void*        inhEdgeInfo;
    };

Members

ca
Coefficient of x.
 
cb
Coefficient of y.
 
cc
Constant.
 
sizeOfInhEdgeInfo
Size of the inherited edge info.
 
inhEdgeInfo
The inherited edge info.
 

Remarks

Defining inequality for the half plane: ca * x + cb * y > cc.

Requirements

Version: 1.0 or later
Header: Polygon2DData.h
