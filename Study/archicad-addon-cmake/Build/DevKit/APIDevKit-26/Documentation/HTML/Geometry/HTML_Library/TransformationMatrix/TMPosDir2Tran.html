Geometry::TMPosDir2Tran

Initializes a transformation matrix that translates the origo to a given point and rotates the selected coordinate axis to a given direction.

     void TMPosDir2Tran (
      short     axis,
      double    posx,
      double    posy,
      double    posz,
      double    dirx,
      double    diry,
      double    dirz,
      TRANMAT*  tran
    );

Parameters

axis
The selected coordinate axis, 1 for x, 2 for y (not yet implemented), 3 for z.
posx
x coordinate of the translation.
posy
y coordinate of the translation.
posz
z coordinate of the translation.
dirx
x coordinate of the rotated axis.
diry
y coordinate of the rotated axis.
dirz
z coordinate of the rotated axis.
tran
Pointer to the initialized transformation matrix.

Requirements

Version: 1.0 or later
Header: TM.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
