Geometry::SetBox3D

Initializes a 3D box with the given values.

     void SetBox3D (
      Box3DType*  abox,
      double      xMin,
      double      yMin,
      double      zMin,
      double      xMax,
      double      yMax,
      double      zMax
    );

Parameters

abox
Pointer to the box.
xMin
Minimal coordinate value of the box along the x axis.
yMin
Minimal coordinate value of the box along the y axis.
zMin
Minimal coordinate value of the box along the z axis.
xMax
Maximal coordinate value of the box along the x axis.
yMax
Maximal coordinate value of the box along the y axis.
zMax
Maximal coordinate value of the box along the z axis.

Return Values

No values are returned.

Requirements

Version: 1.0 or later
Header: Box3DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
