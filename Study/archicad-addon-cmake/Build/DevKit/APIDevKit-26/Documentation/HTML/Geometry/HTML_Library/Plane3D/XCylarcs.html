Geometry::XCylarcs

Calculates the intersection of two ellipses on two different planes. The ellipses are defined by the plane they ara located on and a circle on the xy plane that is projected to the former plane.

     char XCylarcs (
      const Circle*   ci1,
      const PlaneEq*  plane1,
      const Circle*   ci2,
      const PlaneEq*  plane2,
      Sector3D*       xs
    );

Parameters

ci1
Pointer to the circle used for defining the first ellipse.
plane1
Pointer to the equation of the plane of the first ellipse.
ci2
Pointer to the circle used for defining the second ellipse.
plane2
Pointer to the equation of the plane of the second ellipse.
xs
Pointer to the sector with the intersection points as end points.

Return Values

The right end bit is set to one, if the first point of xs is a valid intersection point.
The bit next to the right end bit is set to one, if the second point of xs is a valid intersection point.

Requirements
