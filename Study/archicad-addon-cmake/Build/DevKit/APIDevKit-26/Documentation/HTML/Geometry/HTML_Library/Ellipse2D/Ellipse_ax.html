Geometry::Ellipse_ax

Gives an ellipse arc with a given total length. The ellipse arc is part of an original ellipse arc between a and endAng ellipse parameters.

     double Ellipse_ax (
      double  ax1,
      double  ax2,
      double  a,
      double  endAng,
      double  givenLen
    );

Parameters

ax1
Length of longer axis of ellipse.
ax2
Length of shorter axis of ellipse.
a
Ellipse parameter of the beginning of the ellipse arc.
endAng
Ellipse parameter of the end of the original ellipse arc.
givenLen
Specified length of the ellipse arc.

Return Values

Ellipse parameter of the end of the ellipse arc. Its equaling -1 signs that no solution exists, i.e. the specified length is longer than the length of the original ellipse arc.

Requirements

Version: 1.0 or later
Header: Ellipse2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
