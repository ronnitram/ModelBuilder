NewPGPolyProcExt callback

User-defined callback function which can store a polygon which is the result of an operation.

    typedef GSErr (*NewPGPolyProcExt) (
      PGRESPOLYEXT*        polygons,
      const PGPOLYEXT*     newPoly
      PGPOLYEXT**          storedPoly
    );

Parameters

polygons
The user-privided storage where you should store this new polygon.
newPoly
The polygon you must store in storage.
storedPoly
A pointer you must give back to the operation which will use it for the additional NewEdge calls. This pointer must be valid till the end of the operation.

Return Values

Returns NoError, if it will be successful.

Remarks

Requirements

Version: 1.0 or later
Header: Polygon2DOperations.h
