Geometry::XSplineCircle

Calculates the intersection of a spline and a circle.

     long XSplineCircle (
      long      nCoords
      Coord*    pCoords,
      DirType*  pDirs,
      bool           closed,
      long           segment,
      const Circle*  c,
      bool           elliptic,
      double         angle,
      double         ratio,
      double**       hSect
    );

     long XSplineCircle (
      Coord**        coords,
      DirType**      dirs,
      bool           closed,
      long           segment,
      const Circle*  c,
      bool           elliptic,
      double         angle,
      double         ratio,
      double**       hSect
    );

Parameters

nCoords
Number of Coodinates (at pointer version).
pCoords / coords
Control points of the Bezier quad points of the spline.
pDirs / dirs
Directions of the Bezier quad points of the spline.
closed
If true, spline is closed.
segment
Index of the spline segments used for the calculation. If it is less than 0, all segments are used.
c
Pointer to the circle.
elliptic
If true, circle is an ellipse.
angle
Angle of the main axis of the ellipse.
ratio
Ratio of the lengths of the two ellipse axes.
hSect
Coordinate values of the intersection points.

Return Values

Number of the intersection points.

Requirements

Version: 1.0 or later
Header: Spline2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
