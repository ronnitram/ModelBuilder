Geometry::TMComplementer

Calculates the complemeter of a transformation matrix, C := A * B^-1

     bool TMComplementer (
      TRANMAT*  ctran,
      TRANMAT*  atran,
      TRANMAT*  btran,
      bool      avers
    );

Parameters

ctran
Pointer to the calculated complemetrer matrix.
atran
Pointer to the first matrix for the calculation.
btran
Pointer to the second matrix for the calculation.
avers
If avers is true, C := A^-1 * B

Return Values

false
Operation is not completed successfully, *btran is the identity matrix in case of avers is false or *atran is the identity matrix in case of avers is true.
true
Operation is completed successfully.

Requirements

Version: 1.0 or later
Header: MatrixData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
