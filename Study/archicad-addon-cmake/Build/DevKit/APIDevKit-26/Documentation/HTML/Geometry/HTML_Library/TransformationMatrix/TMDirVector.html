Geometry::TMDirVector

Transforms a unit length direction vector.

     void TMDirVector (
      TRANMAT*  tran,
      double*   x,
      double*   y,
      double*   z
    );

Parameters

tran
Pointer to the transformation matrix.
x
Pointer to the x coordinate of the direction vector.
y
Pointer to the y coordinate of the direction vector.
z
Pointer to the z coordinate of the direction vector.

Remarks

The translation part of the transformation is neglected. After the transformation the result is rescaled to a unit length vector.

Requirements

Version: 1.0 or later
Header: TM.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
