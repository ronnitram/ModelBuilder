NewPGEdgeProcExt callback

User-defined callback function which can give back an edge with random indexing.

    typedef GSErr (*NewPGEdgeProcExt) (
      PGPOLYEXT*         polygon,
      const PGEDGEEXT*   pgedge
    );

Parameters

polygon
The polygon which an edge added.
pgedge
The edge you must store in polygon.

Return Values

Returns NoError, if operation is completed successfully.

Remarks

You can write this callback function so if it is called you store the given edge to the end of the list of edges.

Requirements

Version: 1.0 or later
Header: Polygon2DOperations.h
