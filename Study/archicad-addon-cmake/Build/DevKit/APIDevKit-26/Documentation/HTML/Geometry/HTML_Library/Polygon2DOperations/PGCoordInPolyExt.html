Geometry::PGCoordInPolyExt

Decides if a point inside a polygon, out of a polygon or on an edge.

     PGPositionExt PGCoordInPolyExt (
      const PGPOLYEXT*  polygon,
      Coord*            p,
      bool              mainContOnly
    );

Parameters

polygon
The polygon.
p
The point which is decided to be inside the given polygon.
mainContOnly
If it's true, the function returns that, the point is inside (the main cotour of) the polygon, even if the point is inside a hole of the polygon. If it's false, the function returns that, the point is outside the polygon, if the point is inside a hole of the polygon.

Return Values

Enumerator Value Meaning
PGInsidePoly Entirely inside.
PGOutsidePoly Entirely outside.
PGOnPolyEdg On edge or across edge.

Requirements
