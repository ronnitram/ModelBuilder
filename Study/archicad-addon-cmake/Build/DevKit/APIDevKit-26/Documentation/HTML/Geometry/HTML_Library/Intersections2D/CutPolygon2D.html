CutPolygon2D

Intersects the polygon with a half plane. The result will be regularized.

     bool CutPolygon2D (
      HalfPlane2D*                       cutline,
      const Geometry::Polygon2DData*     poly,
      VAArray<Geometry::Polygon2DData>*  respoly,
      long*                              wrcode
    );

Parameters

cutline
Structure representing the half plane.
poly
The polygon the half plane is to intersect with.
respoly
Array containing the result polygons.
wrcode
Warning error code. There are 8 codes. The first one is that, "Not enough memory to cut a polygon". The rest of them are "Internal errors".

Return Values

true
The half plane and the polygon are disjunct.
false
The half plane and the polygon are intersecting.

Requirements

Version: 1.0 or later
Header: Polygon2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
