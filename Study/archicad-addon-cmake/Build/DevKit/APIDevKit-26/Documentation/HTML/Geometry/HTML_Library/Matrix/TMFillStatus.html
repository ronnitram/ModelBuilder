Geometry::TMFillStatus

Sets the status of a matrix in accordance with its value.

     void TMFillStatus (
      TRANMAT*  tran
    );

Parameters

tran
The matrix to set its status.

Remarks

It is recommended to call this function if you modify the value of a matrix in any other way than calling a Geometry matrix function. You never need to call this function after a Geometry matrix function.

Requirements

Version: 1.0 or later
Header: MatrixData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
