Geometry::IVBox3DType

Converts a Box3DType stored in big-endian byte order to the same Box3DType stored in little-endian byte order, or vice versa.

     void IVBox3DType (
      Box3DType*  b
    );

Parameters

b
Pointer to the box to be converted.

Return Values

No values are returned.

Requirements

Version: 1.0 or later
Header: Box3DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
