Geometry::TangentCircle

Calculates the center of the circle that touches a spline in two given points.

     bool TangentCircle (
      Coord**       coords,
      DirType**     dirs,
      double        t1,
      double        t2,
      const Coord&  f1,
      const Coord&  f2,
      double        arcDiff,
      Coord*        origo,
      bool*         exist
    );

Parameters

opened
true if the spline is opened.
cyclic
true if the spline is cyclic.
t1
Parameter of the first point.
t2
Parameter of the second point.
f1
Coordinates of the first point.
f2
Coordinates of the second point.
arcDiff
Extent of the allowed ambiguity of the result.
origo
Center of the tangent circle.
exist
false, if at least one of the circles described in the remark does not exist.

Return Values

false
No single tangent circle could be found.
false
There is a single tangent circle.

Remarks

Two circles are calculated. Both touch f1 and f2. They are tangent to the spline in f1 and f2, respectively. The result is the average of the two circles above. The result is accepted only when the difference of the two circles is sufficiently low. The extent of this accepted difference is controlled by arcDiff.
