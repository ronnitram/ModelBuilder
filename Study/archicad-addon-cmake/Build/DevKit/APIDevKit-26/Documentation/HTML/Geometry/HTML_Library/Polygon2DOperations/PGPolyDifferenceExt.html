Geometry::PGPolyDifferenceExt

Subtracts two polygons from each other. The result will be regularized.

     GSErr PGPolyDifferenceExt (
      const PGPOLYEXT*  polygon1,
      const PGPOLYEXT*  polygon2,
      PGRESPOLYEXT*     newPolygons,
      PGRelPolyPosExt*  status
    );

Parameters

polygon1
Polygon to subtract from.
polygon2
Subtracted polygon.
newPolygons
Pointer to result polygons.
status
Relative position of the two input polygons.

Return Values

NoError
The function completed with success.

Remarks

There are no result polygons when the two input polygons are disjunct or overlapping at nodes or edges only.

Requirements
