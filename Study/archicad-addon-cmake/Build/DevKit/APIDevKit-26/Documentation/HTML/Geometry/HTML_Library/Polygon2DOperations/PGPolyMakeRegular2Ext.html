Geometry::PGPolyMakeRegular2Ext

Regularizes a polygon. Makes it geometrically valid.

     GSErr PGPolyMakeRegular2Ext (
      const PGPOLYEXT*            originalPolygon,
      PGRESPOLYEXT*             newPolygons,
      bool                     resolveSelfInters,
      PGRegulHoleHandling      holeHandling
    );

Parameters

originalPolygon
Polygon to regularize.
newPolygons
Pointer to polygons resulted from the regularization.
resolveSelfInters
If true, self intersections are resolved
holeHandling
If KeepInvalidHoles, all hole contours are kept regardless their orientation.
If ThrowInvalidHoles, only correctly oriented hole contours are kept.
If KeepMainContour, the main contour is kept regardless its orientation, provided there are no hole contours.

Return Values

NoError
The function completed with success.

Remarks

A polygon valid if

  • Its first boundary contains all the others
  • Oriented correctly (main contour positive, rest negative)
  • Has no self-intersections
  • Its area is not zero
  • Has no zero length edges
