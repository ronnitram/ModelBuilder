Geometry::PGPolyOffsetEdgeFixAreaExt

Translates an edge of a rectangular polygon perpendicularly to its direction. Negative and positive offset values make the edge move inside and outside, respectively. The area of the rectangle is fix, so one of the neighbor edge is also translated. The result will be regularized.

     GSErr PGPolyOffsetEdgeFixAreaExt (
      const PGPOLYEXT*  polygon,
      long              edgeIdx,
      long              vertIdx,
      double            offset,
      double            area,
      PGRESPOLYEXT*     newPolygons
    );

Parameters

polygon
Polygon whose edge is to be translated.
edgeIdx
Index of the edge to be translated.
vertIdx
Index of the fixed vertex.
offset
Distance of the translation.
area
New area of the rectangle.
newPolygons
Pointer to the result polygons.

Return Values

NoError
The function completed with success.

Requirements

Version: 1.0 or later
Header: Polygon2DOperations.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
