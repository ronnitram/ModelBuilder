ChamferPolygon2DDataContour

Makes a chamfer on all vertexes of the given contour of a polygon. The result will be regularized.

     GSErr ChamferPolygon2DDataContour (
      const Polygon2DData&     poly,
      long                     contourIdx,
      double                   chamferRadius,
      VAArray<Polygon2DData>*  resultPolys
    );

Parameters

poly
The polygon.
contourIdx
Index of the contour to be chamfered. If contourIdx is 0 then all contours will be chamfered.
chamferRadius
Radius for chamfering.
resultPolys
Pointer to the array of result polygons.

Return Values

Returns NoError, if operation is completed successfully.

Remarks

The function won't change those vertexes which couldn't be chamfered with chamferRadius value.

Requirements

Version: 1.0 or later
Header: Polygon2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
