Geometry::PointsOnPlane

Decides if a set of points are all located on the same plane and line. If they are on the same plane, the equation of this plane is determined.

     GSErr PointsOnPlane (
      long      n,
      double*   x,
      double*   y,
      double*   z,
      bool*     planar,
      bool*     colinear,
      PlaneEq*  pl
    );

Parameters

n
Number of the points.
x
Pointer to the x coordinates of the points.
y
Pointer to the y coordinates of the points.
z
Pointer to the y coordinates of the points.
planar
true, if the points are on one plane.
colinear
true, if the points are on one line.
pl
Pointer to the plane equation.

Return Values

Returns true, if the operation is completed successfully.

Requirements

Version: 1.0 or later
Header: Plane3DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
