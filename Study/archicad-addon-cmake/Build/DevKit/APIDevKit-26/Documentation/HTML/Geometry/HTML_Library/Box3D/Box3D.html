Box3D

This structure describes a block of the space.

    typedef struct Box3D {
      double  xMin;
      double  yMin;
      double  xMax;
      double  yMax;
      double  zMin;
      double  zMax;
    } Box3D;

Members

xMin
The min. X coordinate of the box (3D).
 
yMin
The min. Y coordinate of the box (3D).
 
xMax
The max. X coordinate of the box (3D).
 
yMax
The max. Y coordinate of the box (3D).
 
zMin
The min. Z coordinate of the box (3D).
 
zMax
The max. Z coordinate of the box (3D).
