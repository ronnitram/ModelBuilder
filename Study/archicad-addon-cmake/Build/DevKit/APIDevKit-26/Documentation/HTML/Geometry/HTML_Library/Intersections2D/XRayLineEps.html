Geometry::XRayLineEps

Calculate the intersection of a ray and a line. Parallel lines has no intersection.

     bool XRayLineEps (
      const Sector&  s1,
      const Sector&  s2,
      Boundary       boundary,
      Coord*         xc,
      double         eps,
      double         radEps
    );

Parameters

s1
Pointer to the sector representing the ray.
s2
Pointer to the sector representing the line.
boundary
The boundary type of the ray end.
xc
Pointer to the coordinates of the intersection point.
eps
Threshold value for calculating the intersection point of the line and the line the ray is located on.
radEps
Threshold value for deciding whether the intersection point is located on the ray.

Return Values

true
The ray and the line are intersecting..
false
The ray and the line are not intersecting..

Requirements

Version: 1.0 or later
Header: Sector2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
