Geometry::CoordsInBox

Determines whether the box passed to the function intersects with the box that encloses all the coordinates in the array pointed to by c. If rot is true, then the latter box encloses the rotated coordinates.

     bool CoordsInBox (
      long             nCoords,
      const Coord*        c,
      const Box2DData*   boundBox,
      bool             rot,
      double           angle
    );

Parameters

nCoords
Number of coordinates to be enclosed in the box.
c
Pointer to the coordinate array.
boundBox
Box that encloses the coordinates.
rot
Whether the coordinates are rotated.
angle
Angle of rotation is -angle.

Return Values

true
The two boxes are intersecting, i.e. boundBox may enclose one or more coordinates.
false
Boxes are disjunct, i.e. no coordinates are inside boundBox.

Remarks

Rotation does not change coordinates pointed to by c.

Requirements

Version: 1.0 or later
Header: Box2DData.h
Import Library: GeometryImp.lib [Win]
Module: Geometry.dll [WIN] GeometryLib [Mac]
