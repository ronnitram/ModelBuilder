GXInitDefaultTransparentPattern

Initializes a pattern to a predefined one, using a floating point transparency value.

    void GXInitDefaultTransparentPattern (
      GSPattern*      pat,
      double          transpar
    );

Parameters

pat
Pattern to be initialized.
transpar
Transparency value of the pattern.
transpar < 0.125
Empty fill.
0.125 ≤ transpar < 0.375
25% fill with slanted lines.
0.375 ≤ transpar < 0.625
50% fill with slanted lines.
0.625 ≤ transpar < 0.875
75% fill with slanted lines.
0.875 ≤ transpar
Solid fill (100%).

Remarks

If you don't want to use patterns, initialize your pattern to solid.
