Pixmap conversion and I/O

The Graphics pixmap has no direct persistency instead you can read-write it to various well-known and well-defined pixmap formats as Jpeg, Png, and so on.

There are two equal packages: the C++ interface which bounds you to the same compiler vendor and version as we using and the binary compatible one, which gives you the freedom to choose compiler for yourself. The latter is a wrapper around the former.
