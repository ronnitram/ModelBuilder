FileTypeManager::ReadID

Read an id from a channel.

    static GeneralID ReadID (
      GS::IChannel&    ic,
      const char*            managerId = NULL
    );

Parameters

ic
Channel to read from.
managerId
Unique identifier of a FileTypeManager object.

Return Values

A valid dynamic id of the type or group, or if the type is not aviable then UnknownType.

Remarks

ReadID can only read data written by WriteID. These two functions must be used in pair. The data written and read by these functions is based on the MIME identifier string of the FileType object, so if this string is empty ReadID will read UnknownType. Also if a type has a valid MIME identifier during write, but this type is not present during read (for example an add-on is loaded during write, but not during read) UnknownType will be returned. The runtime value of the id returned by read will not necessary will be equal to the value of the written id, but the two ids will represent the same FileType.
