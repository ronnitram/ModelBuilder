    FileType (

Simple constructor

    FileType (
      const char*               description,
      const char*               fileExt,
      unsigned long             creator,
      unsigned long             macType,
      int                       iconID,
      FileTypeInspectorCallBack pCallback = NULL
    );

Parameters

source
The source of the copy operation.
description
Description of the file type. Could be used in dialogs such as OpenFile.  
fileExt
Character string that lists the extensions used to identify the file type. Use ';' as separator between extensions. Do not insert separator at the end unless you want to add the empty extension. Only valid file name characters are allowed. (Excluding '/', '\', ':', '*', '?', '"', '<', '>', "|")
creator
Macintosh creator id.
macType
Macintosh file type id.
iconID
Resource id of the icon representing the file type.
pCallback 
Pointer to the callback function, that identifies the file type in case of extension collision.

Remarks

Use the default constructor if you want to create a return object for some function.
