FileTypeManager::GetNextGroup

Finds the next sibling group.

    static GroupID GetNextGroup (
      GroupID      prevGroupID
    );

Parameters

prevGroupID
GroupID previously returned by GetNextGroup or GetFirstGroupChild.

Return Values

Returns UnkonwnGroup if the parent group does not have more group child or in case of error, otherwise it returns a valid GroupID.

Remarks

Use for enumeration of child groups. The returned GroupID should be used in GetNextGroup.

Requirements

Version: Input-Output Library 1.0.0 or later

Import Library: InputOutput.lib [WIN]
Module: InputOutputLib [MAC], InputOutput.dll [WIN]
