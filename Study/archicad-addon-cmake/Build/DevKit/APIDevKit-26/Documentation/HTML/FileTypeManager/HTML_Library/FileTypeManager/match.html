FileTypeManager::Match

Compares a file with a FileTypeManager item. Used as an == operator.

    static bool Match (
      const Location&    loc,
      GeneralID          referenceID,
      GroupID            rootGroupID = RootGroup
    );

Parameters

loc
Location of the file to be identified.
referenceID
Identifier of the type database item.
rootGroupID
The virtual root of the operation. Only items bellow this group are visible to during this operation.

Return Values

Returns true if the type is equal match level to the highest, or the group contains at least one of the highest matching types.

Remarks

Match can be used as an == operator between a type or group and a location. It returns true the item referenced by the referenceID (can be a type or group) is a type with the best mach level or the group that contains at least one best match type.  This is the least strong identifying function, thus the fastest.
