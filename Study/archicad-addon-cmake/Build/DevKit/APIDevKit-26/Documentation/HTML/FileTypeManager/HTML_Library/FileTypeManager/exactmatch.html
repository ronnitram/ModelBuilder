FileTypeManager::ExactMatch

Compares a file with a FileTypeManager item. Used as an == operator.

    static bool ExactMatch (
      const Location&    loc,
      GeneralID          referenceID,
      GroupID            rootGroupID = RootGroup
    );

Parameters

loc
Location of the file to be identified.
referenceID
Identifier of the type database item.
rootGroupID
The virtual root of the operation. Only items bellow this group are visible to during this operation.

Return Values

Returns true if no other type has equally high match level or the specified group contains all the best  matching types.

Remarks

ExactMatch can be used as an == operator between a type or group and a location. It returns true the item referenced by the referenceID (can be a type or group) is the type with the best mach level or the group that contains all best match types. No other types can have the same mach level.

Requirements
