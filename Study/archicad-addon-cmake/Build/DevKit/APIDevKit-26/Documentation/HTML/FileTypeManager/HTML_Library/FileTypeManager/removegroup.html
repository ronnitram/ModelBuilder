FileTypeManager::RemoveGroup

Removes a group from database.

    GSErrCode RemoveGroup (
      GroupID groupID
    );

Parameters

groupID
GroupID of the group to be removed.

Return Values

Value Meaning
NoError Deletion executed.
GroupNotFound Group did not exist in database.
AccessDenied The group specified by groupID was inserted by another FileTypeManager object.

Remarks

Removes the specified group from the static database. The group can not be deleted if the group was inserted by another FileTypeManager. The immediate children of the group will get the parent of the deleted group as new parents.  

Requirements

Version: Input-Output Library 1.0.0 or later

Import Library: InputOutput.lib [WIN]
Module: InputOutputLib [MAC], InputOutput.dll [WIN]
