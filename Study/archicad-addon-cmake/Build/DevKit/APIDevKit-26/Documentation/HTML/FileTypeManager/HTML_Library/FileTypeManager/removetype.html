FileTypeManager::RemoveType

Removes a FileType object from database.

    GSErrCode RemoveType (
      TypeID  typeID
    );

Parameters

typeID
The unique identifier of the FileType. The FileType object must have been inserted with this FileTypeManager instance.

Return Values

Value Meaning
NoError Deletion executed.
TypeNotFound Type did not exsist in database.
AccessDenied The FileType specified by typeID was inserted by another FileTypeManager object.

Remarks

Removes the specified FileType object from the global database. It won't be available to any other FileTypeManager objects after deletion. The typeID must identify a FileType object inserted through this FileTypeManager to be deleted.

Requirements

Version: Input-Output Library 1.0.0 or later

Import Library: InputOutput.lib [WIN]
Module: InputOutputLib [MAC], InputOutput.dll [WIN]
