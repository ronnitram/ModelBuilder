FileType::CheckTypeCallback

Calls the callback function associated with the file type.

    bool CheckTypeCallback (
      IChannel& ic,
      void*     retVal = NULL
    ) const;

Parameters

ic
Input channel to be inspected.
retVal
Callback specific return value. The callback function can optionally use this supplied pointer to a data structure to return type specific values. The callback function should work with NULL.

Return Values

Value Meaning
true The input channel is identified as the given FileType by the associated callback function.
false The input channel is not identified.

Remarks

If no callback is associated with the FileType object it returns false. 

Requirements

Version: Input-Output Library 1.0.0 or later
Header: FileTypeManager.hpp
Import Library: InputOutput.lib [WIN]
Module: InputOutputLib [MAC], InputOutput.dll [WIN]
