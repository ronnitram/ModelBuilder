FileTypeManager::SearchForGroup

Searches for a group.

    static GroupID SearchForGroup (
      const char*  name,
      GroupID      parentGroupID = RootGroup
    );

    static USize SearchForGroup (
      GroupID*     retList,
      USize        listSize,
      const char*  name,
      GroupID      parentGroupID = RootGroup
    );

Parameters

retList
An array of GroupIDs
listSize
Size of array.
name
Referance name for searching.
parentGroupID
Limiting group. Only groups under this group will be examined.

Return Values

The first version returns the first group that matches the reference name.

 

The second version returns the number of found groups in the return value and fills the GroupID array with the result.

Remarks

Use this functions to get the id of a group based on its name.

 

Requirements
