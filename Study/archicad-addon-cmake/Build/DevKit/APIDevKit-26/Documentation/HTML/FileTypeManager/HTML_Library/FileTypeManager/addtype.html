FileTypeManager::AddType

Inserts new FileType.

    TypeID AddType (
      FileType  newType,
      GroupID   parentGroupID = RootGroup
    );

Parameters

newType
FileType object to be copied into database.
parentGroupID
GroupID of the parent group in which the new group to be inserted.

Return Values

Returns UnknownType if operation was not successful, otherwise returns a valid TypeID.

Remarks

Inserts new FileType object into the database. The FileType object will be copied, so it can be a temporary object, even an unnamed one.

Requirements

Version: Input-Output Library 1.0.0 or later

Module: InputOutputLib [MAC], InputOutput.dll [WIN]
