FileTypeManager::GetFirstTypeChild

Finds the first type child of a group.

    static TypeID GetFirstTypeChild (
      GroupID      parentGroupID
    );

Parameters

parentGroupID
GroupID of the parent group for which the first type child item is searched.

Return Values

Returns UnkonwnType if the parent group does not have a type child or in case of error, otherwise it returns a valid TypeID.

Remarks

Use for starting enumeration of child types. The returned TypeID should be used in GetNextType.

Requirements

Version: Input-Output Library 1.0.0 or later
