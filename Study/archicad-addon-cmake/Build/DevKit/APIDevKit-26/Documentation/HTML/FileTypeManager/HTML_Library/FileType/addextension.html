FileType::AddExtension

Add multiple or single extensions to the FileType.

    GSErr AddExtension (
      const char*      extListIni
    );

Parameters

extListIni
Character string that lists the extensions used to identify the file type. Use ';' as separator between extensions. Do not insert separator at the end unless you want to add the empty extension. Only valid file name characters are allowed. (Excluding '/', '\', ':', '*', '?', '"', '<', '>', "|"). If an extension is already added, it will not be repeated in the list.

Return Values

Returns memory error values if a memory error occurs, otherwise NoError. This method will not report error on repeated extensions, although they won't be added repeatedly. Use other FileType methods to determine if an extension is already in the list.

Requirements
