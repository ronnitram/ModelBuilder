FileTypeManager::AddTypeToGroup

Inserts an existing type into a group.

    GSErrCode AddTypeToGroup (
      TypeID  typeID,
      GroupID parentGroupID
    );

Parameters

typeID
The unique identifier of the FileType. The FileType object must have been inserted with this FileTypeManager instance.
parentGroupID
GroupID of the parent group in which the type to be inserted.

Return Values

Value Meaning
NoError Type added to group.
TypeNotFound Type does not exist in database.
GroupNotFound Parent group does not exist.
AccessDenied Neither the type nor the group does belong to this FileTypeManager.

Remarks

Inserts an existing type into a group. Earlier parent of the type won't be lost, because a type can have multiply parents. If the group specified is already in parent relation (grandparent) with the type, then NoError will be returned, but won't insert any new relation into the database. Either the type or group (or both) must belong to this FileTypeManager instance.
