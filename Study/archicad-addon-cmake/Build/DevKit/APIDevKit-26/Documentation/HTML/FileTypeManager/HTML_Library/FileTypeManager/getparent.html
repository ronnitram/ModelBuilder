FileTypeManager::GetParent

Returns the immediate parent of a group.

    static GroupID GetParent (
      GroupID      childGroupID
    );

Parameters

childTypeID
Name of the group. Also could be used as a description in dialogs such as the OpenFile dialog.

Return Values

 Returns UnkonwnGroup if the specified group does not have a parent or in case of error, otherwise it returns a valid GroupID.

Remarks

Use to walk up in group hierarchy.

Requirements

Version: Input-Output Library 1.0.0 or later

Import Library: InputOutput.lib [WIN]
Module: InputOutputLib [MAC], InputOutput.dll [WIN]
