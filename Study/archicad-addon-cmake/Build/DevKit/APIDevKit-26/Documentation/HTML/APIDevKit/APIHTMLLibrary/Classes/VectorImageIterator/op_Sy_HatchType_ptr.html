VectorImageIterator::operator Sy_HatchType*


Returns a reference to the element of the VectorImage that is pointed by the iterator.


operator Sy_HatchType* (
    void
) const;

Return value

The element of the VectorImage that is pointed by the iterator if it is a Sy_HatchType, 0 otherwise.

Description

You can access and modify a Sy_HatchType record placed into a VectorImage.

Requirements

Version: API 10.0 or later, ModelerGraphics 8 or later
Library: ModelerGraphicsImp.lib (Win) or ModelerGraphics.framework (Mac)
