API_HatchQuantity

Describes the different calculated values for fills.

    typedef struct {
        double     surface;
        double     perimeter;
        double     holesPrm;
        double     holesSurf;
        double     filler_1[8];
    } API_HatchQuantity;

 

Members

perimeter

holesPrm
perimeter of holes in the fill
holesSurf
