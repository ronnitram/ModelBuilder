Libraries

 
In the subfolder named Support/Lib you will find the necessary libraries to link to your source files. In most cases you do not have to worry about setting up the communication channels to Archicad, this is handled by these libraries.

  • The ACAP_STAT.lib library should be used for Windows development.
  • The libACAP_STAT.a library should be used for Mac Intel development.
