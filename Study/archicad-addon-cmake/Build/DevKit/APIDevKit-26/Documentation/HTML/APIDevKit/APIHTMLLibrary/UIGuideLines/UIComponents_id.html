User Interface Components

 

There are several possibilities to implement user interface components. The API offers a greatly enhanced array of tools to let developers select the most suitable interface solution for a given problem. The following possibilities are at disposal of developers.

Menus
Dialog Boxes
Archicad Toolbox Additions
Graphic Input
Windows
Other Items

This services allow developers to access the standard Archicad Interface Toolbox and provide an "Archicad look and feel" interface using the standard interface components.
