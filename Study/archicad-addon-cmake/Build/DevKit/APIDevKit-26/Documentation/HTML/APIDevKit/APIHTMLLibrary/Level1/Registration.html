Registration

 

You can use it to evaluate the Archicad Application Programming Interface (API) without any registration required. However add-ons developed without registration will be loaded only by the DEMO version of Archicad.

Once you decide to be a Registered 3rd Party Developer please register at http://archicadapi.graphisoft.com. Upon registration you get your developer ID and authorization key, which enables you to create "real" add-ons.
