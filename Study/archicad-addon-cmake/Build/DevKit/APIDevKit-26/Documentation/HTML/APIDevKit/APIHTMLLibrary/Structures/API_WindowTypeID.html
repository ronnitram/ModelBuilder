Remarks

This structure holds the parameters of a window or database reference.
It must be passed in the case of changing the current window/database (APIDb_ChangeCurrentDatabaseID), or just getting the identification of the active one.(APIDb_GetCurrentWindowID/APIDb_GetCurrentDatabaseID)
In API_PublishFormatData structure it defines the windows in which the publish format is available.
