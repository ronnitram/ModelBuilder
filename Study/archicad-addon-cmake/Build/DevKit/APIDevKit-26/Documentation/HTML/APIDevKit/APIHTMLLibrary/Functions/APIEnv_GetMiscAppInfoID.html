APIEnv_GetMiscAppInfoID

Gets miscellaneous information about application.

    GSErrCode ACAPI_Environment (
        APIEnv_GetMiscAppInfoID,
        API_MiscAppInfo*     miscAppInfo
    );

 

Parameters

miscAppInfo
[out] Miscellaneous information of the application

 

Return Values

NoError
The function has completed with success.
APIERR_BADPARS
miscAppInfo is nullptr.

 

Remarks

This function is used to get version string of application and caption of application's main window.

 

Requirements

Version: API 2.1 or later
Header: APIdefs_Environment.h
