API_GridEdgeInfo

Contains extended information of a API_GridMeshEdge.

        API_GridElemID     id;
        bool               mainAxis;
        bool               filler_1[7];

Members

The identifier of the edge, the info belongs to
mainAxis
Whether it belongs to the main axis or not.
