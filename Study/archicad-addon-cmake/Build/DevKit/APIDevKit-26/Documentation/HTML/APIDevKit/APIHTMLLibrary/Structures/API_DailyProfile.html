API_DailyProfile

Describes a daily profile contained in Operation Profile.

    typedef struct {
        GS::uchar_t     name [API_UniLongNameLen];

        double          maxTemp[24];
        bool            maxTempDefs[24];
        double          occupancyCount[24];
        double          lightGain[24];
        double          equipmentGain[24];
    } API_DailyProfile;
