API_SolidOperationID

The type of the solid operation link between two construction elements.

    typedef enum {
        APISolid_Substract      = 'SSUB',,
        APISolid_SubstUp        = 'SSUU',,
        APISolid_SubstDown      = 'SSUD',,
        APISolid_Intersect      = 'SINT',,
        APISolid_Add            = 'SADD'
   } API_SolidOperationID;
