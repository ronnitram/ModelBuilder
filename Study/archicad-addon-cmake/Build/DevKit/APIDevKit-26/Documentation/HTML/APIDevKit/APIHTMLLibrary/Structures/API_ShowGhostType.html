API_ShowGhostType

Represents the way of the ghost to be shown.

    typedef enum {
        APIGhost_NoFloor,
        APIGhost_ShowOneFloor,
        APIGhost_ShowAboveFloor,
        APIGhost_ShowBelowFloor,
        APIGhost_ShowPreviousFloor,
        APIGhost_ShowAllStructure
    } API_ShowGhostType;
