System Functions


There are several functions to provide information on the system environment. On Windows it is often necessary to know what system the add-on is running on, as well as what is the instance identifier of the add-on. On the Mac to get a resource it is necessary to specify the right resource file as the target.

Those API functions are collected here, which support these services.

ACAPI_GetMainWindow
ACAPI_GetExtensionInstance
ACAPI_GetOwnResModule
ACAPI_GetOwnLocation
ACAPI_SetDllTermHookProc
