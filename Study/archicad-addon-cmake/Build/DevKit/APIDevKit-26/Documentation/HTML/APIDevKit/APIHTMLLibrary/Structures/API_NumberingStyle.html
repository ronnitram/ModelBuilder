API_NumberingStyle

Describes the Layout ID styles.

    typedef enum {
        API_NS_Undefined,
        API_NS_abc,
        API_NS_ABC,
        API_NS_1,
        API_NS_01,
        API_NS_001,
        API_NS_0001,
        API_NS_noID
    } API_NumberingStyle;
