ACAPI_Protection_GetSerialNumber

Retrieves the unique serial number of the current license.

    UInt32 ACAPI_Protection_GetSerialNumber ();

 

Return Value

The numerical representation of the current serial number.

 

Remarks

The serial number of hardware and software licenses can be queried with this function.

 

Requirements

Version: API 26 or later
Header: ACAPinc.h
