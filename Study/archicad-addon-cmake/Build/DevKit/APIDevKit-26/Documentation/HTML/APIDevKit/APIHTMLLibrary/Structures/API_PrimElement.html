API_PrimElement

A union containing all drawing primitives used with ACAPI_Element_ShapePrims and ACAPI_LibPart_ShapePrims.

    typedef union {
        API_Prim_Head      header;
        API_PrimPoint      point;
        API_PrimLine       line;
        API_PrimArc        arc;
        API_PrimText       text;
        API_PrimPLine      pline;
        API_PrimTri        tri;
        API_PrimPoly       poly;
        API_PrimPict       pict;
        API_CWallPanel     panelKind;
    } API_PrimElement;
