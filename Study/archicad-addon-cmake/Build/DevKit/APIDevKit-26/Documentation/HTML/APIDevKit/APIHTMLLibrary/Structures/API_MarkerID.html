API_MarkerID

Describes the various types of dimension markers.

        APIMark_CrossLine,
        APIMark_EmptyCirc,
        APIMark_SlashLine,
        APIMark_OpenArrow30,

        APIMark_SlashLine45,

        APIMark_OpenArrow60,
        APIMark_ClosArrow60,

        APIMark_SlashLine75
