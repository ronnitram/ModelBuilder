API_DimensionAttrType

A dimension style (pseudo-) attribute.

    typedef struct {
        API_Attr_Head          head;
        API_DimensionPrefs     dim;
        Int32                  filler_2[8];
    } API_DimensionAttrType;

 

Members

head
The attribute header.
dim
Various style options for dimensions.

 

Remarks

This attribute stores dimension style options for views. The dimName field in API_NavigatorView refers to this attribute.

 

Requirements

Version: API 10 or later
Header: APIdefs_Attributes.h
