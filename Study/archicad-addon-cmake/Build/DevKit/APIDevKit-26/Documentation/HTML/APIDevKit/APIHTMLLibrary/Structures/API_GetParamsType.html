API_GetParamsType

The changed parameter values of a Library Part.

    typedef struct {
        double             a;
        double             b;
        API_AddParType     **params;
        Int32              filler_1;
    } API_GetParamsType;

 

Members

a
value of the 'A' parameter
b
value of the 'B' parameter
params
other parameters

 

Remarks

This structure is used to get the actual parameters set by the APIAny_ChangeAParameterID function.

Do not forget to dispose the params handle when it is not needed any more. Refer to the ACAPI_DisposeAddParHdl to do that.

 

Requirements

Version: API 2.1 or later
Header: APIdefs_Goodies.h
