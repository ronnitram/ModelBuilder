API_IFCDifference

Contains an IFC Difference.

    typedef struct {
        GS::Array<API_IFCEntityDescriptor>     newIFCEntities;
        GS::Array<API_IFCEntityDescriptor>     modifiedIFCEntities;
        GS::Array<API_IFCEntityDescriptor>     deletedIFCEntities;
        bool                                   isValid;
    } API_IFCDifference;

 

Members

newIFCEntities 
New IFC entities.
modifiedIFCEntities 
Modified IFC entities.
deletedIFCEntities 
Deleted IFC entities.
isValid 
It is false if IFCDifference Generator result is indeterminate.

 

Remarks

This structure is used in ACAPI_IFC_GetIFCDifference and ACAPI_IFC_GetIFCDifferenceState.

 

Requirements

Version: API 23 or later
Header: APIdefs_IFC.h
