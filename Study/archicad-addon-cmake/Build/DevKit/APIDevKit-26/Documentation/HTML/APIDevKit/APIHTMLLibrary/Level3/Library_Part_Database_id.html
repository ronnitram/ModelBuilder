Library Part Database


To have an overview what the Library Parts are, and how they fit into the entire database please refer to the LibPart Overview chapter.

In this papers only the used structures are explained in full details.

 

Structures

A Library Part generally accessed through the data structure below. It refers to the main parameters of a Library Part, such as the reference index and the name.

API_LibPart

Additional data can be accessed using the following structures.

API_AddParID
API_AddParType
API_LibPartDetails
API_LibPartSection
API_LibTypeID
