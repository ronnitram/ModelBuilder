ACAPI_Register_NavigatorAddOnViewPointDataHandler

Registers the add-on to be called when Navigator AddOn viewpoints are merged, saved to older format, or converted from older format to current format.

    GSErrCode  ACAPI_Register_NavigatorAddOnViewPointDataHandler (
        void
    );

 

Return Values

NoError
The function has completed with success.

For other common API errors see the API Errors document.

 

Remarks

Call this function from RegisterInterface to let Archicad know that the AddOn has APINavigatorAddOnViewPointDataMergeHandlerProc, APINavigatorAddOnViewPointDataSaveOldFormatHandlerProc or APINavigatorAddOnViewPointDataConvertNewFormatHandlerProc callback function implemented for merging and converting its Navigator AddOn viewpoint data.

In order these functions to be called on the adequate event, the AddOn needs to pass the function pointers to ACAPI_Install_NavigatorAddOnViewPointDataMergeHandler, ACAPI_Install_NavigatorAddOnViewPointDataSaveOldFormatHandler and ACAPI_Install_NavigatorAddOnViewPointDataConvertNewFormatHandler respectively. Call these install functions in the AddOn's Initialize function.

 

Requirements

Version: API 22 or later
Header: ACAPinc.h
