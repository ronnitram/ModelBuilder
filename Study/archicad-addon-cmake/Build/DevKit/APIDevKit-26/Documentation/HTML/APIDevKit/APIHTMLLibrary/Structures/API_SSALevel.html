API_SSALevel

This is a numeric value representing an SSA subscription level.
Use the named constants from the API_SSALevels namespace:

NoSSA
No SSA subscription.
SSA
Legacy SSA subscription.
Forward
Graphisoft Forward subscription.

 

Remarks

The ACAPI_Protection_IsSSALevelAtLeast function can be used to query the availability of a certain SSA subscription.

 

Requirements

Version: API 26 or later
Header: APIdefs_Registration.h
