API_EdgeTrim

Describes the parameters of a roof or slab edge.
In previous API versions this structure used to be called API_RoofSide.

    typedef struct {
        Int32              filler_0;
        API_EdgeTrimID     sideType;
        double             sideAngle;
    } API_EdgeTrim;

 

Members

sideType
The trim type of the edge (see API_EdgeTrimID).
sideAngle
The angle of the edge trim in radians in case of custom trim types.

 

Requirements

Version: API 15 or later
Header: APIdefs_Elements.h
