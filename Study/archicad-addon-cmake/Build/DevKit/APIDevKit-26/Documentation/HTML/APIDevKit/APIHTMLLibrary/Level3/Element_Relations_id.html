Element Relations

 
The API_...Relation structures describe the connections of a certain wall, beam or room element to other elements.
These structures are used for returning the relation information only, and cannot be used for modifying the database.

To learn more about the usage of element relation structures please refer to the ACAPI_Element_GetRelations function.
