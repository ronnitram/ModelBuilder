API_MarkerData

Describes the various parameters of dimension markers.

    typedef struct {
        API_MarkerID     markerType;
        short            filler_1;
        short            markerPen;
        double           markerSize;
    } API_MarkerData;

 

Members

markerType
Marker type.
markerPen
Pen attribute index for the marker.
markerSize
In scaled drawing size (in mm).

 

Requirements

Version: API 2.1 or later
Header: APIdefs_Elements.h
