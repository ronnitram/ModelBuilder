Windows

 

Add-Ons can open their own document windows. These can be effectively used to display temporary text or graphic reports. The contents of these windows are NOT saved when the project is saved.

The following document windows are supported:

  • Text
  • Drawing

 

Text

Text windows are editable but any changes are lost if the Add-On updates the content. Add-Ons are not notified of changes made by the user.

 

Drawing

Similar to graphic List windows, the content of these windows is not editable.

You can do the following in this window type:

  • Zoom, Pan, Fit in Window
  • Selection with Marquee
  • Copy: Elements are copied as 2D graphics.
