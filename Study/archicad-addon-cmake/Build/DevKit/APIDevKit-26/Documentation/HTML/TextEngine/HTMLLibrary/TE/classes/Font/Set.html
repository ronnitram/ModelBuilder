TE::Font::Set Method

Overloaded method. 

void Set (
    double              size,
    Int32               style,
    const GS::UniString & fontName,
    GSCharCode          charCode = CC_Legacy
);

Sets the Font object by all parameters. You should use the Unicode version instead of it. See the description of Unicode version for more details. 

void Set (const TE::FontFamily & family);

Sets the Font object by all parameters. The new font family determined by fontName and charCode. If TextEngine doesn't find any matching font family then creates one with given name and encoding determined by charCode and subsitutes it with a real font family.

Parameters

size
[in] New typographic size.
style
[in] New style. It contains any combination of flags determined the style. See IFont::FontStyle.
fontName
[in] Name of the new font family.
charCode
[in] New charcode determines the encoding supported by font.
family
[in] Const reference to an existing FontFamily object.
