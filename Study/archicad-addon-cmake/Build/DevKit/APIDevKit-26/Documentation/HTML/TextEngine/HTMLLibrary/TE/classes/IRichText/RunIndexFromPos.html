TE::IRichText::RunIndexFromPos Method

Implemented method. Retrieves the index of the paragraph and the index of the run contains the given character position. 

bool RunIndexFromPos (
    Int32               charPos,
    Int32 *             paragraphIndex,
    Int32 *             runIndex
) const;

Parameters

charPos
[in] Questionable position.
paragraphIndex
[out] Index of the paragraph contains the given position.
runIndex
[out] Index of the run contains the given position.

Return Value

Returns false if charPos is out of range else returns true.

Description

Retrieves the index of the paragraph and the index of the run contains the given character position.

Example

GS::UniString ucontent ("Test\rstring");
Gfx::Color    color;

TE::Font      font;
TE::CharStyle cStyle;
TE::Run       run;
TE::Paragraph paragraph;
TE::RichText  richText;

cStyle.Set (0, color, font);
run.SetStyle (cStyle);
paragraph.SetRun (0, run);

paragraph.SetFirstPos (0);
paragraph.SetRange (4);
paragraph.SetNumOfRuns (1);
TE::IRun* irun;
irun = paragraph.GetRun (0);
irun->SetFirstPos (0);
irun->SetRange (4);
richText.SetParagraph (0, paragraph);

paragraph.SetFirstPos (5);
paragraph.SetRange (6);
paragraph.SetNumOfRuns (2);
irun = paragraph.GetRun (0);
irun->SetFirstPos (0);
irun->SetRange (3);
irun = paragraph.GetRun (1);
irun->SetFirstPos (3);
irun->SetRange (3);
richText.AddParagraph (paragraph);

long parIndex, runIndex;
if (richText.RunIndexFromPos (7, &parIndex, &runIndex))
  DBPRINTF ("%d. paragraph %d. run", parIndex, runIndex);
else
  DBPRINTF ("Out of range");

Output

1. paragraph 1. run
