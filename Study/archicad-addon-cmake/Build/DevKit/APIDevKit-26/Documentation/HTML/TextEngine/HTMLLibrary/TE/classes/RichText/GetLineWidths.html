TE::RichText::GetLineWidths Method

Retrieves the line widths of the given paragraph in pixels. 

bool GetLineWidths (
    Int32               paragraphIndex,
    GS::Array< double > * lineLengths
);
bool GetLineWidths (
    Int32               paragraphIndex,
    GS::Array< double > * lineLengths
) const;

Parameters

paragraphIndex
[in] The index of the paragraph.
lineLengths
[out] Floating point array of line widths in pixels.

Return Value

Return false if the paragraphIndex is wrong else return true.

Description

Retrieves the line widths of the paragraphIndexth paragraph in pixels. RichText has a cache for measured line widths. The cache is filled at the first calling of GetLineWidths (Textengine calls it while drawing). You should change RichText's parameters carefully that don't invalidate cache with unnecessary actions otherwise the performance of drawing can become low.
