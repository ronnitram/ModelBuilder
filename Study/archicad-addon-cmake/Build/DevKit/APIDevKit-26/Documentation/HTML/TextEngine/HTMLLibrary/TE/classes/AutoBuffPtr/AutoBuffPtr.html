TE::AutoBuffPtr Class

AutoBuffPtr is template class for exception safe and quicly handling of buffers (arrays) of simple types.

Type can be a simple type declared in "C". For example: char, short, long, ...

Description

You should use AutoBuffPtr if you needs a buffer for a simple "C" types but the dimension of buffer is known only at runtime and the dimension has not limit (for example edited texts). AutoBuffPtr is an exception safe "smart pointer" class to allocate a temporarly used buffers. It allocates a Type type array with dimension BufferCapacity on the stack. The real size of buffer must specify in the constructor of an instance. If the allocatable size is greater than BufferCapacityAutoBuffPtr allocates memory for buffer. In this case it frees the memory in its destructor.
