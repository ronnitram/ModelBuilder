TE::IGreekOutlineBuilder::LineRect Method

Pure virtual function. Called for outlining the rectangle of a line. 

Int32 LineRect (
    double              x,
    double              y,
    double              w,
    double              h,
    double              angle
) = 0;

Parameters

x
{in} x position of the lower left corner of the box in pixels.
y
{in} y position of the lower left corner of the box in pixels.
w
{in} The width of the box in pixels.
h
{in} The height of the box in pixels.
angle
{in} The rotation angle in radian of the box around the lower left corner from the horizontal line counter-clockwise.

Return Value

Zero if there is no error.

Description

Pure virtual function. When outlining a text in greek mode LineRect is called for outlining the rectangle of each line.
