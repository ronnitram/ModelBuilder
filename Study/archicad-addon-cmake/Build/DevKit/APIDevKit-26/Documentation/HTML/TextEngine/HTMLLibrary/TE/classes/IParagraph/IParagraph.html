Description

What is a paragraph? A paragraph is a subdivision of a written composition. It can contain one or more letters or sentences but can be empty too. Paragraphs are separetad in a rich text by separator character CR. The contained text can have one or more styles (TE::IRun, TE::ICharStyle). A paragraph allways begins on a new line. Therefore an empty paragraph (don't contains text) is an empty line. Important: an empty paragraph has a default style that determines the height of an empty line.

Beside the character level formatting styles the paragraph is formatted by the nexts: first left indent, left indent, right indent, line spacing, custom tabulators.

A non empty paragraph can break into more lines with filling of end of line positions. End of line position is an index to the last character of the line. This position is relative to the paragraph begin position, base is 0.
