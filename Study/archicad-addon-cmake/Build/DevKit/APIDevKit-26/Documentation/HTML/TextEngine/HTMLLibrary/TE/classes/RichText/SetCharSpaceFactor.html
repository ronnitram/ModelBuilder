TE::RichText::SetCharSpaceFactor Method

Sets the current value of character spacing factor. 

void SetCharSpaceFactor (double charSpaceFactor);

Parameters

charSpaceFactor
[in] The character spacing factor.

Description

Character spacing factor is a floating point number to increase or decrease the space between characters, but the width of characters remains unchanged. Default value is 1.0, the minimal value 0.75. The original width of text changes proportionally by setting of this factor.
