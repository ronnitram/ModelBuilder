TE::IOutlineBuilder::AddContour Method

Pure virtual function. Called before outlining every curve in the glyph. 

Int32 AddContour (
    const TE::IOutlineBuilder::PixCoord *    points,
    Int32               numOfPoints
) = 0;

Parameters

points
[in] The points of the contour in an array. The points are in screen coordinate system and connecting them (also the last with the first) will allways draw a closed non self meshing polygon.
numOfPoints
[in] The size of the array containing the points of the contour.

Return Value

Zero if there is no error.

Description

Pure virtual function. Glyphs contain one or more contours that can be the inner and outer curves and the accents. The contours bulid up from points defined by the IOutliner::PixCoord. After each contour the EndPoly is called.
