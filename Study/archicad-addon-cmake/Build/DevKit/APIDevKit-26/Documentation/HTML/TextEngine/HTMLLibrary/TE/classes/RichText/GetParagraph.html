TE::RichText::GetParagraph Method

Returns an interface pointer to the requested paragraph. 

TE::IParagraph * GetParagraph (Int32 paragraphIndex);
const TE::IParagraph * GetParagraph (Int32 paragraphIndex) const;

Parameters

paragraphIndex
[in]ParagraphIndex points to paragraph according to visual order of paragraph, base is 0.

Return Value

The pointer to IParagraph interface of requested paragraph.

Description

Returns an interface pointer to the requested paragraph. Indexes to paragraph growths according to visual order of paragraph, base is 0.
