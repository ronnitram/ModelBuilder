Description

Calling any methods for outlining texts (e.g. OutlineRichText) requires the implementation of a class that evaluates this IOutlineBuilder interface. Such a class will be handled all the points on the outlines of each letter. Texts are built up of glyphs that represent one ore more letters or even a part of one. Glyphs can contain more contours such as the outer and inner curves and the accents.
During a whole text outlining process one after the other all the following methods will be called by textengine in this given order: BeginTextOutline, then for each glyph BeginGlyph, for each contour in a glyph the AddContour and the EndPoly and than the EndGlyph. At the end of the whole text the EndTextOutline. StrikeOut and Underline methods are called separatelly giving all the corners of the line for the line. GetPlotScale will be called to determine the resolution to make for the points on the outlining curves.
