TE::IRichText::CopyAsSimple Method

Implemented method. It has ANSI and Unicode version. Makes a multiline "simple text". Converts texts for Archicad 8.1 and before. 

Int32 CopyAsSimple (
    GS::UniString *     content,
    short *             lStarts,
    EOLType             typeOfParSeparator = EOL_CR
) const;

Parameters

content
[out] In ANSI case CopyAsSimple converts content from Unicode to ANSI using GetGSCharCode. The length of "C" string can be different from Unicode length, it is recommended to set it min. twice as big as the length of Unicode content + 1. 
// Example:
  TE::IRichText*  iRTxt = ....;
  GS::UniString&  uContent = iRTxt->GetContent ();
  USize ansiBufferLength = 2 * uContent.GetLength () + 1;
lStarts
[out] This is a pointer to the first item of the array of line start positions. See IRichText::SimpleToRich.
typeOfParSeparator
[in] Paragraph separator. This separator will be inserted at the boundaries of paragraphs. The default value is CR.

Return Value

Description

Implemented method. It has ANSI and Unicode version. "Simple texts" are stored as a string and an array is filled with the line start positions. See IRichText::SimpleToRich. In ANSI version the line start positions are filled according to ANSI string positions. The conversion inserts a paragraph separator at the boundaries of the paragraphs. The default separator is the CR.
