TE::IRichText::SimpleToRich Method

Implemented method. It has ANSI and Unicode version. Converts multiline text into paragraph. Handling texts of Archicad 8.1 and before. 

void SimpleToRich (
    const GS::UniString & content,
    const short *       lStarts = nullptr,
    Int32               lineNum = 0
);

Parameters

content
[in] In ANSI case SimpleToRich converts to Unicode used GetGSCharCode .
lStarts
[in] This is a pointer to the first item of the array of line start positions. The first item is allways 0. The last item is equal the nunmber of characters in content. Therefore its dimension is allways the number of lines + 1. Each item contains a (0 based) position as an offset from the first character in the paragraph. 
// Example:
  char  *txt = "012345"
  short lStart [4] = {0, 3, 5, 6};
  // txt is displayed in three lines:
  // 012
  // 34
  // 5
lineNum
[in]Dimension of lStarts array.

Description

Implemented method. It has ANSI and Unicode version. "Simple texts" are stored as a string and an array is filled with the line start positions. The texts of Archicad 8.1 and previous versions are stored in this form. SimpleToRich converts "simple texts" into a richtext paragraph keeping the original line break positions. In the ANSI version the line start positions are adjusted if the ANSI-Unicode conversion changes the length of characters. Array of line start positions can be NULL, in this case SimpleToRich is equal with CreateParagraphs
