TE::Paragraph::GetLineWidths Method

Retrieves the line widths of the paragraph in pixels. 

TE::LineWidthArray * GetLineWidths ();
const TE::LineWidthArray * GetLineWidths () const;

Return Value

Floating point array of line widths in pixel.

Description

Retrieves the line widths of the paragraph in pixels. This method is usually called before drawing. Lengths of lines are measured by Textegine::MeasureParagraphLines. Since measuring is a resource demanding operation GetLineWidths gives a chance to gain performance with buffering of calculated data.
