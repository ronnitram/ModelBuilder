TE::IOutlineBuilder::BeginGlyph Method

Pure virtual function. Called before outlining a glyph. 

Int32 BeginGlyph (const TE::ICharStyle * iCharStyle) = 0;

Parameters

iCharStyle
[in] The style of this glyph.

Return Value

Zero if there is no error.

Description

Pure virtual function. When outlining a text BeginGlyph is called before outlining each glyph. The sytle of the glyph is given in the argument by an ICharStyle.
