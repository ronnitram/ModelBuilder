Description

A run describes a continous monostyle piece of a paragraph. It can contain one or more letters or sentences but can be empty too. Runs are not separetad in a paragraph. An empty paragraph (doesn't contain text) - is an empty line - contains an empty run. Important: an empty run has a default style that determines the height of an empty line.
