TE::IParagraph::GetLineWidth Method

Retrieves the given line width of the paragraph in pixels. 

double GetLineWidth (Int32 lineNum) const = 0;

Parameters

lineNum
[in] The given text line of the paragraph, base is 0.

Return Value

Floating point value of lineNumth line width in pixel.

Description

Retrieves the given line width of the paragraph in pixels. This method is usually called before drawing. Lengths of lines are measured by Textegine::MeasureParagraphLines. Since measuring is a resource demanding operation GetLineWidth gives a chance to gain performance with buffering of calculated data.
