TE::UnderlineLiftRate Variable

Constant. Its value is - 0.2200. 

const double UnderlineLiftRate

Description

Constant. Its value is - 0.2200. The product of the a font's capISize and this factor results an offset to the line of "underline" of underlined text. This offset is measured from baseline of normal text.
