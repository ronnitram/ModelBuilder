IParagraph Rich texts consists one or more piece of texts so called paragraph. IParagraph is an interface class through which textengine can handle paragraphs. IParagraph has only pure virtual methods.
IPlainTextBuilder IPlainTextBuilder is an interface for a class that handles the plaintext output of a richtext.
IRichText IRichText is an interface class for handling multistyled text. TextEngine handles rich texts (measuring, drawing,...) through this interface. Beside the declarations of pure virtual methods the interface implements some usefull methods for common problems.
IRun Paragraph consists of one or more piece of monostyle texts called run. IRun is an interface class through which textengine can handle runs. IRun has only pure virtual methods.

Run Run is an implementation of IRun interface class.
Tab A tabulator descriptor.
TextEngine TextEngine contains methods and data for handling fonts, mono- and multistyled texts in a platform independent way.
