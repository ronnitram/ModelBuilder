TE::IRichText::GetLineWidths Method

Pure virtual function. Retrieves the line widths of the given paragraph in pixels. 

bool GetLineWidths (
    Int32               paragraphIndex,
    GS::Array< double > * lineLengths
) = 0;
bool GetLineWidths (
    Int32               paragraphIndex,
    GS::Array< double > * lineLengths
) const = 0;

Parameters

paragraphIndex
[in] The index of the paragraph.
lineLengths
[out] Floating point array of line widths in pixels.

Return Value

Return false if the paragraphIndex is wrong else return true.

Description

Retrieves the line widths of the paragraphIndexth paragraph in pixels. This method is usually called befor drawing. You should buffer the retrieved datas in the implementation of this method.
