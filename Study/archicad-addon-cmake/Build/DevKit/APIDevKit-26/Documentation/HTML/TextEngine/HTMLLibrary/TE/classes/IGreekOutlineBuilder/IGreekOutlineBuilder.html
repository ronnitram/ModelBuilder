Description

GreekDrawing is a simple representation of texts. When the letters are too small to be possible to read them instead of the time consuming process of the proper outling of the letters we just draw the framing rectangle for each whole line. Outlining texts in greek mode (e.g. GreekOutlineRichText) requires the implementation of a class that evaluates this IGreekOutlineBuilder interface. Such a class will be handled all the corner points on the outlines of rectangles of each line. During a whole text outlining process one after the other all the following methods will be called by textengine in this given order: BeginGreekOutline, then for each line LineRect and at the end of the whole text the EndGreekOutline.
