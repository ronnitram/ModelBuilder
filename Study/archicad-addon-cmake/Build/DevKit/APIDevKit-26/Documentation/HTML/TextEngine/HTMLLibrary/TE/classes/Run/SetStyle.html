TE::Run::SetStyle Method

Sets the style of the run. 

void SetStyle (const TE::ICharStyle & style);

Parameters

style
[in] Const reference to ICharStyle.

Description

With this method you can set the style of the run. ICharStyle interface determines the format of a run. Character format consits of a font with simple styles, a color, and an extended style set which determines the representation of a character.
