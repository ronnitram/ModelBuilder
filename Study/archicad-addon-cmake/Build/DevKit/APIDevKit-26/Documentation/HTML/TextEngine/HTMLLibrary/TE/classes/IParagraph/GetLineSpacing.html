TE::IParagraph::GetLineSpacing Method

Returns the factor used to set the height of a line. 

double GetLineSpacing () const = 0;

Return Value

The floating point value of the line spacing factor.

Description

The height of a line is the sum of the ascent, the descent and the leading of the largest font used in the line. See the detailed font metrics description. The spacing between lines can be modified by the lineSpacing factor that is used as a multiplier for the lineheights. With its value of -1.0 the lineheights remain the same. With a value less than -1.0 the original lineheights will simply be multiplied by the absolute value of the factor so that the spaces will only be set greater after each line. With a value greater than -1.0 but less then -0.5 both the ascent and the sum of the descent and leading will separately be multiplied by the absolute value of the spacing factor so both of the spacings before and after each line will be less then the original.

Example

// Example:
  double lineSpace = iPar->GetLineSpacing ();
