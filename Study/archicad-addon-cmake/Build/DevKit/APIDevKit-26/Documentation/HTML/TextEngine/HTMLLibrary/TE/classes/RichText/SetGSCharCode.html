TE::RichText::SetGSCharCode Method

Sets the GSCharCode of rich text. 

void SetGSCharCode (GSCharCode charCode);

Parameters

charCode
[in] The GSCharCode of rich text.

Description

Sets the character code representing a code page on Windows, or a script system on Macintosh. A rich text contains only one GSCharCode used by Unicode-ANSI conversions.
