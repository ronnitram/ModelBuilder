TE::RichText::GetDefaultTab Method

Returns the width of the default tab. 

double GetDefaultTab () const;

Return Value

The default tab width in pixels.

Description

In the case there are no defined tab positions in a paragraph, or after the last defined tabulator in a line each tab positions the text to the next round multiple of this default tab value. TextEngine uses default tab in pixels.
