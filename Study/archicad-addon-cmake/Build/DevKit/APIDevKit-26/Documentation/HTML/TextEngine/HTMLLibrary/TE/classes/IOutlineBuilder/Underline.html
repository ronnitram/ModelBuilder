TE::IOutlineBuilder::Underline Method

Gives the position of an underline. 

Int32 Underline (
    const TE::ICharStyle *  param1,
    const TE::IOutlineBuilder::PixCoord *    param2,
    Int32               param3
);

Parameters

param1
[in] The style of the line.
param2
[in] The array of points. The points are in screen coordinate system and connecting them (also the last with the first) will arlways draw a rectangle.
param3
[in] The number of points in the array. Expected to be 4.

Return Value

Zero if there is no error.

Description

Gives the corner points in an array of the underline that really is a thin box.
