About GSRoot

The GSRoot library provides base infrastructure and various interfaces (managers) to add your program functionality.

Using GSRoot and managers

First include GSRoot.hpp to your source. This file contains the common declarations about calling formats, manager identifiers, etc.

To use Containers, Iterators, Algorithms or Unicode Strings include the headers specified in their respective documentation.

The GSRoot library is initialized and terminated automatically, there is no Init or Close function for GSRoot.
