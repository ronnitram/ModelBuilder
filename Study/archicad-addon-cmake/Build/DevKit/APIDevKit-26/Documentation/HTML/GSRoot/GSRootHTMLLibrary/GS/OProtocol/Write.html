OProtocol::Write

Decomposes simple C++ types (and Object) to an output channel.

    OChannel &          output,
    char                c
) = 0;

) = 0;

    bool                b

    OChannel &          output,

virtual GSErrCode Write (
    OChannel &          output,
    const Object &      obj
) = 0;

Parameters

output
Output channel receiving the binary data (usually using the OChannel::WriteBin method).
c
Character value to be decomposed.
b
Boolean value to be decomposed.
s
Short integer value to be decomposed.
i
Integer value to be decomposed.
l
Long integer value to be decomposed.
f
Float value to be decomposed.
d
Double value to be decomposed.
p
Pointer to be decomposed (for debuging purposes only).
cStr
C string to be decomposed.
obj
Object to be decomposed.

Return Value

Value Meaning
NoError* OK
OutputIsFull* output device is full
WriteError* write error on the output device
Error* unspecified error

* declared in OChannel class scope
* declared in GS namespace, but accessible in the global scope

Description

This method decomposes simple C++ types (and Object) using binary level of the specified output channel.
