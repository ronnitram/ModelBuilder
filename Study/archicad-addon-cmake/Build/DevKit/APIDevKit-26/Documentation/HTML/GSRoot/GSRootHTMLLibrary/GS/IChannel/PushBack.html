IChannel::PushBack

Pushes data back to the input channel. 

virtual GSErrCode PushBack (
    const char *        source,
    USize               nOfChar
) = 0;
virtual GSErrCode PushBack (char c);

Parameters

source
Source buffer containing the data to be pushed back to the input channel.
nOfChar
Number of characters to be pushed back to the input channel.
c
Character to be pushed back to the input channel.

Return Value

Value Meaning
NoError* OK
ErrMemoryFull* out of memory
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

These methods push data (sequence of characters or a single character) back to the input channel, so the subsequent read operations will encounter this data first and old content only after that.
