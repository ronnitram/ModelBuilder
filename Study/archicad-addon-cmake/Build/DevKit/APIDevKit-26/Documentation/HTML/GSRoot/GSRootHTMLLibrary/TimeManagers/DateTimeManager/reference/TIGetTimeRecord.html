TIGetTimeRecord

The TIGetTimeRecord function converts the specified GSTime to the specified time structure.

GSErr TIGetTimeRecord(
  GSTime                gsTime,            // time to convert
  GSTimeRecord  *pTimeRecord,      // buffer receiving the converted
                                // time structure
  GSFlags               formatFlags        // conversion attributes
);

 

Parameters

gsTime
Specifies the time variant to convert. When the formatFlags parameter is set toTI_CURRENT_TIME flag, the gsTime parameter is ignored.
pTimeRecord
Specifies the buffer receiving the result of the conversion.
formatFlags
Specifies how to convert. This parameter can be the combination of the following values:
Value Meaning
TI_LOCAL_TIME Local time
TI_UTC_TIME UTC time
TI_CURRENT_TIME Current time (UTC or local)
If you specify the TI_CURRENT_TIME flag, then the gsTime parameter is ignored.

The TI_LOCAL_TIME or TI_UTC_TIME flag can be combined with the TI_CURRENT_TIME flag to get the current local or UTC time.

Return Values

The return value indicates the success of the conversion with an error code. For more information about the error codes, see GSRoot error codes.

If you specify the TI_CURRENT_TIME flag as formatFlags, the function calls TIGetTime function to get the current time and ignores the value of the gsTime parameter.

Remarks

If you want to convert in opposite direction, call TIGetGSTime.

Requirements

    Header: Declared in GSTime.hpp
    Import Library: Link with GSRootImp.lib
