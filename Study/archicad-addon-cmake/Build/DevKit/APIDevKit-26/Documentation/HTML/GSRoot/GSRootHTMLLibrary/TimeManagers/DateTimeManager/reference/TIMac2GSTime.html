TIMac2GSTime

The TIMac2GSTime function converts the Macintosh time format to platform independent GSTime format.

GSErr TIMac2GSTime(
  unsigned long macTime,      // Macintosh time
  GSTime                *pGSTime      // buffer receiving converted time
);

 

Parameters

macTime
Specifies the time variable to convert.
pGSTime
Specifies the buffer receiving the platform independent time.

Return Values

The return value indicates the success of the conversion with an error code. For more information about error codes, see GSRoot error codes.

Remarks

If you want to convert to the opposite direction, call TIGS2MaxTime.

Requirements

    Header: Declared in GSTime.hpp
    Import Library: Link with GSRootImp.lib
