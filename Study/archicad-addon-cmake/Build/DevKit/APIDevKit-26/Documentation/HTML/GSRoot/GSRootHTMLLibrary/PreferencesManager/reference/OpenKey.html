OpenKey

Opens a root prefrences key.

    RootKeyRef OpenKey (
      RootKeyType       type,                  // Type of the root key
      RootKeyDomain     domain,                        // Domain of the root key
    );

Parameters

type
[in] !!!ToDo
domain
[in] !!!ToDo

Return Values

If the function succeeds, retruns a reference to a root key.

If the function fails, the return value is zero. This function can fail if:

-Not enough memory.

-Invalid falgs/parameters.

Remarks

The pChars and pWChars pointers must not be the same, or overlap. If they do the result of this function is undefined!

The unrepresentable characters are translated to system dependent default replacement character on Windows and to '?' on MacOS.

This function is implemented using CoreFoundation String services on MacOS, and MultiByteToWideChar on Windows.

Archicad 7.0 R1/V1 implementation issues:The maxWChars parameter cannot be 0 on MacOS platform. This bug should be fixed in later revisions.
