Array<Type>::Begin

Returns an iterator referring to the first item in the array. 

Iterator Begin (void);
ConstIterator Begin (void) const;

Return Value

A random iterator or a constant random iterator referring to the first item in the array.

Description

This method allows random enumeration of the array's content. It returns a random iterator or a constant random iterator referring to the first item in the array. Enumeration ordering is index based.
References to items obtained by iterator dereferencing can be used only until the next modifying method on the parent array is invoked.
This method is equivalent with the Enumerate method.

Example

GS::Array<GS::String> array;  // array of strings

GS::Array<GS::String>::Iterator begin = array.Begin ();  // obtaining an iterator referring to the first item
GS::Array<GS::String>::Iterator end   = array.End ();    // obtaining an iterator referring to the one past last item
while (begin != end) {                   // until the end of the sequence (array)
    GS::String s = *begin;               // dereferencing the iterator (returns reference)
    *begin = s;                          // dereferencing the iterator (returns reference)
    USize length = begin->GetLength ();  // dereferencing the iterator (returns pointer)
    ++begin;                             // advance to the next item
}


GS::Sort (array.Begin (), array.End ());  // using iterators in algorithms

This example illustrates the use of iterators obtained from an array.
