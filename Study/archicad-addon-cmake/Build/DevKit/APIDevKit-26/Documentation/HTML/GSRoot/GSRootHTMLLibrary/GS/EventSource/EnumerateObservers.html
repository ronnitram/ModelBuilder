EventSource::EnumerateObservers

Enumerates observers attached to the event source. 

ObserverEnumerator EnumerateObservers (void) const;

Return Value

Enumerator object that can be used to enumerate event observers attached to the event source.

Description

This method can be used for the enumeration of event observers attached to the event source. It returns an enumerator object that has (postfix) operator++ method to iterate over event observers, operator* to obtain the current observer and operator== and operator!= methods to compare against NULL to detect end of the enumeration.
