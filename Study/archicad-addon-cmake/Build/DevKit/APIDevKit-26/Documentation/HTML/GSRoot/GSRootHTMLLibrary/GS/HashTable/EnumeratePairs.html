HashTable<IndexType, ItemType>::EnumeratePairs

Allows enumeration of the hashtable's content. 

PairIterator EnumeratePairs (void);
ConstPairIterator EnumeratePairs (void) const;

Return Value

A forward iterator or a constant forward iterator referring to the first index-item pair in the hashtable.

Description

This method allows forward enumeration of the hashtable's content as sequence of pairs of indices and items (keys and values). It returns a forward iterator or a constant forward iterator referring to the first index-item pair in the hashtable. The pair is a simple struct having two members: 

    const IndexType*  key;
    ItemType*         value;
for non-constant forward iterators and 
    const IndexType*  key;
    const ItemType*   value;
for constant forward iterators, where key points to the current index and value points to the current item. So the current index and current item can be accessed as following: 
    *(iterator->key)    // reference to the index
    *(iterator->value)  // reference to the item
or 
    iterator->key->anIndexMember
    iterator->value->anItemMember
Enumeration order is not defined.
References to pairs obtained by iterator dereferencing can be used only until the next modifying method on the parent hashtable is invoked.
This method is equivalent with the BeginPairs method.

Example

GS::HashTable<GS::String, long> hashTable;  // hashtable of {strings, long} pairs

GS::HashTable<GS::String, long>::PairIterator it = hashTable.EnumeratePairs ();  // obtaining a pair iterator (or hashTable.BeginPairs ())
while (it != NULL) {              // until the end of the sequence (hashtable)
    GS::String s = *(it->key);    // dereferencing the iterator (returns pointer to the pair structure)
    long l       = *(it->value);  // dereferencing the iterator (returns pointer to the pair structure)
    *(it->value) = 4;             // dereferencing the iterator (returns pointer to the pair structure)
    ++it;                         // advance to the next pair
}

This example illustrates the use of pair iterators obtained from a hashtable.
