CHEqualASCII

Compares two null terminated ASCII strings with or without case sensitivity.

    bool CHEqualASCII (
      const char*  p1,
      const char*  p2,
      CaseType     cs = CS_CaseSensitive
    );

Parameters

p1
[in] The C string to be compared. Characters should be in the ascii (0-127) range.
p2
[in] The C string to be compared with. Characters should be in the ascii (0-127) range.
cs
[in] Case sensitivity switch.

Return Values

true if the strings are equal, false otherwise.

Remarks

Don't use this function for comparing non-ASCII text. If either p1 or p2 contains non ascii characters the result is undefined. To compare non-ASCII strings use CHEqualCStrings!

The debug version gives assertion if either string contains a non-ASCII character.
