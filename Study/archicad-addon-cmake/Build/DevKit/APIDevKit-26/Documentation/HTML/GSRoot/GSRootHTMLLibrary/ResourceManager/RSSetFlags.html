RSSetFlags

Sets the values of the various flags, which determine the work of the Resource Manager.

    void RSSetFlags (
      GSFlags  flags
    );

Parameters

flags
The new values of the flags.

Remarks

The values of the RSFlags enumeration can be used as masks to set of the flags. In this version of the Resource Manager only RSF_MissingResourceAlert is defined. If that flag is set, the Resource Manager break into the debugger, when a resource is not found (debug version only). The default state of this flag is set.

Requirements

Version: 1.3 or later
Header: RS.hpp
Import Library: Link with GSRootImp.lib
Module: GSRootLib [MAC], GSRoot.dll [WIN]
