VASpac

The VASpac function increases the size of a Virtual Array by npage elements.

long VASpac(
  DESC             *dptr   // pointer to virtual array
);

 

Parameters

dptr
Pointer to the virtual array of which size is increased.

Return Values

If the function succeeds, the return value is the new length of the Virtual Array. (The lastind member of the DESC structure)

If the function fails, the return value is a Graphisoft error code. For more information about error codes, see GSRoot error codes.

If the dptr Virtual Array is uninitialized then the return value is ErrNilHandle.

Remarks

The VASpac function increases the size of the buffer of the Virtual Array if necessary.

Requirements

    Header: Declared in VA.hpp
    Import Library: Link with GSRootImp.lib
