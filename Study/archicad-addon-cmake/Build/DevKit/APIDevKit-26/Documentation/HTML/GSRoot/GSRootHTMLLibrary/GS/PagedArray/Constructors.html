PagedArray<Type>::PagedArray

PagedArray constructor. 

PagedArray (
    USize               pageSize = 0,
    AllocationStrategy  allocStrategy = Linear
);
PagedArray (const PagedArray & source);
PagedArray (const RandomContainer< Type > & source);
PagedArray (const Container< Type > & source);

Parameters

pageSize
Size of a storage page in items. 0 means the default page size which is currently 128 for the constant allocation strategy, 32 for the linear allocation strategy and 16 for the exponential allocation strategy. If the given page size is not power of 2 then it will be rounded to the next power of 2 (for example 128 yields 128, 65 also yields 128, 129 yields 256). Largest possible page size is 65536.
allocStrategy
Determines the allocation strategy of the PagedArray. The default case is the Linear allocation strategy.
source

Description

Constructs a new array with the specified page size and allocation strategy.
After the construction no memory is allocated. It will be allocated the first time when it is really needed.
