RSSearchResource

The RSSearchResource search for a given resource in the opened resource modules.

    bool RSSearchResource (
      GSResType     resType,
      GSResID       resID,
      GSResModule*  pResModule
    );

Parameters

[in] resType
A four byte resource type identifier.
[in] resID
A numeric resource identifier.
[out] pResModule
Returns the module id of the resource module, which contains the resource. This parameter cannot be NULL.

Return Values

Returns true if this resource is available in list of resource modules opened with RSOpenResourceFile. Returns false if the resource cannot be found.

Remarks

MacOS This function reads the resource into memory, then releases it!
