Description

Container is root class of all container classes. Container is a storage object which stores variable number of items (elements). They may store multiple items with the same value. Containers have size which is number of items they actually store. Every container is able to enumerate its content (items) in some (maybe unspecified) order.
Containers store items by value (of course they can store pointers too - by value). Implementations should specify lifetime of references to these items.
Items are required to have default and copy constructors and assigment operator. In addition if they have comparison operators (== and !=) then value dependent operations (eg. Find, Count, Contains etc.) will be also useable.
