DESC

The DESC structure describes a Virtual Array.

typedef struct DESC{
  long             lastind;
  long             maxind;
  long             npage;
  long             size;
  GSHandle arrhdl;
} DESC;

 

Members

lastind
Specifies the index of the last element in the Virtual Array. This is the length of the Virtual Array. If you want to increase the size of the Virtual Array you can do it by calling the VASpac, VANspac or VAReal function.
maxind
Specifies the last allocated index of the Virtual Array. For more information see Remarks.
npage
Specifies the number of elements by which the Virtual Array is increased or decreased. For more information see Remarks.
size
Specifies the size of the elements stored in the Virtual Array.
arrhdl
Specifies the handle of the buffer where the elements of the Virtual Array are stored.

Remarks

The Virtual Array has an allocated buffer arrhdl which stores the elements of the Virtual Array. The allocated buffer can store maxind elements. VASpac and VANspac functions increase the size of the Virtual Array (but not the size of the buffer), then check weather the new elements fits into the existing allocated buffer. If it fits into the existing buffer, the function increase the lastind pointer of the DESC structure. If it does not fit into the existing buffer, the function increases the size of the buffer continuously and increase the lastind pointer. The size of the buffer is always increased by npage elements.

Note the maxind pointer always must be greater than or equal to the lastind pointer.

By calling VARelease function you can decrease the size of the buffer. The VARelease function checks the size of the allocated buffer and the number of elements stored in it. If the stored elements fit in a smaller buffer then decreases the buffer size by npage elements.

Note, if you clear an element from the Virtual Array by calling the VAClearIndex function, then the size of the Virtual Array is decreased, but the size of the internal buffer remains untouched. If you want to save memory you must call the VARelease function after the VAClearIndex function.
