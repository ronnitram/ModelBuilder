TIGetDoubleClickTicks

The TIGetDoubleClickTicks function retrieves the number of ticks elapsed between the first and the second click of a double-click.

GTicks TIGetDoubleClickTicks();

 

Return Values

The return value is the number of ticks elapsed between the first and the second click of a double-click.

Requirements

   Header: Declared in GSTime.hpp
   Import Library: Link with GSRootImp.lib
