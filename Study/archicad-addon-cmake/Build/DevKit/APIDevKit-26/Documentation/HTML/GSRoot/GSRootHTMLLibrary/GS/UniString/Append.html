UniString::Append

Appends new content to the end of the string. 

void Append (UniChar uc);
void Append (
    const UniString &   str,
    UIndex              strFrom = 0,
    USize               strRange = MaxUSize
);
void Append (const UniChar::Layout * uStr);
void Append (
    const UniChar::Layout * uChars,
    USize               charCount
);
void Append (const char * cStr);
void Append (
    const char *        chars,
    USize               charCount
);
void Append (const unsigned char * pStr);

Parameters

uc
Character to be appended.
str
String to be appended. Substring of this string can be also specified.
strFrom
Start position of the substring in the given string to be appended. Must be beetwen 0 and length of the given string.
strRange
Range of characters in the given string to be appended. strFrom + strRange must be beetwen 0 and length of the given string. The default value MaxUSize indicates range that extends to end of the given string.
uStr
0 terminated sequence of 16 bits unsigned short Unicode characters.
uChars
Sequence of 16 bits unsigned short Unicode characters.
charCount
Number of characters to be appended.
cStr
0 terminated C string to be appended.
chars
Sequence of 8 bits signed characters (char) to be appended.
pStr
Pascal string to be appended.

Description

This method appends new content (character, string, substring, Unicode string, C string, Pascal string etc) to the end of the string. New content is converted to the Unicode representation using the default user locale.
On memory overflow OutOfMemoryException exception will be thrown.
