CHTruncate

Truncate C string. [strncpy]

    long CHTruncate (
      const char*  sourceCString,
      char*        targetCString,
      long         targetSize,
      GSCharCode   charCode = CC_Default
    );

Parameters

sourceCString
Source C string.
targetCString
Destination C string.
targetSize
Length of the destination buffer. This length is in bytes, and it includes the null terminator character!
charCode
Character code.

Return Values

The number of bytes in the targetCString not including the null terminator.

Remarks

targetCString and sourceCString might be the same, but they cannot overlap!

targetCString should be at least targetSize bytes long!

targetSize is the length of destination buffer, and the destination string will be null terminated!
