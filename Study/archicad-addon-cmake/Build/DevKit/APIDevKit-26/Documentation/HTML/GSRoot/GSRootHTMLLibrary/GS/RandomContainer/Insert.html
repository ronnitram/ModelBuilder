RandomContainer<Type>::Insert

Inserts item(s) at the specified position. 

void Insert (
    UIndex              to,
    const Type &        item
);
void Insert (
    UIndex              to,
    ConstForwardIterator< Type > begin
);
void Insert (
    UIndex              to,
    ConstForwardIterator< Type > begin,
    ConstForwardIterator< Type > end
);

Parameters

to
Position where item(s) should be inserted. Must be beetwen 0 and size of the random container.
item
Item to be inserted.
begin
Constant forward iterator referring to the first item of the sequence of items to be inserted.
end
Constant forward iterator referring to the one past last item of the sequence of items to be inserted.

Description

This method inserts an item or sequence of items to the random container at the specified position.
On memory overflow OutOfMemoryException exception will be thrown.
