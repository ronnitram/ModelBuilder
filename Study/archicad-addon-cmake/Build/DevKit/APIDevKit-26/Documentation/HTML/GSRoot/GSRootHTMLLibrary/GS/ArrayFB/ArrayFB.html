Description

ArrayFB is an Array with built-in initial buffer of fixed size. This buffer will be used until its capacity is exceeded. When this happens a new (larger) buffer will be allocated on the heap.
This kind of array is useful when number of items in most cases does not exceed a well-known (smaller) limit. In such cases this class will be fast because it will allocate no memory from the heap. For example it can be used for computations inside methods and functions as local objects, or as member variables in classes when small number of items is expected (from the layout view items will be stored in the containing object).
