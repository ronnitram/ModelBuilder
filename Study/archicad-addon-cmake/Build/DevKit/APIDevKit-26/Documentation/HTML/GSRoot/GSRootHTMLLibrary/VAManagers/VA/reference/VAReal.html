VAReal

The VAReal function modifies directly the size of a Virtual Array.

GSErr VAReal(
  DESC         *dptr,   // pointer to virtual array
  long         lasti,   // the new index of the last used cell 
  long         maxi     // the new size of virtual array given with the number of cells
);

 

Parameters

dptr
Pointer to the virtual array of which size is modified. It can not be NULL.
lasti
The new index of the last used virtual array cell. It must be greater than or equal to zero.
maxi
The new size of the virtual array given with the number of cells. It must to be greater than or equal to lasti.

Return Values

If the function succeeds, the return value is 0 (NoError).

If the function fails, the return value is a Graphisoft error code. For more information about error codes, see GSRoot error codes.

If dptr is NULL then the return value is ErrNilHandle.

Remarks

The VAReal function increases or decreases the size of the allocated buffer of the Virtual Array if necessary.

If the maxi parameter is less than lasti, and lasti fulfills the constraint to be non-negative, the VAReal function modifies maxi to lasti.

Requirements

    Header: Declared in VA.hpp
    Import Library: Link with GSRootImp.lib
