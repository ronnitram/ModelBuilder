UniChar::operator=

Assigment operator. 

UniChar & operator= (Layout u);
UniChar & operator= (char c);
UniChar & operator= (unsigned char c);
UniChar & operator= (const UniChar & source);

Parameters

u
16 bits unsigned short Unicode character.
c
8 bits signed / unsigned character.
source
Another UniChar.

Return Value

Reference to this UniChar.

Description

UniChar assigment operator. Non-Unicode representations are converted to Unicode using the default user locale.
