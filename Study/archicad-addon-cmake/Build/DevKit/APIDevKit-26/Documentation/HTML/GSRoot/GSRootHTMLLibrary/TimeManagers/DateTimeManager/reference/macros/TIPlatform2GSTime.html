TIPlatform2GSTime

The TIPlatform2GSTime macro calls the TIMac2GSTime function over Macintosh and the TIWin2GSTime function over Windows.

#if defined(macintosh)
  #define TIPlatfrom2GSTime TIMac2GSTime
#else
  #define TIPlatfrom2GSTime TIWin2GSTime
#endif

Requirements

    Header: Declared in GSTime.hpp
