BMInit

The BMInit function initializes the Basic Memory Manager.

void BMInit(
  BMAbortProc     *myAbort,                      // abort action
  BMOptProc       *myOptimise,           // optimise action
  GSSize          minMem                 // minimal required size of free memory
);

 

Parameters

myAbort
Pointer to an abort function. If a fatal memory error occurs, the BM calls this function (if it is not NULL) to do something before exiting. For more information see the description of BMAbortProc.
myOptimise
Pointer to an optimize function. If a memory allocation request fails, BM calls this function (if it is not NULL) to give a chance to the application to release some memory before trying the allocation again. For more information see the description of BMOptProc.
minMem
The size of the minimum required free memory. If the size of the free memory (as a result of a memory allocation) becomes less than this value, a BM error occurs, and the allocation request fails.

Remarks

If you finish using BM, call the BMExit function.

Requirements

    Header: Declared in BM.hpp
    Import Library: Link with GSRootImp.lib
