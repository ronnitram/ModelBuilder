IChannel::IsEOI

Returns whether has been the end of input reached. 

virtual bool IsEOI (void);

Return Value

Value Meaning
true end of input has been reached
false end of input has not been reached

Description

This method returns whether has been the end of input reached. After the end of input has been reached the input channel will always return with the EndOfInput error code.
