IVInt64s

Swaps the bytes in an array of 64-bit integers.

    void IVInt64s (
        GS::PlatformSign platform,
        Int64 *pInt64s,
        long nInt64s
    );
    void IVInt64s (
        GS::PlatformSign from,
        GS::PlatformSign to,
        Int64 *pInt64s,
        long nInt64s
    );

Parameters

platform, from
[in] The input platform.
to
[in] The output platform for the second function.
pInt64s
[in, out] Points to an array of Int64. The 8 bytes will be reversed in all Int64-s if necessary.
nInt64s
[in] The number of elements in pInt64s.

Remarks

You can use this function for either a big-endian->little-endian, or little-endian->big-endian conversion.
