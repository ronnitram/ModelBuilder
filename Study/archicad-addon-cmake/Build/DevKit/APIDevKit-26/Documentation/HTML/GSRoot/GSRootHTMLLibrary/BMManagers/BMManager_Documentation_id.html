Basic Memory Manager

The Basic Memory Manager provides Basic Memory management functions. The function are groupped into two managers: the BM Manager and the BN Manager. The BM Manager responses for allocating, deallocateing memory blocks through handles and pointers. The BN Manager provides helper function which operates with memory blocks, e.g. copy, move, compare.

The Basic Memory Manager are used for handling memory blocks in Archicad. This memory management API provides platform independent interface to manage memory in both Windows and Macintosh.
