Fill

Assigns the same new value to every element in a specified range. 

template <class For, class T>
void Fill (
    For                 first,
    For                 last,
    const T &           value
);

Template parameters

For
Forward iterator.
T
T type template parameter.

Parameters

first
A forward iterator addressing the position of the first element in the range to be traversed.
last
A forward iterator addressing the position one past the final element in the range to be traversed.
value
The value to be assigned to elements in the range [first, last).

Description

Fill assigns the value value to every element in the range [first, last). That is, for every iterator i in [first, last), it performs the assignment *i = value.
