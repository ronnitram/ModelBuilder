Return Value

The specified substring. It can be empty.

Description

This method returns a substring of the string. Substring can be specified with its start position and length or with its enclosing characters. In the later case if no appropriate substring is found then empty substring is returned. Enclosing (separator) characters are not part of the substring.
The substring refers to the specified part of the source string, so itself never allocates memory. Substring is a private type so it can be used only inside of the enclosing expression (statement) of the call.

Substrings can be effectively compared with various types (string, substring, C string etc) and can participate in various assigments and concatenations. These operations are always performed on the corresponding parts of the source strings.

Example

GS::UniString us ("Example string");

bool b1 = us (3, 2) == us (5, 2);          // substrings can be compared with each other
bool b2 = us ('{', '}', 0) == "Data";      // substrings can be compared with other types
us (0, 3) = "abcde";                       // substring can participate in assigments
us = us (1, 2) + 'x' + "123" + us (4, 1);  // substring can participate in concatenations

This example illustrates the use of substring.
