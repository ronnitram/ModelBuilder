EventSource::Attach

Attaches a new observer to the event source. 

void Attach (EventObserver & observer);

Parameters

observer
Event observer to be attached to the event source.

Description

This method attaches a new event observer to the event source. The event source will send events to the event observer until it is detached or any of them is destroyed. Order of notifications is not specified.
