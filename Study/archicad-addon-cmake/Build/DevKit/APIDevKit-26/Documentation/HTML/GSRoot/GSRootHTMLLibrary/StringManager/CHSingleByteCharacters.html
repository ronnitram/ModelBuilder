CHSingleByteCharacters

Returns true if the given character code contains single byte characters only.

    bool CHSingleByteCharacters (
      GSCharCode  charCode = CC_Default
    );

Parameters

[in] charCode=CC_Default
The character code to test.

Return Values

This function returns true if the code page or script system represented by the charCode is a singlebyte system.

Remarks

Please avoid using this function. Use functions defined in CH to handle multi-byte text.

Requirements

Version: GSRoot 1.3 or later
Header: CH.hpp
