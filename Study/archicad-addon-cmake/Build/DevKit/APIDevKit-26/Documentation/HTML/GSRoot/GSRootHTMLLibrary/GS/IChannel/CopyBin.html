IChannel::CopyBin

Copies binary data from the input channel. 

virtual GSErrCode CopyBin (
    char *              destination,
    USize               nOfCharToCopy,
    USize *             copied = NULL
) = 0;

Parameters

destination
Destination buffer receiving the binary data.
nOfCharToCopy
Number of characters to be copied into the destination buffer.
copied
On return: if not NULL then number of characters successfully copied from the input channel.

Return Value

Value Meaning
NoError* OK
EndOfInput* end of input has been reached, operation is not completed
ReadError* read error on the input device, operation is not completed
Error* unspecified error

* declared in IChannel class scope
* declared in GS namespace, but accessible in the global scope

Description

This method copies binary data from the input channel. It copies specified number of characters into a destination buffer. These characters are not consumed, that is they are not removed from the input channel. The next read or copy operations will encounter them again.
If there is no enough data left on the input channel then EndOfInput is returned. In such cases the output copied parameter contains the number of characters that has been successfully copied.
Input status will record and hold the first error occurred on the input channel. After an error this method does nothing and immediately returns with the stored error code (until the input status is reseted by the user).
