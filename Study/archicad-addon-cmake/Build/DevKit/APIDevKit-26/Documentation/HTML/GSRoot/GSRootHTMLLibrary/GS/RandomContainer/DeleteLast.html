RandomContainer<Type>::DeleteLast

Deletes the last item equal with a specified item. 

UIndex DeleteLast (const Type & item);

Parameters

item
Specifies the item to be deleted.

Return Value

Index of the deleted item or MaxUIndex if no matching item is found.

Description

This method deletes the last item equal with a specified item from the random container. It requires items to have comparison operators (== and !=).
