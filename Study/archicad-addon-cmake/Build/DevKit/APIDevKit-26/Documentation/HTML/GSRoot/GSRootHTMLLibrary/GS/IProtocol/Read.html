IProtocol::Read

Composes the next simple C++ type (or Object) from an input channel.

Return Value

Value Meaning
NoError* OK
EndOfInput* end of input has been reached
ReadError* read error on the input device
IntegrityError* data read is not valid for the requested type
Error* unspecified error

* declared in IChannel class scope
* declared in GS namespace, but accessible in the global scope

Description

This method composes the next simple C++ type (or Object) using binary level of the specified input channel. If there is no data left on the input channel then IChannel::EndOfInput should be returned. If amount of data left on the input channel is not sufficient for the composition of the requested type or the type can not be composed from the data then GS::IntegrityError should be returned. IChannel::ConvertObjectReadError method makes the error handling a little easier.
