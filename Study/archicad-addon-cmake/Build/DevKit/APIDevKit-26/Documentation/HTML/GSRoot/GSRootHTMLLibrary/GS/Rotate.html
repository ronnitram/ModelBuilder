Rotate

Exchanges the elements in two adjacent ranges. 

template <class For>
void Rotate (
    For                 first,
    For                 mid,
    For                 last
);

Template parameters

For
Forward iterator.

Parameters

first
A forward iterator addressing the position of the first element in the range to be rotated.
mid
A forward iterator defining the boundary within the range that addresses the position of the first element in the second part of the range whose elements are to be exchanged with those in the first part of the range.
last
A forward iterator addressing the position one past the final element in the range to be rotated.

Description

Rotate rotates the elements in a range. That is, the element pointed to by mid is moved to the position first, the element pointed to by mid + 1 is moved to the position first + 1, and so on. One way to think about this operation is that it exchanges the two ranges [first, mid) and [mid, last). Formally, for every integer n such that 0 <= n < last - first, the element *(first + n) is assigned to *(first + (n + (last - mid)) % (last - first)). Rotate returns first + (last - mid).
