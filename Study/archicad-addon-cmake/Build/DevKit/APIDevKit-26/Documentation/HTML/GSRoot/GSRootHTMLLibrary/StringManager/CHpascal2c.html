CHpascal2c

Converts a Pascal string to 0-terminated C-string.

    long CHpascal2c (
      const unsigned char*  pPascalString,
      char*                 pCString
    );

Parameters

pPascalString
[in] The input, a Pascal string.
pCString
[out] The output, a buffer for a C string, large enough to perform the conversion.

Return Values

Returns the number of characters in nPascalString. This value is always less than 256.

Requirements

Version: GSRoot 1.3 or later
