RandomContainer<Type>::operator!=

Compares two random containers. 

bool operator!= (const RandomContainer & rightOp) const;

Parameters

rightOp
Right operand of the comparison.

Return Value

Value Meaning
true containers are not equal
false containers are equal

Description

This method compares two random containers. Two random containers are equal if their sizes are equal and their corresponding items are equal (by comparing them by value).
This method requires items to have comparison operators (== and !=).
