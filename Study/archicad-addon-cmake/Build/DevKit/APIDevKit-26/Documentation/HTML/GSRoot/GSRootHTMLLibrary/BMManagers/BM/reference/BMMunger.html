BMMunger

The BMMunger function searches text for a specified string pattern and replaces it with another string.

long BMMunger(
  GSHandle        h,     // handle to a text string
  long            offset,        // byte offset in the destination string
  const void      *ptr1, // manipulation string
  long            len1,  // length of manipulation string
  const void      *ptr2, // substitution string
  long            len2   // length of substitution string
);

 

Parameters

h
Specifies the handle to the text string to manipulate.
offset
Specifies the byte offset in the destination string at which the BMMunger function begins its operation.
ptr1
Pointer to the first character of the string to search for by BMMunger.
len1
Specifies the number of bytes in the string to search for by BMMunger.
ptr2
Pointer to the first character of the substitution string.
len2
Specifies the number of bytes in the substitution string.

Return Values

If the function succeeds, the return value depends on the actual parameters of the function. For more information see Remarks.

If the function fails to find the designated string, the return value is negative.

Remarks

The BMMunger function takes special action if either of the specified pointer values is NULL or either of the length values is 0.

  • If ptr1 is NULL, BMMunger replaces characters without searching. It replaces len1 characters starting at the offset location with the substitution string.
  • If ptr1 is NULL and len1 is negative, BMMunger replaces all of the characters from the offset location to the end of the string with the substitution string.
  • If len1 is 0, BMMunger inserts the substitution string without replacing anything. BMMunger inserts the string of the offset location and returns the offset of the first byte past where the insertion occurred.
  • If ptr2 is NULL, BMMunger searches but does not replace. In this case, BMMunger returns the offset at which the string was found.
  • If len2 is 0 and ptr2 is not NULL, BMMunger searches and deletes. In this case, BMMunger returns the offset at which deletion was performed.
  • If the portion of the string from the offset location to its end matches the beginning of the string that BMMunger is searching for, BMMunger replaces that portion with the substitution string.
