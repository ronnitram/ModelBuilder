About Timer Manager

The Timer Manager is used to manage timers.

The Timer Manager uses the GSTimer type (structure in C language, class in C++ language) to represent a timer. The GSTimer type is used to store the parameters and state of a timer. For specifying a timer use the GSTimer type in order to exploit the services provided by the Timer Manager.

Use the TPReset function to initialize a timer, to set its name and to reset its value to zero. You can start a timer at the initialization with TPResetAndStart, or later by calling the TPStart function. Once the timer is started, you can obtain its value by calling the TPGetValue function. The value of a timer is the elapsed time in seconds since its startup. Each timer has a name as a parameter. TPGetName function retrieves the name of a timer. If you don't know whether a timer has already started or not, you can check running by calling the TPRun function. You can stop a timer with the TPStop function.

The TPPrintValue function supports debugging; it sends a debug string containing information about the value of the specified timer.

If you use C++, you can use the GSTimer to operate with timers as objects. For more information see the GSTimer class.
