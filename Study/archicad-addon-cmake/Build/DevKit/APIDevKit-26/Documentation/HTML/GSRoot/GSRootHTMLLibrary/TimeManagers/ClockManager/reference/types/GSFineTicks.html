GSFineTicks

GSFineTicks represents a higher resolution tick counter.

typedef unsigned long GSFineTicks;

 

Remarks

Note that if you if you store the elapsed time as GSFineTicks, the time will wrap around zero.

Windows:    1 GSFineTick = 0.0001 second, wrap time = 4.97 days
Macintosh:  1 GSFineTick = 0.0000064 second, wrap time = 3.18 days
