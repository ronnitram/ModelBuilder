VAUnused

The VAUnused function marks a Virtual Array unused.

void VAUnused(
  DESC         *dptr,   // pointer to the virtual array
);

 

Parameters

dptr
Pointer to the virtual array to mark unused.

Remarks

Note that the VAUnused function does not release the buffer of the Virtual Array. It only fills the members of the DESC structure with zeros. You must call the VAFree function to unallocate the buffer of the Virtual Array before you call VAUnused.

Requirements

    Header: Declared in VA.hpp
    Import Library: Link with GSRootImp.lib
