AssociativeContainer<IndexType, ItemType>::Add

Adds a new item with its associated index. 

bool Add (
    const IndexType &   idx,
    const ItemType &    item
);

Parameters

idx
Index of the item. Item will be accessible with this index. Index should be unique in the associative container otherwise the operation will fail.
item
Item to be added.

Return Value

Value Meaning
true item was successfully added
false item was not added because another item with the same index already exists

Description

This method adds a new item with its associated index to the associative container. It will fail if another item with the same index already exists in the container.
On memory overflow OutOfMemoryException exception will be thrown.
