RSAddModule

Adds a resource module to the end of the module list.

    void RSAddModule (
      GSResModule  resModule
    );

Parameters

resModule
The module id of the resource module to be added.

Remarks

RSAddModule does not examine, weather the given module is already in the module list, so a module can be added multiple times.

Requirements

Version: 1.3 or later
Header: RS.hpp
Import Library: Link with GSRootImp.lib
Module: GSRootLib [MAC], GSRoot.dll [WIN]
