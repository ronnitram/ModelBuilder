About BN Manager

The BN Manager provides helper functions to manipulate allocated memory blocks. It provides functions to copy, move and compare memory blocks or to check the validity of pointers.

You can check whether you have read access to a memory block by calling the BNValidReadPtr function. For checking possession of write access to a memory block, call the BNValidWritePtr function. When you want to check having read access to a string pointer, you can use the BNValidStringPtr function.

With the BNFillMemory function you can fill a memory block with a bit pattern. If you want to fill a memory range with zeros, call the BNZeroMemory function. The BNCopyMemory function copies the content of a memory block into an other block. The BNMoveMemory function moves the content of a memory block into an other. You can compare the content of two memory blocks by calling the BNCompareMemory function.
