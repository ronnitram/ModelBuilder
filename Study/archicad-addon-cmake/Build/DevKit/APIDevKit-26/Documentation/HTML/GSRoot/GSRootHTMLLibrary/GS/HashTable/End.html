HashTable<IndexType, ItemType>::End

Returns an iterator referring to the one past last item in the hashtable. 

Iterator End (void);
ConstIterator End (void) const;

Return Value

A forward iterator or a constant forward iterator referring to the one past last item in the hashtable.

Description

This method returns a forward iterator or a constant forward iterator referring to the one past last item in the hashtable.
It can not be dereferenced, it can be only compared against other iterators (see the example).

Example

GS::HashTable<GS::String, long> hashTable;   // hashtable of {strings, long} pairs

GS::HashTable<GS::String, long>::Iterator begin = hashTable.Begin ();  // obtaining an iterator referring to the first item
GS::HashTable<GS::String, long>::Iterator end   = hashTable.End ();    // obtaining an iterator referring to the one past last item
while (begin != end) {  // until the end of the sequence (hashtable)
    long l = *begin;    // dereferencing the iterator (returns reference)
    ++begin;            // advance to the next item
}

This example illustrates the use of item iterators obtained from a hashtable.
