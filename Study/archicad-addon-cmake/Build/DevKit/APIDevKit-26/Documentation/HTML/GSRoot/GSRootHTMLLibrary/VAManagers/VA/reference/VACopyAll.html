VACopyAll

The VACopyAll function copies the content of a Virtual Array into an unused or uninitialized Virtual Array.

GSErr VACopyAll(
  DESC             *todesc,      // pointer to the virtual array to copy to
  DESC             *fromdesc     // pointer to the virtual array to copy from
);

 

Parameters

todesc
Pointer to the Virtual Array to copy to. It can not point to an initialized Virtual Array.
fromdesc
Pointer to the Virtual Array to copy from. It must point to a initialized Virtual Array.

Return Values

If the function succeeds, the return value is 0.

If the function fails, the return value is a Graphisoft error code. For more information about error codes, see GSRoot error codes.

If todesc is already initialized or fromdesc is not initialized yet then the return value is ErrNilHandle.

Remarks

The VACopyAll function implements a deep copy operation for Virtual Arrays. The function copies the content of the members of the DESCand allocates a new buffer for todesc Virtual Array. After allocating the function copies the content from the buffer of fromdesc Virtual Array into the buffer of the todesc Virtual Array. 

Requirements

   Import: Link with GSRootImp.lib
