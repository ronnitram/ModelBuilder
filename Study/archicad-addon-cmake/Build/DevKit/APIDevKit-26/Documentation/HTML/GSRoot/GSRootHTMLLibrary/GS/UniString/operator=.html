UniString::operator=

Assigment operator. 

UniString & operator= (const UniChar::Layout * uStr);
UniString & operator= (const char * cStr);
UniString & operator= (const unsigned char * pStr);
UniString & operator= (const UniString & source);
UniString & operator= (const Substring & source);
UniString & operator= (const ConstSubstring & source);
UniString & operator= (const Concatenation & source);

Parameters

uStr
0 terminated sequence of 16 bits unsigned short Unicode characters to be set.
cStr
0 terminated C string to be set.
pStr
Pascal string to be set.
source
Source string to be set.

Return Value

Reference to this string.

Description

UniString assigment operator. UniString can be set from various representations (another string, substring, Unicode string, C string, Pascal string etc). New content is converted to the Unicode representation using the default user locale.
On memory overflow OutOfMemoryException exception will be thrown.
