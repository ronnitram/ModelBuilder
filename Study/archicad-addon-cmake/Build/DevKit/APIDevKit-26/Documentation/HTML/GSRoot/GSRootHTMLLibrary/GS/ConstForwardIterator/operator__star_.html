ConstForwardIterator<Type>::operator *

Returns constant reference to the item iterator is currently referring to. 

const Type & operator * () const;

Return Value

Constant reference to the item iterator is currently referring to.

Description

This method returns constant reference to the item iterator is currently referring to. It is illegal to dereference an iterator referring to the end of the sequence.
