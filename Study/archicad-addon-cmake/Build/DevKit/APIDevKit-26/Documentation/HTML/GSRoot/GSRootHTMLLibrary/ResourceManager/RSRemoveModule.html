RSRemoveModule

Removes a resource module from the module list. 

    void RSRemoveModule (
      GSResModule  resModule
    );

Parameters

resModule
The module id of the module to be removed.

Remarks

A module can occur multiple times in the module list. In that case RSRemoveModule must be called multiple times.

 If the given resource module is the active resource module, than after removing it from the module list, the module proceeding this module becomes the current module.

Requirements

Version: 1.3 or later
Header: RS.hpp
Import Library: Link with GSRootImp.lib
Module: GSRootLib [MAC], GSRoot.dll [WIN]
