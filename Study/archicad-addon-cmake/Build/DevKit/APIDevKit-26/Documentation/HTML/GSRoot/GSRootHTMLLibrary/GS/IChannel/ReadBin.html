IChannel::ReadBin

Reads binary data from the input channel. 

virtual GSErrCode ReadBin (
    char *              destination,
    USize               nOfCharToRead,
    USize *             read = NULL
) = 0;

Parameters

destination
Destination buffer receiving the binary data.
nOfCharToRead
Number of characters to be read into the destination buffer.
read
On return: if not NULL then number of characters successfully read from the input channel.

Return Value

Value Meaning
NoError* OK
EndOfInput* end of input has been reached, operation is not completed
ReadError* read error on the input device, operation is not completed
Error* unspecified error

* declared in IChannel class scope
* declared in GS namespace, but accessible in the global scope

Description

This method reads binary data from the input channel. It reads specified number of characters into a destination buffer. These characters are consumed, that is removed from the input channel.
If there is no enough data left on the input channel then EndOfInput is returned. In such cases the output read parameter contains the number of characters that has been successfully read.
Input status will record and hold the first error occurred on the input channel. After an error this method does nothing and immediately returns with the stored error code (until the input status is reseted by the user).
