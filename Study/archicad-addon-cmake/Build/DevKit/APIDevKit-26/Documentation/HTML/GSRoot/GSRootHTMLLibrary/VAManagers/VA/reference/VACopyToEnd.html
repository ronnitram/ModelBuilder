VACopyToEnd

The VACopyToEnd function copies a specified range of a Virtual Array to the end of it. The range to copy is specified by a starting index (fromind) and the length (n) of the range.

GSErr VACopyToEnd(
  DESC              *dptr,                      // pointer to a virtual array 
  long              fromind,                    // the starting index of range to copy
  long             n                    // the length of range to copy
);

 

Parameters

dptr
Pointer to the Virtual Array of which specified part is copied.
fromind
The starting index of the range to copy.
n
The  length of the range to copy.

Return Values

If the function succeeds, the return value is 0.

If the function fails, the return value is a Graphisoft error code. For more information about error codes, see GSRoot error codes.

If dptr Virtual Array is uninitialized, the return value is ErrNilHandle.

Remarks

The VACopyToEnd function increases the size of the Virtual Array in order to the newly added elements fit in the buffer of the Virtual Array.

Requirements

    Header: Declared in VA.hpp
    Import Library: Link with GSRootImp.lib
