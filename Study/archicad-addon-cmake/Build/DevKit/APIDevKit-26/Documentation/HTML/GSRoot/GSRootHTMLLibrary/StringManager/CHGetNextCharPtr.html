CHGetNextCharPtr

Retrieves the pointer to the next character of a C string.

    char* CHGetNextCharPtr (
      const char*  pCString,
      GSCharCode   charCode = CC_Default
    );

Parameters

pCString
[in] Pointer to a character in a null-terminated string.
charCode
[in] Identifier of the character code use to recognize multi-byte characters.

Return Values

The return value is a pointer to the next character in the string, or to the terminating null character if at the end of the string.

If pCString points to the terminating null character, the return value is equal to pCString.

Remarks

The returned value can be calculated as pCString + CHCharBytes (*pCString).

Requirements
