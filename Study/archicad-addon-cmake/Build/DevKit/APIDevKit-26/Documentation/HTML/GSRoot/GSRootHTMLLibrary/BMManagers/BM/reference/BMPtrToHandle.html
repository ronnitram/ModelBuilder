BMPtrToHandle

The BMPtrToHandle function copies the data referred by a pointer to a newly allocated relocatable block.

GSErr BMPtrToHandle(
  const void      *pSource,              // pointer to source memory block
  GSHandle        *hDest,                // pointer to destination handle
  GSSize          nBytes         // number of bytes to copy
);

 

Parameters

pSource
Specifies the address of the first byte of source memory block to copy. It can not be NULL.
hDest
Pointer to the handle for which you have not yet allocated any memory. The BMPtrToHandle function allocates memory for the handle and copies nBytes bytes beginning at pSource into it. It can not be NULL.
nBytes
Specifies the number of bytes to copy.

Return Values

If the function succeeds, the return value is 0 (NoError).

If the function fails, the return value is a Graphisoft error code. To get extended error information, call BMError.

If the pSource parameter is NULL, the return value is ErrNilPtr.

If the hDest parameters is NULL, the return value is ErrNilHandle.

Remarks

Note If you pass a handle which refers to an allocated relocatable memory block to this function, then the BMPtrAndHandle function causes memory leaking, because it does not release the allocated block, but simply allocates a new relocatable memory block.

Requirements

    Header: Declared in BM.hpp
    Import Library: Link with GSRootImp.lib
