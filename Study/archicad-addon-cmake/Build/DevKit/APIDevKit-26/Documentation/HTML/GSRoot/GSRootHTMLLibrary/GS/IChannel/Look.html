IChannel::Look

Looks ahead on the input channel to the specified character. 

virtual GSErrCode Look (
    UIndex              idx,
    char *              c
) = 0;

Parameters

idx
Index of character on the input channel to be returned.
c
On return: the specified character.

Return Value

Value Meaning
NoError* OK
EndOfInput* end of input has been reached
ReadError* read error on the input device
Error* unspecified error

* declared in IChannel class scope
* declared in GS namespace, but accessible in the global scope

Description

This method looks ahead on the input channel to the character specified by its index.
If there is no enough data left on the input channel then EndOfInput is returned.
Input status will record and hold the first error occurred on the input channel. After an error this method does nothing and immediately returns with the stored error code (until the input status is reseted by the user).
