CHEqualCStrings

Compare two null terminated C strings with or without case sensitivity.

    bool CHEqualCStrings (
      const char*  p1,
      const char*  p2
    );

Parameters

p1
[in] Pointer to the first null terminated string to be compared.
p2
[in] Pointer to the second null terminated string to be compared.

Return Values

true if the strings are equal, false otherwise.

Remarks

Notice that if the return value is true, the two strings are "equal" identical.
