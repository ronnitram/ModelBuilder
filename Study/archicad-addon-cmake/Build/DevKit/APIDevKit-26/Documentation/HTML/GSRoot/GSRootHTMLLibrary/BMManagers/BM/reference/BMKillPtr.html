BMKillPtr

The BMKillPtr function releases a non-relocatable block of memory.

void BMKillPtr(
  GSPtr           *p     // address of the pointer
);

 

Parameters

p
Specifies the address of the pointer to the non-relocatable block of memory to release.

Remarks

After releasing the allocated memory, BMKillPtr sets the parameter *p to NULL.

Requirements

    Header: Declared in BM.hpp
    Import Library: Link with GSRootImp.lib
