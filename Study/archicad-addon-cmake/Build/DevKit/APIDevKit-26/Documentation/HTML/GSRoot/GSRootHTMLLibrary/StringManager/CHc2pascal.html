CHc2pascal

Converts a 0-terminated C-string to Pascal string.

    long CHc2pascal (
      const char*     pCString,
      unsigned char*  pPascalString
    );

Parameters

pCString
[in] The input, a null terminated C string.
pPascalString
[out] The output, a buffer for a pascal string.

Return Values

Returns the length of the string pPascalString, after the conversion. This is the length of pCString truncated to 255.

Remarks

If the length of pCString is more than 255 the function will succeed, and pPascalString will contain the first 255 characters.
