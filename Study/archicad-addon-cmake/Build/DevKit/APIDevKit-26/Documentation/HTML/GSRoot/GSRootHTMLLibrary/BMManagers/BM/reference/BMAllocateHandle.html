BMAllocateHandle

The BMAllocateHandle function allocates a relocatable block of memory of a specified size and returns a handle referring to the allocated block.

GSHandle BMAllocateHandle(
  GSSize          nBytes,                // number of bytes to allocate
  GSFlags         allocFlags,    // allocation attributes
  short           actionCode     // user specific code
);

 

Parameters

nBytes
Specifies the requested size (in bytes) of the relocatable block.
allocFlags
Specifies how to allocate memory. This parameter can be any combination of the following values.
Value Meaning
ALLOCATE_ACTION User action is enabled.
ALLOCATE_CLEAR Memory block is zeroed.
ALLOCATE_EMPTY Empty handle is allocated.
actionCode
User specific code for differentiating error locations or functions.

Return Values

If the function succeeds, the return value is a handle that refers to the allocated block of the requested size.

If the function fails, the return value is NULL.

Remarks

The new block is unlocked, unpurgeable and not a resource.

Requirements

    Header: Declared in BM.hpp
    Import Library: Link with GSRootImp.lib
