RSLoadResource

Copies the specified resource into memory.

    GSHandle RSLoadResource (
      GSResType    resType,
      GSResID      resID,
      GSResModule  resModule = AllResModule
    );

Parameters

[in] resType
A four byte resource type identifier.
[in] resID
A numeric resource identifier.
[in] resModule=AllResModule
A resource module to get the resource from. If this value is AllResModule then all resource modules opened with RSOpenResourceFile is searched.

Return Values

An unique GSHandle to the copy of the resource data. You should release this handle using BMKillHandle

The function returns NULL if the resource is not found

Remarks

If the resource is not found the debug version will break into the debugger. You can use RSMissingResourceAlert to modify this behaviour.

 Windows only version:

    GSHandle RSLoadResource (
      const char*  resType,
      GSResID      resID,
      GSResModule  resModule = AllResModule,
      bool         lengthPrefix = false
    );

Parameters

[in] resType
An arbitrary string resource type.
resID
A numeric resource identifier.
resModule=AllResModule
A resource module to get the resource from. If this value is AllResModule then all resource modules opened with RSOpenResourceFile is searched.
lengthPrefix=false
If this flag is false then it returns the raw resource. If this flag is true then the first 4-byte of the resource is parsed as the length of the resource. If this value is true the resulting GSHandle won't contain the first 4 byte, nor any extra values appended by the resource compiler. If this value is true you can query the resource size with BMGetHandleSize.

Return Values

An unique GSHandle to the copy of the resource data. You should release this handle using BMKillHandle

Remarks

If the resource is not found the debug version will break into the debugger. You can use RSMissingResourceAlert to modify this behaviour.

Requirements

Version: 1.3 or later
Header: RS.hpp
Import Library: Link with GSRootImp.lib
Module: GSRootLib [MAC], GSRoot.dll [WIN]
