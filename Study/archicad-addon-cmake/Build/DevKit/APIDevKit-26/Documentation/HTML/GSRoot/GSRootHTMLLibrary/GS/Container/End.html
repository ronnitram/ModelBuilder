Container<Type>::End

Returns an iterator referring to the one past last item in the container. 

ConstIterator End (void) const;

Return Value

A constant forward iterator referring to the one past last item in the container.

Description

This method returns a constant forward iterator referring to the one past last item in the container.
It can not be dereferenced, it can be only compared against other iterators (see the example).

Example

GS::Array<GS::String> array;                    // array of strings
GS::Container<GS::String>& container = array;	  // container of strings (refers to the array)

GS::Container<GS::String>::ConstIterator begin = container.Begin ();  // obtaining an iterator referring to the first item
GS::Container<GS::String>::ConstIterator end   = container.End ();    // obtaining an iterator referring to the one past last item
while (begin != end) {                   // until the end of the sequence (container)
    GS::String s = *begin;               // dereferencing the iterator (returns reference)
    USize length = begin->GetLength ();  // dereferencing the iterator (returns pointer)
    ++begin;                             // advance to the next item
}


GS::Find (container.Begin (), container.End (), GS::String ("Test"));  // using iterators in algorithms

This example illustrates the use of iterators obtained from a container.
