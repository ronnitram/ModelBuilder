BNCompareMemory

The BNCompareMemory function compares the bytes (characters) of two memory blocks.

GSSize BNCompareMemory(
  const void      *src1Ptr,              // address of first source block
  const void      *src2Ptr,              // address of second source block
  GSSize          len            // number of bytes to compare
);

 

Parameters

src1Ptr
Specifies the address of the first source block.
src2Ptr
Specifies the address of the second source block.
len
Specifies the number of bytes to compare.

Return Values

The return value indicates the relationship between the memory blocks.

Return Value Relationship of first len bytes of scr1Ptr and src2Ptr
< 0 src1Ptr less than src2Ptr
0 src1Ptr identical to src2Ptr
> 0 src1Ptr greater than src2Ptr

Remarks

If the application is compiled as a debugging version, and the process does not have read access to src1Ptr and src2Ptr (the BNValidReadPtr function returns false), the function causes an assertion fault and breaks into the debugger.

The BNCompareMemory function wraps the memcmp standard C Run-Time function.

Requirements

    Import Library: Link with GSRootImp.lib
