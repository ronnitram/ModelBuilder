Preferences Manager

Preference Manager allows you to store values that are associated with a key that can later be used to look up the preference value when you need it. You should store only noncritical information with Preferences Manager, as these preferences might be edited by the power users! The information is stored in the Registry on windows, and ~/Library/Preferences/*.plist on MacOS.
