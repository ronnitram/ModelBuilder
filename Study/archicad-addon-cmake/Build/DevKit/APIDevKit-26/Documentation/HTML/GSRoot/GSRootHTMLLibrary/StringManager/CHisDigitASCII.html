CHisDigitASCII

Returns whether the specified character is digit in the ASCII range [0-9].

    bool CHisDigitASCII (
      char            aChar
    );

Parameters

aChar
[in] The character to test.
charCode
[in] Character code.

Return Values

Returns true if the character represented by aChar is a digit in the ASCII range [0-9]
