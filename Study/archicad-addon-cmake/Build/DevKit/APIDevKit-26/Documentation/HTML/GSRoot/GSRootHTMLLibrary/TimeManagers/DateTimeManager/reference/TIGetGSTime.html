TIGetGSTime

The TIGetGSTime function converts the specified time record to GSTime format.

GSErr TIGetGSTime(
   const GSTimeRecord  *pTimeRecord,  // time structure to convert
   GSTime               *pGSTime,     // buffer receiving converted time
   GSFlags       formatFlags  // conversion attributes
);

 

Parameters

pTimeRecord
Specifies the time record to convert. When the formatFlags parameter is set to  TI_CURRENT_TIME, the pTimeRecord parameter is ignored.
pGSTime
Specifies the buffer receiving the result of the conversion.
formatFlags
Specifies how to convert. This parameter must be one of the following values:
Value Meaning
TI_LOCAL_TIME Local time
TI_UTC_TIME UTC time
TI_CURRENT_TIME Current time
If you specify the TI_CURRENT_TIME flag, then the pTimeRecord parameter is ignored.

Return Values

The return value indicates the success of the conversion with an error code. For more information about the error codes, see GSRoot error codes.

Remarks

If you want to convert in opposite direction, call TIGetTimeRecord.

If you specify the TI_CURRENT_TIME flag as formatFlags, the function calls TIGetTime function and stores its result inpGSTime parameter. In this case the value of pTimeRecord is ignored.

Requirements

    Header: Declared in GSTime.hpp
    Import Library: Link with GSRootImp.lib
