CHSearchCharRight

Search for the last occurance of a character in a string. (ie. right to left search) [strrchr]

    char* CHSearchCharRight (
      unsigned short  aChar,
      const char*     pChars,
      long            nChars = CStringLen,
      GSCharCode      charCode = CC_Default
    );

Parameters

[in] aChar
The character to search for.
[in] pChars
The string to search in.
[in] nChars
The length of the string. This value can be CStringLen that means that pChars is a null terminated C String.
[in] charCode
Character code.

Return Values

Pointer to the character if found, NULL otherwise.

Requirements

Version: GSRoot 1.3 or later
Header: CH.hpp
Import Library: GSRootImp.lib.lib
Module: GSRoot
