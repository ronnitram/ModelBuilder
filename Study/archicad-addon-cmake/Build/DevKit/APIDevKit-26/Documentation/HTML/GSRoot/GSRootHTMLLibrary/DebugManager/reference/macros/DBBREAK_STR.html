DBBREAK_STR

The DBBREAK_STR macro calls the DBBreak function with predefined parameters.

#define     DBBREAK_STR(s)  DBBreak (__FILE__, __LINE__, s)

 

Parameters

s
Specifies the message. This parameter is optional, i. e. NULL is allowed.
