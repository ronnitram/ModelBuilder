About Performance Monitor Manager

Windows NT only!

The Performance Monitor Manager makes adding own-designed performance counters more comfortable and more convenient. Counters are displayed by the system administrative tool Performance Monitor. The manager provides an easy-to-use interface, in order to hide the difficulty of connecting to Performance Monitor.

The simplest way to get connected to Performance Monitor: add your counters and performance objects (performance counters are grouped into objects), then put them to use. You can add an object to Performance Monitor Manager by calling the PMAddObject function. Once you added an object, you can add counters to the counter set of the object by calling the PMAddCounter function. After creating your object and counter structure you can put them to use with the PMUseCounters function.

After the above procedure you can modify the value of these counters within your program by calling the PMSetCounterValue function. If you want to get a value of a counter within the program, call the PMGetCounterValue function. The Performance Monitor continuously follows the changes made by PMSetCounterValue by displaying the current values of the counters. If the current counter set is no longer needed you can define a new set in run time. Before defining the new set of counters you have to destroy the old one by calling the PMAbuseCounters and the PMDeleteObjects functions.
