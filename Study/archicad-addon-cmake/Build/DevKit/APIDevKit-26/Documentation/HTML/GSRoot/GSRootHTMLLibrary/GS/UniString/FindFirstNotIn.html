UniString::FindFirstNotIn

Searches for the first occurence of any character not specified in a given set of characters. 

UIndex FindFirstNotIn (
    const UniString &   str,
    UIndex              from = 0,
    USize               range = MaxUSize
) const;
UIndex FindFirstNotIn (
    const char *        cStr,
    UIndex              from = 0,
    USize               range = MaxUSize
) const;

Parameters

str
Unicode string containing characters that are excluded from the searching.
from
Start position of the substring to be searched in. Must be beetwen 0 and length of the string.
range
Range of characters to be searched in. from + range must be beetwen 0 and length of the string. The default value MaxUSize indicates range that extends to end of the string.
cStr
C string containing characters that are excluded from the searching.

Return Value

Index of the first occurence of any character that is not specified in a given set of characters. If no occurence is found then the return value is MaxUIndex.

Description

This method searches for the first occurence of any character in the string (or in part of the string) that is not specified in a given set of characters. Character set should be specified as characters of another string.
