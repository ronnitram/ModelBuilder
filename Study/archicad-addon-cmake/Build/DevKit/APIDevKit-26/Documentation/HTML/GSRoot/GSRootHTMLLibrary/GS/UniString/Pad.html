UniString::Pad

Extends the string to reach a given length. 

void Pad (
    USize               toLength,
    UniChar             filler
);

Parameters

toLength
New length of the string to be reached.
filler
Filler character to be used for appending (if necessary).

Description

This method extends the string to reach a given length by appending filler characters. If the current length of the string is already greater than or equal to the given length then this method does nothing.
On memory overflow OutOfMemoryException exception will be thrown.
