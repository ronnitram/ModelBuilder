TIGS2MacTime

The TIGS2MacTime function converts the platform independent GSTime format to Macintosh format.

GSErr TIGS2MacTime(
  GSTime                gsTime,       // platform independent time
  unsigned long *pMacTime     // buffer receiving converted time
);

 

Parameters

gsTime
Specifies the platform independent time variable to convert.
pMacTime
Specifies the buffer receiving the converted Macintosh time.

Return Values

The return value indicates the success of the conversion with an error code. For more information about the error codes, see GSRoot error codes.

Remarks

If you want to convert to the opposite direction, call TIMac2GSTime.

Requirements

    Header: Declared in GSTime.hpp
    Import Library: Link with GSRootImp.lib
