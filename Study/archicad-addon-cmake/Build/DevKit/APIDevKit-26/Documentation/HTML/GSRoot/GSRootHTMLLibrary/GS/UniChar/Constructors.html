UniChar::UniChar

UniChar constructor. 

UniChar ();
UniChar (Layout u);
UniChar (char c);
UniChar (unsigned char c);
UniChar (const UniChar & source);

Parameters

u
16 bits unsigned short Unicode character.
c
8 bits signed / unsigned character.
source
Another UniChar (for the copy constructor).

Description

UniChar constructor. UniChar can be created from various types. Non-Unicode representations are converted to Unicode using the default user locale.
Default constructor sets UniChar to value 0.
