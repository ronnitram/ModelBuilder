VASearch

The VASearch function searches a specified element within a Virtual Array.

long VASearch(
  DESC         *dptr,              // pointer to the virtual array to search in
  void             *data,           // pointer to the element to search 
  CmpFunction      *cmpFunc             // pointer comparison function
);

 

Parameters

dptr
Pointer to the Virtual Array to search in. It can not be NULL.
data
Pointer to the element to find in the Virtual Array dptr.
cmpFunc
Pointer to a function that compares elements of the Virtual Array with the specified element data.

Return Values

If the function succeeds, the return value is the index of the specified element.

If the function fails, the return value is 0.

Remarks

The VASearch function uses the bsearch function from the standard C Run-Time Library to search the specified element. Hence bsearch implements the binary search algorithm, elements must be sorted before using the VASearch function.

Note The Virtual Array is indexed from 1 to the lastind member of the DESC structure.

Requirements

    Import Library: Link with GSRootImp.lib
