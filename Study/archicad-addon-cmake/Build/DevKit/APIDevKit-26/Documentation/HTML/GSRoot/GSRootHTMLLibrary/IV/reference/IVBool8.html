IVBool8

Swaps the byte in a GS::Bool8.

    void IVBool8 (
        GS::PlatformSign platform,
        GS::Bool8* pBool8
    );
    void IVBool8 (
        GS::PlatformSign from,
        GS::PlatformSign to,
        GS::Bool8* pBool8
    );

Parameters

platform, from
[in] The input platform.
to
[in] The output platform for the second function.
pBool8
[in, out] Points to a GS::Bool8.

Remarks

GS::Bool8 is a replacement type for the built-in bool. The GCC compiler define the bool type to be 4 bytes long, whereas the VC++ and the CodeWarrior compilers treat that as a 1-byte type. This made serialization a headache, and thus the new Bool8 type was introduced, which is guaranteed to be 1-byte long for all compilers.
Because of that these functions are no-ops; one byte cannot be swapped. The reason why we supply these functions is to make automatic generation of swapping functions or member functions easier.
