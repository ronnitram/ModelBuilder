RSOpenResourceFile

Opens a resource file. This function has two variations: one for Macintosh and one for Windows.

 Windows only version:
    GSResModule RSOpenResourceFile (
      const char*  pathName
    );

Parameters

[in] pathName

Pointer to a null terminated string that names the resource module. (.dll file)

If the string specifies a path but the file does not exist in the specified directory, the function fails. When specifying a path, be sure to use backslashes (\), not forward slashes (/).

Return Values

If the function succeeds the return value is an identifier for the resource module. You can use the returned value as the resModule parameter of RSLoadResource.

If the function fails, the return value is NULL.

You can use RSCloseResourceFile to close the resource module.

Remarks

This function appends the resource module to the list of resource modules searched by RSSearchResource, and RSLoadResource (If AllResModule is specified as the resModule parameter of RSLoadResource).

 

 Macintosh only version:
    GSResModule RSOpenResourceFile (
      const FSSpec*  fileSpec
    );

Parameters

fileSpec

A pointer to the file specification record specifying the name and location of the file whose resource fork is to be opened. The file system specification record for files is defined by the FSSpec data type and is a standard format in Carbon for identifying a file or directory.


Return Values

If the function succeeds the return value is an identifier for the resource module. You can use the returned value as the resModule parameter of RSLoadResource.

If the function fails, the return value is NULL.

You can use RSCloseResourceFile to close the resource fork of the file.

Remarks

This function appends the resource module to the list of resource modules searched by RSSearchResource, and RSLoadResource (If AllResModule is specified as the resModule parameter of RSLoadResource).

Do not use the returned value, as a MacOS reference number to the resource fork. Don't pass the returned id to Carbon routines!

 

Requirements

Version: 1.3 or later
Header: RS.hpp
Import Library: Link with GSRootImp.lib
Module: GSRootLib [MAC], GSRoot.dll [WIN]
