About Debug Manager

The Debug Manager provides useful debug services. In order to display debug messages a shadowing application is needed. Such a shadowing application is the Debug Monitor.

The DBBreak function sends debug messages to the Debugger and throws debug exceptions. Throwing debug exceptions with DBBreak function breaks the application, and gives the control  to the Debugger. The DBTrace function is similar to DBBreak, it also sends debug messages to the Debugger. DBTrace displays tracing information. The DBPrintf function sends formatted messages to the Debugger of the current application.

Functions DBSetFlags and DBGetFlags  provide a opportunity to configure the Debug Manager.
