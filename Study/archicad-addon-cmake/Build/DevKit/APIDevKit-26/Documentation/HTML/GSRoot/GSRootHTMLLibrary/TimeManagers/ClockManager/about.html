About Clock Manager

The Clock Manager is used to manage clocks in a platform independent manner. The Clock Manager defines two tick counters: a lower and a higher resolution tick counter. The manager provides functions to get the current time - which is actually the value of the a specified counter. These tick counters count the number of ticks elapsed since the system startup.

The clock manager represents the value of the lower resolution counter with the GSTicks type and the value of higher resolution counter with the GSFineTicks type. You can obtain the time elapsed since system startup by calling the TIGetTicks or the TIGetFineTicks function. The TIGetTicks function retrieves the value of the lower resolution counter while the TIGetFineTicks retrieves the value of the higher resolution tick counter. You can get the number of ticks elapsed in a second by calling the TIGetTicksPerSec function, and you can get the fine ticks per second value by calling the TIGetFineTicksPerSec function.

The TIGetDoubleClickTicks function retrieves the number of ticks elapsed between the first and the second click of a double-click. You can use the TIWait function to wait until the specified time has elapsed.
