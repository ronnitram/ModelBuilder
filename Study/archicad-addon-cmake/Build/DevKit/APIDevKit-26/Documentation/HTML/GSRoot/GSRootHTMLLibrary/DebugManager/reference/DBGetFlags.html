DBGetFlags

The DBGetFlags function retrieves the previously specified debug flags set in the Debug Manager.

GSFlags DBGetFlags(
  unsigned long flagCategory       // flag category
);

 

Parameters

flagCategory
Specifies what kind of flags are required. The flag category must be one of the following values:
Value Meaning
DBGeneral General purpose debug flags.

Return Values

If the function succeeds, the return value is the desired flag set.

If the function fails, the return value is 0.

If the flagCategoryparameter specifies an invalid flag category, the return value is 0.
