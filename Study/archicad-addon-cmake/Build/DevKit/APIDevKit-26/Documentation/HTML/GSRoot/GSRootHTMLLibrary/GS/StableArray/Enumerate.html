StableArray<Type>::Enumerate

Allows enumeration of the array's content. 

Iterator Enumerate (void);
ConstIterator Enumerate (void) const;

Return Value

A random iterator or a constant random iterator referring to the first item in the array.

Description

This method allows random enumeration of the array's content. It returns a random iterator or a constant random iterator referring to the first item in the array. Enumeration ordering is index based.
References to items obtained by iterator dereferencing can be used only until the Compact method on the parent array is invoked.
This method is equivalent with the Begin method.

Example

GS::StableArray<GS::String> array;  // array of strings

GS::StableArray<GS::String>::Iterator it = array.Enumerate ();  // obtaining an iterator (or array.Begin ())
while (it != NULL) {                  // until the end of the sequence (array)
    GS::String s = *it;               // dereferencing the iterator (returns reference)
    *it = s;                          // dereferencing the iterator (returns reference)
    USize length = it->GetLength ();  // dereferencing the iterator (returns pointer)
    ++it;                             // advance to the next item
}

This example illustrates the use of iterators obtained from an array.
