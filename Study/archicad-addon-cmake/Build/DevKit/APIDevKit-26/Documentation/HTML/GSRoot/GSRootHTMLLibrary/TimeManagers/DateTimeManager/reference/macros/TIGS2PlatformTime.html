TIGS2PlatfromTime

The TIGS2PlatformTime macro calls the TIGS2MacTime function over Macintosh and the TIGS2WinTime function over Windows.

#if defined(macintosh)
  #define TIGS2PlatfromTime TIGS2MacTime
#else
  #define TIGS2PlatfromTime TIGS2WinTime
#endif

Requirements

    Header: Declared in GSTime.hpp
