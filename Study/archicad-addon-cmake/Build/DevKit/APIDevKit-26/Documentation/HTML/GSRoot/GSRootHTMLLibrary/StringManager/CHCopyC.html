CHCopyC

Copy a null terminated string.

    long CHCopyC (
      const char*  p1CString,
      char*        p2CString
    );

Parameters

p1CString
[in] Null-terminated source string.
p2CString
[out] Destination buffer. Should be lare enough to hold p1CString.

Return Values

The length of the source string (not including the null terminator).

Remarks

The function copies p1CString, including the terminating null character, to the location specified by p2CString.

The order of parameters is reverse of the <stdio.h> functions!

No overflow checking is performed when strings are copied. The behavior is undefined if the source and destination strings overlap.

Requirements
