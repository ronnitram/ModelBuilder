VAClearIndex

The VAClearIndex function clears a cell specified by the index of a Virtual Array.

GSErr VAClearIndex(
  DESC         *dptr,         // pointer to a virtual array
  long              index            // the index of the cell to clear
);

 

Parameters

dptr
Pointer to a Virtual Array of which specified cell is cleared.
index
Specifies the index of the cell to clear. It can not be less than 1 or more than the number of elements in the Virtual Array (lastind member of the DESC structure).

Return Values

If the function succeeds, the return value is 0.

If the function fails, the return value is a Graphisoft error code. For more information about error codes, see GSRoot error codes.

If index less than 1 or more than the number of elements in the Virtual Array (lastind member of the DESC structure), the return value is ErrParam.

Remarks

The function deletes the specified cell, and shifts down the the remainder of the array, that is all indices higher than the cleared one, will decrease by one.

Requirements

    Header: Declared in VA.hpp
    Import Library: Link with GSRootImp.lib
