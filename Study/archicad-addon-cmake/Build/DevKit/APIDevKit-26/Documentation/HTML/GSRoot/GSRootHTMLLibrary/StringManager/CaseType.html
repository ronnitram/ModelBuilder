CaseType

Possible values of case sensitivity for string comparisons.

    typedef enum CaseType {
        CS_CaseInsensitive,
        CS_CaseSensitive
    } CaseType;
Enumerator Value Meaning
CS_CaseInsensitive Comparison is case insensitive.
CS_CaseSensitive Comparison is case sensitive.
