UniString::ReplaceLast

Replaces last occurence of a character or string with another one. 

UIndex ReplaceLast (
    UniChar             oldChar,
    UniChar             newChar
);
UIndex ReplaceLast (
    const UniString &   oldStr,
    const UniString &   newStr
);

Parameters

oldChar
Old character to be replaced.
newChar
New character to be replaced with.
oldStr
Old string to be replaced.
newStr
New string to be replaced with.

Return Value

Index of the character or (sub)string that is replaced. If no occurence is found then the return value is MaxUIndex.

Description

This method replaces last occurence of a character or string with another one in this string.
On memory overflow OutOfMemoryException exception will be thrown.
