UniString::ReplaceFirst

Replaces first occurence of a character or string with another one. 

UIndex ReplaceFirst (
    UniChar             oldChar,
    UniChar             newChar,
    UIndex              from = 0,
    USize               range = MaxUSize
);
UIndex ReplaceFirst (
    const UniString &   oldStr,
    const UniString &   newStr,
    UIndex              from = 0,
    USize               range = MaxUSize
);

Parameters

oldChar
Old character to be replaced.
newChar
New character to be replaced with.
from
Start position of the substring to be replaced in. Must be beetwen 0 and length of the string.
range
Range of characters to be replaced in. from + range must be beetwen 0 and length of the string. The default value MaxUSize indicates range that extends to end of the string.
oldStr
Old string to be replaced.
newStr
New string to be replaced with.

Return Value

Index of the character or (sub)string that is replaced. If no occurence is found then the return value is MaxUIndex.

Description

This method replaces first occurence of a character or string with another one in this string (or in part of this string).
On memory overflow OutOfMemoryException exception will be thrown.
