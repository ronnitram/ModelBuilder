TIGetTicksPerSec

The TIGetTicksPerSec function retrieves the number of ticks within a second.

GSTicks TIGetTicksPerSec();

 

Return Values

The return value is the number of ticks within a second.

Remarks

If you want to know more about tick resolution and storage limits, see Clock Manager Simple Types.

Requirements

    Header: Declared in GSTime.hpp
    Import Library: Link with GSRootImp.lib
