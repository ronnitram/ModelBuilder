UniString::Assign

Assigment with multiply parameters. 

UniString & Assign (
    const UniChar::Layout * uChars,
    USize               charCount
);
UniString & Assign (
    const char *        chars,
    USize               charCount
);
UniString & Assign (
    const UniString &   source,
    UIndex              sourceFrom = 0,
    USize               sourceRange = MaxUSize
);

Parameters

uChars
Sequence of 16 bits unsigned short Unicode characters to be set.
charCount
Number of characters to be set.
chars
Sequence of 8 bits signed characters (char) to be set.
source
Source string to be set. Substring of this string can be also specified.
sourceFrom
Start position of the substring in the source string to be set. Must be beetwen 0 and length of the source string.
sourceRange
Range of characters in the source string to be set. sourceFrom + sourceRange must be beetwen 0 and length of the source string. The default value MaxUSize indicates range that extends to end of the source string.

Return Value

Reference to this string.

Description

UniString assigment method with multiply parameters. New content is converted to the Unicode representation using the default user locale.
On memory overflow OutOfMemoryException exception will be thrown.
