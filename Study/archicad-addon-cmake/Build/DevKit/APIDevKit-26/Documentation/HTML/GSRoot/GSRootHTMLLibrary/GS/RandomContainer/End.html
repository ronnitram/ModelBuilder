RandomContainer<Type>::End

Returns an iterator referring to the one past last item in the random container. 

Iterator End (void);
ConstIterator End (void) const;

Return Value

A random iterator or a constant random iterator referring to the one past last item in the random container.

Description

This method returns a random iterator or a constant random iterator referring to the one past last item in the random container.
It can not be dereferenced, it can be only compared against other iterators (see the example).

Example

GS::Array<GS::String> array;                               // array of strings
GS::RandomContainer<GS::String>& randomContainer = array;  // random container of strings (refers to the array)

GS::RandomContainer<GS::String>::Iterator begin = randomContainer.Begin ();  // obtaining an iterator referring to the first item
GS::RandomContainer<GS::String>::Iterator end   = randomContainer.End ();    // obtaining an iterator referring to the one past last item
while (begin != end) {                   // until the end of the sequence (random container)
    GS::String s = *begin;               // dereferencing the iterator (returns reference)
    *begin = s;                          // dereferencing the iterator (returns reference)
    USize length = begin->GetLength ();  // dereferencing the iterator (returns pointer)
    ++begin;                             // advance to the next item
}


GS::Sort (randomContainer.Begin (), randomContainer.End ());  // using iterators in algorithms

This example illustrates the use of iterators obtained from a random container.
