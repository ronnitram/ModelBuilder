UniString::ToCStr

Returns C string form of the string. 

CStr ToCStr (
    UIndex              from = 0,
    USize               range = MaxUSize
) const;

Parameters

from
Start position of the substring to be returned. Must be beetwen 0 and length of the string.
range
Range of characters to be returned. from + range must be beetwen 0 and length of the string. The default value MaxUSize indicates range that extends to end of the string.

Return Value

A private object which can be implicitly converted to const char*. This pointer remains valid only inside its enclosing expression (statement) so the pointer should not be stored (but can be passed in the same expression as parameter to a funtion or method).

Description

This method returns C string form of the string (or part of the string), that is sequence of 8 bits signed characters terminated with a 0 character. It returns a private object which can be implicitly converted to const char*. This pointer remains valid only inside its enclosing expression (statement) so the pointer should not be stored (but can be passed in the same expression as parameter to a funtion or method). Be aware that if the type of the parameter of the function or method is not const char *, you have to explicitly cast it to const char * especially in case of printf family of functions: 

sprintf (errStr, "%s_%ld", static_cast <const char *>(uniString.ToCStr ()), errCode);
On memory overflow OutOfMemoryException exception will be thrown.
