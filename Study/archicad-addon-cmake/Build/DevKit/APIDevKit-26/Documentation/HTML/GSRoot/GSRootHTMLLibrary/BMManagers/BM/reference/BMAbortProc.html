BMAbortProc

The BMAbortProc is a user defined abort procedure called by the Basic Memory Manager. The BMAbortProc is a function type.

short BMAbortProc(
  short           c,     // code of the BM function
  long            n,     // number of bytes requested
  short           e,     // operating system error code returned
  short           action      // action code
);

 

Parameters

c
Specifies the Basic Memory Manager function where the error occurs. The possible values are:
Value Meaning
1 BMAllocatePtr
2 BMAllocateHandle
4 BMReallocHandle
5 BMKillPtr
6 BMKillHandle
10 BMModifyHandleState
17 BMGetHandleSize
18 BMGetPtrSize
19 BMReallocPtr
n
Specifies the number of bytes requested by the caller Basic Memory Manager function.
e
Specifies the operating system error code of the failed memory action.
action
Specifies the action code that is passed to caller BM Manager (BM) function. If an error occurs in a BM function that has no action parameter BM passes 0 on this parameter.

Return Values

The returned value determines whether the Basic Memory Manager exits or continues the execution of the function where the error occurred. The return value must be true to allow Basic Memory Manager to exit, or false to continue.

Remarks

You can set the BMAbortProc for the Basic Memory Manager in theBMInit function.

Requirements

   Header: Declared in BM.hpp
   Import Library: Link with GSRootImp.lib
