BMPtrAndHandle

The BMPtrAndHandle function concatenates part of a memory block (or the entire block) to the end of a relocatable memory block.

GSErr BMPtrAndHandle(
  const void      *pSource,              // source pointer
  GSHandle        hDest,         // destination handle
  GSSize          nBytes         // number of bytes to concatenate
);

 

Parameters

pSource
Pointer to the beginning of the data that the BM Manager is to concatenate to the end of the relocatable block. It can not be NULL.
hDest
Specifies the handle of the relocatable block, of which size the BM Manager expands in order to concatenate the parameter pSource to the end of this block. It can not be NULL.
nBytes
Specifies the number of bytes of the block referred by pSource to copy.

Return Values

If the function succeeds, the return value is 0 (NoError).

If the function fails, the return value is a Graphisoft error code. To get extended error information, call BMError.

If the pSource parameter is NULL, the return value is ErrNilPtr.

If the hDest parameter is NULL, the return value is ErrNilHandle.

Requirements

    Header: Declared in BM.hpp
    Import Library: Link with GSRootImp.lib
