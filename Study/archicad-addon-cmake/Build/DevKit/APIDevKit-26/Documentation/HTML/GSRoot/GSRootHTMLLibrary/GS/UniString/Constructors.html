Parameters

uStr
0 terminated sequence of 16 bits unsigned short Unicode characters.
uChars
Sequence of 16 bits unsigned short Unicode characters.
charCount
Number of characters.
cStr
0 terminated C string.
chars
Sequence of 8 bits signed characters (char).
pStr
Pascal string.
source
Source string for the copy constructor. Substring of this string can be also specified.
sourceFrom
Start position of the substring in the source string. Must be beetwen 0 and length of the source string.
sourceRange
Range of characters in the source string. sourceFrom + sourceRange must be beetwen 0 and length of the source string. The default value MaxUSize indicates range that extends to end of the source string.

Description

UniString constructor. UniString can be created from various representations (another string, substring, Unicode string, C string, Pascal string etc). Content is converted to the Unicode representation using the default user locale.
On memory overflow OutOfMemoryException exception will be thrown.
