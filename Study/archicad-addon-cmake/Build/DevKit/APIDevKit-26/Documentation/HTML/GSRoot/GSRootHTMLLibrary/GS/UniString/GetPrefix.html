UniString::GetPrefix

Returns a prefix substring of the string. 

Substring GetPrefix (USize range);
ConstSubstring GetPrefix (USize range) const;

Parameters

range
Length of the prefix substring, that is number of leading characters of the string. range must be beetwen 0 and length of the string.

Return Value

The specified prefix substring. It can be empty.

Description

This method returns a prefix substring of the string. Prefix substring can be specified with its length, that is number of leading characters of the string.
The substring refers to the specified part of the source string, so itself never allocates memory. Substring is a private type so it can be used only inside of the enclosing expression (statement) of the call.

Substrings can be effectively compared with various types (string, substring, C string etc) and can participate in various assigments and concatenations. These operations are always performed on the corresponding parts of the source strings.
