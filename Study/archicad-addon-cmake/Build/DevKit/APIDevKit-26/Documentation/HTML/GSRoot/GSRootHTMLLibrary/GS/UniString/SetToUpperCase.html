UniString::SetToUpperCase

Converts string to uppercase. 

void SetToUpperCase (
    UIndex              from = 0,
    USize               range = MaxUSize
);

Parameters

from
Start position of the substring to be converted. Must be beetwen 0 and length of the string.
range
Range of characters to be converted. from + range must be beetwen 0 and length of the string. The default value MaxUSize indicates range that extends to end of the string.

Description

This method converts string (or part of the string) to uppercase using the default user locale. Note that this operation may change length of the string!
On memory overflow OutOfMemoryException exception will be thrown.
