BidirectionalIterator<Type>::operator++

Moves forward to the next item of the sequence. 

BidirectionalIterator & operator++ ();
BidirectionalIterator operator++ (int param1);

Parameters

param1
Not used (don't pass anything).

Return Value

1. version: reference to this iterator.

2. version: old state of the iterator (before the increment operation is performed).

Description

This method moves the iterator forward to the next item of the sequence. It is illegal to move beyond the end of the sequence (beyond the one past last item).
