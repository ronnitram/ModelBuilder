GSCharCode

The character code representing a code page on Windows, or a script system on Macintosh.

    typedef enum GSCharCode {
        CC_Default=0,
        CC_System=1,
 CC_User=2,
  CC_Application=3,
        CC_WestEuropean=11,
        CC_EastEuropean=12,
        CC_Baltic=13,
        CC_Cyrillic=14,
        CC_Greek=15,
        CC_Turkish=16,
        CC_Hebrew=21,
        CC_Arabic=22,
        CC_Thai=31,
        CC_Japanese=32,
        CC_TradChinese=33,
        CC_SimpChinese=34,
        CC_Korean=35,
      CC_UTF8=99,
        CC_UniCode=100
    } GSCharCode;
Enumerator Value Meaning
CC_Default=0 The default character code. You can set/query it with CHSetDefaultCharCode, and CHGetDefaultCharCode.
CC_System=1 The code page/script system currently used by the system. See CHGetSystemCharCode.
CC_User=2 The defult character code used by the user. See CHGetUserCharCode.
CC_Application=3
 The default character code used by the application. 
CC_WestEuropean=11 CodePage 1252 on Windows, smRoman or kCFStringEncodingMacRoman on MacOS.
CC_EastEuropean=12 CodePage 1250 on Windows, smCentralEuroRoman or kCFStringEncodingMacCentralEurRoman on MacOS.
CC_Baltic=13 CodePage 1257 on Windows, smCentralEuroRoman or kCFStringEncodingMacCentralEurRoman on MacOS.
CC_Cyrillic=14 CodePage 1251 on Windows, smCyrillic or kCFStringEncodingMacCyrillic on MacOS.
CC_Greek=15 CodePage 1253 on Windows, smRoman or kCFStringEncodingMacGreekon MacOS.
CC_Turkish=16 CodePage 1254 on Windows, smRoman or kCFStringEncodingMacTurkish on MacOS.
CC_Hebrew=21 CodePage 1255 on Windows, smHebrew or kCFStringEncodingMacHebrew on MacOS.
CC_Arabic=22 CodePage 1256 on Windows, smArabic or kCFStringEncodingMacArabic on MacOS.
CC_Thai=31 CodePage 874 on Windows, smThai or kCFStringEncodingMacThai on MacOS.
CC_Japanese=32 CodePage 932 on Windows, smJapaneese or kCFStringEncodingMacJapaneese on MacOS.
CC_TradChinese=33 CodePage 950 on Windows, smTradChinese or kCFStringEncodingTradChinese on MacOS.
CC_SimpChinese=34 CodePage 936 on Windows, smSimpChinese or kCFStringEncodingSimpChinese on MacOS.
CC_Korean=35 CodePage 949 on Windows, smKorean or kCFStringEncodingMacKorean on MacOS.
CC_UTF8=99 CodePage 65001 on WindowsNT, kCFStringEncodingUTF8 on MacOS. Note that this character code is not supported by some CH routines.
CC_UniCode=100 CodePage 1200 on Windows, kCFStringEncodingUnicode on MacOS. Note that this character code is not supported by some CH routines.
