TPGetValue

The TPGetValue function retrieves the value of a timer. The value of a timer is the elapsed time in seconds since the timer first received a start message.

double TPGetValue(
  GSTimer              *pTimer                // pointer to timer
);

 

Parameters

pTimer
Pointer to the timer of which value is to retrieve.

Return Values

The return value is the elapsed time in seconds since the specified timer first received a start message.

Requirements

    Header: Declared in GSTime.hpp
    Import Library: Link with GSRootImp.lib
