IVEndofLine

Converts the line end character to the given end-of line type.

    long IVEndofLine (
      const char*  pChars,
      long         nChars,
      char*        pNewChars,
      long         maxNewChars,
      EOLType      targetEOL = Platform_EOLType
    );

Parameters

pChars
[in] Input string.
nChars

[in] Specifies the size in bytes of the string pointed to by pChars parameter, or it can be CStringLen if the string is null terminated.

If this string is null terminated, the resulting character string has a null terminator, and the returned length includes the null terminator.

If this parameter is a positive integer, the function processes exactly the specified number of bytes. If the given length does not include a null terminator then the resulting wide character string will not be null terminated, and the returned length does not include a null terminator.

pNewChars
[out] The buffer for the output string.
maxNewChars

[in] Specifies the size, in bytes, of the buffer pointed to by the pNewChars parameter. If this value is zero, the function returns the required buffer size, in bytes, and makes no use of the pNewChars buffer.

targetEOL=Platform_EOLType
The target line end format.

Return Values

If the function succeeds, and maxNewChars is nonzero, the return value is the number of bytes written to the buffer pointed to by pNewChars.

If the function succeeds, and maxNewChars is zero, the return value is the required size, in bytes, for a buffer that can receive the translated string.

If the function fails, the return value is zero. This function fails if the parameters are incorrect or the provided buffer is too small to perform the conversion.

Remarks

The pChars and pNewChars pointers must not be the same, or overlap. If they do the result of this function is undefined!
