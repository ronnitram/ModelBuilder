VATInit

The VAtInit function initializes a VAT (Virtual Array of records with Type field).

GSErr VAtInit(
  VATDESC          *dptr,       // pointer to VAT
  long             npage,       // the number of elements by which
                        // the VAT is increased or decreased 
  long             n     // the size of elements
);

 

Parameters

dptr
Pointer to the VAT to initialize.
npage
Number of elements by which the VAT is increased or decreased. It must be greater than zero.
n
Size of elements stored in the dptr VAT. It must be greater than zero.

Return Values

If the function succeeds, the return value is 0 (NoError).

If the function fails, the return value is a Graphisoft error code. For more information about error codes, see GSRoot error codes.

Remarks

The VATInit function allocates a buffer for the VAT. The allocated buffer can supply the storage of exactly npage elements. The VATInit also initializes the members of the dptr VATDESC structure. The initial value of maxind is npage and the initial value of lastind is 0. The VATInit sets the freeind value to 0.

Requirements

    Import Library: Link with GSRootImp.lib
