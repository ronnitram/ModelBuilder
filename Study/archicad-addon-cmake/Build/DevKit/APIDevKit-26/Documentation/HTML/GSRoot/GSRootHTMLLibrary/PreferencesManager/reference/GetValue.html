GSPref::GetLongValue, GSPref::GetBoolValue, GSPref::GetStringValue, GSPref::GetANSIStringValue

Reads a typed data from the registry onWindows, preference file on MacOS .

    bool GetLongValue (
      RootKeyRef  root,
      const char*   path,
      const char*    valueName,
      long*    oValue,
      const long       iDefValue,
      WriteBackType    iWriteBack
    );

    bool GetBoolValue (
      RootKeyRef  root,
      const char*   path,
      const char*    valueName,
      bool*    oValue,
      const bool       iDefValue,
      WriteBackType    iWriteBack
    );

    bool GetStringValue (
      RootKeyRef  root,
      const char*   path,
      const char*    valueName,
      wchar_t*    oValue,
      unsigned int     iBufferByteSize,
      const wchar_t*   iDefValue,
      WriteBackType    iWriteBack
    );

    bool GetANSIStringValue (
      RootKeyRef  root,
      const char*   path,
      const char*    valueName,
      wchar_t*    oValue,
      unsigned int     iBufferByteSize,
      const wchar_t*   iDefValue, 
      const GSCharCode    iCharCode,
      WriteBackType    iWriteBack     
    );

Parameters

root
[in]Pointer to the root key, created with CreateRootKey. See RootKeyRef.
path
[in] Pointer to a string containing a '/' separated path. The value is read from to the sub-key specified by this path relative to the root. Use "" if you want to read direcly under the root key.

The path name cannot begin or end with a '/'. The size of the path cannot exceed 1024 characters, and cannot be NULL. It can contain only ASCII characters.

valueName
[in] Pointer to a string containing the name of the value to set. If a value with this name is not already present, the function creates it.

The valueName cannot contain '/' characters, and it should contain only ASCII characters. The size of the name cannot exceed 255 characters.

oValue
[out] The requested data is stored int this address.

GetLongValue The address of a 32-bit signed integer to be stored.

GetBoolValue The address of a bool value.

GetStringValue The address of a buffer for holding null-terminated unicode characters. The size (in bytes) of the buffer specified in iBufferByteSize.

GetANSIStringValue The address of a buffer for holding null-terminated multi-byte characters. The string is converted from unicode based on the iCharCode param, and the converted value is stored. The size (in bytes) of the buffer specified in iBufferByteSize. If a character cannot be represented in the given code page, this character is replaced with a System specific replace character. See CHUnicode2ANSI for details.

iBufferByteSize
GetANSIStringValue, GetStringValue [in] The length (in bytes) of the buffer specified by the oValue.

If this value is too small to hold the default value, the result is undefined.

iDefValue
[in] The default value. If the function cannot find the requested value, this data is copyed to the oValue.

GetANSIStringValue, GetStringValue This value cannot be NULL.

iCharCode
GetANSIStringValue [in] The GSCharCode of the parameter oValue. The string is converted from unicode based on the parameter, and the converted value is stored in oValue.
iWriteBack
If this value is WriteBack_True and the function didn't find the requested value, the value defined in iDefValue parameter is written to the registry/preference.

Return Values

True if the function founds the requested value, false otherwise. Even if the function returns false the oValue will contain the iDefValue. If the function fails but WriteBack_True was defined, it tries to write back the defaultValue to the preferences/registry data. If the sub-key defined by path is not already present, and WriteBack_True is defined it is created.

This function may fail in the following cases:
  • The specified registry/prefence value/key was not found.
  • The specified key was found but it contained wrong type of data. Or it was a name of a sub-key.
  • Not enough resources.
  • Not enough rights to read the registry/preference file. (Unlikely)
  • Not enough rights to write the registry. IfwriteBack is GSPref::WriteBack_True.
  • In case ofGetANSIStringValue the UnicodeToANSI string conversion failed.
