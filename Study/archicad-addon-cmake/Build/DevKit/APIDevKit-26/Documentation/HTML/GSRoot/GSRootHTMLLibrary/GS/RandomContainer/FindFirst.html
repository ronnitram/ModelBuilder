RandomContainer<Type>::FindFirst

Searches for the first occurence of a specified item. 

UIndex FindFirst (
    const Type &        item,
    UIndex              from = 0,
    USize               range = MaxUSize
) const;

Parameters

item
Specifies item to be found.
from
Start position of the searching. Must be beetwen 0 and size of the random container.
range
Range of items where searching should be performed. from + range must be beetwen 0 and size of the random container. The default value MaxUSize indicates range that extends to end of the random container.

Return Value

Index of the first occurence of the specified item or MaxUIndex if no matching item is found.

Description

This method searches for the first occurence of a specified item in the random container.
It requires items to have comparison operators (== and !=).
