IVCString

Converts an international string between MacOS and Windows.

    void IVCString (
      char*         pCString,
      GSCharCode    charCode = CC_Default,
      PlatformSign  targetPlatform = Act_Platform_Sign
    );

Parameters

pCString
[in-out] Points to a null terminated c-string represented in the charCode character code of the source platform.
charCode=CC_Default
The character code of the pChars string.
targetPlatform=Act_Platform_Sign
The target, and the source platform specifier.
Value Meaning
Win_Platform_Sign Use this flag to perform a MacOS to Windows conversion.
Mac_Platform_Sign Use this flag to perform a Windows to MacOS conversion.
