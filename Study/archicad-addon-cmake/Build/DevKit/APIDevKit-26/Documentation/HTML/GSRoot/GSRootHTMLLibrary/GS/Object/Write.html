Object::Write

Writes state of the object to the specified output channel. 

virtual GSErrCode Write (OChannel & param1) const;

Parameters

param1

Return Value

Value Meaning
NoError* OK
OutputIsFull* output device is full
WriteError* write error on the output device
Error* unspecified error

* declared in OChannel class scope
* declared in GS namespace, but accessible in the global scope

Description

If is implemented then this method should write state of the object to the specified output channel using its typed level (recommended) or binary level write methods. Default implementation does nothing and returns Error.
