VATDESC

The VATDESC structure defines a VAT.

typedef struct VATDESC{
  long             freeind;
  long             lastind;
  long             maxind;
  long             npage;
  long             size;
  GSHandle arrhdl;
} VATDESC;

 

Members

freeind
Specifies the first free index of the VAT.
lastind
Specifies the last used index of the VAT. This is the internal length of the VAT.
maxind
Specifies the highest index of the VAT, of which a cell is allocated.
npage
Specifies the number of elements by which the VAT is increased or decreased.
size
Specifies the size of the elements stored in the VAT.
arrhdl
Specifies the handle of the buffer where the elements of the VAT are stored.

Remarks

The VAT has an allocated buffer (arrhdl) which stores the elements of the VAT. The allocated buffer can store maxind elements. If you add elements to the VAT, first call the VATFind function which tries to locate a free cell. If VATFind can not find any free cell, it increases the size of the allocated buffer continuously by the size of npage elements and updates the lastind, maxind and freeind members of the VATDESC structure.

Note that the maxind pointer always must be greater than or equal to the lastind pointer.

Note that you must call VATFind before inserting into or adding to a VAT to get the index of the first free cell in the VAT. After it you can complete the add or insert operation at the previously obtained index.
