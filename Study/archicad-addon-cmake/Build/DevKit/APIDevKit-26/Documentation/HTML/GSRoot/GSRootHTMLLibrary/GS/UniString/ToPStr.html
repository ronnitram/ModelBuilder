UniString::ToPStr

Return Pascal string form of the string. 

PStr ToPStr (
    UIndex              from = 0,
    USize               range = MaxUSize
) const;

Parameters

from
Start position of the substring to be returned. Must be beetwen 0 and length of the string.
range
Range of characters to be returned. from + range must be beetwen 0 and length of the string. The default value MaxUSize indicates range that extends to end of the string.

Return Value

A private object which can be implicitly converted to const unsigned char*. This pointer remains valid only inside its enclosing expression (statement) so the pointer should not be stored (but can be passed in the same expression as parameter to a funtion or method).

Description

This method returns Pascal string form of the string (or part of the string), that is sequence of 8 bits unsigned characters preceding with number of characters. Note that maximum length of a Pascal string is 255 characters. It returns a private object which can be implicitly converted to const unsigned char*. This pointer remains valid only inside its enclosing expression (statement) so the pointer should not be stored (but can be passed in the same expression as parameter to a funtion or method).
