Description

EventSource is root ancestor class of all event source classes. An event source is an object that informs its environment about important changes occurred in its state. Event sources notify event observers attached to them via events (event objects).
When an event observer is interested in some event on an event source it should attach (or should be attached by a third participant) to that event source. The event source will send events to the event observer until it is detached. Order of notifications is not specified. When an event source is destroyed, first all of its observers are notified via their SourceDestroyed method, then observers are automatically detached. When an event observer is destroyed then it will be automatically detached from all of event sources it is attached to.
Note that this class declares the root destructor as virtual.
