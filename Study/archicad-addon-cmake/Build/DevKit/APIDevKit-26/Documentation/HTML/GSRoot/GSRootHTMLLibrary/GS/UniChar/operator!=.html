UniChar::operator!=

Case sensitive 'not equal to' comparison of UniChars and other types. 

static bool operator!= (
    Type1               param1,
    Type2               param2
);
GSROOT_DLL_EXPORT friend bool operator!= (
    const UniChar &     leftOp,
    const UniChar &     rightOp
);
GSROOT_DLL_EXPORT friend bool operator!= (
    const UniChar &     leftOp,
    Layout              rightOp
);
GSROOT_DLL_EXPORT friend bool operator!= (
    Layout              leftOp,
    const UniChar &     rightOp
);
GSROOT_DLL_EXPORT friend bool operator!= (
    const UniChar &     leftOp,

);
GSROOT_DLL_EXPORT friend bool operator!= (

GSROOT_DLL_EXPORT friend bool operator!= (

);
GSROOT_DLL_EXPORT friend bool operator!= (
    unsigned char       leftOp,
    const UniChar &     rightOp
);

Parameters

param1
Left operand of the comparison.
param2
Right operand of the comparison.
leftOp
rightOp

Return Value

Value Meaning
true two operands are different
false two operands are not different

Description

This method compares UniChars and other types to determine whether they are different. Comparing is case sensitive and uses the default user locale.
It can compare UniChars, 16 bits unsigned short Unicode characters and 8 bits signed / unsigned characters.
