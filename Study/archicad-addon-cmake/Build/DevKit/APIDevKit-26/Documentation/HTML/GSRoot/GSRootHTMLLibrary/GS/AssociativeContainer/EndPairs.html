AssociativeContainer<IndexType, ItemType>::EndPairs

Returns an iterator referring to the one past last index-item pair in the associative container. 

PairIterator EndPairs (void);
ConstPairIterator EndPairs (void) const;

Return Value

A forward iterator or a constant forward iterator referring to the one past last index-item pair in the associative container.

Description

This method returns a forward iterator or a constant forward iterator referring to the one past last index-item pair in the associative container.
It can not be dereferenced, it can be only compared against other iterators (see the example).

Example

GS::HashTable<GS::String, long> hashTable;                                     // hashtable of {strings, long} pairs
GS::AssociativeContainer<GS::String, long>& associativeContainer = hashTable;  // associative container of {string. long} pairs (refers to the hashTable)

GS::AssociativeContainer<GS::String, long>::PairIterator begin = associativeContainer.BeginPairs ();  // obtaining an iterator referring to the first index-item pair
GS::AssociativeContainer<GS::String, long>::PairIterator end   = associativeContainer.EndPairs ();    // obtaining an iterator referring to the one past last index-item pair
while (begin != end) {               // until the end of the sequence (associative container)
    GS::String s = *(begin->key);    // dereferencing the iterator (returns pointer to the pair structure)
    long l       = *(begin->value);  // dereferencing the iterator (returns pointer to the pair structure)
    *(begin->value) = 4;             // dereferencing the iterator (returns pointer to the pair structure)
    ++begin;                         // advance to the next pair
}

This example illustrates the use of pair iterators obtained from an associative container.
