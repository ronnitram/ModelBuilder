CHGetSystemCharCode

The CHGetSystemCharCode function retrieves the current character code for the system.

    GSCharCode CHGetSystemCharCode (
      void
    );

Return Values

The return value is the current character code identifier for the system, or default identifier if no character code is current.

Version: GSRoot 1.3 or later
