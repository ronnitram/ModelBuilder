VATClear

The VATClear function clears a specified element of a VAT.

GSErr VATClear(
  VATDESC          *dptr,       // pointer to VAT
  long             ind  // index to clear
);

 

Parameters

dptr
Pointer to the VAT.
ind
Index of the element to clear. It must be a valid index, so it must be greater than or equal to 0 and less than lastind.

Return Values

The return value is 0.

Remarks

The VATClear function clears the specified cell by marking it with VAT_FREE flag. The function updates the freeind member of the dptr VAT if necessary.

Requirements

    Header: Declared in VA.hpp
    Import Library: Link with GSRootImp.lib
