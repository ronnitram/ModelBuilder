About String Manager

The String Managers provides functions to convert, search, compare and process international text.

The encoding of the texts in the various functions can be given by a GSCharCode parameter. The default value of this parameter can be get / set by CHGetDefaultCharCode and CHSetDefaultCharCode respectively.

Character strings represented in C and Pascal style can be copied by CHCopyC and CHCopyPascal, converted by CHpascal2c and CHc2pascal.

The encoding of the text can be changed by CHANSI2Unicode, CHUnicode2ANSI, IVChars, IVCString, IVPString. The end of line marker character(s) can be changed by IVEndofLine.

The remaining functions in the String Manager make possible to classify characters (CHisAlpha, CHisAlnum, CHisDigit), search for characters and substrings in texts (CHSearchCharLeft, CHSearchCharRight, CHSearchSubString), compare texts for equality or ordering (CHEqualASCII, CHEqualCStrings, CHEqualFileNames700, CHEqualPFileNames, CHCompareASCII, CHCompareCStrings, CHCompareFileNames, CHCompareTexts).

Remarks:

The String Manager interface for C and C++ differs at a few functions. The documentation contains basically the C++ interface, with some remarks for the C interface.
