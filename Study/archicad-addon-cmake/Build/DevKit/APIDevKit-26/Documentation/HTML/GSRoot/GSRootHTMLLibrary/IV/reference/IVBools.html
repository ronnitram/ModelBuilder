IVBools

Swaps an array of bools.

    void IVBools (
        GS::PlatformSign platform,
        bool* pBools,
        long nBools
    );
    void IVBools (
        GS::PlatformSign from,
        GS::PlatformSign to,
        bool* pBools,
        long nBools
    );

Parameters

platform, from
[in] The input platform.
to
[in] The output platform for the second function.
pBools
[in, out] Points to an array of bools.
nBools
[in] The number of elements in pBools.

Remarks

These functions are no-ops; one byte cannot be swapped. The reason why we supply these functions is to make automatic generation of swapping functions or member functions easier.

Requirements

Version: 2.0 or later
Header: IV.hpp
