OChannel::GetOutputStatus

Returns output status of the output channel. 

virtual GSErrCode GetOutputStatus (void) const = 0;

Return Value

Output status of the output channel.

Description

This method returns output status of the output channel. This status is NoError until an error occures on the output channel. After that the output channel stores the error code (e.g. OutputIsFull or WriteError) in the output status until it is reseted by the user. After an error has been occurred, methods performing write-like operations do nothing and immediately return with the output status. By reseting the output status, these methods will behave normally and will try the writing again.
