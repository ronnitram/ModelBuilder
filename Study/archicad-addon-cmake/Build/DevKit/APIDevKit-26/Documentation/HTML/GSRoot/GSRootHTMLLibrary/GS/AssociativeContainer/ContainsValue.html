AssociativeContainer<IndexType, ItemType>::ContainsValue

Examines whether the associative container contains a specified item. 

bool ContainsValue (const ItemType & item) const;

Parameters

item
Item to be examined for in the associative container.

Return Value

Value Meaning
true the specified item was found
false the specified item was not found

Description

This method examines whether the associative container contains a specified item (by comparing items by value). It requires items to have comparison operators (== and !=).
