CHSearchSubstring

Find a substring.

    char* CHSearchSubstring (
      const char*  pCSubString,
      const char*  pChars,
      long         nChars = CStringLen,
      CaseType     cs = CS_CaseSensitive,
      GSCharCode   cc = CC_Default
    );

Parameters

pCSubString
[in] String to search for. Must be a null terminated C-String.
pChars
[in] String to search.
nChars=CStringLen
The length of the string pChars, or it can be CStringLen if pChars is a null terminated C-string.
cs=CS_CaseSensitive
Case sensitivity parameter. Pass CS_CaseSensitive to perform a case sensitive search, pass CC_CaseInsensitive to perform a case insensitive search.
cc=CC_Default
Character code of pChars and pCSubString parameter.

Return Values

A pointer to the first occurance of pCSubString in pChars.
