VARelease

The VARelease function releases the unused part of a Virtual Array.

GSErr VARelease(
  DESC             *dptr,     // pointer to virtual array
);

 

Parameters

dptr
Pointer to the Virtual Array of which unused part is released. It can't be NULL.

Return Values

If the function succeeds, the return value is 0 (NoError).

If the function fails, the return value is a Graphisoft error code. For more information about error codes, see GSRoot error codes.

If dptr is NULL, the return value is ErrNilHandle.

Remarks

The VARelease function checks whether the elements of the Virtual Array can be stored in a smaller buffer. If they can, the function decreases the size of the buffer of the Virtual Array by npage members.

Note, calling this function is highly recommended after calling the VAClearIndex function.

Requirements

   Header: Declared in VA.hpp
   Import: Link with GSRootImp.lib
