BMGetPtrSize

The BMGetPtrSize retrieves the size of the specified non-relocatable memory block.

GSSize BMGetPtrSize(
  GSPtr           p      // pointer to the memory block
);

 

Parameters

p
Pointer to the non-relocatable block of memory of which size is to retrieve.

Return Values

If the function succeeds, the return value is the size of the specified non-relocatable block of memory in bytes.

If the function fails, the return value is 0. To get extended error information, call BMError.

Requirements

    Header: Declared in BM.hpp
    Import Library: Link with GSRootImp.lib
