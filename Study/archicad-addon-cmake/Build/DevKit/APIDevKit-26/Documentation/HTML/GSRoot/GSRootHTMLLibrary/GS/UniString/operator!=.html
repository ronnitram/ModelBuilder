leftOp

Return Value

Value Meaning
true two operands are different
false two operands are not different

Description

This method compares strings and other types to determine whether they are different. Comparing is case sensitive and uses the default user locale.
It can compare strings, substrings, 0 terminated sequence of 16 bits unsigned short Unicode characters and 0 terminated C strings.
