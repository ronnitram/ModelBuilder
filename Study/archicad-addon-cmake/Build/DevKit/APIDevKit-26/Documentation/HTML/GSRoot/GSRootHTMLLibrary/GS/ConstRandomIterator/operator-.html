ConstRandomIterator<Type>::operator-

Returns a new iterator referring to the specified item or returns difference between two iterators. 

ConstRandomIterator operator- (Offset offset) const;

Returns a new iterator referring to the specified item. 

Offset operator- (const ConstRandomIterator & rightOp) const;

Returns difference between two iterators.

Parameters

offset
Offset of the item to which the new iterator should refer to from the item this iterator is currently referring to.
rightOp
Right operand of the difference operation between iterators.

Return Value

1. version: iterator referring to the specified item.

2. version: (signed) difference between positions of items iterators are referring to.

Description

1. version of this method returns a new iterator referring to the specified item. Base point of the addressing is the item this iterator is currently referring to. The specified (signed) offset is subtracted from the position of the current item, that is the desired item is in this relative distance from the current item.
It is illegal to refer to non-existing items out of (before or after) the sequence.

2. version of this method returns (signed) difference between two iterators, that is difference between positions of items these iterators are referring to.
