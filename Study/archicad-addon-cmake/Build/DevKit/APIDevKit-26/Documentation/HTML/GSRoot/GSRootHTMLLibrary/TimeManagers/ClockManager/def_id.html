Clock Manager

The Clock Manager is used to manage clocks in platform independent manner. The Clock Manager defines two tick counters: a lower and a higher resolution tick counter. The manager gives functions to get the system time which is the value of the specified counters. These tick counters count the number of ticks elapsed since the system startup.
