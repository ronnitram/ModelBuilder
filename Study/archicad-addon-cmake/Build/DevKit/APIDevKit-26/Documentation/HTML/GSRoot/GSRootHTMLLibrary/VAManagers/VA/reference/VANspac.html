VANspac

The VANspac function increases the size of a Virtual Array by n elements.

long VANspac(
  DESC      *dptr,   // pointer to virtual array
  long             n   // number of elements by which virtual array is increased
);

 

Parameters

dptr
Pointer to the Virtual Array whose size is increased.
n
Specifies the number of elements by which the Virtual Array is increased. It must be greater than or equal to zero.

Return Values

If the functions succeeds, the return value is the index of the first unused cell of the Virtual Array. This is the index of the first cell from the newly allocated cells of the Virtual Array.

If the function fails, the return value is a Graphisoft error code. For more information about error codes, see GSRoot error codes.

If dptr is uninitialized then return value is ErrNilHandle.

Remarks

The VANspac function increases the size of the buffer of the Virtual Array if necessary.

If n is less than zero, the function does nothing.

Requirements

    Header: Declared in VA.hpp
    Import Library: Link with GSRootImp.lib
