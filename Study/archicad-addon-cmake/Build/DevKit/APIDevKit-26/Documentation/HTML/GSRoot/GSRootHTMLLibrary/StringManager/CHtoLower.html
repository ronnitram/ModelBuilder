CHtoLower

Converts a string to lowercase. [strlwr]

    void CHtoLower (
      char*       pChars,
      long        nChars = CStringLen,
      GSCharCode  charCode = CC_Default
    );

Parameters

pChars
[in-out] String to convert to lovercase.
nChars
[in] Length of the string. This value can be CStringLen that means that pChars is a null terminated string.
charCode
[in] The character code used to perform the conversion.

Remarks

The CHtoLower function converts any uppercase letters in string to lowercase as determined by the the charCode parameter. Other characters are not affected.
