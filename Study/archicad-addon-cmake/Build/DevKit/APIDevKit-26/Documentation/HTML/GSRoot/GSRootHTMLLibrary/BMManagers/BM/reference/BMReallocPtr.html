BMReallocPtr

The BMReallocPtr function reallocates a non-relocatable block of memory of the specified size.

GSPtr BMReallocPtr(
  GSPtr           p,     // pointer to reallocate
  GSSize          nBytes,                // number of bytes
  GSFlags         reallocFlags   // reallocation attributes
  short           actionCode     // user specified code
);

 

Parameters

p
Specifies the pointer to the non-relocatable block of memory to reallocate.
nBytes
Specifies the desired size (in bytes) of the non-relocatable block
reallocFlags
Specifies how to reallocate memory. This parameter can be any combination of the following values:
Value Meaning
REALLOC_ACTION User action is enabled.
REALLOC_CLEAR Newly allocated memory block is zeroed.
REALLOC_FULLCLEAR The full memory block is zeroed.
REALLOC_SHRINK The new size must be smaller than the old.
REALLOC_DISPOSE Dispose if reallocation fails.
REALLOC_MOVEABLE The new memory block is moveable.
actionCode
User specified code for the BMOptProc function.

Return Values

If the function succeeds, the return value is a handle referring to the allocated block of the requested size.

If the function fails, the return value is NULL.

Remarks

If the pointer refers to an existing block, the BMReallocPtr releases that block before creating a new one.

Requirements

    Header: Declared in BM.hpp
    Import Library: Link with GSRootImp.lib
