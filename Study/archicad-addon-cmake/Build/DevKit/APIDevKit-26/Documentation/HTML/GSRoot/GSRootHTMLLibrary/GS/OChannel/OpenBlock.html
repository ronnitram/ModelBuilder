OChannel::OpenBlock

Opens a new data block extended with data size. 

GSErrCode OpenBlock (void);

Return Value

Value Meaning
NoError* OK
OutputIsFull* output device is full
WriteError* write error on the output device
ErrMemoryFull* out of memory
Error* unspecified error

* declared in OChannel class scope
* declared in GS namespace, but accessible in the global scope

Description

This is a utility method (implemented with block methods) which supports writing data in blocks with the data size placed at the begining of the block. This method opens a new block and reserves (skips) space for the data size. After the data has been written, block should be closed with the CloseBlock method which writes back size of the data (and returns write pointer to the end of the output stream). Data size is written as unsigned long. Blocks can be nested.
