CHisAlphaASCII

Returns whether the specified character is a letter int the ASCII range: [a-zA-Z].

    bool CHisAlphaASCII (
      char  aChar
    );

Parameters

aChar
[in] The character to test.

Return Values

Returns true if the character represented by aChar is alphabetic ASCII character.
