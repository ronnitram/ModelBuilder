CHCompareCStrings

Compare two null terminated C strings with or without case sensitivity, using the specified character code. [strcmp, strcmpi]

    long CHCompareCStrings (
      const char*  p1,
      const char*  p2,
      CaseType     cs = CS_CaseSensitive,
      GSCharCode   cc = CC_Default
    );

Parameters

p1
[in] Pointer to the first null terminated string to be compared.
p2
[in] Pointer to the second null terminated string to be compared.
cs
[in] Case sensitivity switch.
cc
[in] Specifies the character code used for the comparsion. See GSCharCode.

Return Values

If the function fails, the return value is zero.
Value Meaning
<0 The string pointed to by the p1 parameter is less in lexical value than the string pointed to by the p2 parameter.
0 The string pointed to by p1 is equal in lexical value to the string pointed to by p2.
>0 The string pointed to by p1 is greater in lexical value than the string pointed to by p2.

Remarks

Notice that if the return value is 0, the two strings are "equal" in the collation sense, though not necessarily identical.

This function might move (allocate/deallocate) memory.
