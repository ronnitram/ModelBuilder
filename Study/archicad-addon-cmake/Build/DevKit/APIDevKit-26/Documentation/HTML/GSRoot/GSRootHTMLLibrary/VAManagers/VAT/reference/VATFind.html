VATFind

The VATFind function searches for the index of the first free cell in a VAT.

long VATFind(
  VATDESC              *dptr    // pointer to VAT
);

 

Parameters

dptr
Pointer to the VAT of which the first free cell is needed.

Return Values

If the function succeeds, the return value is the index of the first free cell.

If the function fails, the return value is a a Graphisoft error code. For more information about error codes, see GSRoot error codes.

Remarks

The VATFind function increases the size of the allocated buffer of the VAT if it can't find free cell in the buffer. In this case the size of the buffer will be  increased by npage elements.

Note that if you want to insert an element into a VAT or add an element to a VAT, first you must call the VATFind function to obtain the index of a free cell. Then you can write the new element into the free cell identified by the return value of the VATFind.
