AssociativeContainer<IndexType, ItemType>::Get

Item access by index. 

ItemType & Get (const IndexType & idx);
const ItemType & Get (const IndexType & idx) const;
bool Get (
    const IndexType &   idx,
    ItemType *          item
) const;

Parameters

idx
Index of the item to be accessed. In the 1. and 2. case it must be an existing index.
item
On return: the specified item, if it was found.

Return Value

In the 1. and 2. case reference to the specified item. Generally this reference remains valid until the next modifying operation on the container is invoked.
In the 3. case true if the specified item was found (and returned), otherwise false.

Description

This method provides access to the item specified by its index in the associative container.
In the 1. and 2. case the resulting reference generally remains valid until the next modifying operation on the container is invoked. If no matching index is found then IllegalReferenceException exception will be thrown.
