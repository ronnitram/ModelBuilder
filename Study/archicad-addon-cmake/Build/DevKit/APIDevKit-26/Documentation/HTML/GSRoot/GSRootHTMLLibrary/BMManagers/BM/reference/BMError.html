BMError

The BMError function returns the error code produced by the last direct Basic Memory Manager function call within the application.

GSErr BMError();

 

Return Values

The return value is the error code of the last error occurred in the Basic Memory Manager.

Remarks

The most functions in the Basic Memory Manager set the last error code value when they fail. Therefore when a function fails you must immediately call the BMError function in order to get the last error code. Hence the BMError function sets the last error code value to NoError, the second call of the function always returns NoError. For more information about error codes see GSRoot Error Codes.
