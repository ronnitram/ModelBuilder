BMHandleToHandle

The BMHandleToHandle function copies all of the data from a relocatable block to a newly allocated relocatable block.

GSErr BMHandleToHandle(
  GSHandle        inpH,  // source handle
  GSHandle        *outH  // pointer to destination handle
);

 

Parameters

inpH
Specifies the handle of the relocatable memory block of which data is to copy. It can not be NULL and it must be a nonempty handle.
outH
Pointer to the handle of a new relocatable block of which data duplicates the original one. It can not be NULL.

Return Values

If the function succeeds, the return value is 0 (NoError).

If the function fails, the return value is a Graphisoft error code. To get extended error information, call BMError.

If inpH or outH is NULL, the return value is ErrNilHandle.

If inpH handle is an empty handle, the return value is ErrEmptyHandle.

Remarks

If the function is successful in creating a new relocatable block, it does not copy the properties of the original block. The new relocatable block is unlocked, unpurgeable and not a resource.

Requirements
