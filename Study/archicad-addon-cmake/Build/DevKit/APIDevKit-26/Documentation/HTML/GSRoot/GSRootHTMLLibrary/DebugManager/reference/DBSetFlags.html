DBSetFlags

The DBSetFlags function sets the flags of the Debug Manager in the copy the specified flag category.

void DBSetFlags(
  unsigned long flagCategory,      // flag category
  GSFlags               flags              // flags to set
);

 

Parameters

flagCategory
Specifies what kind of flags are subject to change. The flag category must be one of the following values:
Value Meaning
DBGeneral General purpose debug flags.
flags
Specifies the flags to set. This parameter can be any combination of the following values:
Value Meaning
DBEnableCallTrace Enables the tracing of function calls.
DBEnableBMTrace Enables the tracing of Basic Memory Manager operations.
DBEnableLogFile Enables logging into file.
DBEnableTrace Enables the tracing.

Remarks

If the flagCategory parameter specifies an invalid flag category, the function does nothing.
