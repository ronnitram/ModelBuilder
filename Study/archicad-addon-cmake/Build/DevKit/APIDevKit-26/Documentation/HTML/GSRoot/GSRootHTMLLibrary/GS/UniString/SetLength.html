UniString::SetLength

Sets length of the string. 

void SetLength (USize newLength);
void SetLength (
    USize               newLength,
    UniChar             filler
);

Parameters

newLength
New length of the string.
filler
Filler character for new characters if the new length is greater than the old one.

Description

This method sets length of the string. If the new length is less than the old one then trailing characters that don't fit into the new length will be deleted. If the new length is greater than the old one then new characters are appended to the string. Filler for the new characters can be specified otherwise their value will be undefined.
On memory overflow OutOfMemoryException exception will be thrown.
