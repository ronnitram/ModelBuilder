Description

Event is root ancestor class of all event classes. Event objects are issued by event sources and perceived by event observers attached to event sources. Event carries information about important changes from event sources to event observers. Source of the event is an implicit information which is stored in every event.
Note that this class declares the root destructor as virtual.
