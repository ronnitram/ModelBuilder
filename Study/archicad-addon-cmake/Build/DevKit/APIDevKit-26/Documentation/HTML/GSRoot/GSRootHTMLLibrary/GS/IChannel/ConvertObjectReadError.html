IChannel::ConvertObjectReadError

Converts error codes during the type composition. 

static GSErrCode ConvertObjectReadError (GSErrCode ec);

Parameters

ec
Error code to be converted.

Return Value

Value Meaning
IntegrityError* end of input has been reached during the type composition
other values original error codes

* declared in GS namespace, but accessible in the global scope

Description

This static helper method can be used when an error is encountered during the composition (reading) of a type for the error code conversion. If a type can not be composed because the end of input has been reached, then this method returns IntegrityError, otherwise it returns the original error code. Converted value can be passed to the caller.
