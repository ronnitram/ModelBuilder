OChannel::CloseBlock

Closes the last data block. 

GSErrCode CloseBlock (void);

Return Value

Value Meaning
NoError* OK
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

This method closes the last data block opened by the previous OpenBlock call. It calculates and writes back size of the data (not including the size itself) at the begining of the block and returns write pointer to the end of the output stream.
