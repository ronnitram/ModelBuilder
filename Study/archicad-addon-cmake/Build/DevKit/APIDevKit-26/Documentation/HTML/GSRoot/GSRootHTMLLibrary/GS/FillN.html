FillN

Assigns a new value to a specified number of elements in a range beginning with a particular element. 

template <class Out, class T>
void FillN (
    Out                 first,
    USize               range,
    const T &           value
);

Template parameters

Out
Output iterator.
T
T type template parameter.

Parameters

first
An output iterator addressing the position of the first element in the range to be assigned the value value.
range
A signed or unsigned integer type specifying the number of elements to be assigned the value.
value
The value to be assigned to elements in the range [first, first + range).

Description

FillN assigns the value value to every element in the range [first, first + n). That is, for every iterator i in [first, first + n), it performs the assignment *i = value. The return value is first + n.
