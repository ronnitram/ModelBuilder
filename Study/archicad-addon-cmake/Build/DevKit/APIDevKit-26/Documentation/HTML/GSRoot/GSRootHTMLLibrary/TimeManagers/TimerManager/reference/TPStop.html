TPStop

The TPStop function stops a timer.

double TPStop(
  GSTimer               *pTimer            // pointer to timer
);

 

Parameters

pTimer
Pointer to the timer to stop.

Return Values

The return value is the elapsed time in seconds since the specified timer first received a start message.

Remarks

The TPStop function decreases the value of the reference counter of the specified timer. The TPStop stops the specified timer only if the reference counter reached zero.

Requirements

    Header: Declared in GSTime.hpp
    Import Library: Link with GSRootImp.lib
