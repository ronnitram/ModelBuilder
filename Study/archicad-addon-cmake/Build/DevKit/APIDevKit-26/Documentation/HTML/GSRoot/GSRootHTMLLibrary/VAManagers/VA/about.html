About Virtual Arrays

 

The Virtual Array (VA) is a dynamic array of which size can be increased or decreased according to the number of elements stored in it. The Virtual Array is represented by the DESC structure. The DESC structure defines the attributes of a Virtual Array.

The DESC structure contains a memory buffer in which the elements of the Virtual Array are stored. The Virtual Array Manager provides functions to increase or decrease the size of this buffer, but the get/set operations must be done manually. The DESC structure has two indices which point to the last allocated (maxind) and the last used (lastind) cell of the buffer. If you are performing get/set operations on a Virtual Array you must consider this indices. Note that the VA is indexed from 1 to the lastind member of the DESC. The index range is always continuous, the cells of the index range are always in use. If the Virtual Array must be enlarged or shrunken then the size will be increased or decreased by the number of elements specified by the npage member of the DESC structure.

Before using Virtual Arrays you must initialize them by calling the VAInit function. VAInit allocates the buffer, and initializes the members of the DESC structure. After initializing you can increase the size of the Virtual Array with VASpac, VANspac or VAReal functions. The previous functions maintain the members of the DESC structure and increase continuously the size of the buffer if necessary.

Because the index range is continuous, if you want to add a new element to the VA, first you must increase the size of the VA by one with the VASpac function. The VASpac increases the size of the buffer in DESC and updates maxind and lastind members. The new value of the lastind member will be increased by one. Because the lastind member points to the last cell of the VA which is empty (after calling VASpac), you can copy the new element to add into the cell pointed by lastind. Note that you can add new elements only to the end of the VA, that is you can append but not insert elements.

If you want to decrease the size of the Virtual Array you can use the VaDecr and VARelease functions. They update the members of the DESC structure and decrease the size of the buffer if necessary.

After using the Virtual Array you must call the VAFree function to release its buffer. After it you can call the VAUnused function to mark the Virtual Array unused.

If you want to clear an element of the Virtual Array you can do this with the VAClearIndex function. Note that the VAClearIndex decreases the index range, because it releases the cleared cell of the VA. The VACopyAll function copies the content of a Virtual Array into an other VA. If you have to add more than one element to a Virtual Array, you can use the VACopyToEnd and VAAddFromArr copy functions. The VASort function are to sort the members of a Virtual Array whereas the VASearch function are to search the index of an element in a Virtual Array.
