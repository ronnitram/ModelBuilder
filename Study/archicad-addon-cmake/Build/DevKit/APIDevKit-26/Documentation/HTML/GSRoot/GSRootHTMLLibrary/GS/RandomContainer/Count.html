RandomContainer<Type>::Count

Counts number of occurences of a specified item. 

USize Count (
    const Type &        item,
    UIndex              from = 0,
    USize               range = MaxUSize
) const;

Parameters

item
Specifies item to be counted.
from
Start position of the counting. Must be beetwen 0 and size of the random container.
range
Range of items where counting should be performed. from + range must be beetwen 0 and size of the random container. The default value MaxUSize indicates range that extends to end of the random container.

Return Value

Number of occurences of the specified item in the container.

Description

This method counts number of occurences of a specified item in the random container or in the part of the random container (by comparing by value).
This method requires items to have comparison operators (== and !=).
