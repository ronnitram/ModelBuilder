About Date & Time Manager

The Time & Date Manager is used to manage the current date and time in a platform independent manner. The manager defines the platform independent GSTime date-time format to represent the current date and time. The manager provides functions to obtain the current time and date and to convert between time formats for ease of comparison and display. The Time & Date Manager can handle three different date-time formats: the GSTime time format, the Windows time format and the Macintosh time format. The manager provides functions to convert among these time formats.

The Time & Date manager defines the GSTime type as a platform-independent format for representing time. The GSTime is a raw date-time format, for representing the date and time in a easy-use format, use the GSTimeRecord structure which contains each element of the current date and the current time as members within the structure. If you want to convert between the previously mentioned formats, you can use the TIGetGSTime function to convert from GSTimeRecord to GSTime or the TIGetTimeRecord function to convert from GSTime to GSTimeRecord.

The TIGetTime function is used to get the current time in GSTime format. If you want to get the current time in Windows format call the TIGetWinTime function or use the TIGetMacTime function to get it in Macintosh format.

If you want to convert from Macintosh time format to GSTime format then use the TIMac2GSTime function. For the opposite directional conversion call the TIGS2MacTime function. If you want to convert from Windows time format to GSTime format then use the TIWin2GSTime function. For the opposite directional conversion call the TIGS2WinTime function.

The TIGetTimeString function creates a time string from a GSTimeRecord.
