AssociativeContainer<IndexType, ItemType>::Enumerate

Allows enumeration of the associative container's items. 

Iterator Enumerate (void);
ConstIterator Enumerate (void) const;

Return Value

A forward iterator or a constant forward iterator referring to the first item in the associative container.

Description

This method allows forward enumeration of the associative container's items. It returns a forward iterator or a constant forward iterator referring to the first item in the associative container. Enumeration order is not defined.
References to items obtained by iterator dereferencing can be used only until the next modifying method on the parent associative container is invoked.
This method is equivalent with the Begin method.

Example

GS::HashTable<GS::String, long> hashTable;                                     // hashtable of {strings, long} pairs
GS::AssociativeContainer<GS::String, long>& associativeContainer = hashTable;  // associative container of {string. long} pairs (refers to the hashTable)

GS::AssociativeContainer<GS::String, long>::Iterator it = associativeContainer.Enumerate ();  // obtaining an item iterator (or associativeContainer.Begin ())
while (it != NULL) {  // until the end of the sequence (associative container)
    long l = *it;     // dereferencing the iterator (returns reference)
    ++it;             // advance to the next item
}

This example illustrates the use of item iterators obtained from an associative container.
