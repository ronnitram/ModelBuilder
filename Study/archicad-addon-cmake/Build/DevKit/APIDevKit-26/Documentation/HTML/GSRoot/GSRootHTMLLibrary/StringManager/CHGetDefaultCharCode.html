CHGetDefaultCharCode

Returns the default character code.

    GSCharCode CHGetDefaultCharCode (
      void
    );

Return Values

A character code, previously set by GHSetDefaultCharCode.

Remarks

You can set the default char code with GHSetDefaultCharCode. By default this value is CC_System.

Version: GSRoot 1.3 or later
