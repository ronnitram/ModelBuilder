UniString::operator<=

Case sensitive 'less than or equal to' comparison of strings. 

bool operator<= (const UniString & rightOp) const;

Parameters

rightOp
Right operand of the comparison.

Return Value

Value Meaning
true this string is less than or equal to the right operand
false this string is neither less than nor equal to the right operand

Description

This method compares two strings to determine whether is this string less than or equal to the right operand. Comparing is case sensitive and uses the default user locale.
