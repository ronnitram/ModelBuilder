IChannel::Read

Reads the next simple C++ type (or Object) from the input channel. 

GSErrCode Read (char & c);
GSErrCode Read (unsigned char & c);

GSErrCode Read (short & s);

GSErrCode Read (unsigned int & i);

GSErrCode Read (float & f);

Parameters

c
On return: character value read from the input channel.
b
On return: bolean value read from the input channel.
s
On return: short integer value read from the input channel.
i
On return: integer value read from the input channel.
l
On return: long integer value read from the input channel.
f
On return: float value read from the input channel.
d
On return: double value read from the input channel.
cStr
On return: C string read from the input channel.
obj
On return: object read from the input channel.

Return Value

Value Meaning
NoError* OK
EndOfInput* end of input has been reached
ReadError* read error on the input device
IntegrityError* data read by the current IProtocol object is not valid for the requested type
Error* unspecified error

* declared in IChannel class scope
* declared in GS namespace, but accessible in the global scope

Description

This method reads the next simple C++ type (or Object) from the input channel through the current IProtocol object.

If there is no data left on the input channel then EndOfInput is returned. If amount of data left on the input channel is not sufficient for the composition of the requested type then IntegrityError is returned.

Input status will record and hold the first error occurred on the input channel. After an error this method does nothing and immediately returns with the stored error code (until the input status is reseted by the user).
