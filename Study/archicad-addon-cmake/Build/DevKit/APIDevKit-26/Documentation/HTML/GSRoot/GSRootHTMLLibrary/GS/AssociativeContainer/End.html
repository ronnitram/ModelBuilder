AssociativeContainer<IndexType, ItemType>::End

Returns an iterator referring to the one past last item in the associative container. 

Iterator End (void);
ConstIterator End (void) const;

Return Value

A forward iterator or a constant forward iterator referring to the one past last item in the associative container.

Description

This method returns a forward iterator or a constant forward iterator referring to the one past last item in the associative container.
It can not be dereferenced, it can be only compared against other iterators (see the example).

Example

GS::HashTable<GS::String, long> hashTable;                                     // hashtable of {strings, long} pairs
GS::AssociativeContainer<GS::String, long>& associativeContainer = hashTable;  // associative container of {string. long} pairs (refers to the hashTable)

GS::AssociativeContainer<GS::String, long>::Iterator begin = associativeContainer.Begin ();  // obtaining an iterator referring to the first item
GS::AssociativeContainer<GS::String, long>::Iterator end   = associativeContainer.End ();    // obtaining an iterator referring to the one past last item
while (begin != end) {  // until the end of the sequence (associative container)
    long l = *begin;    // dereferencing the iterator (returns reference)
    ++begin;            // advance to the next item
}

This example illustrates the use of item iterators obtained from an associative container.
