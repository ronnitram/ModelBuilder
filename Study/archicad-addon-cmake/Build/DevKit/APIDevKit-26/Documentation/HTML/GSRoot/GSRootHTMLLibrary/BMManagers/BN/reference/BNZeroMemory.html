BNZeroMemory

The BNZeroMemory function fills a memory range with zeros.

void BNZeroMemory(
  void            *destPtr,              // address of memory range to fill
  GSSize          len,           // length of memory range
);

 

Parameters

destPtr
Specifies the address of the memory range to fill.
len
Specifies the length of the memory range to fill.

Remarks

If the application is compiled as a debugging version, and the process does not have write access to destPtr (the BNValidWritePtr function returns false), the function causes an assertion fault and breaks into the debugger.

The BNZeroMemory function wraps the memset standard C Run-Time function.

Requirements

    Header: Declared in BM.hpp
    Import Library: Link with GSRootImp.lib
