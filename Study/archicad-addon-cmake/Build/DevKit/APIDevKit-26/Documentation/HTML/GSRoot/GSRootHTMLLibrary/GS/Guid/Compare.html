GS::Guid::Compare Method

Compares two Guids numerically.

long Compare (const GS::Guid & rightOp) const;

Parameters

rightOp
The Guid object to compare with.

Return Value

  • -1 if rightOp is greater,
  • 1 if rightOp is less,
  • 0 if the two Guids are equal.
