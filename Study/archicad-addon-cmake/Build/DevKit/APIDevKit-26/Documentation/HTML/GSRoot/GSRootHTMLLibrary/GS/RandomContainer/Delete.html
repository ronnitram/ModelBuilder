RandomContainer<Type>::Delete

Deletes item(s) at the specified position. 

void Delete (UIndex idx);
void Delete (
    UIndex              from,
    USize               range
);

Parameters

idx
Index of the item to be deleted. Must be beetwen 0 and size of the random container minus 1.
from
Start position of the deleting. Must be beetwen 0 and size of the random container.
range
Number of items to be deleted. from + range must be beetwen 0 and size of the random container.

Description

This method deletes an item or range of items at specified position from the random container.
