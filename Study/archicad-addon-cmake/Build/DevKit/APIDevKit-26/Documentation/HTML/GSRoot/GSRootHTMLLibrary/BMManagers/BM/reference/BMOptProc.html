BMOptProc

The BMOptProc function is used for customizing the optimization function of the Basic Memory Manager. The BMOptProc is a function type.

void BMOptProc(
  short           action // action code
);

 

Parameters

action
Specifies the action code passed to the caller Basic Memory Manager function.

Remarks

You can set your BMOptProc for the Basic Memory Manager in the BMInit function.

Requirements

    Header: Declared in BM.hpp
    Import Library: Link with GSRootImp.lib
