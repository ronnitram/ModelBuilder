TIWait

The TIWait function waits until the specified time elapsed.

void TIWait(
  double           delaySeconds  // time to wait
);

 

Parameters

delaySeconds
Specifies the time to wait in seconds.

Remarks

The TIWait function converts the delaySecondstime to GSFineTicks, then sleeps the process until the time to wait has elapsed.

Requirements

    Header: Declared in GSTime.hpp
    Import Library: Link with GSRootImp.lib
