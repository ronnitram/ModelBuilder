UniString::FindFirst

Searches for the first occurence of a given character or string. 

UIndex FindFirst (
    UniChar             uc,
    UIndex              from = 0,
    USize               range = MaxUSize
) const;
UIndex FindFirst (
    const UniString &   str,
    UIndex              from = 0,
    USize               range = MaxUSize
) const;

Parameters

uc
Character to be searched for.
from
Start position of the substring to be searched in. Must be beetwen 0 and length of the string.
range
Range of characters to be searched in. from + range must be beetwen 0 and length of the string. The default value MaxUSize indicates range that extends to end of the string.
str
String to be searched for.

Return Value

Index of the first occurence of the given character or string. If no occurence is found then the return value is MaxUIndex.

Description

This method searches for the first occurence of a given character or string in this string (or in part of this string).
