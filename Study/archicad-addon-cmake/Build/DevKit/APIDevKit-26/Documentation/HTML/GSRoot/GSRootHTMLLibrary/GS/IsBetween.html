IsBetween

Returns whether is the given value in a specified range. 

template <class T>
bool IsBetween (
    const T &           value,
    const T &           min,
    const T &           max
);

Template parameters

T
Type of input parameters.

Parameters

value
Value to be examined.
min
Lower bound of the range.
max
Upper bound of the range.

Return Value

Value Meaning
true value is in the range
false value is not in the range

Description

This function returns whether is the given value in a specified range (including bounds).
