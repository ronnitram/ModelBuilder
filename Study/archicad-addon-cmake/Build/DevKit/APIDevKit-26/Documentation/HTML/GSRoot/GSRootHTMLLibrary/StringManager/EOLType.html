EOLType

This type describes the various formats used to represent the line-end in text files.

    typedef enum EOLType {
        EOL_CR=0,
        EOL_LF=1,
        EOL_CRLF=2
    } EOLType;
Enumerator Value Meaning
EOL_CR=0 The line ending is the CharCR character. (0x0D) This line ending is used on MacOS systems.
EOL_LF=1 The line ending is the CharLF character. (0x0A) This line ending is used on Unix systems (including some part of MacOS-X).
EOL_CRLF=2 The line ending is a CharCR followed by a CharLF character. (0x0D,0x0A) This line ending is used on Windows.
