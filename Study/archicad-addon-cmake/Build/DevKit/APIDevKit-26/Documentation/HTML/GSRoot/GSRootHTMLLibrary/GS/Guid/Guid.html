Description

This class represents a Universally Unique Identifier as defined in RFC 4122. It uses system functions to generate version 4 UUIDs on all platforms.
There's a special global constant called GS::NULLGuid which represents an empty (zeroed) Guid. You can use this to test for invalid values.
