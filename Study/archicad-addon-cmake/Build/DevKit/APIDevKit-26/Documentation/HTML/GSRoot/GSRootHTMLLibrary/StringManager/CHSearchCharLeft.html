CHSearchCharLeft

Search for the first occurance of a character in a string. (ie. a left to right search) [strchr]

    char* CHSearchCharLeft (
      unsigned short  aChar,
      const char*     pChars,
      long            nChars = CStringLen,
      GSCharCode      charCode = CC_Default
    );

Parameters

aChar
[in] The character to search for.
pChars
[in] The string to search in.
[in] nChars
The length of the string. This value can be CStringLen that means search in a null terminated C-string.
[in] charCode
Character code.

Return Values

A pointer to the character if found, NULL otherwise.

Requirements

Version: GSRoot 1.3 or later
Header: CH.hpp
Import Library: GSRootImp.lib.lib
Module: GSRoot
