VADecr

The VADecr function decreases the size of a Virtual Array to npage * 10 elements.

void VADecr(
  DESC             *dptr,        // pointer to the virtual array
);

 

Parameters

dptr
Pointer to the virtual array of which size is decreased.

Remarks

The VADecr function decreases the size of the allocated buffer of the Virtual Array.

If the dptr Virtual Array is uninitialized the VADecr function does nothing.

Requirements

    Header: Declared in VA.hpp
    Import Library: Link with GSRootImp.lib
