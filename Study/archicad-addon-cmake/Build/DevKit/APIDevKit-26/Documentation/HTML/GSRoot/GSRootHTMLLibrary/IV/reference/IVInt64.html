IVInt64

Swaps the bytes in a signed 64-bit integer.

    void IVInt64 (
        GS::PlatformSign platform,
        Int64 *pInt64
    );
    void IVInt64 (
        GS::PlatformSign from,
        GS::PlatformSign to,
        Int64 *pInt64
    );

Parameters

platform, from
[in] The input platform.
to
[in] The output platform for the second function.
pInt64
[in, out] Points to a signed 64-bit integer. The 8 bytes will be reversed if necessary.

Remarks

You can use this function for either a big-endian->little-endian, or little-endian->big-endian conversion.

Requirements

Version: 2.0 or later
Header: IV.hpp
Import Library: GSRootImp.lib
