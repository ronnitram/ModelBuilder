Time Managers

The Time Managers provides functions to manage timers, global timers, time & date and clock. The Timer Manager contains four managers: Clock Manager, Date & Time Manager, Timer Manager and Global Timer Manager. The Clock Manager is responsible for managing system clock. The Date & Time Manager provides functions to obtain the current time and to convert among different timer formats. The Timer Manager and the Global Timer Manager provides functions to manage timers.
