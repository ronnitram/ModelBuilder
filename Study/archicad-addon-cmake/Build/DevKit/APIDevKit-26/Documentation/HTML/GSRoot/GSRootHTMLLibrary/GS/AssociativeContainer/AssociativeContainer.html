AssociativeContainer Class

Base class of associative containers. 

template <class IndexType, class ItemType>
class AssociativeContainer : public Container< ItemType >

Template parameters

IndexType
Type of indices (keys) the items in the associative container are accessible with.
ItemType
Type of items (values) the associative container stores.

Description

Associative containers are containers where an unique index (key) must be associated with every item (value) stored in the associative container. Items are directly accessible (referrable) with these indices. At most one item can be associated with every index.
Indices are required to have default and copy constructors, assigment operator and comparison (==, !=) operators.
