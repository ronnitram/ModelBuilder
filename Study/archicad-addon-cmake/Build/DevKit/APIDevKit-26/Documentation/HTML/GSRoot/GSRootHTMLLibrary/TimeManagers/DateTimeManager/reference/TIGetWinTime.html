TIGetWinTime

The TIGetWinTime function retrieves the current time information in Windows format.

long TIGetWinTime();

 

Return Values

The return value is the current date-time information.

Requirements

    Header: Declared in GSTime.hpp
    Import Library: Link with GSRootImp.lib
