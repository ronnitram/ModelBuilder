CHCompareASCII

Compares two null terminated ASCII strings with or without case sensitivity. [strcmp, strcmpi]

    long CHCompareASCII (
      const char*  p1,
      const char*  p2,
      CaseType     cs = CS_CaseSensitive
    );

Parameters

p1
The C string to be compared. Characters should be in the ascii (0-127) range.
p2
The C string to be compared with. Characters should be in the ascii (0-127) range.
cs
Case sensitivity switch.

Return Values

The return value for each of these functions indicates the lexicographic relation of p1 to p2.
Value Relationship of p1 and p2
<0 p1 less than p2
0 p1 identical to p2
>0 p1 greater than p2

Remarks

Don't use this function for comparing non-ASCII text. If either p1 or p2 contains non ascii characters the result is undefined. To compare non-ASCII strings use CHCompareCStrings!

The debug version gives assertion if either string contains a non-ASCII character.
