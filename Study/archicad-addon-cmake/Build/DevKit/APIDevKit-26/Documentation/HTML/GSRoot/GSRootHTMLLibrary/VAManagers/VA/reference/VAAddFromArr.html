VAAddFromArr

The VAAddFromArr function adds an existing array to the end of a Virtual Array.

GSErr VAAddFromArr(
  DESC            *dptr,   // pointer to virtual array to add to 
  void             *arr,        // pointer to an array to copy from 
  long             n    // the number of elements to copy
);

 

Parameters

dptr
Pointer to the Virtual Array of which size the Virtual Array Manager expands so that it can copy the specified number of elements from arr to the end of this Virtual Array.
arr
Pointer to the array to copy from. It can not be NULL.
n
Number of elements to copy.

Return Values

If the function succeeds, the return value is 0 (NoError).

If the function fails, the return value is a Graphisoft error code. For more information about error codes, see GSRoot error codes.

If arr is NULL or dptr Virtual Array is uninitialized, the return value is ErrNilHandle.

Remarks

The VAAddFromArr function expands the size of the Virtual Array in order to the newly added elements fit in the buffer of the Virtual Array.

Requirements

    Header: Declared in VA.hpp
    Import Library: Link with GSRootImp.lib
