TPRun

The TPRun function checks whether a timer is running or not.

bool TPRun(
  GSTimer               *pTimer       // pointer to timer
);

 

Parameters

pTimer
Pointer to the timer to check.

Return Values

If the specified timer is running, the return value is true.

If the specified timer is not running (stopped or not started yet), the return value is false.

Requirements

    Header: Declared in GSTime.hpp
    Import Library: Link with GSRootImp.lib
