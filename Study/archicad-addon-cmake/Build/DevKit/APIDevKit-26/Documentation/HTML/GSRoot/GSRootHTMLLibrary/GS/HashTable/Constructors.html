HashTable<IndexType, ItemType>::HashTable

HashTable constructor. 

HashTable (
    float               averageSeekFactorLimit = 0.0,
    float               minFillFactor = 0.0
);
HashTable (const HashTable & source);

Parameters

averageSeekFactorLimit
Specifies the average seek factor limit for the hashtable. 0.0 means the default average seek factor limit which is currently 2.0. Values less than 1.0 have no sense so they are also interpreted as the default value.
minFillFactor
Specifies the minimum fill factor for the hashtable. 0.0 means the default minimum fill factor which is currently 0.75. Values less than 0.01 are treated unsafe so they are also interpreted as the default value.
source

Description

Constructs a new hashtable with the specified average seek factor limit and minimum fill factor.
After the construction no memory is allocated. It will be allocated the first time when it is really needed.
