VAInit

The VAInit function initializes a Virtual Array.

GSErr VAInit(
  DESC             *dptr,       // pointer to virtual array
  long             npage,       // the number of elements by which the
                        // VA is increased or decreased
  long             size // the size of elements
);

 

Parameters

dptr
Pointer to the Virtual Array to initialize.
npage
Specifies the number of elements by which the size of the Virtual Array is modified. npage must be greater than zero. After VAInit the dptr Virtual Array has npage cells.
size
Specifies the size of elements stored in the dptr VA. It must be greater than zero.

Return Values

If the function succeeds, the return value is 0 (NoError).

It the function fails, the return value is a Graphisoft error code. For more information about error codes, see GSRoot error codes.

Remarks

The VAInit function allocates a buffer for the Virtual Array. In the allocated buffer there is room for npage elements. The VAInit initializes the members of the dptr DESC structure also. The initial value of maxind is npage and the initial value of lastind is 0.

The VAInit returns the error code of the buffer allocating operation.

Requirements

    Header: Declared in VA.hpp
    Import Library: Link with GSRootImp.lib
