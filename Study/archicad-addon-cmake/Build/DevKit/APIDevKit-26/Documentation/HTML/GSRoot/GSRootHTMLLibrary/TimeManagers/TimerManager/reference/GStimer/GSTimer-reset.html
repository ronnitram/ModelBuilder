GSTimer::reset

void reset( const char* name = NULL );

Parameters

name
Specifies the name of the timer.

Remarks

Resets the timer.

Note that the the length of the timer name is limited, so the passed name will be truncated if its length exceeds the limit. The current limit is 32 characters.
