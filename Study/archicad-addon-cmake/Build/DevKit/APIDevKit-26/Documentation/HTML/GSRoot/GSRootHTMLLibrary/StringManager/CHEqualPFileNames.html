CHEqualPFileNames

Test two Pascal file names/paths for equality.

    bool CHEqualPFileNames (
      const unsigned char*  p1PFileName,
      const unsigned char*  p2PFileName,
      GSCharCode            charCode = CC_System
    );

Parameters

p1PFileName
[in] The file name (Pascal string) to be compared.
p2PFileName
[in] The file name (Pascal string) to be compared with.
charCode
Specifies the character code used for the comparsion. See GSCharCode.

Return Values

true if the file names are equal, false otherwise.

Remarks

This function is a string comparsion, that is case sensitive on case sensitive file systems, and non case insensitive, on case insensitive file systems, such as Windows and MacOS.

Notice that if the return value is true, the two strings represents the same file, though not necessarily identical.

Note that if the two string represents paths, the function might return false value, though they represent the same file/directory. Eg. Comparing CHCompareFileNames("etc/../..","..") == false, while CHCompareFileNames("test.txt","Test.txt")==true if the file system is case insensitive.
