BNMoveMemory

The BNMoveMemory function moves a block of memory from one location to an other.

void BNMoveMemory(
  void            *destPtr,              // address of destination memory block
  const void      *srcPtr,               // address of source memory block
  GSize           len            // number of bytes to move
);

 

Parameters

destPtr
Specifies the address of the destination memory block.
srcPtr
Specifies the address of the source memory block.
len
Specifies the number of bytes to move.

Remarks

If the application is compiled as a debugging version, and the process does not have write access to destPtr (the BNValidWritePtr function returns false) or does not have read access to srcPtr, the function causes an assertion and breaks into the debugger.

The BNMoveMemory function wraps the memmove standard C Run-Time function.

Requirements

    Header: Declared in BM.hpp
    Import Library: Link with GSRootImp.lib
