StableArray::CompactionStrategy Enum

Possible compaction strategies. 

enum CompactionStrategy
{
    Manual,
    AtQuarter
};

Value Definitions

Manual
Compaction is performed only by explicit invocation of the Compact method.
AtQuarter
Compaction is performed automatically when fill factor (ratio of the size and capacity) decreases below 0.25 and there are at least two pages.

Description

This type descibes the possible compaction strategies of the StableArray. Compaction is an expensive operation which eliminates holes from pages and frees the unused memory. It changes addresses of stored items, so compaction should not be used if this is not desirable.
