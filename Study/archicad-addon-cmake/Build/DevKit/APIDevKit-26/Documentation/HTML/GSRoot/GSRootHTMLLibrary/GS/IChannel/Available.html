IChannel::Available

Tries to determine number of characters left on the input channel. 

virtual GSErrCode Available (USize * param1);

Parameters

param1

Return Value

Value Meaning
NoError* OK
Error* unspecified error or result can not be determined

* declared in GS namespace, but accessible in the global scope

Description

This method tries to determine number of characters left on the input channel. Because the input channel can be fully sequential or unlimited (e.g. pipes, keyboard etc.) this number can not be always determined. In such cases Error is returned.
