DBERROR_STR

The DBERROR_STR macro calls the DBBreak function with predefined parameters if the condition expression is evaluated to true.

#define	DBERROR_STR(b, s)	((b) ? DBBreak (__FILE__, __LINE__, s) : False)

 

Parameters

b
Specifies a boolean expression (including pointer values).
s
Specifies the message. This parameter is optional, i. e. NULL is allowed.

Return Values

If the b is false, the return value is false.

If the b is true, the return value is the value returned by DBBreak.
