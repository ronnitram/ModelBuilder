GSTimer Class

The GSTimer class represents a programmable timer.

Class Members

 

Construction

GSTimer Constructs a GSTimer object.

 

Initialization

reset Initializes a GSTimer object and resets the timer.

 

Timer Functions

start Starts the timer.
stop Stops the timer.
run Checks whether the timer is running or not.
name Retrieves the name of the timer.
value Retrieves the value of the timer. The value of the timer is the elapsed time in seconds since the timer was started first.

 

Debug Support Functions

print Sends a debug string containing information from the value of the timer.
