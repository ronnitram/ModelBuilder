TPReset

The TPReset function creates a timer with the specified name or reset an existing one.

void TPStart(
  GSTimer               *pTimer,      // timer to reset
  const char    *name              // timer name
);

 

Parameters

pTimer
Pointer to a timer to reset.
name
Specifies the name of the timer.

Remarks

Note that the the length of the timer name is limited, so the name passed as a parameter will be truncated, if its length exceeds the limit. The current limit is 32 character.

If you don't start the timer at creation, you can start it later by calling the TPStart function.

Requirements

    Header: Declared in GSTime.hpp
    Import Library: Link with GSRootImp.lib
