Library parts

If you supply library parts with your add-on, please use the Save as Archive command in Archicad to convert between Mac and Windows (not just simply drag it over to the other platform); in this case the preview pictures and pictures used in user interface scripts will be correct. Also, if you use fonts in the scripts, set the default font both on Windows (e.g. Arial) and on the Mac (e.g. Helvetica); they look much nicer.
