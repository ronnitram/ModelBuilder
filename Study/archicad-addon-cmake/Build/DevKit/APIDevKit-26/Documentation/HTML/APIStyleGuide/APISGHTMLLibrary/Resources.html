Resources

The main axiom: use the GRC format! Especially if you would like to use the fundctionality of the Dialog Manager; it would be very hard to assemble manually the extra resources required by DG.

The resource IDs should start from 32500 (except palettes and modeless dialogs, where they should start from 32400). Don't use resource IDs higher than 32767 (SHORT_MAX). This is also true for bitmaps and icons: their resource IDs should also be taken from the range [32500..32767].

The dialogs/palettes should also have accompanying help resources (even if empty), otherwise some other add-on might have a help resource with the ID of your dialog, and you'll see suprising messages.
