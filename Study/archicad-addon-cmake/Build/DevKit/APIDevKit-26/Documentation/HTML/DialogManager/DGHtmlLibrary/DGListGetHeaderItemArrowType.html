DGListGetHeaderItemArrowType

Retrieves the arrow type of a header item of a list box item.

    short DGListGetHeaderItemArrowType (
      short        dialId,
      short        item,
      short        headerItem
   );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
headerItem
1-based index of the header item.

Return Values

The arrow type of the header item:

Value Arrow type
DG_LHA_NONE No symbol.
DG_LHA_UP Simple triangle pointing upwards.
DG_LHA_DOWN Simple triangle pointing downwards.
DG_LHA_DOUBLEUP Double triangle pointing upwards.
DG_LHA_DOUBLEDOWN Double triangle pointing downwards.

Remarks

The list box header is displayed at the top of the list box. The header can be divided into more header items. The header is used to display information about the list box columns and can be configured to enable the user to dynamically adjust the size and the order of the list box columns. The number of the header items can be retrieved by DGListGetHeaderItemCount. The arrow type of a header item can be set by DGListSetHeaderItemArrowType.

It is possible to indicate the sorting order of a list box coloumn in a header item. Primary and secondary sorting order are shown with simple and double triangles, pointing up or downwards. To show or hide the sorting arrow the DGListSetHeaderItemArrowType function is used. For using the sorting order the header item need to be used as a pushbutton. This means that the isPushable header flag need to be set by the DGListSetHeaderFlags function.
