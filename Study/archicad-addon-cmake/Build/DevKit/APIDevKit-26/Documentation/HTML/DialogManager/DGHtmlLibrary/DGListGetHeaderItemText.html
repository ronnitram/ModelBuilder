DGListGetHeaderItemText

Retrieves the text of a header item of a list box item.

    GS::UniString DGListGetHeaderItemText (
      short        dialId,
      short        item,
      short        headerItem
   );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
headerItem
1-based index of the header item.

Return Values

The header item text in Unicode format.

Remarks

The list box header is displayed at the top of the list box. The header can be divided into more header items. The header is used to display information about the list box columns and can be configured to enable the user to dynamically adjust the size and the order of the list box columns. The number of the header items can be retrieved by DGListGetHeaderItemCount. The content of a header item can be set by DGListSetHeaderItemText.

The header can be configured so that the number and size of the header items are either synchronized with the corresponding tabulator fields by DG or independently handled by the application. The header flags can be set and retrieved by the DGListSetHeaderFlags and DGListGetHeaderFlags functions respectively.

A header item can contain an icon and/or text also, but only one icon can appear in a header item. In case of using an icon with text in a header item, the icon can be set and retrieved by the DGListSetHeaderItemIcon and DGListGetHeaderItemIcon functions respectively. If a header item contains an icon and text also, the icon will be positioned either at the beginning or at the end of the text according to the justification. (The DG_IS_LEFT justification is applied for the header item containing an icon and text also, even if the DG_IS_CENTER justification was given.) For a multi-line text the truncation cannot be applied. The font style and size of each header item can be set by the DGListSetHeaderItemFont function.
