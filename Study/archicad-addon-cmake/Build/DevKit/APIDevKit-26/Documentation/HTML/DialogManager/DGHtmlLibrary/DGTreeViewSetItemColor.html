DGTreeViewSetItemColor

Sets the foreground (text) color of a tree view item.

    void DGTreeViewSetItemColor (
      short   dialId,
      short   item,
      Int32   treeItem,
      UShort  red,
      UShort  green,
      UShort  blue
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
treeItem
ID of the tree view item. Use DG_ALL_ITEMS to change the color of all items.
 
red
Red component of the color (0-65535).
 
green
Green component of the color (0-65535).
 
blue
Blue component of the color (0-65535).

Remarks

Before the item color is set by DGTreeViewSetItemColor, DG uses the system default foreground (text) color.

After the item color is set at least once (no matter whether it is set to the system default), DGTreeViewGetItemColor function returns 1.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
