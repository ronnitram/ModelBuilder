DGDisplayContextHelp

With this function you can force DG to display context sensitive help information.

    bool DGDisplayContextHelp (
      void
    );

Return Values

Value Meaning
true DG was able to find the context sensitive help anchor, and launched the registered help engine.
false DG was unable to find the context sensitive information, or the focused dialog didn't have registered help engine, or some other problems occured. Due to the problems, no action was taken.

Remarks

The function works in the following way. First, it determines which is the focused dialog. If there's no focused dialog, then it tries to find the first visible modeless dialog. If DG cannot find any dialog in this way, it won't do any action and returns false.

If any dialog is found, DG gets the creator's MDID from the dialog data. From the MDID, it can determine the registered help engine path. If no help engine registered for the MDID, no action is taken and the function returns false.If no dialog item is focused, the help engine is launched only with the dialog help anchor. If there is a focused item on the dialog, the help engine is launched with the combined dialog and item help anchor parameter. Both case the function returns true.

Every dialog item (on a Palette, on a Modeless or Modal Dialog, or on a TabPage) can have two types of help information: a short one and an anchor to the detailed information in the separate help engine.

Item tooltips and help anchors are defined in DLGH resources for every type of dialogs (GRC Specification of the Dialog Helps).

On Macintosh, help information is displayed in a balloon help if this feature is enabled through the Macintosh system Help menu. On Windows, tooltips serve to show the short item help. The detailed help information of the item can be invoked by setting the keyboard focus to a focusable dialog item then pressing the [F1] key on Windows or [Help] key on Macintosh; or by clicking the right mouse button on the item; or by clicking the help [?] button in the dialog caption then clicking on the item itself. If tooltips are enabled, the short help information is displayed as usual.

Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
