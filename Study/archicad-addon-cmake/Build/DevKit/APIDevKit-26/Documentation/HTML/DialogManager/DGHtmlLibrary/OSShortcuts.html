Operating System Shortcuts

On Macintosh:

Shortcut Description
COMMAND + SHIFT + number For example: COMMAND + SHIFT + 3 captures the screen.
COMMAND + SPACE Switches script systems.
COMMAND + OPTION + SPACE Switches keyboards inside script systems.
OPT + COMMAND + ESC Force to quit the current application.

On Windows:

Shortcut Description
ALT + ESC Switches to the next application.
ALT + TAB Switches to the next application.
SHIFT + ALT + TAB Switches to the previous application. The user must press and hold down ALT + SHIFT while pressing TAB.
ALT + F4 Closes an application or a window.
ALT + HYPHEN Opens the System menu for a document window.
ALT + SPACE Opens the System menu for an application window.
CTRL + ESC Switches to the Windows Task List.
F1 Starts Help if the application has Help.
PRINT SCREEN Copies an image on the screen into the clipboard.
ALT + PRINT SCREEN Copies the image of the active window into the clipboard.
