DGIsModelessDialogVisible

Retrieves the visibility of a modeless or palette dialog.

    short DGIsModelessDialogVisible (
      short  dialId
    );

Parameters

dialId
ID of the dialog.

Return Values

Value Meaning
1 The specified dialog is visible.
0 The specified dialog is invisible.

Remarks

When a modeless or palette dialog is created by the DGModelessInit function it can be visible or not, according to the value of the show parameter of the function. If this parameter is zero, DGModelessInit does not show the dialog before it returns. Later you can show or hide the dialog by using the DGShowModelessDialog or DGHideModelessDialog functions, respectively.
