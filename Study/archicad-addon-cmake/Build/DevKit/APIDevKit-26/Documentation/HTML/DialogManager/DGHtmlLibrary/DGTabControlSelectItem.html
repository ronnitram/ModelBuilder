DGTabControlSelectItem

Selects a tab control item.

    void DGTabControlSelectItem (
      short  dialId,
      short  item,
      short  tabItem
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
tabItem
1-based index of the tab control item.

Remarks

A disabled tab page cannot be selected by user action in case of a normal tab control (DG_TT_NORMALTAB). However, from program code even disabled tab pages can be activated by DGTabControlSelectItem.

Simple tab controls (DG_TT_SIMPLETAB) are controlled by other dialog items therefore it is the application's responsibility not to let activate a disabled tab page.
