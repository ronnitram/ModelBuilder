DGTreeViewGetTopMostParent

Retrieves the ID of the topmost parent item of a tree view item.

    Int32 DGTreeViewGetTopMostParent (
      short  dialId,
      short  item,
      Int32  treeItem
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
treeItem
ID of the tree view item.

Return Values

The ID of the topmost parent (i.e. the root item() of the specified tree view item.

Remarks

To enumerate all items on a hierarchy level use DG_TVG_NEXT and DG_TVG_PREVIOUS. To go through hierarchy levels use DG_TVG_CHILD and DG_TVG_PARENT. To enumerate all the listed items use DG_TVG_NEXTLISTED and DG_TVG_PREVLISTED. To enumerate all items of a subtree write a recursive algorithm based on DG_TVG_CHILD and DG_TVG_NEXT.

The only way to determine whether an item is a parent is calling DGTreeViewGetItem with the DG_TVG_CHILD flag. If the return value is different from DG_TVI_NONE the tree item has at least one child.

Requirements

Header: DG.h

Module: DGLib [MAC], DG.dll [WIN]
