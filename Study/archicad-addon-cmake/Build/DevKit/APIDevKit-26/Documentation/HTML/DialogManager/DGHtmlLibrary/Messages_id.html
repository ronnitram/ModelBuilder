DG Messages

Messages Description
DG_MSG_ACTIVATE This message is obsolete, use DG_MSG_TOPSTATUSCHANGE message insted. A modeless dialog is activated or deactivated or a palette dialog is activated.
DG_MSG_CHANGE This message has different meaning for different type of dialog.
DG_MSG_CLICK A dialog item is clicked.
DG_MSG_CLOSE The dialog is about to be closed.
DG_MSG_CONTEXTMENU Context menu was called on the dialog or on a specified dialog item.
DG_MSG_DOUBLECLICK A dialog item is double-clicked.
DG_MSG_DRAGDROP A drag and drop event occurs on a dialog or on a dialog item.
DG_MSG_FILTERCHAR A key is pressed while an edit control has the keyboard focus.
DG_MSG_FOCUS A focusable dialog item gains or loses the keyboard focus.
DG_MSG_GROW This message is obsolete, use DG_MSG_RESIZE message insted. The dialog has been resized.
DG_MSG_HOTKEY A registered hotkey is pressed.
DG_MSG_INIT The dialog and its items have been created and the items need to be initialized.
DG_MSG_ITEMHELP Tool tip or context sensitive help message should be displayed by DG.
DG_MSG_LISTHEADERCLICK A header item of a list box item was clicked.
DG_MSG_LISTHEADERDRAG A drag and drop event occurs on a header item of a list box item.
DG_MSG_LISTHEADERRESIZE A header item of a list box item is being resized.
DG_MSG_LISTHEADERBUTTONCLICK The header button of a list box item was clicked.
DG_MSG_MOUSEMOVE The mouse moves over a list box or a user item.
DG_MSG_NULL Modal and modeless dialogs can receive this message if there is no other higher priority message for them.
DG_MSG_RESIZE The dialog is being resized.
DG_MSG_SPLITTERDRAG A drag event occurs on a splitter control.
DG_MSG_TRACK he position of the scroll box of a scroll bar, the position of the slider of a slider bar or the value of a spin control or a user control is changed during tracking.
DG_MSG_TOPSTATUSCHANGE A modeless dialog is activated or deactivated or a palette dialog is activated.
DG_MSG_UPDATE A user item, a tabulator field of a list box or the image of a list view should be redrawn.
DG_MSG_USER An application-defined, own message is sent to the dialog callback function.
DG_MSG_WHEELTRACK Mouse wheel event occured.
