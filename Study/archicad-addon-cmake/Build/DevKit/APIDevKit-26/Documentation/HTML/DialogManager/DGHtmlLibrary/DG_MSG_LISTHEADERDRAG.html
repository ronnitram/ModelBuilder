DG_MSG_LISTHEADERDRAG

A drag and drop event occurs on a header item of a list box item.

Message parameters

dialId
ID of the dialog or tab page.
 
item
1-based dialog item index of the list box control in which a header item is dragged.
 
msgData
Pointer to a DGListHeaderMsgData structure cast to DGMessageData.

Return Values

Set to 1 if the application handled the event otherwise set to zero.

Remarks

The header of a list box can be configured having dragable items. In case of the items are dragable and the user clicks and tracks a header item with the mouse, DG sends DG_MSG_LISTHEADERDRAG message to the application. The index of the dragged item is specified in the DGListHeaderMsgData structure.

To set or reset the dragable property of the header of a list box use the DGListSetHeaderFlags function.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
