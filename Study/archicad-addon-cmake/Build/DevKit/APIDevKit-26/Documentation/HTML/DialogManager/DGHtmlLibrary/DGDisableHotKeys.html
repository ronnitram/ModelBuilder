DGDisableHotKeys

Disables hotkeys for a given dialog.

    void DGDisableHotKeys (
      short  dialId
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog. DG_ALL_DIALOGS disables hotkeys for all dialogs.

Remarks

A hotkey is a keystroke or combination of keystrokes that generates a DG_MSG_HOTKEY message for a dialog. If the hotkeys of a dialog are disabled DG does not send hotkey messages to the dialog callback function.

When hotkeys are registered by calling the DGRegisterHotKey function, they are automatically enabled. The status of all hotkeys belonging to a dialog can be set to disabled by DGDisableHotKeys. Use DGEnableHotKeys to enable these hotkeys again.
