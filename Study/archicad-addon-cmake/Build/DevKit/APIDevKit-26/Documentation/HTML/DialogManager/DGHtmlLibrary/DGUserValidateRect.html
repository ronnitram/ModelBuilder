DGUserValidateRect

Validates a rectangular area of a user item. It is removed in version DGLib 2.2.0.

    void DGUserValidateRect (
      short  dialId,
      short  item,
      short  left,
      short  top,
      short  right,
      short  bottom
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
left
Horizontal position of the upper left corner of the validated rectangular area, in pixels.
 
top
Vertical position of the upper left corner of the validated rectangular area, in pixels.
 
right
Horizontal position of the lower right corner of the validated rectangular area, in pixels.
 
bottom
Vertical position of the lower right corner of the validated rectangular area, in pixels.

Remarks

When the user item is to be redrawn only the invalidated parts are updated. This function validates the specified rectangular part and excludes it from the updated area.

A non-rectangular area of a user item can be validated by the DGUserValidateRgn function.

To invalidate a rectangular or a non-rectangular area use either DGUserInvalidateRect or DGUserInvalidateRgn.

To update the invalid area of a user item immediately use the DGUserUpdateItem function. After updating a user item its entire rectangle is validated. Regardless of the invalidated area the entire user item can be redrawn by the DGUserRedrawItem function.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
