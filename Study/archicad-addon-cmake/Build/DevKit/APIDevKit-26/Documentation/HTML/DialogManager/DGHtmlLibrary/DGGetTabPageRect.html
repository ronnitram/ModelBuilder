DGGetTabPageRect

Retrieves the rectangle of the tab page.

    void DGGetTabPageRect (
      short   dialId,
      short*  left,
      short*  top,
      short*  right,
      short*  bottom
    );

Parameters

dialId
Dialog identifier of the tab page.
 
left
Output parameter: the left value of the tab page rect in pixels.
 
top
Output parameter: the top value of the tab page rect in pixels.
 
right
Output parameter: the right value of the tab page rect in pixels.
 
bottom
Output parameter: the bottom value of the tab page rect in pixels.

Remarks

The retrieved coordinates are relative to the main dialog coordinate system.

Use the DGSetTabPageSize function to modify the size of the tab page.

If any of the pointer parameters is nullptr, that value is not returned.

Requirements

Version: DG 2.0.0 or later
Header: DG.h, DGDefs.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
