DGResAlert

Invokes a standard DG alert given in a GALR resource.

    short DGResAlert (
      GSResModule  resModule,
      short        alertId
    );

Parameters

resModule
The resource module the GALR resource is in.
alertId
GALR resource ID of the alert.

Return Values

Value Meaning
DG_OK The first button is clicked.
DG_CANCEL The second button is clicked.
DG_THIRD The third button is clicked.

If the function parameters are incorrect or the specified resource is not found, zero is returned.

Remarks

DGResAlert has the same behavior as of DGAlert. The only difference is in the way how the icon and the alert and button texts are specified.

Alerts are sized automatically to the minimal size that contains all of the given elements. If the first and/or the second alert text does not fit into one line, DG automatically breaks it/them into more lines. Of course, these texts can contain explicit line breaks as well.

On Windows, if the user clicks the close box in the caption, DGResAlert returns DG_CANCEL. If there is no second button in the alert DG_OK is returned.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
