Dialog::GetClientPosition

Retrieves the client area position. 

Point GetClientPosition (void) const;

Return Value

The position of the upper left corner of the client area. The coordinates are retrieved in the following coordinate system depending on the type of the dialog:

Coordinate system Dialog type
Global Modeless
Screen Modal
Screen Palette

Description

On Macintosh, global and screen coordinates are identical. On Windows global coordinates are relative to the application's client window, screen coordinates are relative to the screen's top left corner.
