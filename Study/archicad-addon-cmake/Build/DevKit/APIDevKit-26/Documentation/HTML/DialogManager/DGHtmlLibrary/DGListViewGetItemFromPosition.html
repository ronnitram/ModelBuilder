DGListViewGetItemFromPosition

Retrieves the ID of a list view item according to the specified mouse position.

    short DGListViewGetItemFromPosition (
      short  dialId,
      short  item,
      short  xPos,
      short  yPos
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
xPos
Horizontal mouse position in screen coordinates.
 
yPos
Vertical mouse position in screen coordinates.

Return Values

1-based index of the list item or 0 if there is no such an item.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
