DGGetDialogPosition

Retrieves the position of a dialog.

    void DGGetDialogPosition (
      short   dialId,
      short   rectType,
      short   posType,
      short*  hPos,
      short*  vPos
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
rectType
Value Meaning
DG_CLIENT hPos and vPos refer to the client area of the dialog.
DG_FRAME hPos and vPos refer to the window area of the dialog (including frame and title).
DG_RESTCLIENT hPos and vPos refer to the client area of the dialog, when the dialog window is restored.
DG_RESTFRAME hPos and vPos refer to the window area of the dialog (including frame and title), when the dialog window is restored.
 
posType
Value Meaning
DG_GLOBAL hPos and vPos are given in global coordinates.
DG_SCREEN hPos and vPos are given in screen coordinates.
 
hPos
Output parameter: Horizontal position of the upper left corner, in pixels. This parameter can be nullptr.
 
vPos
Output parameter: Vertical position of the upper left corner, in pixels. This parameter can be nullptr.

Remarks

On Macintosh, global and screen coordinates are identical. On Windows global coordinates are relative to the application's client window, screen coordinates are relative to the screen's top left corner.

If either of the position parameter pointers are nullptr, that coordinate is not returned.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
