DGUnregisterHotKey

Unregisters a hotkey of a dialog.

    void DGUnregisterHotKey (
      short  dialId,
      short  hotKeyId
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
hotKeyId
Hotkey identifier returned by the DGRegisterHotKey function.

Remarks

A hotkey is a keystroke or combination of keystrokes that generates a DG_MSG_HOTKEY message for a dialog. Hotkeys in DG are similar to menu accelerators and to the Operating System shortcuts. They also provide faster, more direct access to frequently used commands than menus and toolbars do. Moreover, they have some other special features (see the description of the DG_MSG_HOTKEY message).

Hotkeys can be registered to any type of dialogs by calling the DGRegisterHotKey function. If a hotkey is successfully registered, later it has to be referred by the short identifier that DGRegisterHotKey returns. To unregister a previously registered hotkey use the DGUnregisterHotKey function.
