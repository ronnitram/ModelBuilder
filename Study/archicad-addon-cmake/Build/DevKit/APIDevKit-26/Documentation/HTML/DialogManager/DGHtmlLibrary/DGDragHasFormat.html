DGDragHasFormat

Decides, whether the specified drag format is enabled in the current drag and drop operation. A drag format is enabled, if any data of this format is set up, with DGDragSetData or with DGDragSetDelayedData.

    bool DGDragHasFormat (
      DGDragFormat  format
    );

Parameters

format
Int32 value, usually used in it's character representation ('FILE', 'TEXT'). Used to identify the format of the data being drag dropped.
 

Return Values

true if the given drag format has been found in the current drag and drop operation, false otherwise.

Remarks

Use this function to decide whether the drag is accepted or not by a given target. Calling this function don't have any performance implications.

Requirements

Header: DG.h
