DGRichEditChangeData

If the dialog callback function receives a DG_MSG_CHANGE message for a rich edit control dialog item, the message data parameter is a pointer to a DGRichEditChangeData structure that contains information about the user action causing the change.

    struct DGRichEditChangeData {
      short            changeFlag;
      short            filler;
      Int32            oldLength;
      Int32            newLength;
      Int32            oldMinPos;
      Int32            oldMaxPos;
      Int32            newMinPos;
      Int32            newMaxPos;
      DGUniChangeData  uniRich;
    };

Members

changeFlag
Determines what happened to the rich edit control. Possible values are:
changeFlag Meaning
DG_RECF_SELECTION Selection changed. Only the selection related members are filled in the structure.
DG_RECF_CONTENT Content changed. All the structure members (except uniRich) are filled with values.
DG_RECF_RESIZEREQUEST Resizing will happen in an UniRichEdit item. The reqLeft, reqTop, reqWidth, reqHeight structure members of uniRich are filled with the recommended/requested values; all the other fields are 0.
DG_RECF_SCROLLBARPOS UniRichEdit item's content was scrolled. The oldHScrollPos, hScrollPos, oldVScrollPos, vScrollPos structure members are filled with values; all the other fields are 0.
 
filler
Reserved.
 
oldLength
The length of the contained text before the change in unicode characters.
 
newLength
The length of the contained text after the change in unicode characters.
 
oldMinPos
The beginning position of the selection before the change.
 
oldMaxPos
The end position of the selection before the change.
 
newMinPos
The beginning position of the selection after the change.
 
newMaxPos
The end position of the selection after the change.
 
uniRich
Contains information on changes in UniRichEdit items.

Requirements
