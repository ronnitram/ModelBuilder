Components of the Dialog Manager

The Graphisoft's Dialog Manager (DG) package consists of the following components:

On Macintosh:

  • Header and other include files, such as DG.h, DGDefs.h, DGMac.h and DGMacResType.r. If you want to use DG in your application, you have to include the DG.h header in your source files and the DGMacResType.r file in your GRC resource files.
  • A shared library called DGLib. You also have to link the same DGLib to your application.

On Windows:

  • Header files, such as DG.h, DGDefs.h, DGWin.h and DGWinDefs.h. If you want to use DG in your application, you have to include the DG.h header in your source files while the DGDefs.h and DGWinDefs.h headers in your GRC resource files.
  • A dynamic link library called DG.dll. You have to link your application with the import library called DG.lib.
