DGGetItemImageId

Retrieves the resource ID of the image of a dialog item.

    short DGGetItemImageId (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.

Return Values

The resource ID of the image depending on the type of the item. DG_NO_IMAGE is returned if the image of the specified dialog item was set by DGSetItemImage, or if the function parameters are incorrect.

Item type  Macintosh only:   Windows only: 
DG_ITM_BUTTON Resource ID of the icon. Resource ID of the icon.
DG_ITM_CHECKBOX Resource ID of the icon. Resource ID of the icon.
DG_ITM_RADIOBUTTON Resource ID of the icon. Resource ID of the icon.
DG_ITM_ICONITEM Resource ID of the icon. Resource ID of the icon.
DG_ITM_PICTITEM Resource ID of the picture. Resource ID of the bitmap.

Remarks

For specifying an image to a dialog item with the resource ID of the image: see DGSetItemImageId, DGSetItemImageResource. The image of a dialog item can also be set by its image handle with the DGSetItemImage function.

For icon-style dialog items the returned value can be one of the following predefined constants:
