DGListViewGetImage

Retrieves the image associated with a list view control.

    const void* DGListViewGetImage (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 

Return Values

The image data of the specified list view control (cast to void*) depending on the type of the image. nullptr is returned if the image of the list view control was set by DGListViewSetImageId or DGListViewSetImageResource, if the type of the image is DG_LVIT_IMAGEINDEX or if the function parameters are incorrect.

The image data depending on the type of the image:
imageType imageData
DG_LVIT_NOIMAGE nullptr - no image is defined for this item.
DG_LVIT_ICON
 Macintosh only: 
Icon, referenced with NSImage* cast to void*.
 Windows only: 
Icon, referenced with HICON cast to void*.
DG_LVIT_PICTURE
 Macintosh only: 
Picture, referenced with NSImage* cast to void*.
 Windows only: 
Picture, referenced with HBITMAP cast to void*.
 Windows only:
DG_LVIT_DIBITMAP Device Independent Bitmap, referenced with BITMAPINFO* cast to void*.

Remarks

There are two ways to define a list view item image:

  • Assign images to the items individually by DGListViewSetItemImage, DGListViewSetItemImageId or DGListViewSetItemImageResource. These images can be of different types and sizes for each list view item.
  • Associate one image with the list view itself by DGListViewSetImage, DGListViewSetImageId or DGListViewSetImageResource, divide it into smaller sub-images of the same size by giving the number of rows and the total number of sub-images. Assign a sub-image to the list view item by DGListViewSetItemImageIndex. The image index parameter of the function is the simple 1-based row-by-row index of a sub-image. In this case the type of the item's image will be DG_LVIT_IMAGEINDEX. The image data, image ID, or image resource module associated with the list view control can be retrieved DGListViewGetImage or by DGListViewGetImageId, DGListViewGetImageResource, respectively.

DGListViewGetImage retrieves the image associated with the list view control by DGListViewSetImage. This image stores more sub-images that form a rectangular array and all the list view items must have DG_LVIT_IMAGEINDEX image type.

The type of the image can be retrieved by DGListViewGetImageType.
