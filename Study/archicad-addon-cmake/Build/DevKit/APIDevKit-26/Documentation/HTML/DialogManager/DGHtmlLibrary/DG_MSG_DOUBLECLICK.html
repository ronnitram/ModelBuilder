Dialog ManagerVersion: 2.0

DG_MSG_DOUBLECLICK

A dialog item is double-clicked.

Message parameters

dialId
ID of the dialog or tab page.
 
item
1-based index of the dialog item.
 
msgData
Item type msgData
Check Box Not used, always zero.
Icon Not used, always zero.
List Box Pointer to a DGListMsgData structure cast to DGMessageData.
List View 1-based index of the double-clicked list view item.
Picture Not used, always zero.
Push Menu Control Not used, always zero.
Radio Button Not used, always zero.
Static Text Not used, always zero.
User Control Not used, always zero.
User Item Pointer to a DGUserMsgData structure cast to DGMessageData.

Return Values

Not used, should be set to zero.

Remarks

Note that only button-like check boxes and radio buttons can receive DG_MSG_DOUBLECLICK messages.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
