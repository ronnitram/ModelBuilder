DGStringToDouble

Converts a formatted string into a double value according to the current unit settings.

    Int32 DGStringToDouble (
      const GS::UniString&       str,
      double&                    value,
      short                      type
    );

Parameters

str
The formatted string which holds the value to convert.
 
value
Reference to the converted double value.
 
type
Type of double value. Possible values of type (see subtypes of edit controls): DG_ET_REAL, DG_ET_ANGLE, DG_ET_POLARANGLE, DG_ET_LENGTH, DG_ET_AREA, DG_ET_VOLUME, DG_ET_MM_OR_PT.

Return Values

If the function succeeds, 1 is returned.
If the function parameters are incorrect, zero is returned.

Remarks

This function converts a formatted string to a double value depending on the units currently set in DG. To use different unit settings for the conversion, call DGStringToDoubleUnit. For the description of units in DG see DGUnitData.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
