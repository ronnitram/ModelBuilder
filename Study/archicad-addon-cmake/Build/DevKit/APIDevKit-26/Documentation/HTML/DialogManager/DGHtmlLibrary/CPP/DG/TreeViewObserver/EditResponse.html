TreeViewObserver::EditResponse Enum

The response after finishing the editing of a tree view item text. 

enum EditResponse
{
    SetNewLabel =  DG_TVCR_SETNEWTEXT,
    KeepOldLabel =  DG_TVCR_KEEPOLDTEXT,
    EditLabel =  DG_TVCR_EDITTEXT
};

Value Definitions

SetNewLabel
The new text is accepted.
KeepOldLabel
The original text will be kept.
EditLabel
Editing will continue.
