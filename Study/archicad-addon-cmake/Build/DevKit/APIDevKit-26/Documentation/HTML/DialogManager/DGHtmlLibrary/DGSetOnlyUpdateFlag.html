DGSetOnlyUpdateFlag

Enable or disable receiving only update messages for modeless dialogs.

    void DGSetOnlyUpdateFlag (
      bool  onlyUpdate
    );

Parameters

onlyUpdate
Value Meaning
true Only update messages will be delivered to modeless and palette dialogs that have normal (DG_DS_NORMAL) status.
false All dialog related messages are handled.

Remarks

There is a global flag in DG called onlyUpdate. This function sets the value of this flag. If onlyUpdate is true, the modeless and palette dialogs, which have the status DG_DS_NORMAL, will receive only the update messages. If this flag is false, all messages will be received by these dialogs.

Every time, when DGModelessHandler is called, it sets this internal DG flag to the value if its onlyUpdate parameter. So in most cases it is enough to call DGModelessHandler in the event-loop with the required onlyUpdate value. However rarely this flag may be needed to set before or instead of calling DGModelessHandler.

Examples, when you are suggested to use the DGSetOnlyUpdateFlag function:

  • You use a special, own message loop that does not call DGModelessHandler, but you need to disable your modeless and palette dialogs with DG_DS_NORMAL state for a while.
  • You need to disable your modeless and palette dialogs with DG_DS_NORMAL state, but you are going to call DGModelessHandler later.

The usage of this DG flag can be explained by the following example. Suppose that an application begins a long process such as a time-consuming calculation. In this case, it is advisable to show a process indicator dialog with a Stop button until the calculation is finished. When handling system-specific messages during this period, DGModelessHandler should be called with onlyUpdate of value true. This ensures that modeless dialogs of DG_DS_NORMAL state do not handle any messages except the update messages (this way they do not respond to any user actions). The status of the process indicator dialog should be set to DG_DS_ENABLED in order to enable the user to press the Stop button and cancel the process.
