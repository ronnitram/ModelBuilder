DGUserUpdateData

If the dialog callback function receives a DG_MSG_UPDATE message for a user item, the msgData parameter is a pointer to a DGUserUpdateData structure that contains information about the update event.

    struct DGUserUpdateData {
      void*  drawContext;
      void*  updateRgn;
      short  updateType;
      short  left;
      short  top;
      short  right;
      short  bottom;
      short  filler;
    };

Members

drawContext
The drawing context of the image field. Cast this value to CGContextRef on Macintosh or HDC on Windows. The origin of the drawing context is set to the upper left corner of the user item.
 
updateRgn
The update region. Cast this value to HRGN on Windows and on Macintosh it is always nullptr.
 
updateType
The type of the update message. This value can be one of the following constants:
Value Meaning
DG_UUT_NORMAL Normal update
DG_UUT_PARTIAL Partial update
DG_UUT_INUPDATE A previous update is still processing
 
left
Horizontal position of the upper left corner of the item client rectangle in pixels.
 
top
Vertical position of the upper left corner of the item client rectangle in pixels.
 
right
Horizontal position of the lower right corner of the item client rectangle in pixels.
 
bottom
Vertical position of the lower right corner of the item client rectangle in pixels.
 
filler
Reserved.

Remarks

On Macintosh the updateRgn member of the structure is always nullptr and the drawContext member to CGContextRef, since the application can draw just on the CGContextRef. On Windows, updateRgn should be cast to HRGN* and the drawContext to HDC*, since the application can draw just on the hDC. The origin of the drawing context is set to the upper left corner of the user item.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
