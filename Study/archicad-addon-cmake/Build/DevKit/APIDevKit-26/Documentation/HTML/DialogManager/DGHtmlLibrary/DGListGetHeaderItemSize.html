DGListGetHeaderItemSize

Retrieves the size of a header item of a list box item.

    short DGListGetHeaderItemSize (
      short        dialId,
      short        item,
      short        headerItem
   );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
headerItem
1-based index of the header item.

Return Values

The size of the header item or zero if the function parameters are incorrect.

Remarks

The list box header is displayed at the top of the list box. The header can be divided into more header items. The header is used to display information about the list box columns and can be configured to enable the user to dynamically adjust the size and the order of the list box columns. The number of the header items can be retrieved by DGListGetHeaderItemCount. The size of a header item can be set by DGListSetHeaderItemSize.

The header can be configured so that the number and size of the header items are either synchronized with the corresponding tabulator fields by DG or independently handled by the application. The header flags can be set and retrieved by the DGListSetHeaderFlags and DGListGetHeaderFlags functions respectively.
If the isSynchron flag is set on the list box header, the DGListSetHeaderItemSize function does not affect the header item properties.

Header: DG.h

Module: DGLib [MAC], DG.dll [WIN]
