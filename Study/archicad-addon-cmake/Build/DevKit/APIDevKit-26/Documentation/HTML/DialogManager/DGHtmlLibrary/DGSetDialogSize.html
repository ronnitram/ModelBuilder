DGSetDialogSize

Sets the width and height of a dialog.

    void DGSetDialogSize (
      short  dialId,
      short  rectType,
      short  hSize,
      short  vSize,
      short  fixPoint,
      bool   keepOld
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
rectType
Value Meaning
DG_CLIENT hSize and vSize refer to the client area of the dialog.
DG_FRAME hSize and vSize refer to the window area of the dialog (including frame and title).
 
hSize
Horizontal size in pixels.
 
vSize
Vertical size in pixels.
 
fixPoint
This parameter determines the reference point that does not move during resizing, and defines whether the old content of the dialog is to be kept at its original place.
Value Meaning
DG_TOPLEFT Upper left corner of the dialog.
DG_BOTTOMLEFT Lower left corner of the dialog.
DG_TOPRIGHT Upper right corner of the dialog.
DG_BOTTOMRIGHT Lower right corner of the dialog.
DG_CENTER Center of the dialog.
 
keepOld
If true, keeps the old content of the dialog.

Remarks

DGSetDialogSize sets the dialog width and height to absolute values. Use the DGGrowDialog function to resize the dialog with given relative values.

A dialog cannot be shrunk to arbitrarily small only until it reaches its minimal sizes given in GRC or re-set by DGSetDialogMinSize.

A dialog can be resized only in the directions allowed by the dialog grow type flag, given in GRC or specified by DGSetDialogGrowType.

During the DGSetDialogSize call DG_MSG_RESIZE and DG_MSG_GROW messages are sent to the dialog callback function. When handling these messages, the application can move and/or resize dialog items according to the new size of the dialog. When the callback function receives the DG_MSG_GROW message, the dialog already has its new size.

Dialog positions and sizes in DG are stored in system preferences (Macintosh) or in the system registry (Windows). This means that a dialog remembers its previous size and position when it is reopened next time, even after exiting and restarting the application.

Current size of a dialog can be retrieved by DGGetDialogSize.
