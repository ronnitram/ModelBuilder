DGAppendDialogItem

Appends a dialog item to the end of the item list of a dialog.

    short DGAppendDialogItem (
      short  dialId,
      short  itemType,
      short  itemSubType,
      short  itemData,
      short  hPos,
      short  vPos,
      short  hSize,
      short  vSize
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
itemType
Type of the dialog item.
 
itemSubType
Subtype of the dialog item.
 
itemData
Additional data of the dialog item.
 
hPos
Horizontal position of the dialog item in pixels, relative to the dialog client rectangle.
 
vPos
Vertical position of the dialog item in pixels, relative to the dialog client rectangle.
 
hSize
Horizontal size of the dialog item in pixels.
 
vSize
Vertical size of the dialog item in pixels.

Return Values

1-based index of the newly created dialog item (equals to the current number of the dialog items), or zero if an error occurs.

Remarks

Dialog items can be appended to dialogs created either from GRC or from program code. If you create an empty dialog by calling the DGModalDialog or DGModelessInit functions with zero resId parameter, you can use DGAppendDialogItem to fill the dialog with items when handling the DG_MSG_INIT message in the dialog callback function.

The following table summarizes the dialog item types that can be created by DGAppendDialogItem. The table contains the possible values of the function's itemType and itemSubType parameters. Some type of dialog items can have one or more modifier flags that can also be given in the itemSubType parameter using the bitwise or (|) operator. For the details of these modifier flags see the description of the dialog items. The last column explains the meaning of the additional itemData parameter.

Other properties (text, font, value, etc.) of the appended dialog items can be set using DG functions after the item has been created. In these functions use the item index returned by DGAppendDialogItem.
