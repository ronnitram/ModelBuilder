DGPopUpSetItemIconResource

Sets the icon of a pop-up control item with the icon ID and the resource module ID of the icon.

    void DGPopUpSetItemIconResource (
      short        dialId,
      short        item,
      short        popupItem,
      GSResModule  resModule,
      short        iconId
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
popupItem
1-based index of the item in the pop-up control. DG_POPUP_TOP specifies the first item of the pop-up. DG_POPUP_BOTTOM specifies the last item of the pop-up. Use DG_ALL_ITEMS to change the icon of all items.
 
resModule
The resource module ID containing the icon.
 
iconId
The new icon ID of the pop-up control item.

Remarks

A pop-up control item can have only one icon. The icon is always displayed on the left side of the pop-up item text. If there is no icon in the pop-up control item, on Macintosh the item text is aligned to the left, on Windows, however, the text position is kept.

 Windows only:

Left position of the item text can be defined in GRC only.

If the icon is located in the resource module of the dialog, the icon can also be set with DGPopUpSetItemIconId. The resource module of the icon in the pop-up item can be retrieved by DGPopUpGetItemIconResource. The resource module of the dialog can be set and retrieved by DGSetDialogResourceModule or DGSetDialogResourceModules and DGGetDialogResourceModule or DGGetDialogIconResourceModule, respectively.

The icon of the pop-up control item can also be specified with its icon handle: see DGPopUpSetItemIcon for details.
