DGSplitterIsDragEnabled

Retrieves the drag status of a splitter control.

    bool DGSplitterIsDragEnabled (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.

Return Values

Value Meaning
true The drag status of the splitter control item is enabled.
false The drag status of the splitter control item is disabled.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
