DG_MSG_FOCUS

A focusable dialog item gains or loses the keyboard focus.

Message parameters

dialId
ID of the dialog or tab page.
 
item
1-based index of the dialog item.
 
msgData
Value Meaning
0 The item loses the keyboard focus.
1 The item gains the keyboard focus.

Return Values

Not used, should be set to zero.

Remarks

The keyboard focus and the focusable dialog item categories in DG essentially have the same meaning as on the Macintosh operating system (i.e. the same type of dialog items are focusable in DG and on Macintosh). On Windows, however, more dialog items can have the keyboard focus. Focusable dialog items of DG form a subset of focusable dialog items of Windows.

Focusable dialog items of DG are the following: date & time control, edit control, list box, list view, rich edit control and tree view.

When the keyboard focus is changed as a result of user actions or the DGSetFocus function call, DG first sends a DG_MSG_FOCUS message with msgData of value 0 to the previously focused dialog item (if any), then it sends a DG_MSG_FOCUS message with msgData of value 1 to the currently focused dialog item.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
