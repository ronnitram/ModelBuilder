DGTreeViewGetItemRect

Retrieves the item rectangle of a tree view item.

    short DGTreeViewGetItemRect (
      short   dialId,
      short   item,
      Int32   treeItem,
      short*  left,
      short*  top,
      short*  right,
      short*  bottom
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
treeItem
ID of the tree view item.
 
left
Output parameter: horizontal position of the upper left corner of the item rectangle in pixels, relative to the dialog client rectangle. This parameter can be nullptr.
 
top
Output parameter: vertical position of the upper left corner of the item rectangle in pixels, relative to the dialog client rectangle. This parameter can be nullptr.
 
right
Output parameter: horizontal position of the lower right corner of the item rectangle in pixels, relative to the dialog client rectangle. This parameter can be nullptr.
 
bottom
Output parameter: vertical position of the lower right corner of the item rectangle in pixels, relative to the dialog client rectangle. This parameter can be nullptr.

Return Values

Value Meaning
DG_TVR_VISIBLE The tree view item is visible, i.e. it is listed and its client rectangle is at least partly in the tree view client rectangle.
DG_TVR_LISTED The tree view item is listed but it is not in the client rectangle of the tree view.
DG_TVR_NOTLISTED The tree view item is not listed.

If the function parameters are incorrect, DG_TVR_NOTLISTED is returned.

Remarks

Tree view item coordinates are always returned in the main dialog coordinate system even if the item is on a tab page.

If either of the position parameter pointers are nullptr, that coordinate is not returned.

Requirements

Header: DG.h

Module: DGLib [MAC], DG.dll [WIN]
