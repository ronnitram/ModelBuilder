Dialog::GetOriginalClientHeight

Retrieves the original client area height of the dialog. 

short GetOriginalClientHeight (void) const;
static short GetOriginalClientHeight (short resId);

If the dialog is defined in the resource script this method can be used even if the dialog is not opened.

Parameters

resId
Resource ID of the dialog.

Return Value

The original client area height of the dialog.

Description

Original frame size of a dialog is the size either given in the resource file or the creation size. Original client size is the client area size which belongs to the original frame size of a dialog.
Both methods can be used regardless of the dialog was created with or without resource.
