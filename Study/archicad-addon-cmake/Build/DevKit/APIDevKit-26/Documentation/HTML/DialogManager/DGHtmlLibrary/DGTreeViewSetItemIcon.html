DGTreeViewSetItemIcon

Sets the icon data of the state or the normal icon in a tree view item.

    void DGTreeViewSetItemIcon (
      short        dialId,
      short        item,
      Int32        treeItem,
      short        iconType,
      const void*  iconData
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
treeItem
ID of the tree view item. Use DG_ALL_ITEMS to change all of the given type of icons.
 
iconType
The type of icon to be changed in the tree view item:
Value Meaning
DG_TVIT_NORMAL The normal icon of the item.
DG_TVIT_STATE The state icon of the item.
 
iconData
 Macintosh only: 
NSImage* pointer of the new icon (the normal or the state icon depending on the iconType parameter) cast to void*.
 Windows only: 
HICON handle of the new icon (the normal or the state icon depending on the iconType parameter) cast to void*.

Remarks

A tree view item can have two types of icon: the state icon and the normal icon. The icons are always on the left side of the item text. If the tree view has both types of icon, the state icon is the leftmost.

The number of different state icons in a tree view is limited to 12. There is no such limitation for the normal icons.

There are two ways to set an icon of a tree view item. One possibility is to specify the icon with its resource ID by DGTreeViewSetItemIconId or by DGTreeViewSetItemIconResource, the other is to set it with its icon data by DGTreeViewSetItemIcon. If the icon was set by DGTreeViewSetItemIcon the icon data can be retrieved by DGTreeViewGetItemIcon, in this case DGTreeViewGetItemIconId returns DG_NO_ICON and DGTreeViewGetItemIconResource returns InvalidResModule.

If the icon is not set yet, an empty area is displayed instead of shifting the content of the item to the left.

Either of the icon types can be omitted by setting its size to zero either from resource or by a call to DGTreeViewSetIconSize. These settings apply to the whole tree view, i.e. to all tree items.
