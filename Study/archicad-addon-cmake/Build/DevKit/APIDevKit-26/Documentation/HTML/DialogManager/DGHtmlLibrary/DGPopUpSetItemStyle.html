DGPopUpSetItemStyle

Sets the style of a pop-up control item.

    void DGPopUpSetItemStyle (
      short  dialId,
      short  item,
      short  popupItem,
      short  style
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
popupItem
1-based index of the item in the pop-up control. DG_POPUP_TOP specifies the first item of the pop-up. DG_POPUP_BOTTOM specifies the last item of the pop-up. Use DG_ALL_ITEMS to change the style of all items.
 
style
One of the following values:
Value Meaning
DG_IS_PLAIN Item text is plain.
DG_IS_BOLD Item text is bold.
DG_IS_ITALIC Item text is italic.
DG_IS_UNDERLINE Item text is underlined.

Remarks

Before the text style is set by DGPopUpSetItemStyle, all the pop-up items have the DG_IS_PLAIN style.

Only the text style can be set item-by-item. The font size can be changed by DGSetItemFont, for all the items.

The style of a separator line cannot be set.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
