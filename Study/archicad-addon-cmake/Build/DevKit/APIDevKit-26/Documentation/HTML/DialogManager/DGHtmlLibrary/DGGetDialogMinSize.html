DGGetDialogMinSize

Retrieves the allowed minimum width and height of the dialog.

    void DGGetDialogMinSize (
      GSResModule    dialResModule,
      short          dialId,
      short          rectType,
      short*         hMinSize,
      short*         vMinSize
    );

Parameters

dialResModule
The resource module the original dialog resource, used when rectType is DG_ORIGCLIENT or DG_ORIGFRAME, and the dialog is not accessible.
dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
rectType
Value Meaning
DG_CLIENT hMinSize and vMinSize refer to the client area of the dialog.
DG_FRAME hMinSize and vMinSize refer to the window area of the dialog (including frame and title).
 
hMinSize
Output parameter: minimum horizontal size in pixels. This parameter can be nullptr.
 
vMinSize
Output parameter: minimum vertical size in pixels. This parameter can be nullptr.

Remarks

If any of the pointer parameters is nullptr, that minimum size is not returned.

The minimal width and height of a dialog is initially equals to the dialog size given in GRC. These values can be modified by DGSetDialogMinSize.

The dialog can be shrunk either by user action or using DGGrowDialog or DGSetDialogSize until the dialog reaches its minimal sizes.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
