Effect Enum

The possible effects during drag and drop. 

enum Effect
{
    Reject =  DG_DRAG_REJECT,
    Accept =  DG_DRAG_ACCEPT,
    Copy =  DG_DRAG_ACCEPT_COPY,
    Move =  DG_DRAG_ACCEPT_MOVE,
    Link =  DG_DRAG_ACCEPT_LINK,
    Delete =  DG_DRAG_DELETE
};

Value Definitions

Reject
The drag and drop operation is rejected.
Accept
The drag and drop operation is accepted.
Copy
The drag and drop operation is accepted for copy.
Move
The drag and drop operation is accepted for move.
Link
The drag and drop operation is accepted for link.
Delete
The drag and drop operation is accepted with deleting the source.

Description

When DG sends events to the application during drag and drop, usually it requests a value, which can be one of the enumeration values. This value tells, what the application wants to do with this event.
