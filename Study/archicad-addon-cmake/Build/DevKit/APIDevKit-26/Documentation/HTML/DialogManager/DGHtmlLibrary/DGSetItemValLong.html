DGSetItemValLong

Sets the Int32 value of a dialog item.

    void DGSetItemValLong (
      short  dialId,
      short  item,
      Int32  value
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
value
The new value.
Item type Meaning
Check Box 0: Unchecks the control.
1: Checks the control.
Date & Time Control Value of the date & time control.
On Macintosh: The number of seconds since January 1, 1904.
On Windows: The number of seconds since January 1, 1970.
Integer-type Edit Control Value of the edit control.
Pop-up Control 1-based index of the pop-up item to be selected.
Progress Bar Value of the progress bar.
Push Menu Control 0: Unchecks the control.
1: Checks the control.
Radio Button 0: Unchecks the control.
1: Checks the control.
Scroll Bar Value of the scroll bar.
Slider Bar Value of the slider bar.
Spin Control Value of the spin control.
Tab Control 1-based index of the tab page to be activated.
User Control Value of the user control.

Remarks

The edit control accepts values entered by the user according to the limits given in GRC or set by DGSetItemMinLong and DGSetItemMaxLong. If a value outside this range is entered or set by DGSetItemValLong, it is automatically changed by DG to the maximum or minimum.

Position of a progress bar, of the scroll box or of a slider, respectively, is derived from the current item value based on the minimal and maximal value specified in GRC or set by DG functions later. The relative position is calculated using the following formula: relative position = (value - minimum) / (maximum - minimum).

From DG version 1.1.1 use DGPushMenuCheck to check or uncheck a push menu control.
