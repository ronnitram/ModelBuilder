DGListSetTabFieldCount

Sets the number of tabulator fields in a list box.

    bool DGListSetTabFieldCount (
      short  dialId,
      short  item,
      short  nTabFields
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
nTabFields
Number of tabulator fields. This value must be at least 1.

Return Values

true if the function succeeds, otherwise false.

Remarks

The tabulator field data is assigned to the list box not to its items.

List box items can be divided into more columns. This function is used to set the number of columns in the list box items. In this documentation the notation "tabulator field" is used instead of "column". The reason for this is that texts of columns can be separated within a list item by inserting tabulator ('\t') characters into its text. The number of tabulator fields is given by nTabFields. The properties of a tabulator field are set up by DGListSetTabFieldData.

By default every list box has one tabulator field which extends from the left of the list's client area to the scrollbar. Its content is left aligned and truncated at the end.

The content of a tabulator field can be retrieved and set by DGListGetTabItemText, DGListGetTabItemIcon, DGListGetTabItemIconId and DGListSetTabItemText, DGListSetTabItemIcon, DGListSetTabItemIconId respectively.
