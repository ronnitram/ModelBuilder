FileDialog::ExtensionOptions Enum

Option for displaying extensions in filter selection popup. 

enum ExtensionOptions
{
    SystemDefault,
    DisplayExtensions,
    DontDisplayExtensions
};

Value Definitions

SystemDefault
The default behavior of the operating system. On Windows extension are displayed, but on Macintosh not.
DisplayExtensions
Displays extensions on all systems.
DontDisplayExtensions
Does not display extensions on neither system.
