DGFontData

The DGFontData structure contains information about font typeface, size and style of a rich edit control.

    struct DGFontData {
      float  angle;
      short  size;
      UChar  bold;
      UChar  italic;
      UChar  underline;
      UChar  outline;       // Macintosh only!
      UChar  shadow;        // Macintosh only!
      UChar  strikeOut;     // Windows only!
      UChar  charSet;       // Windows only!
      char   filler[3];
      char   faceName[256];
    };

Members

angle
Angle of base line from x-axis in radians.
 
size
Character size of the font in points.
 
bold
Bold flag. Value 1 means bold, 0 means normal.
 
italic
Italic flag. Value 1 means italic, 0 means normal.
 
underline
Underline flag. Value 1 means underlined, 0 means normal.
 
filler
Reserved.
 
faceName
Typeface name of the font.
 
 Macintosh only:
 
outline
Outline flag. Value 1 means outlined, 0 means normal.
 
shadow
Shadow flag. Value 1 means shadowed, 0 means normal.
 
 Windows only:
 
strikeOut
Strikeout flag. Value 1 means stricken-out, 0 means normal.
 
charSet
Character set of the font (ANSI, Latin-2/Central European, etc.).
 

Remarks

The DGFontData structure is used only by the DGRichEditGetFont and DGRichEditSetFont functions.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
