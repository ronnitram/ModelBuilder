DGTabControlSetItemIconId

Sets the icon ID of a tab control item.

    void DGTabControlSetItemIconId (
      short  dialId,
      short  item,
      short  tabItem,
      short  iconId
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
tabItem
1-based index of the tab control item.
 
iconId
The new icon ID of the tab control item.

Remarks

A normal tab control, unlike a simple tab control may have an optional icon and text displayed on its label. The icon in the label is always adjusted to the left of the label text.

DGTabControlSetItemIconId function uses the resource module associated with the dialog or tab page containing the tab control. Use this function if you are sure that the icon is in the same resource module as the dialog. But if the resource module of the icon might differ from the resource module of the dialog, than use the DGTabControlSetItemIconResource function to set the resource module containing the icon. The resource module of the dialog can be set and retrieved by DGSetDialogResourceModule or DGSetDialogResourceModules and DGGetDialogResourceModule or DGGetDialogIconResourceModule, respectively.

The icon ID of a tab control item can be retrieved by DGTabControlGetItemIconId.

An icon can also be specified to the tab control item with its icon handle: see DGTabControlSetItemIcon for details.
