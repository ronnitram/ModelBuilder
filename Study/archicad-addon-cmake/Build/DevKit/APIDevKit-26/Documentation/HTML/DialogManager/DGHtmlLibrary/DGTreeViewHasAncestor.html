DGTreeViewHasAncestor

Tests whether the given item is an ancestor of a tree view item or not.

    bool DGTreeViewHasAncestor (
      short  dialId,
      short  item,
      Int32  treeItem,
      Int32  ancestorItem
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
treeItem
ID of the given tree view item.
 
ancestorItem
ID of the given ancestor item.

Return Values

true if the given ancestor item is an ancestor of the tree view item, false otherwise.

Remarks

To enumerate all items on a hierarchy level use DG_TVG_NEXT and DG_TVG_PREVIOUS. To go through hierarchy levels use DG_TVG_CHILD and DG_TVG_PARENT. To enumerate all the listed items use DG_TVG_NEXTLISTED and DG_TVG_PREVLISTED. To enumerate all items of a subtree write a recursive algorithm based on DG_TVG_CHILD and DG_TVG_NEXT.

Requirements

Module: DGLib [MAC], DG.dll [WIN]
