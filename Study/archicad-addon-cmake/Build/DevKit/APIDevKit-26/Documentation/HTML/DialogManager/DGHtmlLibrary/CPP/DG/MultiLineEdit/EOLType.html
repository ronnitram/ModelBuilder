MultiLineEdit::EOLType Enum

The type of the end-of-line character. 

enum EOLType
{
    Default =  DG_EOL_DEFAULT,
    CR =  DG_EOL_CR,
    CRLF =  DG_EOL_CRLF
};

Value Definitions

Default
Default end-of-line character. It is CRLF on Windows, CR on Macintosh.
CR
CR (\x0D) end-of-line character.
CRLF
CRLF (\x0D\x0A) end-of-line character pair.
