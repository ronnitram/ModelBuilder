DGFontTypeToNameAndSize

Retrieves the font family name and size from the DG font size constant.

    void DGFontTypeToNameAndSize (
      short            fontSize,
      GS::UniString*   name,
      short*           size
    );

Parameters

fontType
Font size constant (see DGSetItemFont).
 
name
Output parameter: the font family name.
 
size
Output parameter: the font size.

Remarks

In the size output parameter the function returns with character height: in Textengine it will be considered as character height (negativ value).

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
