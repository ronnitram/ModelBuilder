DGGetPrevMenuItem

If the checked state or the selected menu item of a push menu control has been changed, the dialog callback function receives a DG_MSG_CHANGE message. Use this macro to retrieve the previously selected menu item of the push menu control from the message data.

    short DGGetPrevMenuItem (
      DGMessageData  msgData
    );

Parameters

msgData
The message data value passed to the callback function.

Return Values

1-based index of the previously selected menu item.

Remarks

The DGGetPrevMenuItem macro is defined as follows:

    #if defined (WINDOWS)
        #define DGGetPrevMenuItem(longData)  ((short) (((ULong) (longData) >> 16) & 0xFFFF))
    #endif
    #if defined (macintosh)
        #define DGGetPrevMenuItem(longData)  ((short) ((Int32) (longData) >> 16))
    #endif

To retrieve the currently selected menu item of the push menu control use DGPushMenuGetSelected.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
