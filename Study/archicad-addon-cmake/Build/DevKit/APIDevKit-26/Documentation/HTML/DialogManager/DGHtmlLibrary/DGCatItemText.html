DGCatItemText

Concatenates text to a dialog item's text.

    void DGCatItemText (
      short                 dialId,
      short                 item,
      const GS::UniString&  text
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
text
The text to be concatenated to the end of the text in the dialog item.

Dialog item types

Type Subtypes
DG_ITM_BUTTON DG_BT_ICONTEXT, DG_BT_PUSHTEXT, DG_BT_SPLIT
DG_ITM_CHECKBOX DG_BT_TEXT, DG_BT_PUSHTEXT
DG_ITM_RADIOBUTTON DG_BT_TEXT, DG_BT_PUSHTEXT
DG_ITM_EDITTEXT DG_ET_TEXT, DG_ET_MULTILINETEXT, DG_ET_PASSWORD
DG_ITM_STATICTEXT all
DG_ITM_GROUPBOX all

Remarks

If you concatenate text to a multi-line edit control and the end-of-line markers are not the platform specific ones (CR on Macintosh, CRLF on Windows), DG automatically converts end-of-line markers in the text before concatenates it to the item.

To concatenate text to a rich edit control, use DGRichEditCatText.
