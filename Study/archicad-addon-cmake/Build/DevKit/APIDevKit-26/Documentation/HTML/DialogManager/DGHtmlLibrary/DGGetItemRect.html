DGGetItemRect

Retrieves a dialog item's position and size.

    void DGGetItemRect (
      short   dialId,
      short   item,
      short*  left,
      short*  top,
      short*  right,
      short*  bottom
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
left
Output parameter: Horizontal position of the upper left corner of the item rectangle, in pixels. This parameter can be nullptr.
 
top
Output parameter: Vertical position of the upper left corner of the item rectangle, in pixels. This parameter can be nullptr.
 
right
Output parameter: Horizontal position of the lower right corner of the item rectangle, in pixels. This parameter can be nullptr.
 
bottom
Output parameter: Vertical position of the lower right corner of the item rectangle, in pixels. This parameter can be nullptr.

Remarks

If any of the pointer parameters is nullptr, that coordinate is not returned.

Item coordinates are always returned in the main dialog coordinate system, even if the item is on a tab page. The sizes include the frame of the item if any.

Sizes of the same dialog item can be different on Macintosh and on Windows. This is why there is only relative positioning and sizing routines in DG for dialog items.

After a call to DGBeginMoveGrowItems the subsequent DGGrowItem, DGMoveItem and DGMoveGrowItem calls do not actually change the size and/or position of the item until DGEndMoveGrowItems is called. Therefore DGGetItemRect will not retrieve the new coordinates before calling DGEndMoveGrowItems.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
