DGDestroyTabPage

Removes a tab page from a tab control item and destroys it.

    short DGDestroyTabPage (
      short  dialId
    );

Parameters

dialId
ID of the tab page assigned to a tab control.

Remarks

Tab pages are special dialogs. They are assigned to tab control items. When a tab control item is selected the corresponding tab page appears in the display area of the tab control. DGDestroyTabPage removes the tab page from a tab control item and destroys it regardless of it was created from resource of from program code. Once a tab page is assigned to a tab control item, it should be destroyed prior the tab control item is deleted. If a dialog is closed its tab control items and the assigned tab pages are automatically destroyed by DG.

Tab pages which are assigned to a tab control of a dialog in the GRC script are automatically created when the dialog is invoked by the application. Use the DGCreateTabPage or the DGCreateBlankTabPage function to create and assign a tab page to an empty or a newly inserted tab control item.

The ID of a tab page assigned to a tab control item can be retrieved by the DGTabControlGetTabPageId function.

Requirements

Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
