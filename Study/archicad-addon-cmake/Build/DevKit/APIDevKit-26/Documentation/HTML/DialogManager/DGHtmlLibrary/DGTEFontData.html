DGTEFontData

The DGTEFontData structure contains information about font typeface, size and style of a rich edit control.

    class DGTEFontData {
      float     angle;
      TE::Font  font;
      bool      strikeOut;     // Windows only!
      UChar     charSet;       // Windows only!
    };

Members

angle
Angle of base line from x-axis in radians.
 
font
Rich description of the font, including its style, size, and family. See the TextEngine documentation for more details.
 Windows only:
 
strikeOut
Strikeout flag. True means stricken-out, false means normal.
 
charSet
Character set of the font (ANSI, Latin-2/Central European, etc.).
 

Remarks

The DGTEFontData class is used only by the DGRichEditGetFont and DGRichEditSetFont functions.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
