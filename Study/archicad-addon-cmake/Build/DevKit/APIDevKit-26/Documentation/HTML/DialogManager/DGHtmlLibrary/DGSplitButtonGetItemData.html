DGSplitButtonGetItemData

Retrieves the value assigned to a split button popup item.

    DGUserData DGSplitButtonGetItemData (
      short                 dialId,
      short                 item,
      short                 popupItem
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
popupItem
1-based indexed of the popup menu item.

Return Value

Returns the value assigned to the specific split button popup item.

Remarks

Retrieves the value assigned to a split button popup item.

Requirements

Version: DG 18.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
