DGModelessClose

Closes a modeless or palette dialog.

    void DGModelessClose (
      short  dialId
    );

Parameters

dialId
ID of the modeless dialog.

Remarks

DGModelessClose stops event processing and then destroys the dialog. Before the dialog is destroyed, a DG_MSG_CLOSE message with DG_CLOSEBOX as the item parameter is sent to the dialog callback function.

If your application has allocated resources to that dialog or to one of its items (see DGSetItemImage and DGSetItemValLong), do not forget to free them when handling the DG_MSG_CLOSE message.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
