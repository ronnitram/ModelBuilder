Modifier Enum

The possible key modifiers. 

enum Modifier
{
    NoModifier =  0,
    Command =  DG_MOD_COMMAND,
    Option =  DG_MOD_OPTION,
    Shift =  DG_MOD_SHIFT
};

Value Definitions

NoModifier
No modifier is pressed.
Command
The Command (Control) key is pressed.
Option
The Option (Alt) key is pressed.
Shift
The Shift key is pressed.
