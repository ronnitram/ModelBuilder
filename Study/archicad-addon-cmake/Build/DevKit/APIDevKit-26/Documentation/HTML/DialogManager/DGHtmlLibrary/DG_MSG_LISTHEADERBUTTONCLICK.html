DG_MSG_LISTHEADERBUTTONCLICK

The header button of a list box item was clicked.

Message parameters

dialId
ID of the dialog or tab page.
 
item
1-based dialog item index of the list box control in which a header item was clicked.
 
msgData
Value Meaning
1 The header button was clicked.
0 Otherwise.

Return Values

Set to 1 if the application handled the event otherwise set to zero.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
