PopUp::EnableItem

Enables a popup item. 

void EnableItem (short popupItem);

Parameters

popupItem
1-based index of the popup item to be enabled, TopItem or BottomItem of Item type specifies the first or last item of the popup control. Use AllItems of Item type to enable all items in the popup control.
