DG_MSG_TREEITEMCLICK

A normal or a state icon of a tree item is clicked.

Message parameters

dialId
ID of the dialog or tab page.
 
item
1-based index of the clicked tree dialog item.
 
msgData
Pointer to a DGTreeViewItemClickData structure cast to DGMessageData.

Return Values

Return 1 if you handle the click yourself; this also means that the selection won't change.

Remarks

Tree view controls receive this message after a DG_MSG_CLICK message if the user clicks the normal or a state icon of a tree item.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
