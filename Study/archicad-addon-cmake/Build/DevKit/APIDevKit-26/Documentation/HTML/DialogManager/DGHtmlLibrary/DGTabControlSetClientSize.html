DGTabControlSetClientSize

Sets the display area size of a tab control.

    void DGTabControlSetClientSize (
      short  dialId
      short  item
      short  hSize
      short  vSize
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
hSize
Horizontal size of the tab control's display area in pixels.
 
vSize
Vertical size of the tab control's display area in pixels.

Remarks

This function resizes the tab control so that its display (client) area is modified to the specified size. Sizes and positions of the assigned tab pages are modified according to the individual tab page fix points and grow types. See DGSetTabPageFixPoint and DGSetTabPageGrowType for details.

The display area size and rectangle of the tab control can be retrieved by the DGTabControlGetClientSize and DGTabControlGetClientRect functions.

In case of frameless simple tab controls the dialog item rectangle is the same as the display area rectangle.
