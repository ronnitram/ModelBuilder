DGModified

Retrieves whether a given dialog item has been modified since the dialog was initialized or not.

    short DGModified (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.

Return Values

Value Meaning
1 The given dialog item has been modified.
0 The given dialog item has not been modified yet.

Remarks

The modification flag of a dialog item is set by user actions that generate DG_MSG_CLICK, DG_MSG_CHANGE or DG_MSG_TRACK messages. Different functions can also set the flag except if they are called when handling the DG_MSG_INIT message in the dialog callback function.

The following table summarizes which functions and messages set the modification flag:

Item type Messages Functions
Check Box DG_MSG_CHANGE DGSetItemValLong
Date & Time Control DG_MSG_CHANGE DGSetItemValLong, DGSetItemMinLong, DGSetItemMaxLong
Edit Control (text value) DG_MSG_CHANGE DGSetItemText, DGCatItemText
Edit Control (double value) DG_MSG_CHANGE DGSetItemValDouble, DGSetItemMinDouble, DGSetItemMaxDouble
Edit Control (Int32 value) DG_MSG_CHANGE DGSetItemValLong, DGSetItemMinLong, DGSetItemMaxLong
List Box DG_MSG_CHANGE DGListSelectItem, DGListDeselectItem
List View DG_MSG_CHANGE DGListViewSelectItem, DGListViewDeselectItem
Pop-up Control DG_MSG_CHANGE DGPopUpSelectItem
Progress Bar none DGSetItemValLong, DGSetItemMinLong, DGSetItemMaxLong
Push Button DG_MSG_CLICK none
Push Menu Control DG_MSG_CHANGE DGPushMenuCheck
Radio Button DG_MSG_CHANGE DGSetItemValLong
Rich Edit Control DG_MSG_CHANGE DGCatItemUniText, DGGetItemUniText, DGRichEditCatText, DGRichEditSetText, DGRichEditSetTextRange, DGSetItemUniText
Scroll Bar DG_MSG_CHANGE, DG_MSG_TRACK DGSetItemValLong, DGSetItemMinLong, DGSetItemMaxLong
Slider Bar DG_MSG_CHANGE, DG_MSG_TRACK DGSetItemValLong, DGSetItemMinLong, DGSetItemMaxLong
Spin Control DG_MSG_CHANGE, DG_MSG_TRACK DGSetItemValLong, DGSetItemMinLong, DGSetItemMaxLong
Tab Control DG_MSG_CLICK, DG_MSG_CHANGE DGSetItemValLong
Tree View DG_MSG_CHANGE DGTreeViewSelectItem, DGTreeViewDeselectItem
User Control DG_MSG_CHANGE, DG_MSG_TRACK DGSetItemValLong, DGSetItemMinLong, DGSetItemMaxLong

When the dialog callback function returns after handling the DG_MSG_INIT message, DG clears the modification flags of all items in the dialog. Whenever it is necessary, the application can also clear the flag of one or more items by calling the DGResetModified function.

DGModified is typically called from the callback function of a modal dialog when handling the DG_MSG_CLOSE message. This is the simplest and most elegant way to test whether a dialog item has been modified during the lifetime of the dialog or not.
