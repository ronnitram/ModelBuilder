DGGetDialogResourceModule

Retrieves the resource module associated with a dialog.

    GSResModule DGGetDialogResourceModule (
      short  dialId
    );

Parameters

dialId
The Id of the dialog.

Return Values

The resource module associated with the dialog.

Remarks

DG associates the currently active resource module to a dialog when it is being created by DGModalDialog or DGModelessInit. The application can affect which module will be associated with a dialog by calling DGSetDialogResourceModule or DGSetDialogResourceModules before creating the dialog. DG uses the associated module whenever it wants to load a resource belonging to the dialog.

If the application itself needs some dialog related resources, use DGGetDialogResourceModule to retrieve handle of the module containing the resources.
