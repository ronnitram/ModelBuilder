DGGetItemUniText

Retrieves the text of a unicode richedit item.

    Int32 DGGetItemUniText (
      short    dialId,
      short    item,
      UShort*  text,
      Int32    size,
      short    eolType
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
text
Pointer to buffer to receive the unicode text. This parameter can be nullptr.
 
size
Size of the text buffer. If text is nullptr, this parameter is ignored.
 
eolType
Value Meaning
DG_EOL_DEFAULT DGGetItemUniText returns text with the end-of-line markers of the current platform: CR (0x0D) on Macintosh and CRLF (0x0D 0x0A) on Windows.
DG_EOL_CR DGGetItemUniText returns text with CR (0x0D) end-of-line markers.
DG_EOL_CRLF DGGetItemUniText returns text with CRLF (0x0D 0x0A) end-of-line markers.

Return Values

The length of the item text (including the 0 terminator), or 0 if size is less than the size of the item text.

Remarks

Use nullptr for the text parameter to determine the necessary buffer size.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
