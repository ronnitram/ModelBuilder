DGGetTabPageFixPoint

Retrieves the fix (anchor) point of a tab page.

    short DGGetTabPageFixPoint (
      short  dialId
    );

Parameters

dialId
ID of the tab page subdialog.

Return Values

One of the following values: DG_TOPLEFT, DG_TOPCENTER, DG_TOPRIGHT, DG_CENTERLEFT, DG_CENTER, DG_CENTERRIGHT, DG_BOTTOMLEFT, DG_BOTTOMCENTER, DG_BOTTOMRIGHT.

Remarks

The fix point of the tab page determines how the position of the tab page in the tab control's display area is modified when the tab control is being resized. By default tab pages are centered in the tab control's display area. While resizing the tab control the fix point of the tab page keeps its place in the tab control and the size of the tab page is modified according to the grow type. The fix point and the grow type can be set with the DGSetTabPageFixPoint and DGSetTabPageGrowType functions.
