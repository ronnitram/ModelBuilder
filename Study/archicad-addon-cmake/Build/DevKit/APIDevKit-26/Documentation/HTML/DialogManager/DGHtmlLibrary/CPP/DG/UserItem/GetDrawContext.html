UserItem::GetDrawContext

Retrieves the drawing context of the user item. 

void * GetDrawContext (void) const;

Return Value

A pointer to the platform-dependent drawing context of the user item to be updated.
The drawing context indicates:

Platform Meaning
On Macintosh: GrafPtr cast to void*.
On Windows: HDC cast to void*.
