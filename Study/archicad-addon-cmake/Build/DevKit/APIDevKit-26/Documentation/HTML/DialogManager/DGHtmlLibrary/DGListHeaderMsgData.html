DGListHeaderMsgData

If the dialog callback function receives a DG_MSG_LISTHEADERCLICK, DG_MSG_LISTHEADERDRAG or DG_MSG_LISTHEADERRESIZE message for a list box header item, the msgData parameter is a pointer to a DGListHeaderMsgData structure that contains information about the user action.

    struct DGListHeaderMsgData {
      short  subMessage;
      short  itemIndex;
      short  newIndex;
      short  growSize;
    };

Members

subMessage
message subMessage
DG_MSG_LISTHEADERCLICK 0
DG_MSG_LISTHEADERDRAG 0
DG_MSG_LISTHEADERRESIZE DG_LHM_RESIZEENTER, DG_LHM_RESIZING, DG_LHM_RESIZEEXIT
 
itemIndex
message itemIndex
DG_MSG_LISTHEADERCLICK 1-based index of the clicked header item.
DG_MSG_LISTHEADERDRAG 1-based index of the dragged header item.
DG_MSG_LISTHEADERRESIZE 1-based index of the resized header item.
 
newIndex
1-based new index of the dragged header item in case of a header item drag event. Otherwise it is 0.
 
growSize
Size change of the resized header item in case of a header item resize event. Otherwise it is 0.

Requirements

Version: DG 2.2.0 or later
