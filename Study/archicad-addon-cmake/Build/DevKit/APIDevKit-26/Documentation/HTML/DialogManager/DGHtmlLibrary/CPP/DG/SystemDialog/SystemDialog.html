Description

The SystemDialog class provides support for handling system specific dialogs in a very similar way as native DG dialogs handled, although these dialogs are not created and managed by DG directly. Additionally this class enables the customization of the system dialogs with a native DG panel (SystemDialogPanel). This panel is inserted into a specific place in each dialog and operating system.
Some of the methods of the SystemDialogPanel class and its derived classes can be used only before and after the dialog is displayed, others only during the modal state of the dialog. When a restriction applies to a method it is noted in its description. For example the PostCloseRequest method is only useable when the dialog is displayed. Other methods like SetOKButtonText are restricted for initialization use only, usually because of some operating system restriction.
