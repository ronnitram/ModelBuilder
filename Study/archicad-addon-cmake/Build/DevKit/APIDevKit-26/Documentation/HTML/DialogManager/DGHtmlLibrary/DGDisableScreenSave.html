DGDisableScreenSave

Disables screen save under modal dialogs.

    void DGDisableScreenSave (
      short  disableFlag
    );

Parameters

disableFlag
Disable screen saving of the following windows:
Flag Description
DG_DSS_DIALOGS Disables screen save of dialogs (Windows only).
DG_DSS_WINDOWS Disables screen save of application windows.

Remarks

When a modal dialog is opened, only those areas of the screen are saved that belong to the application. While the modal dialog is open, the screen is redrawn from this saved image whenever the application should be updated.

If DG_DSS_WINDOWS flag is specified in DGDisableScreenSave, nothing is restored after the call but everything is updated.

This feature can be useful when a modal dialog is used to modify content of windows or dialogs under it. If the screen saving is not disabled in this case, the modified content cannot be displayed to reflect the changes while the modal dialog is open.

When the DG dialog manager is initialized, screen save is automatically enabled.

On Macintosh, when the screen is restored under a modal dialog from the previously saved image, areas of other dialogs of the application are not restored, they are updated instead.

On Windows, however, all the application areas are restored if the screen save is turned on. If DGDisableScreenSave is called with DG_GSS_DIALOGS, other modal dialogs of the application are not restored after the call, only the application's windows. This is the default behavior on Macintosh.
