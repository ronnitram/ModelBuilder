DGRetrieveUserDataFromRegistry

Retrieves the stored userdata from the registry from Dialog Manager Custom Data key under the current Archicad key.

    ULong DGRetrieveUserDataFromRegistry (
      const GS::UniString&  keyName,
      const void*           data
    );

Parameters

keyName
The keyname of the registry (on Windows) or preferences (on Macintosh).
 
data
The stored userdata to be retrieved.
 

Return Values

Retrieves the size of the stored data.

Remarks

Custom userdata in DG can be stored in system preferences (Macintosh) or in the system registry (Windows). This means that a given data can be stored and retrieved by the DGStoreUserDataToRegistry and the DGRetrieveUserDataFromRegistry functions respectively.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
