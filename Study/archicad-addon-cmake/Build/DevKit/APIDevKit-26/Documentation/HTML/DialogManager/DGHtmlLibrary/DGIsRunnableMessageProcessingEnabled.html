DGIsRunnableMessageProcessingEnabled

Tests whether the execution of the posted runnable messages is enabled or not.

    bool DGIsRunnableMessageProcessingEnabled (
      void
    );

Return Values

Value Meaning
true The execution of the posted runnable messages in the application's message loop is enabled.
false The execution of the posted runnable messages in the application's message loop is disabled.

Remarks

Runnable messages can be posted to the application's message loop by the DGPostCommandToApplication function. So as not to interrupt certain processes, the execution of the runnable messages can be disabled. In this case the messages are executed only after the process was finished, e.g. the modal dialog was closed. To enable or disable the execution of the runnable messages use the DGEnableRunnableMessageProcessing and DGDisableRunnableMessageProcessing functions respectively. By default the execution of the runnable messages is enabled.
