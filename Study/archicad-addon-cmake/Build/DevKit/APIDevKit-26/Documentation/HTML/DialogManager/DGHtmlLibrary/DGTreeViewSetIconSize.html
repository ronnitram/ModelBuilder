DGTreeViewSetIconSize

Sets the size of the tree view icons depending on the specified icon type.

    void DGTreeViewSetIconSize (
      short  dialId,
      short  item,
      short  iconType,
      short  hSize,
      short  vSize
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
iconType
Type of tree view icons to be adjusted:
Value Meaning
DG_TVIT_NORMAL The normal icons.
DG_TVIT_STATE The state icons.
 
hSize
Horizontal size of the icon in pixels.
 
vSize
Vertical size of the icon in pixels.

Remarks

These settings apply to the whole tree view, i.e. to all tree items.

A tree view item can have two type of icons: the state icon and the normal icon. The icons are always on the left side of the item text. If the tree view has both type of icons, the state icon is the leftmost.

Either of the icon types can be omitted by setting its size to zero either from resource or by a call to DGTreeViewSetIconSize.

Sizes of icons can be set only if the tree view control is empty.

Requirements

Version: DG 1.1.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
