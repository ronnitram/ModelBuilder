DG_MSG_LISTHEADERRESIZE

A header item of a list box item is being resized.

Message parameters

dialId
ID of the dialog or tab page.
 
item
1-based dialog item index of the list box control in which a header item is being resized.
 
msgData
Pointer to a DGListHeaderMsgData structure cast to DGMessageData.

Return Values

Set to 1 if the application handled the event otherwise set to zero.

Remarks

DG_MSG_LISTHEADERRESIZE messages are sent to the dialog callback function when a header item of a list box is resized. The major difference between the other messages and DG_MSG_LISTHEADERRESIZE is that this message is sent not only by user action but also by program code.

By every resizing action the dialog callback function receives three or more DG_MSG_LISTHEADERRESIZE messages with the following submessages:

Value Meaning
DG_LHM_RESIZEENTER This message is sent before the header item is resized.
DG_LHM_RESIZING This message is sent when the header item is being resized.
DG_LHM_RESIZEEXIT This message is sent after the header item is resized.

By the DG_MSG_LISTHEADERRESIZE message the application has the possibility to adjust the tabulator fields according to the header items. If the isSyncron flag is set on the list box header and the application does not handle the DG_MSG_LISTHEADERRESIZE message (0 is returned), the list box adjusts the tabulator fields automatically. In this case if there was space among the tabulator fields, it will be removed.
If the application handles this message, it should return 1. In this case the application should change the tabulator field sizes. They are not changed automatically.

If the isSyncron flag is set to false on the list box header, DG never changes the size of the tabulator fields.

To change the isSyncron flag of the header of a list box use the DGListSetHeaderFlags function.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
