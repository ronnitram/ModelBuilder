DGSetAutoCloseState

An alert or a modal dialog usually terminates by the result of a user action, i.e. click on the OK or Cancel button, or the Close Box (Windows only). However, there is a feature in DG to close these types of dialogs automatically (without user action) if a given interval expires. Use this macro to enable or disable automatic close of modal dialogs and alerts.

    void DGSetAutoCloseState (
      bool  state
    );

Parameters

state
Value Meaning
true Enable automatic close.
false Disable automatic close.

Remarks

The DGSetAutoCloseState macro is defined as follows:

#define DGSetAutoCloseState(a)  if (a) dgFt->dgEnableAutoClose (); else dgFt->dgDisableAutoClose ();

To enable or disable automatic close of modal dialogs and alerts the DGEnableAutoClose and DGDisableAutoClose functions can be used also.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
