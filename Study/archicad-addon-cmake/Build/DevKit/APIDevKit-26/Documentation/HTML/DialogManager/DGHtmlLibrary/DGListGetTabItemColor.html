DGListGetTabItemColor

Retrieves the foreground (text) color of a tabulator field of a list box item.

    bool DGListGetTabItemColor (
      short            dialId,
      short            item,
      short            listItem,
      short            tabIndex,
      unsigned short*  red,
      unsigned short*  green,
      unsigned short*  blue
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
listItem
1-based index of the list item. DG_LIST_TOP specifies the first item of the list, DG_LIST_BOTTOM specifies the last item of the list.
 
tabIndex
1-based index of the tabulator field.
 
red
Output parameter: red component of the color (0-65535). This parameter can be nullptr.
 
green
Output parameter: green component of the color (0-65535). This parameter can be nullptr.
 
blue
Output parameter: blue component of the color (0-65535). This parameter can be nullptr.

Return Values

Value Meaning
false The tabulator field has the system default foreground (text) color.
true The tabulator field's foreground (text) color was previously set by DGListSetTabItemColor.

If the function parameters are incorrect or the specified list item is a separator, false is returned and the color components are set to zero.

Remarks

If either of the color parameter pointers are nullptr, that component is not returned.

Requirements

Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
