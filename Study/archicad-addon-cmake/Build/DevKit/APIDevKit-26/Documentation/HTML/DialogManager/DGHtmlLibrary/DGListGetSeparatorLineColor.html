DGListGetSeparatorLineColor

Retrieves the separator line color of a list box item.

    void DGListGetSeparatorLineColor (
      short    dialId,
      short    item,
      UShort*  red,
      UShort*  green,
      UShort*  blue
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
red
Output parameter: red component of the color (0-65535). This parameter can be nullptr.
 
green
Output parameter: green component of the color (0-65535). This parameter can be nullptr.
 
blue
Output parameter: blue component of the color (0-65535). This parameter can be nullptr.

Remarks

After the list box is created, DGListEnableSeparatorLines should be called to include a one pixel-wide separator line between each two list box items.
The separator line status for the list box is retrieved by the DGListHasSeparatorLines function.

If either of the color parameter pointers are nullptr, that component is not returned.

Requirements

Header: DG.h

Module: DGLib [MAC], DG.dll [WIN]
