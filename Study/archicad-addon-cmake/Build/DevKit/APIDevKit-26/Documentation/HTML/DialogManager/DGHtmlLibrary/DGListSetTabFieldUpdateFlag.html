DGListSetTabFieldUpdateFlag

Sets the update flag of a tabulator field of a list box.

    void DGListSetTabFieldUpdateFlag (
      short  dialId,
      short  item,
      short  tabIndex,
      bool   updateFlag
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
tabIndex
1-based index of the tabulator field.
 
updateFlag
Value Meaning
false If the tabulator field is drawn by DG.
true The tabulator field must be drawn by the application.

Remarks

The content of list items can be divided into more columns. In this documentation the notation "tabulator field" is used instead of "column". The reason for this is that texts of columns can be separated within a list item by inserting tabulator ('\t') characters into its text.

Call this routine only after you set the number of tabulator fields and their properties by DGListSetTabFieldCount and DGListSetTabFieldData.

After the update flag of a tabulator field is set to true, the dialog callback function receives a DG_MSG_UPDATE message whenever the tabulator field must be updated. The message data parameter is a pointer to a DGListUpdateData structure that contains all the information needed to update the tabulator field.

The update flag is assigned to the list and not to its items. This means that the tabIndex tabulator field of all list items must be updated by the application.

If the list box has horizontal scrollbar then the extent of the horizontal scrollbar is calculated depending the properties of the last tabulator field. If the truncation flag or the user update flag is turned on in this field then the scrollbar's extent equals the end position of the last tabulator field, otherwise it is dynamically calculated from the content of the longest listbox item.
