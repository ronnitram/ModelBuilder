DGListViewSetHelpStyle

Sets the style of the list view.

    void DGListViewSetHelpStyle (
      short  dialId,
      short  item,
      Int32  helpStyle
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
helpStyle
One of the following values:
Value Meaning
DG_LVHS_NORMAL Only one help text will be displayed for the list.
DG_LVHS_FORITEM Help text will be displayed for each listItem separately.

Remarks

The detailed help information of the item can be invoked by setting the keyboard focus to a focusable dialog item then pressing the [F1] key on Windows or [Help] key on Macintosh; or by clicking the right mouse button on the item; or by clicking the help [?] button in the dialog caption then clicking on the item itself. If tooltips are enabled, the short help information is displayed as usual.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
