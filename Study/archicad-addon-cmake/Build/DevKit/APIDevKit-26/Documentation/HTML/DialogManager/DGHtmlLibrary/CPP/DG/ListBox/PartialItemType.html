ListBox::PartialItemType Enum

The display of partial items. 

enum PartialItemType
{
    PartialItems =  0,
    NoPartialItems =  DG_LT_NOPARTIALITEMS
};

Value Definitions

PartialItems
Partial items will be displayed.
NoPartialItems
Partial items will not be displayed.

Description

Specifies, that DG should or should not display partial items at the edges of the list box.
