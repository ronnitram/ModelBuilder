DGListGetTabFieldData

Retrieves the properties of a tabulator field in a list box.

    void DGListGetTabFieldData (
      short   dialId,
      short   item,
      short   tabIndex,
      short*  begPos,
      short*  endPos,
      short*  justFlag,
      short*  truncFlag,
      bool*   hasSeparator,
      bool*   status
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
tabIndex
1-based index of the tabulator field.
 
begPos
Output parameter: tabulator field left position in pixels, relative to the client rectangle of the list box. This parameter can be nullptr.
 
endPos
Output parameter: tabulator field right position in pixels, relative to the client rectangle of the list box. This parameter can be nullptr.
 
justFlag
Output parameter: justification of the content in the tabulator field. For left/center/right justification DG_IS_LEFT, DG_IS_CENTER, DG_IS_RIGHT is returned. This parameter can be nullptr.
 
truncFlag
Output parameter: How the text of the tabulator field is truncated if it does not fit into the tabulator field. This parameter can be nullptr
.
Possible values are:
Flag Action
DG_IS_NOTRUNC No truncation.
DG_IS_TRUNCEND Truncates at the end.
DG_IS_TRUNCMIDDLE Truncates in the middle (leaves the same number of characters from both ends of the text).
 
hasSeparator
Output parameter: if the tabulator field has a separator line on its right edge this flag is set.
 
status
Output parameter: if the tabulator field is disabled, this flag is set to false.

Remarks

The properties of the tabulator fields can be set and modified with DGListSetTabFieldData. By default every list box has one tabulator field which extends from the left of the list's client area to the left edge of the vertical scrollbar. The content of this item is left aligned and truncated at the end if the list box has no horizontal scrollbar otherwise not truncated.

If any of the pointer parameters is nullptr, that value is not returned.
