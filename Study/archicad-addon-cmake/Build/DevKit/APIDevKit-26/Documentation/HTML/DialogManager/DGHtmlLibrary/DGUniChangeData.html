DGUniChangeData

If the dialog callback function receives a DG_MSG_CHANGE message for a unicode edit control item, the msgData parameter is a pointer to a DGRichEditChangeData structure (which has a DGUniChangeData member) that contains information about the user action.

    struct DGUniChangeData {
      short  reqWidth;
      short  reqHeight;
      Int32  oldHScrollPos;
      Int32  oldVScrollPos;
      Int32  hScrollPos;
      Int32  vScrollPos;
    };

Members

reqWidth
The width of requested/recommended unirichedit control's rectangle, in logical units.
 
reqHeight
The height of requested/recommended unirichedit control's rectangle., in logical units
 
oldHScrollPos
Previous horizontal scrollbar position, in logical units.
 
oldVScrollPos
Previous vertical scrollbar position, in logical units.
 
hScrollPos
Current horizontal scrollbar position, in logical units.
 
vScrollPos
Current vertical scrollbar position, in logical units.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
