DGListSetTabData

This function is obsolete, use DGListSetTabFieldCount and DGListSetTabFieldData instead.

Sets the number and the properties of tabulator fields in a list box.

    bool DGListSetTabData (
      short                dialId,
      short                item,
      short                nTabFields,
      const DGListTabData* tabData
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
nTabFields
Number of tabulator fields. This value must be at least 1.
 
tabData
Pointer to the tabulator fields data.

Return Values

true if the function succeeds, false otherwise. If the tabulator fields overlap or do not fit the list box item then they will be set to equal size, forced to fit the listbox item and the function returns false. If the list box has horizontal scrollbar then the tabulator fields are allowed to extend the list item size.

Remarks

The tabulator field data is assigned to the list box not to its items.

List box items can be divided into more columns. This function is used to set the number of columns and set up the properties of each column simultaneously. In this documentation the notation "tabulator field" is used instead of "column". The reason for this is that texts of columns can be separated within a list item by inserting tabulator ('\t') characters into its text. The number of tabulator fields is given by nTabFields and their size, alignment and truncation properties are defined in the DGListTabData structures of tabData.

By default every list box has one tabulator field which extends from the left of the list's client area to the left edge of the vertical scrollbar. Its content is left aligned and truncated at the end.

The content of a tabulator field can be retrieved and set by DGListGetTabItemText, DGListGetTabItemIcon, DGListGetTabItemIconId and DGListSetTabItemText, DGListSetTabItemIcon, DGListSetTabItemIconId respectively.

There is another way to modify the text of a tabulator field: first retrieve the text of a list item by DGListGetItemText and look for the content of the specified tabulator field by searching for tabulator ('\t') characters in the item text, then replace the old text between the appropriate two tabulator characters with the new one using string manipulating functions and set this new item text by DGListSetItemText. This method is obsolete and kept for backward compatibility only.
