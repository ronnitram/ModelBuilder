ListBoxDragSourceObserver::ListBoxDragStarting

Event handler for the drag starting notification. 

virtual void ListBoxDragStarting (
    const ListBoxDragSourceEvent & ev,
    bool *              canStart
);

Parameters

ev
A reference to the event.
canStart
false by default. If true, the drag and drop operation can start from the list box controls.

Description

If this event rise it is time to decide to let the drag and drop start from this control or not.

Example

// Method Example:
void	DragTab2Observer::ListBoxDragStarting (const DG::ListBoxDragSourceEvent& /*ev*/, bool* canStart)
{
	*canStart = true;
}

Granting drag starting in a list box control.
