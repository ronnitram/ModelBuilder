DG_MSG_ACTIVATE

This message is obsolete, and kept for backward compatibility only. In new applications the DG_MSG_TOPSTATUSCHANGE message should be used which gives more information about the activate event.

A modeless dialog is activated or deactivated or a palette dialog is activated.

Message parameters

dialId
ID of the dialog.
 
item
Not used, always zero.
 
msgData
Value Meaning
0 The modeless dialog is deactivated.
1 The modeless or palette dialog is activated.

Return Values

Not used, should be set to zero.

Remarks

The dialog callback function receives this message as a result of a user action or of calling the DGActivateModelessDialog function. When a dialog is activated, DG first sends a DG_MSG_ACTIVATE message with msgData of value 0 to the previously active dialog, then it sends a DG_MSG_ACTIVATE message with msgData of value 1 to the currently activated dialog.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
