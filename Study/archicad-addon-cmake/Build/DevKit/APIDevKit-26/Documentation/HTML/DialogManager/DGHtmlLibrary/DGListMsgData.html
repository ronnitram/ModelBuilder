DGListMsgData

If the dialog callback function receives a DG_MSG_CLICK or DG_MSG_DOUBLECLICK message for a list box item, or a DG_MSG_MOUSEMOVE message for a list box item, the msgData parameter is a pointer to a DGListMsgData structure that contains information about the user action.

    struct DGListMsgData {
      short  listItem;
      short  tabIndex;
      short  xPos;
      short  yPos;
      short  keyFlag;
    };

Members

listItem
1-based index of the list item.
message listItem
DG_MSG_CLICK List item where the click occurred.
DG_MSG_DOUBLECLICK List item where the double click occurred.
DG_MSG_MOUSEMOVE List item under the mouse.
 
tabIndex
1-based index of the tabfield where the event occured.
 
xPos
Horizontal mouse position in pixels, relative to the list item rectangle.
 
yPos
Vertical mouse position in pixels, relative to the list item rectangle.
 
keyFlag
Status of the modifier keys. This value can be any combination of the following constants:
flag Macintosh Windows
DG_MOD_COMMAND Command (Apple) Ctrl
DG_MOD_OPTION Option Alt
DG_MOD_SHIFT Shift Shift

Requirements

Version: DG 2.0.0 or later
Header: DG.h
