DGResetModified

Resets the state of a given dialog item to unmodified.

    void DGResetModified (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item. DG_ALL_ITEMS specifies all items of the given dialog.

Remarks

The modification flag of a dialog item is set by user actions that generate DG_MSG_CLICK, DG_MSG_CHANGE or DG_MSG_TRACK messages. Different functions can also set the flag except if they are called when handling the DG_MSG_INIT message in the dialog callback function. See DGModified for a complete list of functions and messages that set the modification flag of different dialog items.

When the dialog callback function returns after handling the DG_MSG_INIT message, DG clears the modification flags of all items in the dialog. Whenever it is necessary, the application can also clear the flag of one or more items by calling the DGResetModified function.

DGModified is typically called from the callback function of a modal dialog when handling the DG_MSG_CLOSE message. This is the simplest and most elegant way to test whether a dialog item has been modified during the lifetime of the dialog or not.
