DGListEnableSeparatorLines

Sets or resets the list box' horizontal separator line flag.

    void DGListEnableSeparatorLines (
      short  dialId,
      short  item,
      bool   hasSeparatorLine
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
hasSeparatorLine
true to include a horizontal separator line at the bottom of each list box items, false to remove them.

Remarks

Call DGListEnableSeparatorLines after the list is created to include a one pixel-wide separator line between each two list box item. The separator line color is set by the DGListSetSeparatorLineColor function.

Having separator lines reduces the height of an item by one pixel reported by function DGListGetItemRect. Function DGListGetItemHeight is not affected by separator lines. In other words the formula DGListGetItemHeight * DGListGetItemCount returns the list box' client height.
