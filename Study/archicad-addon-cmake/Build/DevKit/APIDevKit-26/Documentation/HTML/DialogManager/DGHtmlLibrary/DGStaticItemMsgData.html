DGStaticItemMsgData

If the dialog callback function receives a DG_MSG_MOUSEMOVE message for a static text item, the msgData parameter is a pointer to a DGStaticItemMsgData structure that contains information about the user action.

    struct DGStaticItemMsgData {
      short  left;
      short  top;
      short  right;
      short  bottom;
      short  xPos;
      short  yPos;
      short  keyFlag;
      short  buttonFlag;
    };

Members

left
Dialog or item static item rectangle left position.
 
top
Dialog or item static item rectangle top position.
 
right
Dialog or item static item rectangle right position.
 
bottom
Dialog or item static item rectangle bottom position.
 
xPos
Horizontal mouse position in pixels, relative to the static item rectangle.
 
yPos
Vertical mouse position in pixels, relative to the static item rectangle.
 
keyFlag
Status of the modifier keys. This value can be any combination of the following constants:
flag Macintosh Windows
DG_MOD_COMMAND Command (Apple) Ctrl
DG_MOD_OPTION Option Alt
DG_MOD_SHIFT Shift Shift
DG_MOD_CTRL Ctrl Ctrl
 
buttonFlag
Tells which mouse button is pressed.
Flag Mouse button
DG_BUTTON_LEFT Left button
DG_BUTTON_RIGHT Right button
DG_BUTTON_WHEEL Mouse wheel

Requirements

Version: DG 2.0.0 or later
Header: DG.h
