DGGetModelessToClose

Retrieves ID of a modeless dialog or a palette dialog which is to be closed.

    short DGGetModelessToClose (
      void
    );

Return Values

Dialog identifier to be closed or zero.

Remarks

This function should be called in the event loop of the application.

If the user clicks the close button in the caption of the modeless dialog or palette, or returns a nonzero value in the dialog callback function while processing DG_MSG_CLICK message then an internal flag will be set instead of closing the dialog. In the event loop the application should call DGGetModelessToClose. This function examines the internal flag of every modeless and palette dialog and returns the dialog ID if the flag is set. In this case the dialog should be closed by calling the DGModelessClose function.

Requirements

Version: DG 2.0.0 or later
Header: DG.h, DGDefs.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
