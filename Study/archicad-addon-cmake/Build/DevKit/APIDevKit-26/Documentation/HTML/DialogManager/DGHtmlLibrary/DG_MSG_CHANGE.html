DG_MSG_CHANGE

This message has different meaning for different type of dialog items:

Item type Meaning
Check Box The checked state of a check box is changed.
Date & Time Control The date or time information is changed.
Edit Control The text of a text-type edit control or the value of a numeric-type edit control is changed.
List Box The selection is changed in a list box or a list item is dragged to a new position.
List View The selection is changed in a list view.
Pop-up Control The selected item in a pop-up control is changed.
Push Menu Control The checked state or the selected menu item of a push menu control is changed.
Radio Button A radio button is checked (switched on).
Rich Edit Control The content or the selection of a rich edit control has changed.
Scroll Bar The position of a scroll box is changed.
Slider Bar The position of a slider is changed.
Spin Control The value of a single spin control is changed. Note that edit spin controls do not receive this message.
Tab Control The current (displayed) tab page of a normal tab control is changed. Note that simple tab controls do not receive this message.
Tree View A tree view item is edited or dragged by the user, or new item(s) is/are selected in the tree view.
User Control The value of a user control is changed.

Message parameters

dialId
ID of the dialog or tab page.
 
item
1-based index of the dialog item.
 
msgData
Item type msgData
Check Box Previous state of the check box (0: unchecked; 1: checked).
Date & Time Control Previous date and time information (in the standard C language system time format).
Edit Control Zero for text-type edit controls. The control's previous value or a pointer to a double precision variable holding the previous double value for integer- or double-type edit controls.
List Box Zero if the selection is changed. The previous position if a list item is dragged to a new position.
List View Not used, always zero.
Pop-up Control 1-based index of the previously selected pop-up item.
Push Menu Control The previous state and the previously selected menu item of the push menu control packed into a DGMessageData value.
Radio Button 1-based index of the previously selected radio button in the same group.
Rich Edit Control Pointer to a DGRichEditChangeData structure cast to DGMessageData.
Scroll Bar Previous position of the scroll box.
Slider Bar Previous position of the slider.
Spin Control Previous value of the control.
Tab Control 1-based index of the previously displayed tab.
Tree View Pointer to a DGTreeViewChangeData structure cast to DGMessageData.
User Control Previous value of the control.

Return Values

Not used, should be set to zero.

Remarks

If a list box receives the DG_MSG_CHANGE message with non-zero message data, use DGListGetOldDragPos and DGListGetNewDragPos macros to retrieve the old and new index of the dragged list item.

For push menu controls the message data parameter contains the previous state and the previously selected menu item of the control packed into a DGMessageData value. The DGGetPrevItemVal and DGGetPrevMenuItem macros can be used to separate the packed information.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
