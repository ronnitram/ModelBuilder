DGListHasSeparatorLines

Returns the separator line status for a list box.

    bool DGListHasSeparatorLines (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 

Return Values

true if the list box has horizontal separator lines, false otherwise.

Requirements

Version: DG 2.1.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
