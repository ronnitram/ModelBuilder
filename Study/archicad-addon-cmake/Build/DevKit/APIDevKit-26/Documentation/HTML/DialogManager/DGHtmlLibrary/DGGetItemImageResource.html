DGGetItemImageResource

Retrieves the resource module ID of the image of a dialog item.

    GSResModule DGGetItemImageResource (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.

Return Values

The resource module ID of the image. InvalidResModule is returned if the image was set by the DGSetItemImage function or if the function parameters are incorrect.

Remarks

The resource module ID and the resource ID of the image of the dialog item can be set by DGSetItemImageResource. The resource ID of the image can be retrieved by DGGetItemImageId.

The resource module of the dialog can be set and retrieved by DGSetDialogResourceModule or DGSetDialogResourceModules and DGGetDialogResourceModule or DGGetDialogIconResourceModule, respectively.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
