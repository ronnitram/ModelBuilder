DGListSetHeaderItemMinSize

Sets the minimal size (width) of a header item of a list box item.

    void DGListSetHeaderItemMinSize (
      short        dialId,
      short        item,
      short        headerItem,
      short        minSize
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
headerItem
1-based index of the header item.
 
short
The minimal size of the header item in pixels.

Remarks

The list box header is displayed at the top of the list box. The header can be divided into more header items. The header is used to display information about the list box columns and can be configured to enable the user to dynamically adjust the size and the order of the list box columns. The number of the header items can be retrieved by DGListGetHeaderItemCount. The minimal width of a header item can be retrieved by DGListGetHeaderItemMinSize.

A header item can be resized by mouse interaction, only if the isSizeable flag is set to true on the list box header item. This flag can be set and retrieved by the DGListSetHeaderItemFlags and DGListGetHeaderItemFlags functions respectively. The defult minimal size of a header item is 0.
