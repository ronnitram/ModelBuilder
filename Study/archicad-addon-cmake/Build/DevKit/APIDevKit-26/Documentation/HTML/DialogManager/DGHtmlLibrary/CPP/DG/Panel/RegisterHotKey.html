Panel::RegisterHotKey

Registers a hotkey to the panel. 

short RegisterHotKey (
    Key::Special        keyCode,
    Key::Modifier       mod1 = Key::NoModifier,
    Key::Modifier       mod2 = Key::NoModifier,
    Key::Modifier       mod3 = Key::NoModifier
);
short RegisterHotKey (
    short               keyCode,
    Key::Modifier       mod1 = Key::NoModifier,
    Key::Modifier       mod2 = Key::NoModifier,
    Key::Modifier       mod3 = Key::NoModifier
);

Parameters

keyCode
A character code or a predefined key code.
mod1
A key modifier.
mod2
A key modifier.
mod3
A key modifier.

Return Value

1-based identifier of the hotkey if the registration is successful, or zero if an error occurs.
