DragSourceEventProperty::ModifyBaseFormat

Used to extend tree view, list view or list box drag and drop. 

bool ModifyBaseFormat (
    DGDragFormat        format,
    DragDrop::Scope     scope
);

Parameters

format
DGDragFormat type parameter. Used to specify the drag format in character style like: 'TEXT', 'FILE' etc.
scope
Scope of the data: DragDrop::Item, DragDrop::Dialog, DragDrop::Module, DragDrop::Application, DragDrop::System.

Return Value

true if base format modification was successful, otherwise false.

Description

This member can be used to slightly modify tree view, list view or list box drag and drop behaviour. The mentioned controls by default have only dialog wide drag and drop scope. By using this member function it is possible to modify this scope.

Example

// Method Example:
void	DragTab2Observer::ListBoxDragStarted (const DG::ListBoxDragSourceEvent& ev, UShort* /*effect*/)
{
	if (modelessMode) {
		DG::DragSourceEventProperty* evProp = ev.GetDragData ();
		evProp->ModifyBaseFormat ('LBDR', DG::DragDrop::Module);
	}
}

Modifying the base 'LBEX' format to 'LBDR' of a list box control.
