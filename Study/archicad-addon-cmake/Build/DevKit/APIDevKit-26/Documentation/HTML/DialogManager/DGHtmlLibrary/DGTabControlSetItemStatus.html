DGTabControlSetItemStatus

Sets the status of a tab control item.

    void DGTabControlSetItemStatus (
      short  dialId,
      short  item,
      short  tabItem,
      bool   enable
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
tabItem
1-based index of the tab control item.
 
enable
The new status of the item:
Value Meaning
true The item is enabled.
false The item is disabled.

Remarks

This function enables or disables all dialog items on the tab page attached to the specified tab control item as well. A disabled tab page cannot be selected by user action in case of a normal tab control (DG_TT_NORMALTAB). However, from program code even disabled tab pages can be activated by DGTabControlSelectItem.

Simple tab controls (DG_TT_SIMPLETAB) are controlled by other dialog items therefore it is responsibility of the application not to let a disabled tab page be activated.
