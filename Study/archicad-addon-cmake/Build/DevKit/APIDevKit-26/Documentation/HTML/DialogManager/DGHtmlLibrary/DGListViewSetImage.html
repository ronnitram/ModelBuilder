DGListViewSetImage

Sets the image associated with a list view control.

    void DGListViewSetImage (
      short         dialId,
      short         item,
      short         imageType,
      const void*   imageData,
      short         nImages,
      short         nRows
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
imageType
Type of the image:
imageType imageData
DG_LVIT_NOIMAGE No image is defined for this item.
DG_LVIT_ICON
 Macintosh only: 
Icon, referenced with NSImage* cast to void*.
 Windows only: 
Icon, referenced with HICON cast to void*.
DG_LVIT_PICTURE
 Macintosh only: 
Picture, referenced with NSImage* cast to void*.
 Windows only: 
Picture, referenced with HBITMAP cast to void*.
 Windows only:
DG_LVIT_DIBITMAP Device Independent Bitmap, referenced with BITMAPINFO* cast to void*.
 
imageData
The image data, depending on the imageType parameter.
 
nImages
Number of the sub-images in the image.
 
nRows
Number of the rows of sub-images in the image.

Remarks

There are two ways to define a list view item image:

  • Assign images to the items individually by DGListViewSetItemImage, DGListViewSetItemImageId or DGListViewSetItemImageResource. These images can be of different types and sizes for each list view item.
  • Associate one image with the list view itself by DGListViewSetImage, DGListViewSetImageId or DGListViewSetImageResource, divide it into smaller sub-images of the same size by giving the number of rows with nRows and the total number of sub-images with nImages. Assign a sub-image to the list view item by DGListViewSetItemImageIndex. The image index parameter of the function is the simple 1-based row-by-row index of a sub-image. In this case the type of the item's image will be DG_LVIT_IMAGEINDEX.

If an image is associated with the list view control by DGListViewSetImage, all the list view items must have DG_LVIT_IMAGEINDEX image type, i.e. all the list view item images are sub-images of that image (all item images must be assigned to the item with the DGListViewSetItemImageIndex function). If there is no associated image with the list view, DGListViewSetItemImageIndex cannot be used.

The image of a list view control can be set either with its image data by DGListViewSetImage, or with its resource ID by DGListViewSetImageId or DGListViewSetImageResource. If the image was set by DGListViewSetImage the image data can be retrieved by DGListViewGetImage, in this case DGListViewGetImageId returns DG_NO_IMAGE and DGListViewGetImageResource returns InvalidResModule.
