DGSetTabPageMinSize

Sets the allowed minimum width and height of the tab page.

    void DGSetTabPageMinSize (
      short  dialId
      short  hMinSize
      short  vMinSize
    );

Parameters

dialId
ID of the tab page.
 
hMinSize
Minimum horizontal size of the tabpage in pixels.
 
vMinSize
Minimum vertical size of the tabpage in pixels.

Remarks

The minimal width and height of a tab page initially equals to the tab page size given in the GRC script or the creation size if the tab page was created in the program code. These values can be modified by DGSetTabPageMinSize.

Tab pages can be resized by the DGSetTabPageSize function. Growable tab pages are also resized by modifying the tab control size with DGGrowItem which they are assigned to.

The minimal sizes of a tab page can be retrieved with the DGGetTabPageMinSize function.
