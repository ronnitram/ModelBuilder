RelativePos Enum

The relative position for inserting the dragged items at list box, list view and tree view controls. 

enum RelativePos
{
    InsertOn =  DG_DRAG_INSERT_ON,
    InsertUnder =  DG_DRAG_INSERT_UNDER
};

Value Definitions

InsertOn
The application should insert the dragged items onto the target item.
InsertUnder
The application should insert the dragged items under the target item.

Description

At drag over, drag enter, drag drop events, this value is set in the drag event, to tell the application to where it should insert the dragged items.
