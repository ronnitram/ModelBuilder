DGSetFocus

Sets the keyboard focus to the given focusable dialog item.

    void DGSetFocus (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item. DG_NO_ITEM removes the keyboard focus from the currently focused item.

Remarks

The keyboard focus and the focusable control categories in DG essentially have the same meanings as in the Macintosh operating system (i.e. the same type of controls are focusable in DG and on Macintosh). On Windows, however, more controls can have keyboard focus. Focusable controls of DG form a subset of focusable controls of Windows. The DGGetFocus and DGSetFocus functions work on focusable controls of DG only.

The following controls are focusable in DG: Date & Time Control, Edit Control, List Box, List View, Rich Edit Control, Tree View.

If the newly focused item is an edit control, its entire text will be selected and the caret is moved to the end of the selection. For rich edit controls, the previous selection, if any, is highlighted.

After calling DGSetFocus with DG_NO_ITEM value there will be no focused item in the system until the keyboard focus is set again by user action or by this function.
