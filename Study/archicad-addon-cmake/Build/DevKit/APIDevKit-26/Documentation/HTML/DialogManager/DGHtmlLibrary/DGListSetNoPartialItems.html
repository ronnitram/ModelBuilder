DGListSetNoPartialItems

Resizes a list box to hide the last partially visible list item.

    void DGListSetNoPartialItems (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.

Remarks

Use this function only if the DG_LT_NOPARTIALITEMS flag is not already set.

A list can be resized with pixel precision by DGGrowItem. As a result of resizing, the last list item can often become partially visible. The same can happen when the list is created from resource or by the DGAppendDialogItem function call and the height of its client area is not the multiple of the list item height. In most cases, these partially visible last items are unacceptable. There are two ways to avoid this problem:

  • Call DGListSetNoPartialItems after the list is created and every time when it is resized by DGGrowItem.
  • Set the DG_LT_NOPARTIALITEMS flag in the GRC, or combine it to the itemSubType parameter of DGAppendDialogItem. This flag tells DG to call DGListSetNoPartialItems internally whenever the size of the list is changed.
