DGListGetHeaderItemFlags

Retrieves the justification, truncation and resizeability settings of a header item of a list box item.

    void DGListGetHeaderItemFlags (
      short        dialId,
      short        item,
      short        headerItem,
      short*       style,
      bool*        isSizeable
   );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
headerItem
1-based index of the header item.
 
style
Output parameter: Item style which is DG_IS_DEFAULT or any bitwise OR (|) combination of the justification and truncation flags:
 
Justification of the content in the header item:
Value Meaning
DG_IS_LEFT The item content is aligned to the left.
DG_IS_CENTER The item content is centred.
DG_IS_RIGHT The item content is aligned to the right.
 
How the text of the header item should be truncated if it does not fit into the item. Possible values are:
Value Meaning
DG_IS_NOTRUNC No truncation.
DG_IS_TRUNCEND Item content is truncated at the end..
DG_IS_TRUNCMIDDLE Item content is truncated in the middle. (Leaves the same number of characters from both ends of the text.)
 
isSizeable
Output parameter: if the size of the header item can be modified by mouse interaction, this flag is set.

Remarks

The list box header is displayed at the top of the list box. The header can be divided into more header items. The header is used to display information about the list box columns and can be configured to enable the user to dynamically adjust the size and the order of the list box columns. The number of the header items can be retrieved by DGListGetHeaderItemCount. Special properties of a header item can be set by DGListSetHeaderItemFlags.

If the isSynchron flag is set to true on the list box header, the properties of the header items are synchronized with the corresponding tabulator fields by DG, otherwise they should be handled independently by the application. These properties are: the number and size of the header item, the font type and size, the alignment, the truncation of the text of the header item. If this flag is set, these properties are automatically updated when a property of a tabulator field is changed.
If the isSynchron flag is set to false and the style flag is set to DG_IS_DEFAULT, the defult settings are applied for the header item ie.: left alignment and no truncation. If the isSynchron flag is set to false the following functions won't effect the list box header items DGListSetHeaderItemFlags, DGListSetHeaderItemCount, DGListSetHeaderItemFont, DGListSetHeaderItemSize.
The header flags can be set and retrieved by the DGListSetHeaderFlags and DGListGetHeaderFlags functions respectively.

A header item can contain an icon and/or text also, but only one icon can appear in a header item. The text and the icon of a header item can be set and retrieved by the DGListSetHeaderItemText, DGListSetHeaderItemIcon and DGListGetHeaderItemText DGListGetHeaderItemIcon functions respectively. If a header item contains an icon and text also, the icon will be positioned either at the beginning or at the end of the text according to the justification. (The DG_IS_LEFT justification is applied for the header item containing an icon and text also, even if the DG_IS_CENTER justification was given.) For a multi-line text the truncation cannot be applied. The font style and size of each header item can be set by the DGListSetHeaderItemFont function.

The default setting for the isSizeable flag is false. If the isSizeable flag is set to true on the list box header, the header item can be resized by mouse interaction. The minimal size of a header item can be set and retrieved by the DGListSetHeaderItemMinSize and DGListGetHeaderItemMinSize functions respectively.
