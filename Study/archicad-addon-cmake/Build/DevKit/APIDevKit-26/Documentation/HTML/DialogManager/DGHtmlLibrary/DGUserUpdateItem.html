DGUserUpdateItem

Updates the invalid area of a user item. It is removed in version DGLib 2.2.0.

    void DGUserUpdateItem (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.

Remarks

This function updates the invalid area of the user item immediately regardless of the partial or normal update type of the user item. After updating the user item its entire rectangle is validated.

If the entire user item should be redrawn use the DGUserRedrawItem function.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
