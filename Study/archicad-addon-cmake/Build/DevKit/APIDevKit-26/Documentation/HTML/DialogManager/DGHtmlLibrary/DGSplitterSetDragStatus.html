DGSplitterSetDragStatus

Sets the drag status of a splitter control.

    void DGSplitterSetDragStatus (
      short  dialId,
      short  item,
      bool   isEnabled
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
isEnabled
The new drag status of the splitter:
Value Meaning
true The drag status of the splitter control is enabled.
false The drag status of the splitter control is disabled.

Remarks

Splitter controls make possible to resize the dialog interior division, without increasing the whole dialog size.

The drag status can also be enabled or disabled by the DGSplitterEnableDrag and the DGSplitterDisableDrag functions respectively.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
