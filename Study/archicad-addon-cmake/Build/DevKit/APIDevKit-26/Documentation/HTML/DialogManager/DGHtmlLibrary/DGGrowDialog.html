DGGrowDialog

Resizes a dialog with the given relative values.

    void DGGrowDialog (
      short  dialId,
      short  hGrow,
      short  vGrow
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
hGrow
Horizontal growth (shrink if hGrow is negative).
 
vGrow
Vertical growth (shrink if vGrow is negative).

Remarks

DGGrowDialog resizes a dialog with relative values. Use the DGSetDialogSize function to set the dialog width and height to given absolute values.

A dialog cannot be shrunk to arbitrarily small only until it reaches its minimal sizes given in GRC or re-set by DGSetDialogMinSize.

A dialog can be resized only in the directions allowed by the dialog grow type flag, given in GRC or specified by DGSetDialogGrowType.

During the DGGrowDialog call a DG_MSG_GROW message is sent to the dialog callback function. When handling this message, the application can move and/or resize dialog items according to the new size of the dialog. When the callback function receives this message, the dialog already has its new size.

Dialog positions and sizes in DG are stored in system preferences (Macintosh) or in the system registry (Windows). This means that a dialog remembers its previous size and position when it is reopened next time, even after exiting and restarting the application.

Current size of a dialog can be retrieved by DGGetDialogSize.
