DGWheelMsgData

If the dialog callback function receives a DG_MSG_WHEELTRACK message for a dialog item, the msgData parameter is a pointer to a DGWheelMsgData structure that contains information about the wheel event.

    struct DGWheelMsgData {
      short  subMessage;
      short  itemId;
      short  xPos;
      short  yPos;
      short  xTrackValue;
      short  yTrackValue;
      short  keyFlag;
    };

Members

subMessage
It can be one of the following values:
subMessage meaning
DG_TSM_TRACKENTER Tracking the wheel above the specified dialog item has just begun.
DG_TSM_TRACKING The user is tracking the wheel above the specified dialog item.
DG_TSM_TRACKEXIT Tracking the wheel above the specified dialog item has been finished.
 
itemId
1-based index of the dialog item, otherwise zero if there was no dialog item where the event occured.
 
xPos
Horizontal mouse position relative to the dialog rectangle.
 
yPos
Vertical mouse position relative to the dialog rectangle.
 
xTrackValue
Value of the wheel rotation.
 
yTrackValue
Value of the wheel rotation.
 
keyFlag
Status of the modifier keys. This value can be any combination of the following constants:
flag Macintosh Windows
DG_MOD_COMMAND Command (Apple) Ctrl
DG_MOD_OPTION Option Alt
DG_MOD_SHIFT Shift Shift

Remarks

In case of DG_TSM_TRACKENTER and DG_TSM_TRACKEXIT submessages the xPos, yPos, xTrackValue, yTrackValue, keyFlag are zero.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
