DGListViewGetImageId

Retrieves the resource ID of the image associated with a list view control.

    short DGListViewGetImageId (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 

Return Values

The image resource ID of the list view control. The resource ID of the picture is returned if the type of the image is DG_LVIT_PICTUREID and the icon ID is returned if the type of the image is DG_LVIT_ICONID (for possible value of icon ID see DGListViewSetImageId for details). DG_NO_IMAGE is returned if the image of the list view control was set by DGListViewSetImage, if the type of the image is DG_LVIT_IMAGEINDEX or if the function parameters are incorrect.

Remarks

There are two ways to define a list view item image:

  • Assign images to the items individually by DGListViewSetItemImage, DGListViewSetItemImageId or DGListViewSetItemImageResource. These images can be of different types and sizes for each list view item.
  • Associate one image with the list view itself by DGListViewSetImage, DGListViewSetImageId or DGListViewSetImageResource, divide it into smaller sub-images of the same size by giving the number of rows and the total number of sub-images. Assign a sub-image to the list view item by DGListViewSetItemImageIndex. The image index parameter of the function is the simple 1-based row-by-row index of a sub-image. In this case the type of the item's image will be DG_LVIT_IMAGEINDEX. The image data, image ID, or image resource module associated with the list view control can be retrieved by DGListViewGetImage, DGListViewGetImageId or DGListViewGetImageResource.
