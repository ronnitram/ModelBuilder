DGDoubleToString

Converts a double value to a formatted string depending on the units currently set in DG.

    GS::UniString DGDoubleToString (
      double  value,
      short   type
    );

Parameters

value
A double value to convert.
 
type
Type of double value. Possible values of type (see subtypes of edit controls): DG_ET_REAL, DG_ET_ANGLE, DG_ET_POLARANGLE, DG_ET_LENGTH, DG_ET_AREA, DG_ET_VOLUME/tt>, DG_ET_MM_OR_PT.

Return Values

The formatted string in Unicode format.

If the function parameters are incorrect, zero is returned.

Remarks

This function converts a double value to a formatted string depending on the units currently set in DG. To use different unit settings for the conversion, call DGDoubleToStringUnit. For the description of units in DG see DGUnitData.

If value is greater than DG_DBL_MAX or less than -DG_DBL_MAX, five asterisks ("*****") are displayed instead of the number.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
