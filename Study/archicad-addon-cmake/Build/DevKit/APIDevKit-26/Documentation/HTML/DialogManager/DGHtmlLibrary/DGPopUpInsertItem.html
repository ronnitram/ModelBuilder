DGPopUpInsertItem

Inserts an item into a pop-up control.

    void DGPopUpInsertItem (
      short  dialId,
      short  item,
      short  popupItem
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
popupItem
1-based index of the item in the pop-up control, before which the new item should be inserted. DG_POPUP_TOP specifies the first item of the pop-up, DG_POPUP_BOTTOM inserts the new item after the last item of the pop-up. If popupItem is the number of items in the pop-up plus 1, it has the same effect as DG_POPUP_BOTTOM.

Remarks

If the pop-up control was empty before inserting the new item, this item will become selected.

The new item will have the following default properties:

Property Value
Icon ID DG_NO_ICON
Long 0
Status DG_IS_ENABLE
Style DG_IS_PLAIN
Text empty

These default item properties can be modified by the pop-up control specific functions.

Use the DGPopUpDisableDraw and DGPopEnableDraw functions if you insert more items to avoid unwanted screen updates. To insert a separator line item use DGPopUpInsertSeparator.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
