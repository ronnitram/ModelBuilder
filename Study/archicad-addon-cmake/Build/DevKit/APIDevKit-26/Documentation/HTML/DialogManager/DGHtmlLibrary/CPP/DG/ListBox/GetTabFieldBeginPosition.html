ListBox::GetTabFieldBeginPosition

Retrieves the begin position of a tabulator field. 

short GetTabFieldBeginPosition (short tabIndex) const;

Parameters

tabIndex
1-based index of the tabulator field.

Return Value

The begin position of the tabulator field in pixels, relative to the client rectangle of the list box. This parameter can be nullptr.
