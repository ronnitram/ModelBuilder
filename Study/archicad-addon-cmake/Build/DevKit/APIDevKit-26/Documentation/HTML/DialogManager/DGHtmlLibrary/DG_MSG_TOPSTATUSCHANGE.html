DG_MSG_TOPSTATUSCHANGE

A modeless dialog is activated or deactivated or a palette dialog is activated.

Message parameters

dialId
ID of the dialog.
 
item
Not used, always zero.
 
msgData
Pointer to a DGTopStatusMsgData structure cast to DGMessageData.

Return Values

Not used, should be set to zero.

Remarks

The dialog callback function receives this message as a result of a user action or of a DGActivateModelessDialog function call. If a dialog is activated, a DG_MSG_TOPSTATUSCHANGE message with DG_TSM_TOPSTATUSLOST submessage is sent to the currently active dialog (which will be deactivated), then a DG_MSG_TOPSTATUSCHANGE message with DG_TSM_TOPSTATUSGAINED submessage is sent to the dialog which is being activated.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
