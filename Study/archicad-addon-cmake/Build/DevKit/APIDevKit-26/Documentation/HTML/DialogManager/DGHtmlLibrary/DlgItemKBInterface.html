Dialog Item Keyboard Interface

The following tables describe the keyboard interfaces of the focusable dialog items of DG.

Keyboard interface for Date & Time Controls

Key Action
UP Decreases the value of the highlighted item by one.
DOWN Increases the value of the highlighted item by one.
'0'..'9' Sets the value of the highlighted item.
 Windows only:
'-' Decreases the value of the highlighted item by one.
'+' Increases the value of the highlighted item by one.
HOME Sets the value of the highlighted item to the minimum.
END Sets the value of the highlighted item to the maximum.
LEFT Highlights the previous item in the control.
RIGHT Highlights the next item in the control.

Keyboard interface for Edit Controls and Rich Edit Controls

Key Action
UP Moves the caret to the previous line (multi-line and rich edit controls only).
DOWN Moves the caret to the next line (multi-line and rich edit controls only).
LEFT Moves the caret one character left.
RIGHT Moves the caret one character right.
BACKSPACE Deletes the character left to the caret.
RETURN Breaks the line at the caret position and creates a new line (multi-line and rich edit controls only).
Any character Inserts the typed character into the control.
 Macintosh only:
COMMAND + 'X' Cut the selection into the clipboard.
COMMAND + 'C' Copy the selection into the clipboard.
COMMAND + 'V' Paste the content of the clipboard into the control.
 Windows only:
CTRL + 'X' Cut the selection into the clipboard.
CTRL + 'C' Copy the selection into the clipboard.
CTRL + 'V' Paste the content of the clipboard into the control.
SHIFT + DELETE Cut the selection into the clipboard.
CTRL + INSERT Copy the selection into the clipboard.
SHIFT + INSERT Paste the content of the clipboard into the control.
HOME Moves the caret to the beginning of the current line.
END Moves the caret to the end of the current line.
PGUP Scrolls a page up (multi-line and rich edit controls only).
PGDOWN Scrolls a page down (multi-line and rich edit controls only).
CTRL + LEFT Moves the caret one word left.
CTRL + RIGHT Moves the caret one word right.
CTRL + HOME Moves the caret to the beginning of the content.
CTRL + END Moves the caret to the end of the content.
DELETE Deletes the character right to the caret.
SHIFT + arrow keys Selection according to the caret move.

Keyboard interface for List Boxes

Key Action
UP Selects the previous list item.
DOWN Selects the next list item.
PGUP Scrolls a page up.
PGDOWN Scrolls a page down.
Any character Looks for the list item beginning with the pressed character.
 Windows only:
LEFT Selects the previous list item.
RIGHT Selects the next list item.
HOME Selects the first list item.
END Selects the last list item.
SHIFT + arrow keys Selects items in a multiple-selection list.

Keyboard interface for List Views

Key Action
UP Selects the list view item above the currently selected one.
DOWN Selects the list view item below the currently selected one.
LEFT Selects the list view item left to the currently selected one.
RIGHT Selects the list view item right to the currently selected one.
PGUP Scrolls a page up.
PGDOWN Scrolls a page down.
Any character Looks for the list view item beginning with the pressed character.
 Windows only:
HOME Selects the first list view item.
END Selects the last list view item.

Keyboard interface for Tree Views

Key Action
UP Selects the previous tree view item.
DOWN Selects the next tree view item.
PGUP Scrolls a page up.
PGDOWN Scrolls a page down.
Any character Looks for the tree view item beginning with the pressed character.
 Macintosh only:
HOME Scrolls to the first tree view item.
END Scrolls to the last tree view item.
COMMAND + LEFT Collapses the selected parent tree view item(s).
COMMAND + RIGHT Expands the selected parent tree view item(s).
 Windows only:
HOME Selects the first tree view item.
END Selects the last tree view item.
LEFT Collapses the selected parent tree view item.
RIGHT Expands the selected parent tree view item.
CTRL + UP Scrolls a line up.
CTRL + DOWN Scrolls a line down.
CTRL + HOME Scrolls to the first tree view item.
CTRL + END Scrolls to the last tree view item.
