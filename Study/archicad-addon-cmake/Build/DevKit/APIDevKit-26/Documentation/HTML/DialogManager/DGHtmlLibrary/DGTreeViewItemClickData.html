DGTreeViewItemClickData

If the dialog callback function receives a DG_MSG_TREEITEMCLICK message for a tree view item, the msgData parameter is a pointer to a DGTreeViewItemClickData structure that contains information about the user action.

    struct DGTreeViewItemClickData {
      Int32  treeItem;
      short  partId;
    };

Members

treItem
The 1-based ID of the tree view item.
 
partId
Tells which part of the tree view item was clicked. Possible values are:
partId Meaning
DG_TVPI_STATEICON The user clicked the state icon of the tree view item.
DG_TVPI_NORMALICON The user clicked the normal icon of the tree view item.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
