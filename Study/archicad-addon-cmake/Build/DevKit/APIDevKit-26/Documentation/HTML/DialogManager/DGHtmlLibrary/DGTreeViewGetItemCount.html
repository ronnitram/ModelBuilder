DGTreeViewGetItemCount

Retrieves the number of items in a tree view.

    Int32 DGTreeViewGetItemCount (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.

Return Values

Number of the items in the tree view.

Requirements

Version: DG 1.1.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
