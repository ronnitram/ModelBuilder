DGTreeViewSetItemIconResource

Sets the icon ID of the state or the normal icon in a tree view item and the resource module ID of the corresponding icon.

    void DGTreeViewSetItemIconResource (
      short        dialId,
      short        item,
      Int32        treeItem,
      short        iconType,
      GSResModule  resModule,
      short        iconId
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
treeItem
ID of the tree view item. Use DG_ALL_ITEMS to change all of the given type of icons.
 
iconType
The type of icon to be changed in the tree view item:
Value Meaning
DG_TVIT_NORMAL The normal icon of the item.
DG_TVIT_STATE The state icon of the item.
 
resModule
The resource module ID containing the icon.
 
iconId
New icon ID for one of the tree view item's icons, depending on the iconType parameter.
iconID can be any positive resource ID or one of the following predefined constants:
 
Value Meaning Macintosh Windows
DG_NO_ICON  No icon, empty space.
DG_TVIC_FLOPPY  Floppy icon.
DG_TVIC_HDDRIVE  HDDrive icon.
DG_TVIC_CDDRIVE  CDDrive icon.
DG_TVIC_FOLDER  Folder icon.
DG_TVIC_FILE  File icon.

Remarks

A tree view item can have two types of icon: the state icon and the normal icon. The icons are always on the left side of the item text. If the tree view has both types of icon, the state icon is the leftmost.

The number of different state icons in a tree view is limited to 12. There is no such limitation for the normal icons.

There are two ways to set an icon of a tree view item. One possibility is to specify the icon with its resource ID by DGTreeViewSetItemIconResource or by DGTreeViewSetItemIconId, the other is to set it with its icon handle by DGTreeViewSetItemIcon. If the icon was set by its resource ID, the icon ID can be retrieved by DGTreeViewGetItemIconId and the resoruce module ID can be retrieved by DGTreeViewGetItemIconResource, in this case DGTreeViewGetItemIcon returns nullptr.

If the icon is not set yet, an empty area is displayed instead of shifting the content of the item to the left.

Either of the icon types can be omitted by setting its size to zero either from resource or by a call to DGTreeViewSetIconSize. These settings apply to the whole tree view, i.e. to all tree items.

If the icon is located in the resource module of the dialog, the icon can also be set with DGTreeViewSetItemIconId. The resource module of the dialog can be set and retrieved by DGSetDialogResourceModule or DGSetDialogResourceModules and DGGetDialogResourceModule or DGGetDialogIconResourceModule, respectively.
