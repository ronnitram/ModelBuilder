DGShowContextHelpWithAnchor

With this function you can launch context sensitive help browser by the parameters.

    bool DGShowContextHelpWithAnchor (
      Int32                  developerId,
      Int32                  localId,
      const GS::UniString&   helpAnchor
    );

Parameters

developerId
Input parameter: The DeveloperID part of the AddOn's MDID.
 
localId
Input parameter: The LocalID part of the AddOn's MDID.
 
helpAnchor
Input parameter: The anchor to be reached.

Return Values

Value Meaning
true DG was able to find the location of the registered help engine.
false DG was unable to find the registered help engine, or some other problems occured. Due to the problems, no action was taken.

Requirements

Version: DG 2.1.0.66 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
