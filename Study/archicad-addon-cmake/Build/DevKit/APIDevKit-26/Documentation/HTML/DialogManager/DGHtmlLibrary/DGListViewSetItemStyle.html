DGListViewSetItemStyle

Sets the style of a list view item text.

    void DGListViewSetItemStyle (
      short  dialId,
      short  item,
      short  listItem,
      short  style
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
listItem
1-based index of the list item. DG_LIST_TOP specifies the first item of the list, DG_LIST_BOTTOM specifies the last item of the list. Use DG_ALL_ITEMS to change the style of all items.
 
style
One of the following values:
Value Meaning
DG_IS_PLAIN Item text is plain.
DG_IS_BOLD Item text is bold.
DG_IS_ITALIC Item text is italic.
DG_IS_UNDERLINE Item text is underlined

Remarks

Before the text style is set by DGListViewSetItemStyle, all the list view items have the text style of the list view defined in the resource or set by the DGSetItemFont function.

Only the text style can be set item-by-item. The font size can be changed by DGSetItemFont, for all the items.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
