DGUserSetPartialUpdate

This function is obsolete, use the DGUserEnablePartialUpdate function instead. It is removed in version DGLib 2.2.0.

Sets or clears the partial update flag of a user item.

    void DGUserSetPartialUpdate (
      short  dialId,
      short  item,
      short  flag
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
flag
Value Meaning
0 Clear the partial update flag.
1 Set the partial update flag.

Remarks

On Windows, user items usually get only DG_MSG_UPDATE messages when they should be updated. After you call DGUserSetPartialUpdate and set the partial update flag to 1, user items will get DG_MSG_PARTIALUPDATE messages, too. The partial update flag can also be set in the GRC. It is highly recommended to set this flag if updating the user item is a very time consuming process.

If full drag mode is on in the system, DG sends DG_MSG_PARTIALUPDATE messages while the user moves or sizes the parent dialog of the user item and sends one DG_MSG_UPDATE as soon as the user finishes moving or sizing. When the dialog callback function receives a DG_MSG_PARTIALUPDATE message, the application can do a partial update, e.g. you can erase the background and draw only the frame of the item while it is resized, but when a user item receives DG_MSG_UPDATE you must redraw it entirely.

On Macintosh, this function has no effect: user items always get DG_MSG_UPDATE messages only.
