DGUserGetBackgroundColor

Retrieves the background color of the user item.

    bool DGUserGetBackgroundColor (
      short    dialId,
      short    item,
      UShort*  red,
      UShort*  green,
      UShort*  blue

    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
red
Output parameter: Red component of the RGB color. This parameter can be nullptr.
 
green
Output parameter: Green component of the RGB color. This parameter can be nullptr.
 
blue
Output parameter: Blue component of the RGB color. This parameter can be nullptr.

Return Values

Once the background color was modified by the application the return value is true, otherwise false.

Remarks

The default background color is applied when DG erases an invalid area of the user item.

Use DGUserSetBackgroundColor to modify the current background color.

If any of the pointer parameters is nullptr, that component is not returned.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
