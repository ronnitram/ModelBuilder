DGTabControlDeleteItem

Deletes an item from the tab control.

    bool DGTabControlDeleteItem (
      short  dialId,
      short  item,
      short  tabItem
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
tabItem
1-based index of the tab control item to be deleted. DG_TAB_FIRSTITEM specifies the first item of the tab control, DG_TAB_LASTITEM specifies the last item of the tab control. Use DG_TAB_ALLITEMS to delete all items.

Return Values

true if the function was successful otherwise false.

Remarks

Before deleting a tab control item the assigned tab page must be destroyed with the DGDestroyTabPage function.

To insert new items to the tab control use the DGTabControlInsertItem function. A tab page can be assigned to a tab control item either with the DGCreateTabPage or with the DGCreateBlankTabPage function.

The number of tab control items can be retrieved by DGTabControlGetItemCount.
