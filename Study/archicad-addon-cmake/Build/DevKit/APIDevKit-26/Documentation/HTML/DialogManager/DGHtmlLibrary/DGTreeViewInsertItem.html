DGTreeViewInsertItem

Inserts a new tree item into a tree view.

    Int32 DGTreeViewInsertItem (
      short  dialId,
      short  item,
      Int32  parentItem,
      Int32  treeItem
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
parentItem
ID of the parent item under which the new item should be placed as a child item. DG_TVI_ROOT specifies the root of the tree view. If treeItem is DG_TVI_ROOT, this parameter is ignored.
 
treeItem
The tree view item after which the new item should appear on the same hierarchy level. You can use the following predefined values, too.
Value Meaning
DG_TVI_ROOT The new item will be the root of the tree view.
DG_TVI_TOP Insert the new item at the top of the level below parentItem.
DG_TVI_BOTTOM Insert the new item at the bottom of the level below parentItem.

Return Values

ID of the new tree view item, or DG_TVI_NONE if the function failed.

Remarks

The new item will have the following default properties:

Property Value
Color The system default foreground (text) color.
Icon (normal) DG_NO_ICON
Icon (state) DG_NO_ICON
Long 0
Status DG_IS_ENABLE
Style The style of the font of the tree view.
Text empty

These default item properties can be modified by the tree view specific functions.

Use the DGTreeViewDisableDraw and DGTreeViewEnableDraw functions if you insert more items to avoid unwanted screen updates.

Requirements

Version: DG 1.1.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
