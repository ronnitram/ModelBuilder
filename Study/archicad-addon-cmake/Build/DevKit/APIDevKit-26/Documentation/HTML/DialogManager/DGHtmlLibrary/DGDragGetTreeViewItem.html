DGDragGetTreeViewItem

Gets the specified dragged tree view item.

    Int32 DGDragGetTreeViewItem (
      ULong  index
    );

Parameters

index
0 based index of the tree view item being dragged. It is the index of the tree view item in the dragged tree view item's array.
 

Return Values

Tree view index of the specified dragged item.

Remarks

When tree view items are being dragged, this function can be used to get the tree view item index of the dragged tree view item. To identify the drag source item use the DGDragGetSourceDialog, DGDragGetSourceItem functions.

Requirements

Header: DG.h
Import Library: DG.lib [WIN]
