DGSetEditUpdateDelay

Sets the update delay time of edit controls.

    void DGSetEditUpdateDelay (
      ULong  delay
    );

Parameters

delay
The update delay time of edit controls in milliseconds.

Remarks

In the DG dialog manager edit controls have a special behavior: by default edit controls are validated automatically in a short time (the update delay time) after the last key down. If the update delay time is set to large enough, edit controls are practically validated only if they lose the keyboard focus. You can reach the same behavior by setting the noUpdate flag for an edit control item in the GDLG resource.

When the DG dialog manager is initialized, the update delay time is set to the value of 1000, i.e. 1 second.
