DG_MSG_CLICK

A dialog item is clicked.

Message parameters

dialId
ID of the dialog or tab page.
 
item
1-based index of the clicked dialog item or DG_CLOSEBOX if the close box of a modeless or a palette dialog was clicked.
 
msgData
Item type msgData
Icon Not used, always zero.
List Box Pointer to a DGListMsgData structure cast to DGMessageData.
Picture Not used, always zero.
Push Button Not used, always zero.
Static Text Not used, always zero.
Tab Control 1-based index of the tab that the user clicked.
Tree View The ID of the clicked tree item.
User Item Pointer to a DGUserMsgData structure cast to DGMessageData.

Return Values

Item type Value
Icon Not used, should be set to zero.
List Box Not used, should be set to zero.
Picture Not used, should be set to zero.
Push Button Set it to zero if you do not want to close the dialog. If you want to close the dialog set it to a non-zero value (usually the index of the clicked dialog item). This value will be passed to the callback function with the DG_MSG_CLOSE message.
Static Text Not used, should be set to zero.
Tab Control Set it to 1 to avoid changing tab page, or zero to let changing tab page.
Tree View Set it to 1 to keep the previous selection and break further processing of the click event.
User Item Not used, should be set to zero.

Remarks

When the user clicks a push button on a tab control page, the dialog is not closed even if the application returns a non-zero value in the dialog callback function. The user can close a dialog only if she/he clicks a push button of the main dialog.

When the user clicks the close box of a modeless or palette dialog, the item parameter of the message equals to DG_CLOSEBOX. In this case return this value in the dialog callback function to close the dialog.

If the user clicks the mouse on a selected list item of a list box or user list, the control receives a DG_MSG_CLICK message only. However, when the user clicks on a non-selected item, DG_MSG_CLICK is preceded by a DG_MSG_CHANGE message.

Tree view controls receive DG_MSG_CLICK message and a DG_MSG_TREEITEMCLICK message if the user clicks the normal or state icon of a tree item.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
