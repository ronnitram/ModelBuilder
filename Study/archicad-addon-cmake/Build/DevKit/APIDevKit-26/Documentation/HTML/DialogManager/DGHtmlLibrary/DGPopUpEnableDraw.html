DGPopUpEnableDraw

Enables screen update of a pop-up control. It is used after inserting, deleting or modifying more pop-up items.

    void DGPopUpEnableDraw (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.

Remarks

It is advised to disable screen updates for as short period as possible, i.e. place DGPopUpDisableDraw right before the first pop-up modification and call DGPopUpEnableDraw right after the last pop-up modification. These two functions must always be used in pairs.

DGPopUpEnableDraw itself does not update the pop-up control. You should either call DGInvalidateItem or DGRedrawItem.

Requirements

Version: DG 1.1.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
