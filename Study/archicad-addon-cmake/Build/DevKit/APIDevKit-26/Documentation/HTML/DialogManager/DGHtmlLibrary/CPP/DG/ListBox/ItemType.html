ListBox::ItemType Enum

Special list box item, or collection of items. 

enum ItemType
{
    AllItems =  DG_ALL_ITEMS,
    TopItem =  DG_LIST_TOP,
    BottomItem =  DG_LIST_BOTTOM
};

Value Definitions

AllItems
All items of the list box.
TopItem
Top item of the list box.
BottomItem
Bottom item of the list box.

Description

Some methods expect listItem, which can be one of these enumeration values too. Typically, set methods accept all values (including AllItems), get methods accept only TopItem and BottomItem.
