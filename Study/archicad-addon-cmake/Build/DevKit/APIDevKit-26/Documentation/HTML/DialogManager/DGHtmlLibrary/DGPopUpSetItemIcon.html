DGPopUpSetItemIcon

Sets the icon data of a pop-up control item.

    void DGPopUpSetItemIcon (
      short        dialId,
      short        item,
      short        popupItem,
      const void*  iconData
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
popupItem
1-based index of the item in the pop-up control. DG_POPUP_TOP specifies the first item of the pop-up. DG_POPUP_BOTTOM specifies the last item of the pop-up. Use DG_ALL_ITEMS to change the icon of all items.
 
iconData
 Macintosh only: 
NSImage* pointer of the new icon cast to void*.
 Windows only: 
HICON handle of the new icon cast to void*.

Remarks

A pop-up control item can have only one icon. The icon is always displayed on the left side of the pop-up item text. If there is no icon in the pop-up control item, on Macintosh the item text is aligned to the left, on Windows, however, the text position is kept.

 Windows only:

Left position of the item text can be defined in GRC only.

The icon data of a pop-up control item can be retrieved by DGPopUpGetItemIcon.

An icon can also be specified to the pop-up control item with its resource ID: see DGPopUpSetItemIconId and DGPopUpSetItemIconResource for details.

Requirements

Header: DG.h
