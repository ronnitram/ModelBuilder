DGGetNextModalParent

 Windows only:

Retrieves the parent window of the next invoked alert or modal dialog. This function is applied for the very next dialog only.

    void* DGGetNextModalParent (
      void
    );

Return Values

Window handle (HWND) of the next dialog's parent window.

Remarks

On Macintosh, this function has no effect.

On Windows, it is important to choose the right parent for a modal dialog because the user can switch to another window from an open modal dialog if that window is not an ancestor of the modal dialog.

The default algorithm in DG for determining the parent is the following:

  1. If there is at least one open DG modal dialog, the last one will be the parent.
  2. If there is no DG modal dialog open, the last active pop-up window is chosen. This can be a non-DG modal dialog, for example File dialogs.
  3. Otherwise the application's frame window is used.

The parent retrieved by DGGetNextModalParent applies for the next alert function (DGAlert, DGResAlert or DGResAlertParam) or DGModalDialog call only.

To modify the parent window use the DGSetNextModalParent function.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
