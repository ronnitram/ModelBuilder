DGShowModelessDialog

Shows a modeless or palette dialog.

    void DGShowModelessDialog (
      short  dialId,
      short  refDialId
    );

Parameters

dialId
ID of the dialog.
 
refDialId
ID of the reference dialog in front of which the dialog need to be displayed.

Remarks

If the dialID refers to a modeless dialog, and it is the topmost document window, DGShowModelessDialog not only shows it, but makes it active, too. A reference dialog should always be given above which the dialog need to be displayed.

You can hide a modeless or palette dialog by using the DGHideModelessDialog function. To test whether a modeless or palette dialog is visible call DGIsModelessDialogVisible.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
