DGGetActiveWindow

Get the window handle of the first visible window.

    void* DGGetActiveWindow (
      void
    );

Return Values

Dialog pointer (NSWindow*) of the dialog on Macintosh, window handle (HWND) of the dialog on Windows.

Remarks

Use this function to get the window handle of the first visible window.

Requirements

Version: DG 2.2.0 or later
Header: DG.h, DGDefs.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
