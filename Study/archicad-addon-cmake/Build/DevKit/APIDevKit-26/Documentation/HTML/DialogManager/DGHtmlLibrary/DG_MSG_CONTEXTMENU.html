DG_MSG_CONTEXTMENU

Context menu was called on the dialog or on a specified dialog item.

Message parameters

dialId
ID of the dialog.
 
item
1-based index of the dialog item. If the context menu event occured on the dialog, this value is zero.
 
msgData
Pointer to a DGContextMenuMsgData structure cast to DGMessageData.

Return Values

Set to 1 if the application handled the event otherwise set to zero.

Remarks

The dialog callback function receives DG_MSG_CONTEXTMENU messages when the context menu is called above a dialog item (or the dialog). The msgData parameter tells the position of the event in screen coordinates. If the event occured above a list box item, list box header item, list view or a tree view item, the 1-based item index is specified also. If there were no such item, the item index is 0.

If the application returns zero value in the dialog callback function, DG displays a context sensitive help window. See the DG_MSG_ITEMHELP message for details.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
