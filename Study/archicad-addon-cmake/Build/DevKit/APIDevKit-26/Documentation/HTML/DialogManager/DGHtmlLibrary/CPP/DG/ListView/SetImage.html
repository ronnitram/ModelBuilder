ListView::SetImage

Sets the type and data of the global image of a list view control. 

void SetImage (
    ImageType           imageType,
    const void *        imageData,
    short               nImages,
    short               nRows
);

Parameters

imageType
The type of the global image. (Only types listed at imageData are accepted.)
imageData
The data of the global image depending on the imageType parameter.
imageType imageData
ListView::NoImage No image is defined for this item.
ListView::Icon
Macintosh only:
Icon, referenced with NSImage* cast to void*.
Windows only:
Icon, referenced with HICON cast to void*.
ListView::Picture
Macintosh only:
Picture, referenced with NSImage* cast to void*.
Windows only:
Picture, referenced with HBITMAP cast to void*.
Windows only:
ListView::DIBitmap
Device Independent Bitmap, referenced with BITMAPINFO* cast to void*.
nImages
Number of the sub-images in the image.
nRows
Number of rows of the sub-images in the image.
