DGUserInvalidateRgn

Invalidates an area of a user item. It is removed in version DGLib 2.2.0.

    void DGUserInvalidateRgn (
      short        dialId,
      short        item,
      const void*  rgn,
      bool         erase
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
rgn
The region to invalidate. The type of this value is HRGN on Windows and RgnHandle on Macintosh.
 
erase
true if the background of the user item must be erased otherwise false.

Remarks

When the user item is to be redrawn only the invalidated parts are updated. This function validates the specified part and excludes it from the updated area.

A rectangular area of a user item can be invalidated by the DGUserInvalidateRect function.

To validate a rectangular or a non-rectangular area use either DGUserValidateRect or DGUserValidateRgn.

To update the invalid area of a user item immediately use the DGUserUpdateItem function. After updating a user item its entire rectangle is validated. Regardless of the invalidated area the entire user item can be redrawn by the DGUserRedrawItem function.

Requirements

Header: DG.h
