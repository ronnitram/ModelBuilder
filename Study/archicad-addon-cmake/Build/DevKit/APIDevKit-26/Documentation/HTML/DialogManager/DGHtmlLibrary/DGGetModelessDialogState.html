DGGetModelessDialogState

Retrieves the state of a modeless dialog.

    short DGGetModelessDialogState (
      short  dialId
    );

Parameters

dialId
ID of the dialog.

Return Values

The state of a modeless dialog.

Value Meaning
DG_DST_RESTORED The dialog is in normal state, or it is not a modeless dialog, or the parameter was not correct.
DG_DST_MINIMIZED The modeless dialog is minimized.
DG_DST_MAXIMIZED The modeless dialog is maximized.

Remarks

To modify the state of a modeless dialog use the DGSetModelessDialogState function.

The dialog callback function receives DG_MSG_RESIZE messages, if the dialog state is modified.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
