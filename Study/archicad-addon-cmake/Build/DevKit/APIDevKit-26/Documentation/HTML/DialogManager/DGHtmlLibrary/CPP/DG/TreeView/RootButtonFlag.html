TreeView::RootButtonFlag Enum

The RootButton/NoRootButton flag of the tree view. 

enum RootButtonFlag
{
    NoRootButton =  0,
    RootButton =  DG_TVT_ROOTBUTTON
};

Value Definitions

NoRootButton
The root item has no plus (+) or minus (-) button on Windows.
RootButton
The root item has plus (+) or minus (-) button.
