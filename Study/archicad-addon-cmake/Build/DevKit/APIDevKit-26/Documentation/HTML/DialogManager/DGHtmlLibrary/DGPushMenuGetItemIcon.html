DGPushMenuGetItemIcon

Retrieves the icon data of a menu item in a push menu control.

    const void* DGPushMenuGetItemIcon (
      short  dialId,
      short  item,
      short  menuItem
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
menuItem
1-based index of the menu item.

Return Values

The icon data of the specified push menu control item (cast to void*).

The icon data indicates:
 Macintosh only: 
NSImage* pointer of the icon cast to void*
 Windows only: 
HICON handle of the icon cast to void*

nullptr is returned if the icon was set by DGPushMenuGetItemIconId or DGPushMenuGetItemIconResource, or if the function parameters are incorrect.

Remarks

For specifying an icon to a push menu control item with its icon data, see DGPushMenuSetItemIcon, for specifying with its icon ID, see DGPushMenuSetItemIconId for details.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
