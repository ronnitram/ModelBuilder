DGTabControlSetItemIconResource

Sets the icon of a tab control item with the icon ID and the resource module ID of the icon.

    void DGTabControlSetItemIconResource (
      short        dialId,
      short        item,
      short        tabItem,
      GSResModule  resModule,
      short        iconId
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
tabItem
1-based index of the tab control item.
 
resModule
The resource module ID containing the icon.
 
iconId
The new icon ID of the tab control item.

Remarks

A normal tab control, unlike a simple tab control may have an optional icon and text displayed on its label. The icon in the label is always adjusted to the left of the label text.

If the icon is located in the resource module of the dialog, the icon can also be set with DGTabControlSetItemIconId. The resource module of the icon in the tab control item can be retrieved by DGTabControlGetItemIconResource. The resource module of the dialog can be set and retrieved by DGSetDialogResourceModule or DGSetDialogResourceModules and DGGetDialogResourceModule or DGGetDialogIconResourceModule, respectively.

The icon of the tab control item can also be specified with its icon handle: see DGTabControlSetItemIcon for details.
