DGGetDialogIconResourceModule

Retrieves the resource module of a dialogs' images.

    GSResModule DGGetDialogIconResourceModule (
      short  dialId
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 

Return Values

The resource module of the dialogs' icons.

Remarks

You can use the DGSetDialogResourceModules to set the resource module of the icons.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
