List Box control Drag and Drop

You can drag and drop list box items, between list boxes or other controls (with drag and drop support), using the drag and drop capability of the Dialog Manager module.

Remarks

The default drag format is 'LBEX' its scope is dialog wide. When a drag and drop operation starts from a list box, DG sets up data with 'LBEX' format. This data block contains the selected list box items. To alter the scope or the base format of the drag and drop use DGDragModifyBaseFormat. The maximum scope can be application wide: DG_DRAG_SCOPE_APP. If you want to specify DG_DRAG_SCOPE_SYSTEM scope, user defined drag and drop data must be specified which can be interpreted by the targeted application.

It is possible to find out the list box drag source by using the DGDragGetSourceDialog and the DGDragGetSourceItem functions. Then use the DGDragGetListBoxItem to get the source list box items.

Requirements

Version: DG 2.0.0 or later
