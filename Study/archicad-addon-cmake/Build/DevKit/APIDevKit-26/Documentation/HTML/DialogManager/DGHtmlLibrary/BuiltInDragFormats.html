Built in Drag Formats

Format Meaning
'TEXT' Simple text type drag and drop format.
'FILE' File type drag and drop format.
'TVEX' Default tree view drag and drop format.
'LVEX' Default list view drag and drop format.
'LBEX' Default list box drag and drop format.

Description

The 'TEXT' and 'FILE' drag formats scope are system wide. These formats identify text or file drag and drop from the system or your own application. 'TVEX', 'LVEX' and 'LBEX' formats are specific to Dialog Manager drag and drop operations which scope are dialog wide. To extend this scope use the DGDragModifyBaseFormat function.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
