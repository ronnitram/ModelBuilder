DGSetItemMaxLong

Sets the maximum value of a dialog item.

    void DGSetItemMaxLong (
      short  dialId,
      short  item,
      Int32  value
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
value
The new maximum value.
Item type Possible range Meaning
Date & Time Control 0 - LONG_MAX Maximal value of the date & time control.
Integer-type Edit Control 0 - LONG_MAX Maximal value of the edit control.
Pop-up Control 0 - SHRT_MAX Number of pop-up items.
Progress Bar 0 - SHRT_MAX Maximal value of the progress bar.
Push Menu Control 0 - SHRT_MAX Number of menu items in the push menu control.
Scroll Bar 0 - SHRT_MAX Maximal value of the scroll bar.
Slider Bar 0 - SHRT_MAX Maximal value of the slider bar.
Spin Control SHRT_MIN - SHRT_MAX Maximal value of the spin control.
Tab Control 0 - SHRT_MAX Number of tab pages.
User Control SHRT_MIN - SHRT_MAX Maximal value of the user control.

Remarks

The edit control accepts values entered by the user according to the limits given in GRC or set by DGSetItemMinLong and DGSetItemMaxLong. If a value outside this range is entered, it is automatically changed by DG to the maximum or minimum.

If either the current or the minimal value is greater than the given maximum, it is/they are automatically set to the maximum by DG.

Position of a progress bar, of the scroll box or of a slider, respectively, is derived from the current item value based on the minimal and maximal value specified in GRC or set by DG functions later. The relative position is calculated using the following formula: relative position = (value - minimum) / (maximum - minimum). On Windows if maximal and minimal values are equal, the progress bar is empty, the scroll box or the slider is placed at the beginning of the control. However, on Macintosh these controls become disabled in the same situation.

If minimal and maximal values of a spin control are equal, the control becomes disabled.

From DG version 1.1.1 use DGPushMenuSetItemCount to set the number of menu items in a push menu control.
