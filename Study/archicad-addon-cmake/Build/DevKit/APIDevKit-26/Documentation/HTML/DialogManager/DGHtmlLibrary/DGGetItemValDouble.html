DGGetItemValDouble

Retrieves the current double value of an edit control.

    double DGGetItemValDouble (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.

Return Values

The current value.
Edit control subtype Meaning
DG_ET_REAL Simple real value.
DG_ET_ANGLE Angle value given in degrees.
DG_ET_POLARANGLE Polar angle value given in degrees.
DG_ET_LENGTH Length value.
DG_ET_AREA Area value.
DG_ET_VOLUME Volume value.
DG_ET_MM_OR_PT Scale independent length value given in mm, shown in mm or point.

If the function parameters are incorrect, 0.0 is returned.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
