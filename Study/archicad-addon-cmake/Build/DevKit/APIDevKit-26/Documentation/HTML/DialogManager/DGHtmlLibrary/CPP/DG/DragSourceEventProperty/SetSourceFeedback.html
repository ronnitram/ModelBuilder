DragSourceEventProperty::SetSourceFeedback

Use this function to set the source feedback. 

void SetSourceFeedback (
    void *              feedback,
    const DG::Rect *    metaRect
);

Parameters

feedback
Customized drag source feedback specification is platform specific. Windows: Enhanced metafile pointer. Macintosh: Region handle.
metaRect
In the case of a Windows specific customized feedback, this rect specifies the rect of the enhanced metafile.

Description

This member can be used to provide drag source feedback. Under the WINDOWS platform, a pointer to an enhanced metafile is specified, on the macintosh platform a region handle. Beside the enhanced metafile, you must specify its drawing rectangle.

Example

// Method Example:
void	DragTab3Observer::UserItemDragStarted (const DG::UserItemDragSourceEvent& ev, UShort* /*effect*/)
{
	DG::DragSourceEventProperty* dragSourceData = ev.GetDragData ();
	DG::Point pos = dragSourceData->GetSourcePosition ();
	DG::Rect metaRect (0, 0, 300, 300);
	dragSourceData->SetSourceFeedback (CreateUserItemFeedback (pos), &metaRect);
}
