DGEnableMessage

Enables a message to be sent to a dialog or a dialog item.

    void DGEnableMessage (
      short  dialId,
      short  item,
      short  message,
      short  modifier
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item. For DG_MSG_DRAGDROP message, DG_ALL_ITEMS specifies that the message is enabled for all items (with the exception of edit controls, which are handled internally), otherwise DG_ALL_ITEMS specifies that the message will be sent to the dialog and not to a particular item. This means that in the dialog callback function the message will arrive with zero item ID.
 
message
One of the allowed DG messages.
 
modifier
One of the following values:
Value Meaning
DG_EMM_NONE The application does not send the DG_MSG_NULL messages when the application is inactive.
DG_EMM_SENDFORINACTIVEAPP The application sends the DG_MSG_NULL messages even if the application is inactive.

Remarks

Allowed messages and dialog item types:

Message Dialog item type
DG_MSG_DRAGDROP Allowed for dialogs (item is DG_ALL_ITEMS), and for tab pages.
DG_MSG_MOUSEMOVE Allowed for DG_ITM_LISTBOX and DG_ITM_USERITEM.
DG_MSG_NULL Allowed for modal dialogs (item is DG_ALL_ITEMS).

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
