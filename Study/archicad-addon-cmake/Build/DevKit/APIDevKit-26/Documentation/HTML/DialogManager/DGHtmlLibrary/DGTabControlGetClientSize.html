DGTabControlGetClientSize

Retrieves the display (client) area size of a tab control.

    void DGTabControlGetClientSize (
      short   dialId
      short   item
      short*  hSize
      short*  vSize
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
hSize
Output parameter: horizontal size of the tab control's display area in pixels.
 
vSize
Output parameter: vertical size of the tab control's display area in pixels.

Remarks

To retrieve the location and size of the display area in the main dialog coordinate system use the DGTabControlGetClientRect function.

In case of frameless simple tab controls the dialog item rectangle is the same as the display area rectangle.

The display area size of the tab control can be set by the DGTabControlSetClientSize function.

Requirements

Version: DG 2.0.0 or later
Header: DG.h, DGDefs.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
