DGGetDialogTitle

Retrieves the title text of a dialog.

    GS::UniString DGGetDialogTitle (
      short  dialId
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.

Return Values

The dialog title in Unicode format.

Remarks

Palette dialogs with left caption can have a title text, although it is not displayed.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
