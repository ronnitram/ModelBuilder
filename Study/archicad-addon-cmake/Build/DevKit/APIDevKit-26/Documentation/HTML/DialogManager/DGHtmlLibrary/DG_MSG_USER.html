DG_MSG_USER

An application-defined, own message is sent to the dialog callback function. The ID of the message should be greater or equal to DG_MSG_USER.

Message parameters

dialId
ID of the dialog or tab page.
 
item
1-based index of the dialog item.
 
msgData
Any message related data.

Return Values

Any appropriate short value.

Remarks

The application can send own messages to a dialog only by the DGSendMessage function. In contrast to the standard DG messages, own messages can also be sent when handling the DG_MSG_INIT message.

Sending application-defined messages is a convenient method to collect all the dialog related tasks into its callback function. An application can define as many own messages as it needs, but the ID of these messages should be greater or equal to DG_MSG_USER. The message data parameter of the own messages can be freely chosen by the application.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
