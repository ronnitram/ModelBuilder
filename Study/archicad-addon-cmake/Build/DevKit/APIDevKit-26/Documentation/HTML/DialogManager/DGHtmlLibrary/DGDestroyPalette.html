DGDestroyPalette

Destroys a palette dialog.

    void DGDestroyPalette (
      short  dialId
    );

Parameters

dialId
ID of the palette dialog.

Remarks

This function destroys the window and the related data of that palette dialog. Once the event provessing was started for that palette, it should be stopped before calling this function. See DGEndProcessEvents for details. If there are application allocated resources for that palette or for one for its items, they should be freed before calling DGDestroyPalette.

To destroy modeless dialogs use the DGDestroyModelessDialog function.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
