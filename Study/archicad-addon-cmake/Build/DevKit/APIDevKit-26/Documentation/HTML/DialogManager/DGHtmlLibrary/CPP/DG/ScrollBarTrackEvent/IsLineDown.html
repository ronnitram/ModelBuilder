ScrollBarTrackEvent::IsLineDown

Checks if one line down event occured. 

bool IsLineDown (void) const;

Return Value

true, if one down left event occured.

Description

At one line down event, the scroll box moves one unit down. This event can be raised by clicking on the down-arrow of the scroll bar, or on Windows pressing the down arrow key (when the scroll bar has the keyboard focus).
