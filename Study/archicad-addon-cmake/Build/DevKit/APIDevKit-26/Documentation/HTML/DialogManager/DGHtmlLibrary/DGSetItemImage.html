DGSetItemImage

Sets the image of a dialog item.

    void DGSetItemImage (
      short        dialId,
      short        item,
      const void*  imageData
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
imageData
Item type  Macintosh only:   Windows only: 
DG_ITM_BUTTON NSImage* pointer of the icon cast to void*. HICON handle of the icon cast to void*.
DG_ITM_CHECKBOX NSImage* pointer of the icon cast to void*. HICON handle of the icon cast to void*.
DG_ITM_RADIOBUTTON NSImage* pointer of the icon cast to void*. HICON handle of the icon cast to void*.
DG_ITM_ICONITEM NSImage* pointer of the icon cast to void*. HICON handle of the icon cast to void*.
DG_ITM_PICTITEM NSImage* pointer of the picture cast to void*. HBITMAP handle of the bitmap cast to void*.

Remarks

This function does not make a copy of the image data, the application must manage this image and delete it when it is no longer necessary. This also means that the image must not be deleted as long as it is assigned to the dialog item.

The image of a dialog item can be also assigned by the resource ID of the image, see DGSetItemImageId and DGSetItemImageResource for details. If the image was set by its data by DGSetItemImage, the image data can be retrieved by DGGetItemImage.

For item types DG_ITM_BUTTON, DG_ITM_CHECKBOX and DG_ITM_RADIOBUTTON this function works only for icon-style buttons, i. e. DG_BT_ICON and DG_BT_PUSHICON.

If the item type is DG_ITM_ICONITEM or DG_ITEMPICTITEM, the dialog item is automatically resized to the size of the image. During this operation the center of the dialog item is not moved and the possible frame is taken into account.
