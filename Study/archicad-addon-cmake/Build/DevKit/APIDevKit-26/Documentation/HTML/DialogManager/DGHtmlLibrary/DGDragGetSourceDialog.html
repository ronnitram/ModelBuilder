DGDragGetSourceDialog

Gets the dialog ID of the drag source dialog.

    short DGDragGetSourceDialog (
      void
    );

Return Value

The dialog ID of the drag source dialog.

Remarks

Call this function when the drag data scope is maximum application wide (DG_DRAG_SCOPE_APP). It is mainly used if tree view, list view or list box drag and drop event occurs between dialogs of the same application.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
