Feedback Enum

Built-in feedback functionality. 

enum Feedback
{
    AcceptSelAndIns =  0,
    RejectSelection =  DG_DRAG_REJECT_SEL,
    RejectInsertion =  DG_DRAG_REJECT_INS,
    RejectSelAndIns =  DG_DRAG_REJECT_SEL | DG_DRAG_REJECT_INS
};

Value Definitions

AcceptSelAndIns
Built-in feedback functionality is accepted for both selection and insertion.
RejectSelection
Built-in feedback functionality is rejected for selection.
RejectInsertion
Built-in feedback functionality is rejected for insertion.
RejectSelAndIns
Built-in feedback functionality is rejected for both selection and insertion.
