DGTreeViewIsItemExpanded

Tests whether a tree view item is expanded or collapsed.

    short DGTreeViewIsItemExpanded (
      short  dialId,
      short  item,
      Int32  treeItem
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
treeItem
ID of the tree view item.

Return Values

Value Meaning
1 The subtree of the tree view item is expanded (visible).
0 The subtree is collapsed (not visible).

If this function is called for a non-parent tree item, 0 is returned.

Remarks

The expanded/collapsed state of a tree view item is simply a flag. If the item is currently not visible (e.g. one of its ancestors is collapsed) it still can be expanded.
