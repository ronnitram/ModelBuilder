DGSetDialogUserData

Sets the user-specified data of a modal or modeless dialog.

    void DGSetDialogUserData (
      short        dialId,
      DGUserData   userData
    );

Parameters

dialId
ID of the dialog.
 
userData
The new value of the user data.

Remarks

The application can assign a DGUserData value to every modal or modeless dialog. This value can be used to store any dialog specific user data. It can be given when creating the dialog by DGModelessInit or DGModalDialog as the user data parameter of these functions, or can be modified later by DGSetDialogUserData. When handling DG messages in the dialog callback function, this dialog-specific data is available as the user data parameter of the message. Otherwise you can retrieve this value by DGGetDialogUserData.

It is advised to gather all the dialog related data into a user-defined structure. After filling this structure, give DGUserData to DGModelessInit or DGModalDialog as the user data parameter.

If a dialog has one ore more tab pages, they have a common user data. DGSetDialogUserData can be called only for main dialogs and should not be used for tab pages.

Requirements

Version: DG 1.1.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
