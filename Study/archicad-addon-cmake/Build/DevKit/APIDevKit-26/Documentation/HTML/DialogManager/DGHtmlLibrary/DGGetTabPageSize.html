DGGetTabPageSize

Retrieves the size of a tab page.

    void DGGetTabPageSize (
      short   dialId
      short*  hSize
      short*  vSize
    );

Parameters

dialId
ID of the tab page subdialog.
 
hSize
Output parameter: horizontal size of the tab page.
 
vSize
Output parameter: vertical size of the tab page.

Remarks

Tab pages are assigned to tab control items. They appear in the display area of the tab control. Use this function to retrieve the horizontal and vertical size of a tab page. The size can be modified with the DGSetTabPageSize function.

Requirements

Version: DG 2.0.0 or later
Header: DG.h, DGDefs.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
