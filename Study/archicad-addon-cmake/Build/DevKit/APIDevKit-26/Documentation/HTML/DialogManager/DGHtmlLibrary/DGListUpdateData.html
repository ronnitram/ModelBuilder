DGListUpdateData

If the update flag of a list box tabulator field / list item has been set to true by the DGListSetTabFieldUpdateFlag / DGListSetItemOwnerDrawFlag function and the content of the tabulator field / list item should be updated, the dialog callback function receives a DG_MSG_UPDATE message. The message data parameter of this message is a pointer to a DGListUpdateData structure that contains all the necessary information to update the tabulator field / list item by the application.

    struct DGListUpdateData {
      void*          drawContext;
      short          listItem;
      bool           isHighlighted;
      short          tabIndex;
      short          width;
      short          height;
      short          justFlag;
      short          truncFlag;
      GS::UniString  tabText;
      Gfx::Color     fgColor;
      Gfx::Color     bkColor;
    };

Members

drawContext
The drawing context of the tabulator field. Cast this value to CGContextRef on Macintosh or HDC on Windows.
 
listItem
1-based index of the list item.
 
isHighlighted
If this flag is set to true, the list item is highlighted.
 
tabIndex
If the value is equal to 0, the whole listItem is drawable. Otherwise the given value is the index of the choosen tabulator field.
Tabulator fields are 1-base indexed.
 
width
Width of the tabulator field / list item in pixels.
 
height
Height of the tabulator field / list item in pixels.
 
justFlag
Justification of the content in the tabulator field. For left/center/right justification DG_IS_LEFT, DG_IS_CENTER, DG_IS_RIGHT is returned.
 
truncFlag
How the text of the tabulator field / list item should be truncated. Possible values are:
Flag Action
DG_IS_NOTRUNC No truncation.
DG_IS_TRUNCEND Truncates at the end.
DG_IS_TRUNCMIDDLE Truncates in the middle.
 
tabText
Text of the tabulator field.
 
fgColor
Foreground color of the updated list item.
 
bkColor
Background color of the updated list item.

Remarks

The origin of the drawing context defined by the drawContext member is set to the upper left corner of the tabulator field.

Requirements
