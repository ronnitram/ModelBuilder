DGPopUpGetItemIcon

Retrieves the icon data of a pop-up control item.

    const void* DGPopUpGetItemIcon (
      short  dialId,
      short  item,
      short  popupItem
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
popupItem
1-based index of the item in the pop-up control. DG_POPUP_TOP specifies the first item of the pop-up. DG_POPUP_BOTTOM specifies the last item of the pop-up.

Return Values

The icon data of the specified pop-up item (cast to void*).

The icon data indicates:
 Macintosh only: 
NSImage* pointer of the icon cast to void*
 Windows only: 
HICON handle of the icon cast to void*

nullptr is returned if the specified pop-up item has no icon, if it is a separator, if the icon was set by DGPopUpSetItemIconId or DGPopUpSetItemIconResource, or if the function parameters are incorrect.

Remarks

For specifying an icon to a pop-up item with its icon data, see DGPopUpSetItemIcon, for specifying with its icon ID, see DGPopUpSetItemIconId for details.
