DGListGetTabItemText

Retrieves the text of a tabulator field of a list box item.

    GS::UniString DGListGetTabItemText (
      short  dialId,
      short  item,
      short  listItem,
      short  tabIndex
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
listItem
1-based index of the list item. DG_LIST_TOP specifies the first item of the list, DG_LIST_BOTTOM specifies the last item of the list.
 
tabIndex
1-based index of the tabulator field.

Return Values

The tabulator field text in Unicode format.

Remarks

The content of list items can be divided into more columns. A DG listbox can be imagined as a two-dimensional list consisting of rows and columns. One column is called a "tabulator field", a "tabulator item" is the intersection of a tabulator field and a row. The names "tabulator field" and "tabulator item" are inherited from the previous versions of DG. In the earlier versions texts of columns could be separated within a list item by inserting tabulator ('\t') characters into its text. The content of a tabulator item can be set by DGListSetTabItemText. An icon can also be specified to the tabulator item: see DGListSetTabItemIcon and DGListSetTabItemIconId for details.

A tabulator item can contain an icon and/or text also, but only one icon can appear in a tabulator item. In case of using an icon with text in a tabulator item, the item text can be set with the DGListSetTabItemText function. If a tabulator item contains an icon and text also, the icon will be positioned either at the beginning or at the end of the text according to the justification. (The DG_IS_LEFT justification is applied for the tabulator item containing an icon and text also, even if the DG_IS_CENTER justification was given.)

Call this function only after you set the number of tabulator fields and their properties by DGListSetTabFieldCount and DGListSetTabFieldData.
