DGListSetTabItemBackgroundColor

Sets the background color of a tabulator field of a list box item.

    void DGListSetTabItemBackgroundColor (
      short                 dialId,
      short                 item,
      short                 listItem,
      short                 tabIndex,
      unsigned short        red,
      unsigned short        green,
      unsigned short        blue
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
listItem
1-based index of the list item. DG_LIST_TOP specifies the first item of the list, DG_LIST_BOTTOM specifies the last item of the list.
 
tabIndex
1-based index of the tabulator field.
 
red
Red component of the color (0-65535).
 
green
Green component of the color (0-65535).
 
blue
Blue component of the color (0-65535).

Remarks

The background color of a separator line cannot be set.

Before the tabulator field background color is set by DGListSetTabItemBackgroundColor, DG uses the system default background color.

After the tabulator field background color is set at least once (no matter whether it is set to the system default), the DGListGetTabItemBackgroundColor function returns true.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
