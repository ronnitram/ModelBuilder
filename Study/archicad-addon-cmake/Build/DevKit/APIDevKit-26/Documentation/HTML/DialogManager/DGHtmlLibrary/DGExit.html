DGExit

Exits the DG dialog manager.

    void DGExit (
      void
    );

Remarks

Before terminating the application, do not forget to call DGExit to free internally allocated system resources.

If an application has some dynamically linking components (shared libraries on Macintosh, DLLs on Windows) it is forbidden to use DGInit or DGExit once again from the library. These components use the dialog manager initialized in the main program.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
