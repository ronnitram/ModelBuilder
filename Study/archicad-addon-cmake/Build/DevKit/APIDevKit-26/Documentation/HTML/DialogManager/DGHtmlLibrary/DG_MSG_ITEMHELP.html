DG_MSG_ITEMHELP

Tool tip or context sensitive help message should be displayed by DG.

Message parameters

dialId
ID of the dialog.
 
item
1-based index of the dialog item.
 
msgData
Pointer to a DGItemHelpMsgData structure cast to DGMessageData.

Return Values

Set to 1 if the event was handled by the application otherwise set to zero.

Remarks

The dialog callback function receives DG_MSG_ITEMHELP messages either when the user called the context sensitive help, or when a tooltip message must be displayed above a dialog item. Actually, the application is asked to give the required help text for DG to show. If the message is not handled, DG tries to get the help text from the GRC resource. The DGItemHelpMsgData contains the dialog item identifier, and the submessage. The application should fill either the tooltip text, or the context sensitive help text in the DGItemHelpMsgData structure according to the submessage.

The submessage can be one of the following values:

Value Meaning
DG_HSM_TOOLTIP Tooltip text is going to be displayed.
DG_HSM_CONTEXTHELP Context sensitive help text is going to be displayed.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
