DGIsAutoCloseEnabled

An alert or a modal dialog usually terminates by the result of a user action, i.e. click on the OK or Cancel button, or the Close Box (Windows only). However, there is a feature in DG to close these types of dialogs automatically (without user action) if a given interval expires. This function retrieves whether the automatic close of modal dialogs and alerts is enabled or not.

    bool DGIsAutoCloseEnabled (
      void
    );

Return Values

Value Meaning
true Automatic close of modal dialogs and alerts is enabled.
false Automatic close of modal dialogs and alerts is disabled.

Remarks

To enable or disable the automatic closing of alerts and dialogs use the DGEnableAutoClose and DGDisableAutoClose functions respectively. By default the automatic closing is disabled.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
