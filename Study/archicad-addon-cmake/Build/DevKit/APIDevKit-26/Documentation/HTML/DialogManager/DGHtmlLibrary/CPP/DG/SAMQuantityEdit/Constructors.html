DG_ET_SAM_PLFORCE,

DG_ET_SAM_ELFORCE,
DG_ET_SAM_ELMOMENT,
DG_ET_SAM_SLFORCE
frame
The frame type of the SAM quantity control.
absRel
The absolut/relativ numerical input flag of the SAM quantity control.
update
The update type of the SAM quantity control.
readOnly
The readonly/editable flag of the SAM quantity control.
