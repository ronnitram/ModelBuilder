ListBox::GetTabItemColor

Retrieves the foreground (text) color of a tabulator field of a list box item. 

bool GetTabItemColor (
    short 		listItem,
    short 		tabIndex,
    Gfx::Color * 	color
) const;

Parameters

listItem
1-based index of the list box item, use TopItem or BottomItem of Item type to specify the first or last item of the list box control.
tabIndex
1-based index of the tabulator field.
color
Output parameter: A pointer to the text color of the tabulator field.

Return Value

Value Meaning
false The tabulator field has the system default foreground (text) color.
true The foreground (text) color of the tabulator field was previously set by SetTabItemColor.
