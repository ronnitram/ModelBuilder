DGListSetTabFieldData

Sets the properties of a tabulator field in a list box.

    void DGListSetTabFieldData (
      short  dialId,
      short  item,
      short  tabIndex,
      short  begPos,
      short  endPos,
      short  justFlag,
      short  truncFlag,
      bool   hasSeparator,
      bool   status
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
tabIndex
1-based index of the tabulator field.
 
begPos
Tabulator field left position in pixels, relative to the client rectangle of the list box.
 
endPos
Tabulator field right position in pixels, relative to the client rectangle of the list box.
 
justFlag
Justification of the content in the tabulator field. For left/center/right justification use DG_IS_LEFT, DG_IS_CENTER, DG_IS_RIGHT.
 
truncFlag
How the text of the tabulator field should be truncated if it does not fit into the tabulator field. Possible values are:
Flag Action
DG_IS_NOTRUNC No truncation.
DG_IS_TRUNCEND Truncates at the end.
DG_IS_TRUNCMIDDLE Truncates in the middle (leaves the same number of characters from both ends of the text).
 
hasSeparator
If this flag is set, a separator line is drawn at the right edge of this tabulator field in each list box item. The separator line color can be set or modified by the DGListSetSeparatorLineColor function.
 
status
If this flag is set to false, the tabulator field is disabled, otherwise it is enabled.

Remarks

The number of tabulator fields in a list box is set by DGListSetTabFieldCount. By default every list box has one tabulator field which extends from the left of the list's client area to the left edge of the vertical scrollbar. The content of this item is left aligned and truncated at the end if the list box has no horizontal scrollbar otherwise not truncated.

If the tabulator fields overlap or do not fit the list box item then they will be set to equal size and forced to fit the listbox item. If the list box has horizontal scrollbar then the tabulator fields are allowed to extend the list item size.

The extent of the horizontal scrollbar is calculated depending the properties of the last tabulator field. If the truncation flag or the user update flag is turned on in this field then the scrollbar's extent equals the end position of the last tabulator field, otherwise it is dynamically calculated from the content of the longest listbox item.

Use DGListGetTabFieldData to retrieve the current properties of a tabulator field.
