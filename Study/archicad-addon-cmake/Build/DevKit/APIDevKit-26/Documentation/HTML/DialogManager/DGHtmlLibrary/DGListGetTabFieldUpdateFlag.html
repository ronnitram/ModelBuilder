DGListGetTabFieldUpdateFlag

Retrieves the update flag of a tabulator field of a list box.

    bool DGListGetTabFieldUpdateFlag (
      short  dialId,
      short  item,
      short  tabIndex
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
tabIndex
1-based index of the tabulator field.

Return Values

true if the tabulator field must be drawn by the application, false otherwise.

Remarks

The content of list items can be divided into more columns. In this documentation the notation "tabulator field" is used instead of "column". The reason for this is that texts of columns can be separated within a list item by inserting tabulator ('\t') characters into its text. The update flag of a tabulator field can be modified by DGListSetTabFieldUpdateFlag.

After the update flag of a tabulator field is set to true, the dialog callback function receives a DG_MSG_UPDATE message whenever the tabulator field must be updated. The message data parameter is a pointer to a DGListUpdateData structure that contains all the information needed to update the tabulator field.

The update flag is assigned to the list and not to its items. This means that the tabIndex tabulator field of all list items must be updated by the application.
