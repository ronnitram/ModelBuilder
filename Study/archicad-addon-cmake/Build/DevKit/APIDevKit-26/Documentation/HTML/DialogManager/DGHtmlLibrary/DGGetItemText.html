DGGetItemText

Retrieves a dialog item's text.

    GS::UniString DGGetItemText (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.

Return Values

The item text in Unicode format.

Dialog item types

Type Subtypes
DG_ITM_BUTTON DG_BT_ICONTEXT, DG_BT_PUSHTEXT, DG_BT_SPLIT
DG_ITM_CHECKBOX DG_BT_TEXT, DG_BT_PUSHTEXT
DG_ITM_RADIOBUTTON DG_BT_TEXT, DG_BT_PUSHTEXT
DG_ITM_EDITTEXT DG_ET_TEXT, DG_ET_PASSWORD
DG_ITM_STATICTEXT all
DG_ITM_GROUPBOX all

Remarks

Use nullptr for the text parameter to determine the necessary buffer size.

To retrieve the text of a multi-line or a rich edit control, use DGGetMultiLineText or DGRichEditGetText, respectively.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
