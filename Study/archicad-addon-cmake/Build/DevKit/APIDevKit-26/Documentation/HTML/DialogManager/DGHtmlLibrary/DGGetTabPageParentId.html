DGGetTabPageParentId

Retrieves the identifier of the main dialog the given tab page belongs to.

    short DGGetTabPageParentId (
      short  dialId
    );

Parameters

dialId
ID of the tab page.

Return Values

ID of the main dialog of the tab page.

Remarks

Tab controls define multiple pages for the same area of a dialog. Each page consists of a group of controls that the application displays when the user selects the corresponding tab. The pages of a tab control are either defined in GRC in a special dialog resource format (the so called tab page resource format) of DG or created dynamically from code. The main dialog of a tab page is the dialog that contains the tab control. DGGetTabPageParentId returns the ID of this main dialog.

Tab pages are special dialogs, because they have no dialog callback function. Messages for controls on tab pages are sent to the callback function of the main dialog. For these messages, however, the ID of the tab page subdialog is passed to the callback function in the dialId parameter instead of the ID of the main dialog. The tab page IDs passed to the callback function are usually the same as the IDs defined in GRC, but in special cases they can be different. You can retrieve these IDs by the DGTabControlGetTabPageId function when handling the DG_MSG_INIT message in the dialog callback function.
