DGListGetNewDragPos

After an item of a list box is dragged to a new position, the dialog callback function receives a DG_MSG_CHANGE message with a non-zero message data. Use this macro to retrieve the new index of the dragged list item from the message data.

    short DGListGetNewDragPos (
      DGMessageData  msgData
    );

Parameters

msgData
The message data parameter of the DG_MSG_CHANGE message.

Return Values

1-based index of the new position of the dragged list item.

Remarks

The DGListGetNewDragPos macro is defined as follows:

    #if defined (WINDOWS)
        #define DGListGetNewDragPos(longData)  ((short) (((ULong) (longData) >> 16) & 0xFFFF))
    #endif
    #if defined (macintosh)
        #define DGListGetNewDragPos(longData)  ((short) ((Int32) (longData) >> 16))
    #endif

To allow item reordering in a list box by dragging items with the mouse you need to do the following steps:

  • Enable DG_MSG_MOUSEMOVE for the list box item by DGEnableMessage.
  • When handling DG_MSG_MOUSEMOVE and the cursor is above a list tabulator field, which contains an icon with the ID DG_LIST_DRAGICON, the callback function should return DG_LIST_INDRAGAREA to enable dragging.
  • If the positions of the item before and after the dragging are different, the callback function receives a DG_MSG_CHANGE message with a non-zero message data.

Requirements

Header: DG.h

Module: DGLib [MAC], DG.dll [WIN]
