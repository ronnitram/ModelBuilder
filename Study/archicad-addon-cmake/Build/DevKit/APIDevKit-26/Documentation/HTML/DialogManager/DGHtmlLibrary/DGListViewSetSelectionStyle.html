DGListViewSetSelectionStyle

Sets the selection style of the list view.

    void DGListViewSetSelectionStyle (
      short  dialId,
      short  item,
      Int32  selectionStyle
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
selectionStyle
One of the following values:
Value Meaning
DG_LVSS_NORMAL The selected list items are displayed in the ordinary way.
DG_LVSS_PUSH The selected list items has a 'button-like' appearance.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
