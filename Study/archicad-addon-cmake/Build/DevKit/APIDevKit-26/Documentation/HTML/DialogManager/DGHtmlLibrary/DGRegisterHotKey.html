DGRegisterHotKey

Registers a hotkey for a given dialog.

    short DGRegisterHotKey (
      short  dialId,
      short  modifier,
      short  keyCode
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
modifier
For the desired modifier keys use any combination of the following values:
Value Macintosh Windows
DG_MOD_COMMAND Command (Apple) Ctrl
DG_MOD_OPTION Option Alt
DG_MOD_SHIFT Shift Shift
 
keyCode
A character key code or one of the following special key codes:
Value Meaning
DG_KEY_BACK Backspace key.
DG_KEY_TAB Tab key.
DG_KEY_ENTER Enter key.
DG_KEY_RETURN Return key.
DG_KEY_ESCAPE Escape key.
DG_KEY_LEFT Left arrow key.
DG_KEY_RIGHT Right arrow key.
DG_KEY_UP Up arrow key.
DG_KEY_DOWN Down arrow key.
DG_KEY_DELETE Delete key.
DG_KEY_HOME Home key.
DG_KEY_END End key.
DG_KEY_PAGEUP Page up key.
DG_KEY_PAGEDOWN Page down key.

Return Values

1-based index of the hotkey if the registration is successful, or zero if an error occurs.

Remarks

A hotkey is a keystroke or combination of keystrokes that generates a DG_MSG_HOTKEY message for a dialog. Hotkeys in DG are similar to menu accelerators and to the Operating System shortcuts. They also provide faster, more direct access to frequently used commands than menus and toolbars do. Moreover, they have some other special features (see the description of the DG_MSG_HOTKEY message).

Hotkeys can be registered to any type of dialogs by calling the DGRegisterHotKey function. If a hotkey is successfully registered, later it has to be referred by the short identifier that DGRegisterHotKey returns. A hotkey can be registered to more than one dialog. If a hotkey has been previously registered to a dialog and DGRegisterHotKey is called for an other dialog, it returns the same hotkey identifier. To unregister a previously registered hotkey use the DGUnregisterHotKey function.

The following key combinations cannot be registered as hotkeys (these combinations basically are the union of the Operating System shortcuts and the dialog keyboard interface):

Note that the DG_KEY_LEFT, DG_KEY_RIGHT, DG_KEY_UP and DG_KEY_DOWN keys without any modifier cannot be registered as hotkeys for modal dialogs only. Normal key codes (such as character and numerical keys) cannot be used with the DG_MOD_SHIFT modifier. For example, use the capital 'A' character key instead of the small 'a' key with the DG_MOD_SHIFT modifier.
