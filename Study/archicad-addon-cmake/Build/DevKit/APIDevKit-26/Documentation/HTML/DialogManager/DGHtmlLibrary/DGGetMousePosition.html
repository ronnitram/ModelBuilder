DGGetMousePosition

Retrieves mouse position data if the mouse is above the dialog.

    short DGGetMousePosition (
      short            dialId,
      DGMousePosData*  mPosData
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
mPosData
Pointer to a DGMousePosData structure.

Return Values

Value Meaning
-1 The mouse is not above the dialog.
0 The mouse is above the dialog but there is no dialog item under it.
Any positive value 1-based index of the dialog item under the mouse.

If the function parameters are incorrect, -1 is returned.

Remarks

Besides the mouse position coordinates, this function also retrieves the state of the mouse buttons and the modifier keys.

DGGetMousePosition is usually called when handling a DG_MSG_NULL message in the dialog callback function. When DG sends a DG_MSG_MOUSEMOVE message, similar information can be retrieved from the message data parameter therefore there is no need to call DGGetMousePosition.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
