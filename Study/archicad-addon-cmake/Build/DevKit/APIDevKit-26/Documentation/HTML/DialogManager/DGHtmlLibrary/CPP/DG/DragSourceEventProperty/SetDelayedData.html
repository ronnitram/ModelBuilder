DragSourceEventProperty::SetDelayedData

Sets delayed drag and drop data to a drag source. 

bool SetDelayedData (
    DGDragFormat        format,
    DragDrop::Scope     scope
);

Parameters

format
DGDragFormat type parameter. Used to specify the drag format in character style like: 'TEXT', 'FILE' etc.
scope
Scope of the data: DragDrop::Item, DragDrop::Dialog, DragDrop::Module, DragDrop::Application, DragDrop::System.

Return Value

true if data was set successfully, false otherwise.

Description

Use this member to set a delayed type drag and drop data. When the target needs this type of data, it will fire an event toward the drag source. The drag source than will have to set the requested type of data.

Example

// Method Example:
void	DragTab3Observer::UserItemDragStarted (const DG::UserItemDragSourceEvent& ev, UShort* /*effect*/)
{
	DG::DragSourceEventProperty* dragSourceData = ev.GetDragData ();
	dragSourceData->SetDelayedData ('TEXT', DG::DragDrop::Application);
}
