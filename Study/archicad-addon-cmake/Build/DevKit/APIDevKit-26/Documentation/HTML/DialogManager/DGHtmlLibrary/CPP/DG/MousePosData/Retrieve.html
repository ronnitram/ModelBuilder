MousePosData::Retrieve

Retrieves the mouse and keyboard data. 

bool Retrieve (const Panel & panel);

Parameters

panel
The panel on which mouse information is retrieved.

Return Value

true, if the data retrieving was successful, otherwise false.

Description

Retrieves the mouse and keyboard data, if the mouse is above the specified panel. After this, the retrieved data is available in this object.
