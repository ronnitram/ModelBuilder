Dialog::GrowType Enum

The grow type of the dialog. 

enum GrowType
{
    NoGrow =  DG_DLG_NOGROW,
    HGrow =  DG_DLG_HGROW,
    VGrow =  DG_DLG_VGROW,
    HVGrow =  DG_DLG_HGROW | DG_DLG_VGROW
};

Value Definitions

NoGrow
The dialog is not growable.
HGrow
Horizontal grow is enabled.
VGrow
Vertical grow is enabled.
HVGrow
Both horizontal and vertical grow is enabled.
