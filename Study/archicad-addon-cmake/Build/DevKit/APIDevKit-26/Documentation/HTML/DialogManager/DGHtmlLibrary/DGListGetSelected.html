DGListGetSelected

For single-selection list box retrieves the index of the selected item. For multiple-selection lists retrieves the index of the next selected list item.

    short DGListGetSelected (
      short  dialId,
      short  item,
      short  listItem
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
listItem
1-based index of the list item where the search begins. Use DG_LIST_TOP to search from the first item. For single-selection lists this parameter is ignored.

Return Values

For single-selection lists, the 1-based index of the selected item or zero if there is no selected item.

For multiple-selection lists, the 1-based index of the first selected item after listItem, including the starting item, too. If there is no such selected item, the function returns zero.

If the function parameters are incorrect, zero is returned.

Remarks

To get the next item for consecutive retrieval of the selected items, increase the returned index by 1.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
