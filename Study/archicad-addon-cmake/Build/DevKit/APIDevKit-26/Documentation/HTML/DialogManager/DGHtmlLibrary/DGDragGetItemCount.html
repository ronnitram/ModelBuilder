DGDragGetItemCount

Gets the number of the dragged sub-items in a tree view, list view or list box.

    ULong DGDragGetItemCount (
      void
    );

Return Values

Number of tree view, list view or list box items being dragged.

Remarks

Use this function to get the number of the tree view, list view or list box items being dragged. Then you can retrieve these items by using the following functions: DGDragGetTreeViewItem, DGDragGetListBoxItem, DGDragGetListViewItem.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
