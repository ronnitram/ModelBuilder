DGSetItemMinDouble

Sets the minimum double value of an edit control.

    void DGSetItemMinDouble (
      short   dialId,
      short   item,
      double  value
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
value
The new minimum value.
Edit control subtype Meaning
DG_ET_REAL Simple real value.
DG_ET_ANGLE Angle value.
DG_ET_POLARANGLE Polar angle value.
DG_ET_LENGTH Length value.
DG_ET_AREA Area value.
DG_ET_VOLUME Volume value.
DG_ET_MM_OR_PT Scale independent length value given in mm, shown in mm or point.

Remarks

The edit control accepts values entered by the user according to the limits given in GRC or set by DGSetItemMinDouble and DGSetItemMaxDouble. If a value outside this range is entered, it is automatically changed by DG to the maximum or minimum.

If either the current or the maximal value is less than the given minimum, it is/they are automatically set to the minimum by DG.

The angle value must be given in degrees for angle-type edit controls (DG_ET_ANGLE, DG_ET_POLARANGLE).

Requirements

Header: DG.h

Module: DGLib [MAC], DG.dll [WIN]
