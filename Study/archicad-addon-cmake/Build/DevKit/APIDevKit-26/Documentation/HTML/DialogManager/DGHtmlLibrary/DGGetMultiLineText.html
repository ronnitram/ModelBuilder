DGGetMultiLineText

Retrieves the text of a multi-line edit control with the given end-of-line markers.

    GS::UniString DGGetMultiLineText (
      short  dialId,
      short  item,
      short  eolType
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
eolType
Type of the end-of-line markers:
Value Meaning
DG_EOL_DEFAULT DGGetMultiLineText returns the text with the end-of-line markers of the current platform: CR (0x0D) on Macintosh and CRLF (0x0D 0x0A) on Windows.
DG_EOL_CR DGGetMultiLineText always returns the text with CR (0x0D) end-of-line markers.
DG_EOL_CRLF DGGetMultiLineText always returns the text with CRLF (0x0D0x0A) end-of-line markers.

Return Values

The item text.

Remarks

Use nullptr for the text parameter to determine the necessary buffer size.

To retrieve text of a rich edit control, use DGRichEditGetText.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
