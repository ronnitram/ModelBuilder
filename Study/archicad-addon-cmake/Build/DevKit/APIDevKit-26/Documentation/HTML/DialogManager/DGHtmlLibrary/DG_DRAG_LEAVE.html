DG_DRAG_LEAVE

Submessage of a drag and drop operation. This is a drag target related submessage.

Description

This event is rised, when a the mouse leaves the target control during dragging. Use this event to cancel custom target feedback.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
