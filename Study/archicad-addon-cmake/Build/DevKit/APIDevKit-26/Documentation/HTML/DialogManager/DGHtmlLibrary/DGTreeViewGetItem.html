DGTreeViewGetItem

Retrieves the ID of a tree view item according to the specified flag.

    Int32 DGTreeViewGetItem (
      short  dialId,
      short  item,
      Int32  treeItem,
      short  itemFlag
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
treeItem
If itemFlag is one of DG_TVG_NEXT, DG_TVG_PREVIOUS, DG_TVG_PARENT, DG_TVG_CHILD, DG_TVG_NEXTLISTED or DG_TVG_PREVLISTED, treeItem is the ID of a tree view item. Otherwise this parameter is ignored.
 
itemFlag
The return value of the function is determined by this flag:
Value Meaning
DG_TVG_FIRST The first item in the tree view.
DG_TVG_NEXT The next item of the tree view (the one after treeItem) on the same hierarchy level.
DG_TVG_PREVIOUS The previous item of the tree view (the one before treeItem) on the same hierarchy level.
DG_TVG_PARENT The parent of treeItem, i.e. the item one level higher in the hierarchy.
DG_TVG_CHILD The (first) child item of treeItem, i.e. the first item one level lower in the hierarchy.
DG_TVG_FIRSTLISTED The same as DG_TVG_FIRST.
DG_TVG_LASTLISTED The last listed (visible) item in the tree view.
DG_TVG_NEXTLISTED The next listed (visible) item in the tree view.
DG_TVG_PREVLISTED The previous listed (visible) item in the tree view.

Return Values

The ID of the referenced tree view item (according to itemFlag) or DG_TVI_NONE if there is no such an item. DG_TVI_ROOT is returned if itemFlag is DG_TVG_PARENT and the specified item has no parent, i.e. it is the root item.

Remarks

A tree item is considered as "listed (visible)" if its hierarchy level is expanded. However, an item can be listed as visible even if it is currently not in the client rectangle of the tree view.

To enumerate all items on a hierarchy level use DG_TVG_NEXT and DG_TVG_PREVIOUS. To go through hierarchy levels use DG_TVG_CHILD and DG_TVG_PARENT. To enumerate all the listed items use DG_TVG_NEXTLISTED and DG_TVG_PREVLISTED. To enumerate all items of a subtree write a recursive algorithm based on DG_TVG_CHILD and DG_TVG_NEXT.

The only way to determine whether an item is a parent is calling DGTreeViewGetItem with the DG_TVG_CHILD flag. If the return value is different from DG_TVI_NONE the tree item has at least one child.
