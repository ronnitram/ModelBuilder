DGActivateModelessDialog

Activates a modeless or palette dialog.

    void DGActivateModelessDialog (
      short  dialId
    );

Parameters

dialId
ID of the dialog.

Remarks

The DGActivateModelessDialog function activates a dialog and brings it into the front only if its application is in the foreground at the time the function is called.

Dialog type Action
Modeless dialog Moves the dialog to front document window position (behind the last palette dialog).
Palette dialog Moves the dialog to frontmost window position.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
