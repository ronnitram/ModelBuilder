DGGetDialogId

Retrieves the identifier of a dialog that has the specified type and relationship to the specified reference dialog.

    short DGGetDialogId (
      short  refDialId,
      short  dialFlag,
      short  dialType
    );

Parameters

refDialId
ID of the dialog used as reference dialog.
 
dialFlag
Specifies the relationship between the reference dialog and the dialog whose ID is to be retrieved. This parameter can be one of the following values.
Value Meaning
DG_DF_FIRST The retrieved ID identifies the first dialog of the specified dialog type. The refDialId parameter is ignored.
DG_DF_FIRSTVISIBLE The retrieved ID identifies the first visible dialog of the specified dialog type. The refDialId parameter is ignored.
DG_DF_LAST The retrieved ID identifies the last dialog of the specified dialog type. The refDialId parameter is ignored.
DG_DF_LASTVISIBLE The retrieved ID identifies the last visible dialog of the specified dialog type. The refDialId parameter is ignored.
DG_DF_NEXT The retrieved ID identifies the dialog below the reference dialog in the Z order.
DG_DF_NEXTVISIBLE The retrieved ID identifies the next visible dialog below the reference dialog in the Z order.
DG_DF_PREV The retrieved ID identifies the dialog above the reference dialog in the Z order.
DG_DF_PREVVISIBLE The retrieved ID identifies the previous visible dialog above the reference dialog in the Z order.
 
dialType
Specifies the type of the dialog whose ID is to be retrieved. This parameter can be one of the following values.
Value Meaning
DG_DT_MODAL The retrieved ID identifies a modal dialog.
DG_DT_MODELESS The retrieved ID identifies a modeless dialog.
DG_DT_PALETTE The retrieved ID identifies a palette dialog.

Return Values

The identifier of the specified dialog. If no dialog exists with the given parameters or there is a parameter error, the return value is zero.

Remarks

If the dialFlag parameter is DG_DF_NEXT, DG_DF_NEXTVISIBLE, DG_DF_PREV or DG_DF_PREVVISIBLE the type of the reference dialog has to be equal to the type specified by the dialType parameter.

The position of a dialog in the Z order can be set by using the DGSetDialogZPosition function.

Requirements

Version: DG 2.0.0 or later
Header: DG.h, DGDefs.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
