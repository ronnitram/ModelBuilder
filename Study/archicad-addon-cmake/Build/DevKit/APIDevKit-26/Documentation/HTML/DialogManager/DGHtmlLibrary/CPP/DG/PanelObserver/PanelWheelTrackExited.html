PanelObserver::PanelWheelTrackExited

Event handler for mouse wheel track exited notification. 

virtual void PanelWheelTrackExited (
    const PanelWheelEvent & ev,
    bool *              processed
);

Parameters

ev
A reference to the event.
processed
Output parameter: if the application has processed this event, this value should be set to true, to avoid the default event processing. By default the value of processed is false.
