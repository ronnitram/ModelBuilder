DGSetTabPageGrowType

Sets the grow type of a tab page.

    void DGSetTabPageGrowType (
      short  dialId,
      short  growType
    );

Parameters

dialId
ID of the tab page subdialog.
 
growType
Grow type of the specified tabpage subdialog. The following values are available:
 
Value Meaning
DG_DLG_NOGROW No size change.
DG_DLG_HGROW Horizontal size can be changed.
DG_DLG_VGROW Vertical size can be changed.
DG_DLG_HGROW | DG_DLG_VGROW Both horizontal and vertical size can be changed.

Remarks

Tab pages are placed in the display area of a tab control according to the fix (anchor) point and the grow type of the tab page. By default tab pages are centered in the display area. While resizing the tab control the fix point of the tab page keeps its place in the tab control and the size of the tab page is modified according to the grow type. The grow type and the fix point can be retrieved by the DGGetTabPageGrowType and DGGetTabPageFixPoint functions.
