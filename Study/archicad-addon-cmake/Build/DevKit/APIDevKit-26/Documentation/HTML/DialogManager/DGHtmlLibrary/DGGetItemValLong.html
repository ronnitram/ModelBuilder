DGGetItemValLong

Retrieves the current Int32 value of a dialog item.

    Int32 DGGetItemValLong (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.

Return Values

The current value.
Item type Meaning
Check Box 0: The control is not checked.
1: The control is checked.
Date & Time Control Current value of the date & time control.
On Macintosh: The number of seconds since January 1, 1904.
On Windows: The number of seconds since January 1, 1970.
Integer-type Edit Control Current value of the edit control.
Pop-up Control 1-based index of the currently selected pop-up item.
Progress Bar Current value of the progress bar.
Push Menu Control 0: The control is not checked.
1: The control is checked.
Radio Button 0: The control is not checked.
1: The control is checked.
Scroll Bar Current value of the scroll bar.
Slider Bar Current value of the slider bar.
Spin Control Current value of the spin control.
Tab Control 1-based index of the currently active tab page.
User Control Current value of the user control.

If the function parameters are incorrect, zero is returned.

Remarks

From DG version 1.1.1 use DGPushMenuIsChecked to retrieve the checked state of a push menu control.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
