DG_MSG_SPLITTERDRAG

A drag event occurs on a splitter control.

Message parameters

dialId
ID of the dialog or tab page.
 
item
1-based dialog item index of the dragged splitter control.
 
msgData
Pointer to a DGSplitterMsgData structure cast to DGMessageData.

Return Values

Not used, should be set to zero.

Remarks

DG_MSG_SPLITTERDRAG messages are sent to the dialog callback function when a splitter control is dragged.

By every dragging action the dialog callback function receives three or more DG_MSG_SPLITTERDRAG messages with the following submessages:

Value Meaning
DG_SPL_DRAGSTART This message is sent before the splitter control is dragged.
DG_SPL_DRAG This message is sent when the splitter control is being dragged.
DG_SPL_DRAGEND This message is sent after the splitter control is drop.

To change the dragable status of the splitter control use the DGSplitterEnableDrag and the DGSplitterDisableDrag functions.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
