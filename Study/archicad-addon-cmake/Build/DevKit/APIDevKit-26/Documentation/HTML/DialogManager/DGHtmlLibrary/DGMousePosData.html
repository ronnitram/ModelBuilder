DGMousePosData

The DGMousePosData structure contains information about the mouse position, the status of the modifier keys and the related item or dialog client rectangle. The structure is used only by the DGGetMousePosition function.

    struct DGMousePosData {
      short  dialId;
      short  itemId;
      short  left;
      short  top;
      short  right;
      short  bottom;
      short  xPos;
      short  yPos;
      short  keyFlag;
      short  buttonFlag;
    };

Members

dialId
ID of the dialog or tab page above which the mouse currently is.
 
itemId
1-based index of the dialog item above which the mouse currently is. A value of 0 means there is no item under the mouse.
 
left
Horizontal position of the upper left corner of the item or dialog client rectangle in pixels.
 
top
Vertical position of the upper left corner of the item or dialog client rectangle in pixels.
 
right
Horizontal position of the lower right corner of the item or dialog client rectangle in pixels.
 
bottom
Vertical position of the lower right corner of the item or dialog client rectangle in pixels.
 
xPos
Horizontal mouse position in pixels, relative to the client rectangle.
 
yPos
Vertical mouse position in pixels, relative to the client rectangle.
 
keyFlag
Status of the modifier keys. This value can be any combination of the following constants:
Flag Macintosh Windows
DG_MOD_COMMAND Command (Apple) Ctrl
DG_MOD_OPTION Option Alt
DG_MOD_SHIFT Shift Shift
DG_MOD_CTRL Ctrl Ctrl
 
buttonFlag
This flag represents the state of the mouse button(s) at the moment of the DGGetMousePosition function call:
Flag Macintosh Windows
DG_BUTTON_LEFT The button is pressed. The left button is pressed.
DG_BUTTON_RIGHT The rigth button is pressed. (only at OSX) The right button is pressed.
DG_BUTTON_WHEEL The wheel button is pressed. (only at OSX) The wheel button is pressed.

Remarks

If the mouse is not above a dialog item, left and top is set to zero, right and bottom is set to the dialog client area width and height, respectively.

If itemId is not zero, the rectangle defined by left, top, right and bottom is given in the main dialog client coordinate system.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
