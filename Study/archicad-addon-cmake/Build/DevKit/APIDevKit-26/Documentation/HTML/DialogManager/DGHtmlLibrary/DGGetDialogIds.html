DGGetDialogIds

Retrieves the identifiers of the dialogs that has the specified type and relationship to the specified reference dialog.

    short DGGetDialogIds (
      short   refDialId,
      bool    visibleOnly,
      bool    reversed,
      short   dialType,
      short*  dialIds
    );

Parameters

refDialId
ID of the dialog used as reference dialog.
 
visibleOnly
Visibility flag of the dialog.
 
reversed
The retrieved ID identifiers of the dialogs above or below the reference dialog in the Z order depending on the reversed flag.
 
dialType
Specifies the type of the dialog whose ID is to be retrieved. This parameter can be one of the following values.
Value Meaning
DG_DT_MODAL The retrieved ID identifies a modal dialog.
DG_DT_MODELESS The retrieved ID identifies a modeless dialog.
DG_DT_PALETTE The retrieved ID identifies a palette dialog.
short*
Output parameter: the retrieved ID identifies. This parameter cannot be nullptr.
 

Return Values

The number of the retrieved dialogs. If no dialog exists with the given parameters, the return value is zero.

Remarks

The position of a dialog in the Z order can be set by using the DGSetDialogZPosition function.

Requirements

Version: DG 2.2.0 or later
Header: DG.h, DGDefs.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
