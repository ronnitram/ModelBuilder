DGSetItemText

Sets a dialog item's text.

    void DGSetItemText (
      short                 dialId,
      short                 item,
      const GS::UniString&  text
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
text
The new text of the dialog item.

Remarks

The DGSetItemText function is applicable on the following type of dialog items only.

Type Subtypes
Check Box DG_BT_TEXT, DG_BT_PUSHTEXT
Edit Control DG_ET_TEXT, DG_ET_MULTILINETEXT, DG_ET_PASSWORD
Group Box all
Push Button DG_BT_ICONTEXT, DG_BT_PUSHTEXT, DG_BT_SPLIT
Radio Button DG_BT_TEXT, DG_BT_PUSHTEXT
Static Text all

On Macintosh, the item text is the title of the control for buttons, check boxes, radio buttons and group boxes.

If you set text of a multi-line edit control and the end-of-line markers are not the platform specific ones (CR on Macintosh, CRLF on Windows), DG automatically converts end-of-line markers in the text before sets it to the item.

To set text of a rich edit control, use DGRichEditSetText.

Requirements

Header: DG.h
