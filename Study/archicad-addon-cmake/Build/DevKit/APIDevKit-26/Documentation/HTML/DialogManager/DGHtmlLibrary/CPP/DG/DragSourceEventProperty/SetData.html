DragSourceEventProperty::SetData

Sets data to a drag source. 

bool SetData (
    DGDragFormat        format,
    void *              data,
    ULong               size,
    DragDrop::Scope     scope
);

Parameters

format
DGDragFormat type parameter. Used to specify the drag format in character style like: 'TEXT', 'FILE' etc.
data
Drag and drop data of the specified format.
size
Size of the drag and drop data.
scope
Scope of the data: DragDrop::Item, DragDrop::Dialog, DragDrop::Module, DragDrop::Application, DragDrop::System.

Return Value

true if data was set successfully, false otherwise.

Description

Use this function to attach custom data to the drag and drop operation. This member is usually called when the Drag Started event rise. Use this function to set delayed data when is requested.

Example

// Method Example:
void	TextControl::RichEditDragStarted (const DG::RichEditDragSourceEvent& ev, UShort* /*effect*/)
{
	if (ev.GetSource () == &controlledView->editControl) {
		DG::DragSourceEventProperty*	dsep = ev.GetDragData ();
		dsep->SetData ('YDXX', &sourceDocument, sizeof (TextDocument*), DG::DragDrop::Module);
	}
}

First you get a pointer to a DragSourceEventProperty object, than use this object to set the custom 'YDXX' type data to the drag operation.
