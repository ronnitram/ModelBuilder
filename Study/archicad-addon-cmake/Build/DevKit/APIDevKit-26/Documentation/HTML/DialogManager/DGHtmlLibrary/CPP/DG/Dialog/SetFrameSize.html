Dialog::SetFrameSize

Sets the size of the dialog frame rectangle. 

void SetFrameSize (
    short               width,
    short               height,
    FixPoint            fixPoint = TopLeft,
    bool                keepOld = false
);

Parameters

width
The width of the dialog frame rectangle.
height
The height of the dialog frame rectangle.
fixPoint
This parameter determines the reference point that does not move during resizing, and defines whether the old content of the dialog is to be kept at its original place.
keepOld
If true, the dialog keeps its old content. It means that the old content is not invalidated nor erased during the resize.

Description

This method resizes the dialog so that its frame rectangle gains the specified size.
