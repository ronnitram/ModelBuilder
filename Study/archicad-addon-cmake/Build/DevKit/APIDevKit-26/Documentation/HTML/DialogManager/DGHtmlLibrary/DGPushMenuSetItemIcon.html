DGPushMenuSetItemIcon

Sets the icon data of a menu item in a push menu control.

    void DGPushMenuSetItemIcon (
      short        dialId,
      short        item,
      short        menuItem,
      const void*  iconData
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
menuItem
1-based index of the menu item.
 
iconData
 Macintosh only: 
NSImage* pointer of the new icon cast to void*.
 Windows only: 
HICON handle of the new icon cast to void*.

Remarks

This function does not make a copy of the icon data, the application must manage this icon and delete it when it is no longer necessary. This also means the image must not be deleted as long as the icon is assigned to the push menu control.

The icon data of a menu item in a push menu control can be retrieved by DGPushMenuGetItemIcon.

An icon can also be specified to the push menu control item with its resource ID: see DGPushMenuSetItemIconId and DGPushMenuSetItemIconResource for details.

Requirements

Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
