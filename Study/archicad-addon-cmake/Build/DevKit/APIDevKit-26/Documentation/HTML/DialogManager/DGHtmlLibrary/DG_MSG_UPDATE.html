DG_MSG_UPDATE

A user item, a tabulator field of a list box or the image of a list view should be redrawn.

Message parameters

dialId
ID of the dialog or tab page.
 
item
1-based index of the dialog item.
 
msgData
Item type msgData
User Item Pointer to a DGUserUpdateData structure cast to DGMessageData.
List Box Pointer to a DGListUpdateData structure cast to DGMessageData.
List View Pointer to a DGListViewUpdateData structure cast to DGMessageData.

Return Values

Not used, should be set to zero.

Remarks

The dialog callback function receives DG_MSG_UPDATE messages in the following cases:

  • A user item should be updated. In this case the msgData parameter is a pointer to a DGUserUpdateData
  • structure that contains all the necessary information to update the user item by the application.
  • The update flag of one or more tabulator fields of a list box has been set to true by the DGListSetTabFieldUpdateFlag function, and these fields should be redrawn. Or the update flag of one or more list items of a list box has been set to true by the DGListSetItemOwnerDrawFlag function, and these whole list items should be redrawn. The origin of the drawing context defined by the drawContext member of the DGListUpdateData structure is set to the upper left corner of the context. The application should just draw the desired content; graying and highlighting are handled by DG after the dialog callback function returns.
  • The image type of one or more list view items are DG_LVIT_OWNERDRAW and these items should be redrawn. The origin of the drawing context defined by the drawContext member of the DGListViewUpdateData structure is set to the upper left corner of the image rectangle of the list view item. Within this rectangle the application can draw the image anywhere and in arbitrary size. The application should just draw the image; graying and blending are handled by DG after the dialog callback function returns. From DG version 1.1.1, in order to blend the right area only, the application can set left and top and modify width and height members of the DGListViewUpdateData structure on return to refer the actual rectangle of the drawn image.

Requirements

Version: DG 18.0.0 or later
Header: DG.h
