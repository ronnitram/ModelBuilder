DGListViewGetImageType

Retrieves the type of the image associated with a list view control.

    short DGListViewGetImageType (
      short  dialId,
      short  item
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 

Return Values

The returned image type can be one of the following values:

imageType imageData
DG_LVIT_NOIMAGE No image defined for this item.
DG_LVIT_ICON
 Macintosh only: 
Icon, referenced with NSImage* cast to void*.
 Windows only: 
Icon, referenced with HICON cast to void*.
DG_LVIT_ICONID Icon, referenced with its resource ID.
DG_LVIT_IMAGEINDEX Image, referenced with an index of a sub-image.
DG_LVIT_OWNERDRAW Image, managed and drawn by the application.
DG_LVIT_PICTURE
 Macintosh only: 
Picture, referenced with NSImage* cast to void*.
 Windows only: 
Picture, referenced with HBITMAP cast to void*.
DG_LVIT_PICTUREID Picture, referenced with its resource ID.
 Windows only:
DG_LVIT_DIBITMAP Device Independent Bitmap, referenced with BITMAPINFO* cast to void*.

If the function parameters are incorrect, DG_LVIT_NOIMAGE is returned.

Remarks

The image type depends on the used image-associating function.

function image type
DGListViewSetItemImage
  • DG_LVIT_NOIMAGE
  • DG_LVIT_ICON
  • DG_LVIT_PICTURE
  • DG_LVIT_DIBITMAP
DGListViewSetItemImageId or
DGListViewSetItemImageResource
  • DG_LVIT_NOIMAGE
  • DG_LVIT_ICONID
  • DG_LVIT_PICTUREID
DGListViewSetItemImageIndex
  • DG_LVIT_IMAGEINDEX
DGListViewSetItemOwnerDrawFlag
  • DG_LVIT_OWNERDRAW
