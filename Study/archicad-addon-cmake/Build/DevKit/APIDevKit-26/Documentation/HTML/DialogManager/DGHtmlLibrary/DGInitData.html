DGInitData

The DGInitData structure contains information about general DG settings and preferences.

        GS::UniString                   secondSymb;

Members

mainVersion
Main version number of DG library (0-9) as decimal number [and not as character ('0'-'9')]. When calling DGInit use the DGLIB_MAINVERSION define.
 
subVersion
Sub version number of DG library (0-9) as decimal number [and not as character ('0'-'9')]. When calling DGInit use the DGLIB_SUBVERSION define.
 
revision
Revision number of DG library (0-9) as decimal number [and not as character ('0'-'9')]. When calling DGInit use the DGLIB_REVISION define.
 
filler1
Reserved.
 
northSymb
North symbol for angle type edit controls. If northSymb is an empty string, DG uses the internally stored "N ".
 
southSymb
South symbol for angle type edit controls. If southSymb is an empty string, DG uses the internally stored "S ".
 
eastSymb
East symbol for angle type edit controls. If eastSymb is an empty string, DG uses the internally stored "E ".
 
westSymb
West symbol for angle type edit controls. If westSymb is an empty string, DG uses the internally stored "W ".
 
gradSymb
Grad symbol for angle type edit controls. If gradSymb is an empty string, DG uses the internally stored "G ".
 
radSymb
Radian symbol for angle type edit controls. If radSymb is an empty string, DG uses the internally stored "R ".
 
degSymb
Degree #1 symbol for angle type edit controls. If degSymb is an empty string, DG uses the internally stored "D ".
 
degreeSymb
Degree #2 symbol for angle type edit controls. If degreeSymb is an empty string, DG uses the internally stored "° ".
 
minuteSymb
Minute symbol for angle type edit controls. If minuteSymb is an empty string, DG uses the internally stored "´ ".
 
secondSymb
Second symbol for angle edit control. If secondSymb is an empty string, DG uses the internally stored "" ".
 
internalUnit
Application's internal length unit in mm.
 
contextMenuResModule
If the user finishes a drag&drop operation with the right mouse button, or clicks on an edit control item with the right mouse button, or clicks on a non-edit control item with the right mouse button, a context menu appears on the screen to choose an action. contextMenuResModule is the resource module of the module which contains the a special stringlist resource. The various context menus are constructed from the items of this stringlist. If contextMenuResModule is InvalidResModule or 0, DG uses the internal resource module.
 
allFilesText
'All Files' text for file dialog. If it is empty, DG uses the internally stored English text "All Files"
 
allTypesText
'All types' text when serializing command layouts. If it is empty, DG uses the internally stored English text "All types"
 
menuChoiceTexts
'Select Item', 'Select', and 'Cancel' texts for the dynamic menu item choice dialog. If any of them is empty, DG uses the internally stored English texts "Select Item", "Select", and "Cancel".
 
fileNameAlertResModule
If the user gives a too long path or filename in a filedialog, an alert will appear.fileNameAlertResModule is the resource module of this file name alert.
 
fileNameAlertId
If the user gives a too long path or filename in a filedialog, an alert will appear.fileNameAlertResModule is the resource ID of this file name alert.
 
rulerStrings
These texts appear when the user drags a tabulator in the ruler.
Text Meaning

the default is "mm".
cmText 'cm' unit text;
the default is "cm".
mText 'm' unit text;
the default is "m".
kmText 'km' unit text;
the default is "km".
leftTabText Text describing the left tabulator type;
the default is the English text "Left".
rightTabText Text describing the right tabulator type;
the default is the English text "Right".
middleTabText Text describing the middle tabulator type;
the default is the English text "Middle".
decimalTabText Text describing the decimal tabulator type;
the default is the English text "Decimal".
firstLineIndentText Text describing that the first line is indented;
the default is the English text "First Line Indent".
leftIndentText Text describing the line is indented to the left;
the default is the English text "Left Indent".
rightIndentText Text describing that the line is indented to the right;
the default is the English text "Right Indent".
 
openingBracket
Appended as an opening bracket to a text which describes a shortcut. If empty, the default " (" is used.
 
closingBracket
Appended as a closing bracket to a text which describes a shortcut. If empty, the default ")" is used.
 
 Macintosh only:
 
pPrefsFileLoc
DG saves the screen position/size of every dialog when closing them. When dialogs are reopened they appear at their last position. These positions are stored in a preferences file in the system Preferences folder. pPrefsFileLoc is the location of this file. If pPrefsFileLoc is nullptr, DG does not save the positions and sizes of dialogs.

the default is the English text "Hide".
hideothersText 'Hide Others' menu item name;
the default is the English text "Hide Others".
showAllText 'Show All' menu item name;
the default is the English text "Show All".
quitText 'Quit [application name]' menu item prefix;
the default is the English text "Quit".
helpMenuText 'Help' menu title;
the default is the English text "Help".
 
colorPanelStrings
These texts appear on modal color panels.
Text Meaning
okText 'OK' button text;
the default is the English text "OK".
cancelText 'Cancel' button text;
the default is the English text "Cancel".
 
fileTypeTitleOpen
Static text shown in front of the file type popup on custom Open dialog panels. If empty then uses the default English text "Enable:".
 
fileTypeTitleSave
Static text shown in front of the file type popup on custom Save dialog panels. If empty then uses the default English text "Format:".
 
chooseButtonTitle
Title and text of the default button in folder selection dialogs. If empty then uses the default English text "Choose".
 
fontCollections
Menu text of the font collections item in font popups. If empty then uses the default English text "Font Collections".
 
 Windows only:
 
contextMenuStrListId
contextMenuStrListId is the resource ID of the stringlist id of the localized context menu items stored in a stringlist. If contextMenuStrListId is 0, DG uses the internally stored English default menus. The stringlist has a strict format. The items are in the following order, and has the corresponding meaning: "Help", "Undo", "Cut", "Copy", "Paste", "Clear", "Select All", "Copy Here", "Move Here", "Cancel". If the user clicks on an edit control item with the right mouse button, a context menu appears on the screen to choose an action (Help/Undo/Cut/Copy/Paste/Clear/SelectAll). If the user finishes a drag&drop operation with the right mouse button, a context menu appears on the screen to choose an action (Copy/Move/Cancel). If the user clicks on a non-edit control item with the right mouse button, a context menu appears on the screen with a single item (Help).
 
applIconId
Resource ID of the application's icon. This icon appears on the alert, modal and modeless dialog captions.
 
fontCharSet
Character set of dialog fonts. If fontCharSet is 0, DG uses ANSI_CHARSET.
 
largeFontSize
Character size of the large dialog font in points. If largeFontSize is 0, DG uses 13.
 
smallFontSize
Character size of the small dialog font in points. If smallFontSize is 0, DG uses 13.
 
xSmallFontSize
Character size of the extra small dialog font in points. If xSmallFontSize is 0, DG uses 12.
 
largeFontWeight
Weight of the bold large font (it must be in the range of 400-1000). If largeFontWeight is 0, DG uses 700 (FW_BOLD).
 
smallFontWeight
Weight of the bold small font (it must be in the range of 400-1000). If smallFontWeight is 0, DG uses 600 (FW_SEMIBOLD).
 
xSmallFontWeight
Weight of the bold extra small font (it must be in the range of 400- 1000). If xSmallFontWeight is 0, DG uses 700 (FW_BOLD).
 
largeFontName
Typeface name of the large dialog font. If largeFontName is an empty string, DG uses "MS Sans Serif".
 
smallFontName
Typeface name of the small dialog font. If smallFontName is an empty string, DG uses "Arial".
 
xSmallFontName
Typeface name of the extra small font. If xSmallFontName is an empty string, DG uses "Arial".
 
closeMenuText
Close menu item text in the system menu of modeless dialogs. If closeMenuText is an empty string, DG uses the internally stored English text "&Close".
 
nextMenuText
Next window menu item text in the system menu of modeless dialogs. If nextMenuText is an empty string, DG uses the internally stored English text "Nex&t".
 
errorText
Title of error alert created by DGAlert. If errorText is an empty string, DG uses the internally stored English text "Error!".
 
warningText
Title of warning alert created by DGAlert. If warningText is an empty string, DG uses the internally stored English text "Warning!".
 
infoText
Title of information alert created by DGAlert. If infoText is an empty string, DG uses the internally stored English text "Information".
 
applRegKey
DG saves the screen position/size of every dialog when closing them. When dialogs are reopened they appear at their last position. These positions are stored in the Windows system registry under the application's registry key. applRegKey is the full path of this registry key. If applRegKey is an empty string, DG does not save the positions and sizes of dialogs.
 
rtlOrder
Specify for systems which use right to left writing order (Arabic, Hebrew). Affects the creation of text-like dialog items, the behavior of the ruler. Its value can be 0 (non-RTL) or 1 (RTL).

Remarks

The DGInitData structure is used by the DGInit function. The structure should be zeroed before calling DGInit because in case of empty fields DG uses internal defaults. Only fields with different values from the above mentioned defaults are to be set.

There is a version check in the DG dialog manager when calling the DGInit function. The relation between version numbers given in the DGInitData structure (external versions) and the ones in the DG dialog manager (internal versions) must be the following: mainVersion members must be equal, internal subVersion must be greater or equal than the external one, and if subVersions are equal the internal revision must be greater than or equal to the external one. This procedure ensures that no older version of the DG dialog manager can be used.
