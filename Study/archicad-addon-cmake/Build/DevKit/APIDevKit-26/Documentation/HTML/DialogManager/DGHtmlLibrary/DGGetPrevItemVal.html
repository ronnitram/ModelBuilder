DGGetPrevItemVal

If the checked state or the selected menu item of a push menu control has been changed, the dialog callback function receives a DG_MSG_CHANGE message. Use this macro to retrieve the previous state of the push menu control from the message data.

    short DGGetPrevItemVal (
      DGMessageData  msgData
    );

Parameters

msgData
The message data value passed to the callback function.

Return Values

The previous state of the push menu control. For check box-type push menu controls zero means that the control was previously unchecked, 1 means that it was previously checked. For radio button-type push menu controls: 1-based index of the previously checked radio button in the same group.

Remarks

The DGGetPrevItemVal macro is defined as follows:

    #define DGGetPrevItemVal(a)    ((short) (a))

The current state of the push menu control can be retrieved by DGPushMenuIsChecked.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
