DGGrowItem

Resizes a dialog item with the given relative values.

    void DGGrowItem (
      short  dialId,
      short  item,
      short  hGrow,
      short  vGrow
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
hGrow
Horizontal growth (shrink if hGrow is negative).
 
vGrow
Vertical growth (shrink if vGrow is negative).

Remarks

If dialog item type is DG_ITM_LISTBOX and the DG_LT_NOPARTIALITEMS flag is set, after growing the dialog item the list box is resized to hide the last partially visible list item. See DGListSetNoPartialItems for details.

Sizes of the same dialog item can be different on Macintosh and on Windows. This is why there is only relative positioning and sizing routines in DG for dialog items.

If an item must be both resized and moved within a DGBeginMoveGrowItems - DGEndMoveGrowItems block, you have to use DGMoveGrowItem instead of subsequent DGGrowItem and DGMoveItem calls. Otherwise it is not necessary to use DGMoveGrowItem but is still recommended.

Current size of a dialog item can be retrieved by DGGetItemRect.
