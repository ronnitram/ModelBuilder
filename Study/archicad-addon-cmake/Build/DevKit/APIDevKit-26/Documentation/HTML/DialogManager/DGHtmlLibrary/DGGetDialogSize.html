DGGetDialogSize

Retrieves the width and height of a dialog.

    void DGGetDialogSize (
      GSResModule    dialResModule,
      short          dialId,
      short          rectType,
      short*         hSize,
      short*         vSize
    );

Parameters

dialResModule
The resource module the original dialog resource, used when rectType is DG_ORIGCLIENT or DG_ORIGFRAME, and the dialog is not accessible.
dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
rectType
Value Meaning
DG_CLIENT Current size of the dialog client area.
DG_FRAME Current size of the dialog window (including frame and title).
DG_ORIGCLIENT Original size of the dialog client area, as given in the resource.
DG_ORIGFRAME Original size of the dialog window (including frame and title), as given in the resource.
DG_RESTCLIENT The size of the dialog client area, when the dialog window is restored.
DG_RESTFRAME The size of the dialog window (including frame and title), when the dialog window is restored.
 
hSize
Width of dialog. This parameter can be nullptr.
 
vSize
Height of dialog. This parameter can be nullptr.

Remarks

If rectType is DG_ORIGCLIENT or DG_ORIGFRAME, DGGetDialogSize retrieves the width and height even if the dialog is not open.

If either of the size parameter pointers are nullptr, that value is not returned.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
