DGTreeViewGetItemFromPosition

Retrieves the ID of a tree view item according to the specified mouse position.

    Int32 DGTreeViewGetItemFromPosition (
      short  dialId,
      short  item,
      short  xPos,
      short  yPos
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
xPos
Horizontal mouse position in screen coordinates.
 
yPos
Vertical mouse position in screen coordinates.

Return Values

The ID of the referenced tree view item or DG_TVI_NONE if there is no such an item.

Remarks

A tree item is considered as "listed (visible)" if its hierarchy level is expanded. However, an item can be listed as visible even if it is currently not in the client rectangle of the tree view.

To enumerate all items on a hierarchy level use DGTreeViewGetItem function.

Requirements

Version: DG 1.1.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
