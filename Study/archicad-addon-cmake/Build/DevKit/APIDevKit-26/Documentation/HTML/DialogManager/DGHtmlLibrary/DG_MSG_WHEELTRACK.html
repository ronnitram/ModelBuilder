DG_MSG_WHEELTRACK

Mouse wheel event occured.

Message parameters

dialId
ID of the dialog.
 
item
1-based index of the dialog item, otherwise zero if there was no dialog item where the event occured.
 
msgData
Pointer to a DGWheelMsgData structure cast to DGMessageData.

Return Values

Set to 1 if the event was handled by the application otherwise set to zero.

Remarks

DG_MSG_WHEELTRACK messages are sent to the dialog callback function when the mouse wheel was tracked by the user. By every mouse wheel action the dialog callback function receives three or more DG_MSG_WHEELTRACK messages with the following submessages:

Value Meaning
DG_TSM_TRACKENTER This message is sent when the wheel tracking action begins.
DG_TSM_TRACKING This message is sent continuously during the user tracks the mouse wheel.
DG_TSM_TRACKEXIT This message is sent when the wheel tracking action ends.

If the user moves the mouse to an other dialog item while a mouse tracking process is still going on, then the tracking process ends and a DG_TSM_TRACKEXIT message is sent to the dialog callback function.

See the DGWheelMsgData parameter for more details about the event.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
