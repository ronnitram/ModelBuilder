Group Box

A group box control is a rectangle that groups other controls together on a dialog or tab page. The controls are grouped by drawing a border around them and displaying a specified caption text in the upper-left corner.

On Macintosh, there are two types of group boxes: the so called primary and secondary ones. Primary and secondary group boxes differ from each other in their appearance. A secondary-type group box is usually applied when dialog items should be grouped within another group box or on a tab control page. On Windows, however, there is only one type of group box.

Sometimes a simple rectangle may be more sufficient to group dialog items than a normal group box. In this case use a rectangle-style separator item instead of a group box with empty caption text.

Item type

DG_ITM_GROUPBOX

Subtypes

Subtype Meaning
DG_GT_PRIMARY Simple group box.

  
DG_GT_SECONDARY Nested group box.

  

GRC Specification

    GroupBox    x  y  dx  dy  fontSpec  groupStyle  "text"

where x and y are the pixel coordinates of the upper left corner of the group box, dx and dy are the width and height of the control in pixels. fontSpec declares the font size and style of the caption text. groupStyle specifies the appearance of the control on Macintosh. This flag has no effect on Windows.

Possible values of fontSpec:

Value DG constant combination used in DGSetItemFont
ExtraSmall DG_IS_EXTRASMALL
SmallPlain DG_IS_SMALL | DG_IS_PLAIN
SmallBold DG_IS_SMALL | DG_IS_BOLD
SmallItalic DG_IS_SMALL | DG_IS_ITALIC
SmallUnderline DG_IS_SMALL | DG_IS_UNDERLINE
LargePlain DG_IS_LARGE | DG_IS_PLAIN
LargeBold DG_IS_LARGE | DG_IS_BOLD
LargeItalic DG_IS_LARGE | DG_IS_ITALIC
LargeUnderline DG_IS_LARGE | DG_IS_UNDERLINE

Possible values of groupStyle:

Value Meaning
Primary The DG_GT_PRIMARY subtype is set.
Secondary The DG_GT_SECONDARY subtype is set.

Remarks

The text of a group box can be set, modified and retrieved by DGSetItemText, DGCatItemText and DGGetItemText, respectively. The group box can display its text with any type of DG fonts. The font can be specified not only in GRC, but by the DGSetItemFont function as well. Use DGGetItemFont to retrieve the font currently assigned to a group box.

Requirements
