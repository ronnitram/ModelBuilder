DGIsDialogOpen

Returns whether a dialog is open or not.

    short DGIsDialogOpen (
      short  dialId
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.

Return Values

Value Meaning
Non-zero The specified dialog is open and the returned value is its dialog ID.
0 The specified dialog is not open or it is a tab page.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
