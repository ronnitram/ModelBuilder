DGSetDialogMinSize

Sets the allowed minimum width and height of the dialog.

    void DGSetDialogMinSize (
      short  dialId,
      short  rectType,
      short  hMinSize,
      short  vMinSize
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.
 
rectType
Value Meaning
DG_CLIENT hMinSize and vMinSize refer to the client area of the dialog.
DG_FRAME hMinSize and vMinSize refer to the window area of the dialog (including frame and title).
 
hMinSize
Minimum horizontal size in pixels.
 
vMinSize
Minimum vertical size in pixels.

Remarks

The minimal width and height of a dialog initially equals to the dialog size given in GRC. These values can be re-set by DGSetDialogMinSize.

The dialog can be shrunk either by user action or using DGGrowDialog or DGSetDialogSize until the dialog reaches its minimal sizes.

Requirements

Version: DG 1.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
