DGListViewGetImageLayout

Gets the image layout of a list view control.

    void DGListViewGetImageLayout (
      short   dialId,
      short   item,
      short*  nImages,
      short*  nRows
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
nImages
Output parameter: The number of the sub-images of the whole image. This parameter can be nullptr.
 
nRows
Output parameter: The number of the rows in the whole image. This parameter can be nullptr.
 

Remarks

The number of the sub-images and the number of the rows in the associated image can set by DGListViewSetImage, DGListViewSetImageId and DGListViewSetImageResource.

A sub-images can be assigned to a list view item by DGListViewSetItemImageIndex.

The size of the associated image can set and retrieved by DGListViewSetImageSize and DGListViewGetImageSize, respectively.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
