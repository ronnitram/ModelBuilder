DGGetCheckedRadio

Retrieves the checked radio button in a radio button group.

    short DGGetCheckedRadio (
      short  dialId,
      short  groupId
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
groupId
Identifier of radio button group.

Return Values

1-based item index of the checked radio button.

Remarks

A radio button can be grouped with other radio buttons of the same dialog or tab page. The grouping information can be specified in the GRC only. When switching on a radio button of a group (either by user action or from program code by DGSetItemValLong or DGSelectRadio), DG automatically switches off the other radio buttons of the group.

If a radio button in a group is switched on by user action, the dialog callback function is called with DG_MSG_CHANGE only for the radio button pressed. However, the modification flag of all the radio buttons in the group are set (DGModified will return 1).

Requirements

Version: DG 1.0.0 or later
Header: DG.h

Module: DGLib [MAC], DG.dll [WIN]
