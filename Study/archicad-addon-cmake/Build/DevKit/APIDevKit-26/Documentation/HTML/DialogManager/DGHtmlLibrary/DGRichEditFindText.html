DGRichEditFindText

Searches for a specified text in a given range of characters in a rich edit control.

    short DGRichEditFindText (
      short            dialId,
      short            item,
      Int32*           startPos,
      Int32*           endPos,
      GS::UniString*   text,
      short            findFlag
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
startPos
Meaning
On input: Character position where the search begins. Use DG_REP_TEXTBEGIN for the first character, DG_REP_TEXTEND for the end of the text.
On return: The position of the first character of the text found if the return value is 1.
 
endPos
Meaning
On input: Character position where the search stops. Use DG_REP_TEXTBEGIN for the first character, DG_REP_TEXTEND for the end of the text.
On return: The position of the last character of the text found if the return value is 1.
 
text
The text to search for.
 
findFlag
Search flag. This flag controls search direction and behavior.
 
The search direction values are exclusive.
Value Meaning
DG_RTF_DOWN Search down (from the start to the end).
DG_RTF_UP Search up (from the end to the start).
 
The search behavior flags can be combined to the search direction value.
Flag Description
DG_RTF_WHOLEWORD Match whole words only.
DG_RTF_MATCHCASE Case sensitive search.

Return Values

Value Meaning
-1 Error.
0 The specified text was not found.
1 The specified text was found.

Remarks

For position values use DG_REP_TEXTBEGIN, DG_REP_TEXTEND or the value returned by DGRichEditGetLinePos, DGRichEditGetSelect or this function.

Requirements

Version: DG 1.1.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
