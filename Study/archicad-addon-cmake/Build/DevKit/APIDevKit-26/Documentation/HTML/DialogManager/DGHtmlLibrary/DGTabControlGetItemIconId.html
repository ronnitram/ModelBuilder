DGTabControlGetItemIconId

Retrieves the icon ID of a tab control item.

    short DGTabControlGetItemIconId (
      short  dialId,
      short  item,
      short  tabItem
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
tabItem
1-based index of the tab control item.

Return Values

The icon ID of the specified tab control item. DG_NO_ICON is returned if the specified tab control item has no icon, if the tab control is a simple tab control, if the icon was set by DGTabControlSetItemIcon, or if the function parameters are incorrect.

Remarks

For specifying an icon to a tab control item with the icon ID: see DGTabControlSetItemIconId and DGTabControlSetItemIconResource for details.

The icon of a tab control item can also be set by its icon handle with the DGTabControlSetItemIcon function.
