DGTopStatusMsgData

If the dialog callback function receives a DG_MSG_TOPSTATUSCHANGE message for a modeless or palette dialog, the msgData parameter is a pointer to a DGTopStatusMsgData structure that contains information about the top status change event.

    struct DGTopStatusMsgData {
      short  subMessage;
      short  prevDialId;
      short  nextDialId;
      bool   byClick;
      Int8   filler;
    };

Members

subMessage
It can be one of the following values:
subMessage meaning
DG_TSM_TOPSTATUSLOST The dialog has lost the top-status.
DG_TSM_TOPSTATUSGAINED The dialog has gained the top-status.
 
prevDialId
ID of the dialog losing the top status.
 
nextDialId
ID of the dialog gaining the top status.
 
byClick
true if the dialog gained the top status by a mouse click.
 
filler
Reserved.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
