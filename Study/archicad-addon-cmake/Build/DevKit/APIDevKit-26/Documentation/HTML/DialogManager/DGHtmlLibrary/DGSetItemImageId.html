DGSetItemImageId

Sets the resource ID of the image of a dialog item.

    void DGSetItemImageId (
      short  dialId,
      short  item,
      short  imageId
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
imageId
Item type  Macintosh only:   Windows only: 
DG_ITM_BUTTON Resource ID of the icon. Resource ID of the icon.
DG_ITM_CHECKBOX Resource ID of the icon. Resource ID of the icon.
DG_ITM_RADIOBUTTON Resource ID of the icon. Resource ID of the icon.
DG_ITM_ICONITEM Resource ID of the icon. Resource ID of the icon.
DG_ITM_PICTITEM Resource ID of the picture. Resource ID of the bitmap.

Remarks

DGSetItemImageId function uses the resource module associated with the dialog or tab page containing the dialog item. Use this function if you are sure that the image is in the same resource module as the dialog. But if the resource module of the image might differ from the resource module of the dialog, than use the DGSetItemImageResource function to set the resource module containing the image. The resource module of the dialog can be set and retrieved by DGSetDialogResourceModule or DGSetDialogResourceModules and DGGetDialogResourceModule or DGGetDialogIconResourceModule, respectively.

The image of a dialog item can be also assigned with the image data by DGSetItemImage. If the image was set by its resource ID, the ID can be retrieved by DGGetItemImageId.

If the item type is DG_ITM_ICONITEM or DG_ITEMPICTITEM, the dialog item is automatically resized to the size of the image. During this operation the center of the dialog item is not moved and the possible frame is taken into account.
