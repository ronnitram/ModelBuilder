PropertyHint Enum

Identifiers for the global DG events. 

enum PropertyHint
{
    UnitSet,
    OnlyUpdate,
    NextModalParent,
    EditUpdateDelay,
    AutoCloseEnabled,
    AutoCloseDisabled,
    AutoCloseDelay,
    NormalUpdateEnabled,
    NormalUpdateDisabled,
    ScreenSaverEnabled,
    ScreenSaverDisabled,
    ScreenSaveOnModalDisabled,
    ToolTipsEnabled,
    ToolTipsDisabled,
    DialogRectangleReset,
    SelectionClear,
    SelectionCopy,
    SelectionCut,
    SelectionPaste,
    UndoEdit
};

Value Definitions

UnitSet
The unit has been changed with the SetUnit function.
OnlyUpdate
The onlyUpdate flag has been changed with the SetOnlyUpdateFlag function.
NextModalParent
The parent window of the next invoked alert, or modal dialog has been changed with the SetNextModalParent function.
EditUpdateDelay
The update delay time of the edit controls has been changed with the SetEditUpdateDelay function.
AutoCloseEnabled
The automatic close of alerts or modal dialogs has been enabled with the EnableAutoClose function.
AutoCloseDisabled
The automatic close of alerts or modal dialogs has been disabled with the DisableAutoClose function.
AutoCloseDelay
The automatic close delay time of alerts or modal dialogs has been changed with the SetAutoCloseDelay function.
NormalUpdateEnabled
The normal update mechanism of partially updateable user items has been enabled with the EnableNormalUpdate function.
NormalUpdateDisabled
The normal update mechanism of partially updateable user items has been disabled with the DisableNormalUpdate function.
ScreenSaverEnabled
The screen saver has been enabled with the EnableScreenSave function.
ScreenSaverDisabled
The screen saver has been disabled with the DisableScreenSave function.
ScreenSaveOnModalDisabled
The screen saver has been disabled at modal dialogs with the DisableScreenSaveOnModalDialogs function.
ToolTipsEnabled
The tooltips have been enabled with the EnableToolTips function.
ToolTipsDisabled
The tooltips have been disabled with the DisableToolTips function.
DialogRectangleReset
The dialog rectangles have been reset to their default values with the ResetDialogRectangles function.
SelectionClear
The selection in the focused edit control or rich edit control has been cleared with the ClearSelection function.
SelectionCopy
The selection in the focused edit control or rich edit control has been copied to the clipboard with the CopySelection function.
SelectionCut
The selection in the focused edit control or rich edit control has been cut to the clipboard with the CutSelection function.
SelectionPaste
The selection in the focused edit control or rich edit control has been replaced with the clipboard content with the PasteSelection function.
UndoEdit
The last operation of the focused edit control or rich edit control has been undone with the UndoEditOperation function.
