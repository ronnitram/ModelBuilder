DGListViewSetItemImageResource

Sets the image resource ID of a list view item and the resource module ID of the image.

    void DGListViewSetItemImageResource (
      short        dialId,
      short        item,
      short        listItem,
      short        imageType,
      GSResModule  resModule,
      short        imageId
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
listItem
1-based index of the list item. DG_LIST_TOP specifies the first item of the list, DG_LIST_BOTTOM specifies the last item of the list. Use DG_ALL_ITEMS to change the image resource ID of all items.
 
imageType
Type of the image:
Value Meaning
DG_LVIT_NOIMAGE No image is defined for this item.
DG_LVIT_ICONID Icon, referenced with its resource ID.
DG_LVIT_PICTUREID Picture, referenced with its resource ID.
 
resModule
The resource module ID of the image.
 
imageId
The resource ID of the image. If the imageType parameter is DG_LVIT_ICONID, the imageId can be any positive resource ID or one of the following predefined constants:

Remarks

There are two ways to define a list view item image:

  • Assign images to the items individually by DGListViewSetItemImage, DGListViewSetItemImageId or DGListViewSetItemImageResource. These images can be of different types and sizes for each list view item.
  • Associate one image with the list view itself by DGListViewSetImage, DGListViewSetImageId or DGListViewSetImageResource, divide it into smaller sub-images of the same size by giving the number of rows and the total number of sub-images. Assign a sub-image to the list view item by DGListViewSetItemImageIndex. The image index parameter of the function is the simple 1-based row-by-row index of a sub-image. In this case the type of the item's image will be DG_LVIT_IMAGEINDEX.

The image of a list view control item can be set either with its resource ID by DGListViewSetItemImageId or DGListViewSetItemImageResource, or with its image data by DGListViewSetItemImage. If the image was set by its resource ID, the image ID can be retrieved by DGListViewGetItemImageId and the resource module ID can be retrieved by DGListViewGetItemImageResource, in this case DGListViewGetItemImage returns nullptr.

If the image is located in the resource module of the dialog, the image can also be set with DGListViewSetItemImageId. The resource module of the dialog can be set and retrieved by DGSetDialogResourceModule or DGSetDialogResourceModules and DGGetDialogResourceModule or DGGetDialogIconResourceModule, respectively.
