DGListRedrawTabItem

Redraws a tabulator field of a list box item.

    void DGListRedrawTabItem (
      short  dialId,
      short  item,
      short  listItem,
      short  tabIndex
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
listItem
1-based index of the list item. DG_LIST_TOP specifies the first item of the list, DG_LIST_BOTTOM specifies the last item of the list.
 
tabIndex
1-based index of the tabulator field.

Remarks

The content of list items can be divided into more columns. In this documentation the notation "tabulator field" is used instead of "column". The reason for this is that texts of columns can be separated within a list item by inserting tabulator ('\t') characters into its text. Use DGListRedrawTabItem to redraw a tabulator field defined this way.

Call this routine only after you set the number of tabulator fields and their properties by DGListSetTabFieldCount and DGListSetTabFieldData.

If more parameters of list items (i.e. text, color, style, status, etc.) are modified simultaneously by calling list specific functions, it is advisable to enclose these function calls by DGListDisableDraw and DGListEnableDraw to avoid unwanted screen updates. As DGListEnableDraw itself does not update the list, it is the application's responsibility to redraw the invalid areas. If almost the whole list becomes invalid, use DGInvalidateItem or DGRedrawItem to update the list. However, only if a few list items or tabulator fields must be repainted use DGListRedrawItem or DGListRedrawTabItem.
