DGSetScrollBarPageSize

This function is obsolete, use the DGScrollBarSetPageSize function instead.

Sets the page size of a scroll bar control.

    void DGSetScrollBarPageSize (
      short  dialId,
      short  item,
      short  pageSize
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
pageSize
The new page size value of the scroll bar. This parameter must be greater than zero.

Remarks

When the user clicks in the gray area of a scroll bar, or on Windows presses the PageUp or PageDown keys while the scroll bar has the keyboard focus, the scroll box position is modified by the value of the page size. The page size is initialized from GRC by the following formula: pageSize = (maximum - minimum) / stepVal (stepVal must be greater than zero). When the range of the scroll bar is altered by DGSetItemMinLong or DGSetItemMaxLong, the pageSize value is automatically recalculated by DG.

Sometimes the above method of defining the page size is not sufficient. If you want to set the page size to a particular value use DGSetScrollBarPageSize. Note that after this function is called at least once, the page size is not recalculated from the control range and the stepVal value anymore. In this manner the page size can be made range independent. To retrieve the current page size use DGGetScrollBarPageSize.
