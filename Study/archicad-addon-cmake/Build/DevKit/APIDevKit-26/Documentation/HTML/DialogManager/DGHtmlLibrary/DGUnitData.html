DGUnitData

The DGUnitData structure contains information about the units, metrics and accuracy parameters used in DG.

    struct DGUnitData {
      double  north;
      Int32   lengthType;
      Int32   lengthDigits;
      Int32   lengthFractions;
      Int32   areaType;
      Int32   areaDigits;
      Int32   volumeType;
      Int32   volumeDigits;
      Int32   nonUnitDigits;
      Int32   angleType;
      Int32   angleDigits;
      Int32   angleAccuracy;
      Int32   polarAngleType;
      Int32   polarAngleDigits;
      Int32   polarAngleAccuracy;
      bool    show0whole;
      bool    show0inch;
    };

Members

north
Angle of the project North in radians, used in polar angle type edit controls (subtype: DG_ET_POLARANGLE) if polarAngleType is DG_UNIT_SURVEYORS. Otherwise not used.
 
lengthType
Unit of length-type edit controls (subtype: DG_ET_LENGTH). Possible values are:
Value Meaning
DG_UNIT_M metre
DG_UNIT_CM centimetre
DG_UNIT_MM millimetre
DG_UNIT_KM kilometre
DG_UNIT_DECINCH decimal inch
DG_UNIT_FOOT_INCH foot and fractional inch
DG_UNIT_INCH fractional inch
DG_UNIT_FOOT_DECINCH foot and decimal inch
DG_UNIT_DECFOOT decimal foot
 
lengthDigits
Number of decimal digits for length-type edit controls (subtype: DG_ET_LENGTH), used if lengthType is DG_UNIT_M, DG_UNIT_CM, DG_UNIT_MM, DG_UNIT_KM, DG_UNIT_DECINCH, DG_UNIT_FOOT_DECINCH or DG_UNIT_DECFOOT (so anything except fractional inch units).
 
lengthFractions
Denominator of fraction for length-type edit controls (subtype: DG_ET_LENGTH), used if lengthType is DG_UNIT_FOOT_INCH or DG_UNIT_INCH.
 
areaType
Unit of area-type edit controls (subtype: DG_ET_AREA). Possible values are:
Value Meaning
DG_UNIT_MM2 square millimetre
DG_UNIT_CM2 square centimetre
DG_UNIT_DM2 square decimetre
DG_UNIT_M2 square metre
DG_UNIT_SQ_INCH square inch
DG_UNIT_SQ_FOOT square foot
 
areaDigits
Number of decimal digits for area-type edit controls (subtype: DG_ET_AREA), used if areaType is one of the mentioned values.
 
volumeType
Unit of volume-type edit controls (subtype: DG_ET_VOLUME). Possible values are:
Value Meaning
DG_UNIT_CM3 cubic centimetre
DG_UNIT_DM3_LITER cubic decimetre (litre)
DG_UNIT_M3 cubic metre
DG_UNIT_CU_INCH cubic inch
DG_UNIT_CU_FOOT cubic foot
DG_UNIT_CU_YARD cubic yard
DG_UNIT_US_GALLON US gallon
 
volumeDigits
Number of decimal digits for volume-type edit controls (subtype: DG_ET_VOLUME), used if volumeType is one of the mentioned values.
 
nonUnitDigits
Number of digits in unitless edit controls (subtype: DG_ET_REAL, DG_ET_MM_OR_PT).
 
angleType
Unit of angle type edit controls (subtype: DG_ET_ANGLE). Possible values are:
Value Meaning
DG_UNIT_DECDEGREE decimal degree
DG_UNIT_DEGMINSEC degree, minute, second
DG_UNIT_GRAD grad ("new degree")
DG_UNIT_RADIAN radian
 
angleDigits
Number of decimal digits for angle type edit controls (subtype: DG_ET_ANGLE), used if angleType is DG_UNIT_DECDEGREE, DG_UNIT_GRAD or DG_UNIT_RADIAN.
 
angleAccuracy
Accuracy for angle type edit controls (subtype: DG_ET_ANGLE), used if angleType is DG_UNIT_DEGMINSEC. Possible values are:
Value Meaning
DG_ANGLE_DD degree
DG_ANGLE_DDMM degree, minute
DG_ANGLE_DDMMSS degree, minute, second
> 3 number of digits of seconds (angleAccuracy - 2)
 
polarAngleType
Unit of polar angle type edit controls (subtype: DG_ET_POLARANGLE). Possible values are:
Value Meaning
DG_UNIT_DECDEGREE decimal degree
DG_UNIT_DEGMINSEC degree, minute, second
DG_UNIT_GRAD grad ("new degree")
DG_UNIT_RADIAN radian
DG_UNIT_SURVEYORS surveyor unit
 
polarAngleDigits
Number of decimal digits for polar angle type edit controls (subtype: DG_ET_POLARANGLE), used if polarAngleType is DG_UNIT_DECDEGREE, DG_UNIT_GRAD or DG_UNIT_RADIAN.
 
polarAngleAccuracy
Accuracy for polar angle type edit controls (subtype: DG_ET_POLARANGLE), used if polarAngleType is DG_UNIT_DEGMINSEC, DG_UNIT_SURVEYORS. Possible values are:
Value Meaning
DG_ANGLE_DD degree
DG_ANGLE_DDMM degree, minute
DG_ANGLE_DDMMSS degree, minute, second
> 3 number of digits of seconds (polarAngleAccuracy - 2)
 
show0whole
If lengthType is DG_UNIT_FOOT_INCH or DG_UNIT_FOOT_DECINCH and this flag is true, zero foot is displayed before inches.
 
show0inch
If lengthType is DG_UNIT_FOOT_INCH or DG_UNIT_FOOT_DECINCH and this flag is true, zero inch is displayed after feet.

Remarks

The structure is used by the DGGetUnit, DGSetUnit, DGDoubleToStringUnit and DGStringToDoubleUnit functions.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
