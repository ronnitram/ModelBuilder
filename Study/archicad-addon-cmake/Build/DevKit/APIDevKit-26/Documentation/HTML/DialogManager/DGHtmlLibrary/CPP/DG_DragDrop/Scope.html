Scope Enum

The drag and drop operation scope. 

enum Scope
{
    Item =  DG_DRAG_SCOPE_ITEM,
    Dialog =  DG_DRAG_SCOPE_DIALOG,
    Module =  DG_DRAG_SCOPE_MODULE,
    Application =  DG_DRAG_SCOPE_APP,
    System =  DG_DRAG_SCOPE_SYSTEM
};

Value Definitions

Item
The drag and drop operation is item-wide.
Dialog
The drag and drop operation is dialog-wide.
Module
The drag and drop operation is module-wide.
Application
The drag and drop operation is application-wide.
System
The drag and drop operation is system-wide.
