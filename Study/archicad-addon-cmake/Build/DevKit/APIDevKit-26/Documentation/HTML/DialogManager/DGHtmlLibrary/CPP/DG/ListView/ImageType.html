ListView::ImageType Enum

The type of the image. 

enum ImageType
{
    NoImage =  DG_LVIT_NOIMAGE,
    Icon =  DG_LVIT_ICON,
    Picture =  DG_LVIT_PICTURE,
    DIBitmap =  DG_LVIT_DIBITMAP,
    IconId =  DG_LVIT_ICONID,
    PictureId =  DG_LVIT_PICTUREID,
    ImageIndex =  DG_LVIT_IMAGEINDEX,
    OwnerDraw =  DG_LVIT_OWNERDRAW
};

Value Definitions

NoImage
No image is specified.
Icon
The image is an icon given with HICON cast to void* on Windows, NSImage* cast to void* on Macintosh.
Picture
The image is a picture given with HBITMAP cast to void* on Windows, NSImage* cast to void* on Macintosh.
DIBitmap
Device independent bitmap given with BITMAPINFO* cast to void* (Windows).
IconId
The image is an icon given with its resource ID.
PictureId
The image is a picture given with its resource ID.
ImageIndex
The image is a part of the image attached to the list view, and the image is referenced with image index.
OwnerDraw
The image has an own draw method.
