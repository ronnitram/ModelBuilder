Dialog Keyboard Interface

On Macintosh:

Key Action
SHIFT + TAB Moves the keyboard focus to the previous focusable dialog item.
TAB Moves the keyboard focus to the next focusable dialog item.
ENTER Has the same effect as pressing the default button.
RETURN Has the same effect as pressing the default button, if the keyboard focus is not in a multi-line edit control or in a rich edit control.
ESC Has the same effect as pressing the cancel button.
COMMAND + '.' Has the same effect as pressing the cancel button.

On Windows:

Key Action
ALT + mnemonic Moves the keyboard focus to the first dialog item (having the WS_TABSTOP style) after the static control that contains the given mnemonic.
UP Moves the keyboard focus to the previous dialog item (having the WS_TABSTOP style) in the same group.
DOWN Moves the keyboard focus to the next dialog item (having the WS_TABSTOP style) in the same group.
LEFT Moves the keyboard focus to the previous dialog item (having the WS_TABSTOP style) in the same group.
RIGHT Moves the keyboard focus to the next dialog item (having the WS_TABSTOP style) in the same group.
TAB Moves the keyboard focus to the next dialog item (having the WS_TABSTOP style).
SHIFT + TAB Moves the keyboard focus to the previous dialog item (having the WS_TABSTOP style).
ENTER Has the same effect as pressing the default button, if the keyboard focus is not in a multi-line edit control or in a rich edit control.
ESC Has the same effect as pressing the cancel button.
