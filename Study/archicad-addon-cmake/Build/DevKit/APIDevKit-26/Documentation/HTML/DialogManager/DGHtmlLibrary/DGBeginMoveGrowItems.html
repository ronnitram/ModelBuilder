DGBeginMoveGrowItems

This function is used to disable dialog update while moving or resizing more dialog items to avoid screen flickering.

    void DGBeginMoveGrowItems (
      short  dialId
    );

Parameters

dialId
ID of the dialog. DG_TOP_MODAL specifies the topmost modal dialog.

Remarks

Always use this function in pairs with DGEndMoveGrowItems.

These functions are typically used in a dialog callback function when handling a DG_MSG_RESIZE or DG_MSG_GROW message. However, they can be used anywhere if more items need to be resized/moved at the same time.

After a call to DGBeginMoveGrowItems the subsequent DGGrowItem, DGMoveItem and DGMoveGrowItem calls do not actually change the size and/or position of the item until DGEndMoveGrowItems is called. Therefore DGGetItemRect will not retrieve the new coordinates before calling DGEndMoveGrowItems.

If an item must be both resized and moved within a DGBeginMoveGrowItems - DGEndMoveGrowItems block, you have to use DGMoveGrowItem instead of subsequent DGGrowItem and DGMoveItem calls.

This function cannot be used on a tab page.

Requirements

Version: DG 1.1.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
