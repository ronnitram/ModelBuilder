DGEditTextGetSelect

Retrieves the starting and ending character positions of the current selection in a edit control or rich edit control.

    void DGEditTextGetSelect (
      short   dialId,
      short   item,
      Int32*  selStart,
      Int32*  selEnd
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
selStart
Output parameter: the starting position of the selection. This parameter can be nullptr.
 
selEnd
Output parameter: the position of the first non-selected character after the end of the selection. This parameter can be nullptr.

Remarks

If either of the character position pointers are nullptr, that value is not returned.

Requirements

Version: DG 2.2.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
