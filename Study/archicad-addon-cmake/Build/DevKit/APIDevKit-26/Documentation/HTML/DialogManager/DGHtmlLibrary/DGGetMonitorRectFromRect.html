DGGetMonitorRectFromRect

Retrieves the rectangle of the monitor which mostly intersects the input rectangle.

    void DGGetMonitorRectFromRect (
      short*  left,
      short*  top,
      short*  right,
      short*  bottom
    );

Parameters

left
Input and output parameter. Left part of the rectangle.
 
top
Input and output parameter. Top part of the rectangle.
 
right
Input and output parameter. Right part of the rectangle.
 
bottom
Input and output parameter. Bottom part of the rectangle.
 

Return Values

The result rectangle is returned in the input rectangle parameters.

Remarks

Multi monitor systems: the result is the rectangle of the monitor which mostly intersects the input rectangle.
Single monitor system: the rectangle of the monitor.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
