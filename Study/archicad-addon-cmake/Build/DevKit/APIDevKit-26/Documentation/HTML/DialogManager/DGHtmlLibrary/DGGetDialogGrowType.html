DGGetDialogGrowType

Retrieves the number of items of the specified dialog.

    short DGGetDialogGrowType (
      short  dialId
    );

Parameters

dialId
ID of the dialog to count items.
 

Return Values

The grow type of the dialog, or combinations of them.

Flag Description
DG_DLG_NOGROW The dialog is not resizable.
DG_DLG_HGROW The dialog can be resized horizontally.
DG_DLG_VGROW The dialog can be resized vertically.

Remarks

If both horizontal and vertical resize is allowed, DG_DLG_HGROW | DG_DLG_VGROW is returned.

A dialog can be resized only in the directions allowed by the dialog grow type flag, given in GRC or specified by DGSetDialogGrowType.
