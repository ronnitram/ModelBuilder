DGPushMenuSetItemEnable

Enables or disables a menu item in a push menu control.

    void DGPushMenuSetItemEnable (
      short  dialId,
      short  item,
      short  menuItem
      bool   enable
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
menuItem
1-based index of the push menu item.
 
enable
Value Meaning
true Enable the specified push menu item.
false Disable the specified push menu item.

Remarks

The DGPushMenuSetItemEnable macro is defined as follows:

#define DGPushMenuSetItemEnable(a, b, c, d)     if (d) dgFt->dgPushMenuEnableItem (a, b, c); else dgFt->dgPushMenuDisableItem (a, b, c);

To retrieve the current state of a push menu item use the DGPushMenuIsItemEnabled function.
