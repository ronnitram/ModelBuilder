DGSetTabPageSize

Sets the size of a tab page.

    void DGSetTabPageSize (
      short  dialId
      short  hSize
      short  vSize
    );

Parameters

dialId
ID of the tab page subdialog.
 
hSize
Horizontal size of the tabpage.
 
vSize
Vertical size of the tabpage.

Remarks

Tab pages are assigned to tab control items. They appear in the display area of the tab control. Use this function to set the horizontal and vertical size of a tab page. To retrieve the current size use the DGGetTabPageSize function.

Requirements

Version: DG 2.0.0 or later
Header: DG.h, DGDefs.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
