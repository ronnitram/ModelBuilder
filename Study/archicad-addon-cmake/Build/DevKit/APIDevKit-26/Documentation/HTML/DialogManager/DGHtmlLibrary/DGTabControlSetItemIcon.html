DGTabControlSetItemIcon

Sets the icon data of a tab control item.

    void DGTabControlSetItemIcon (
      short        dialId,
      short        item,
      short        tabItem,
      const void*  iconData
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
tabItem
1-based index of the tab control item.
 
iconData
 Macintosh only: 
NSImage* pointer of the new icon cast to void*.
 Windows only: 
HICON handle of the new icon cast to void*.

Remarks

A normal tab control, unlike a simple tab control may have an optional icon and text displayed on its label. The icon in the label is always adjusted to the left of the label text.

The icon data of a tab control item can be retrieved by DGTabControlGetItemIcon.

An icon can also be specified to the tab control item with its resource ID: see DGTabControlSetItemIconId and DGPTabControlSetItemIconResource for details.

Requirements

Header: DG.h
