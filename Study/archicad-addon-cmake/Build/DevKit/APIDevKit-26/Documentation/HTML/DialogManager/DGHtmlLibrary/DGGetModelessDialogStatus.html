DGGetModelessDialogStatus

Retrieves the status of a modeless or palette dialog.

    short DGGetModelessDialogStatus (
      short  dialId
    );

Parameters

dialId
ID of the dialog.

Return Values

This status flag determines how the system-specific messages referring to the dialog are handled in DGModelessHandler.

Value Meaning
DG_DS_NORMAL If the onlyUpdate parameter of DGModelessHandler is false, all the messages are handled for the dialog.
If onlyUpdate is true in DGModelessHandler, only update messages are handled for the dialog.
DG_DS_ENABLED All the messages are handled for the dialog, value of onlyUpdate is omitted.
DG_DS_DISABLED No messages are handled for the dialog, value of onlyUpdate is omitted.

Remarks

When an application begins a long process such as a time-consuming calculation it is advisable to show a process indicator dialog with a Stop button. When handling system-specific messages during this period, applications should call DGModelessHandler with onlyUpdate of value true. In this case modeless dialogs with DG_DS_NORMAL state do not handle any messages except the update messages. However, the status of the process indicator dialog should be set to DG_DS_ENABLED by DGSetModelessDialogStatus in order to enable the user to press the button canceling the process this way.
