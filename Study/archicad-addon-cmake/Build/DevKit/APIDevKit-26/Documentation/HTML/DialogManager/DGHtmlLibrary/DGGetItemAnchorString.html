DGGetItemAnchorString

Reads anchor string of a given dialog item from DLGH-resource.

    GS::UniString DGGetItemAnchorString (
      short   dialId,
      short   item,
      GSResModule    dialResModule
    );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
dialResModule
The resource module the original dialog or tab page resource. Can be InvalidResModule when the dialog is accessible.

Return Values

Empty text if the anchor string cannot be found.
The anchor string in Unicode format otherwise.

Remarks

Every dialog item (on a Palette, on a Modeless or Modal Dialog, or on a TabPage) can have two types of help information: a short one and an anchor to the detailed information in the separate help engine.

Detailed help information of a dialog item can be invoked by setting the keyboard focus to a focusable dialog item then pressing the [F1] key on Windows or [Help] key on Macintosh; or by clicking the right mouse button on the item; or by clicking the help [?] button in the dialog caption then clicking on the item itself. The help information is displayed in a browser window. The dialog anchor and item anchor jointly refer to a specified part of the help content.

Item tooltips and help anchors are defined in DLGH resources for every type of dialogs (GRC Specification of the Dialog Helps).

Each dialog item can have more than one help string with different help index. This makes it possible to show different help strings in different situations for the same dialog item. Use DGSetItemHelpIndex to change the help index of a dialog item. When a dialog is created each item's help index is set to 1. If help index is changed with DGSetItemHelpIndex, the corresponding anchor string will be returned.
