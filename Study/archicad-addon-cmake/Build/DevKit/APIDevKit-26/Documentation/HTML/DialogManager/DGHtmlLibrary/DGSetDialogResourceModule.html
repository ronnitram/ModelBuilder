DGSetDialogResourceModule

Sets the resource module associated with the dialog.

    void DGSetDialogResourceModule (
      short        dialId
      GSResModule  resModule
    );

Parameters

dialId
The Id of the dialog.
resModule
Platform Meaning
Macintosh File reference number of an opened resource file cast to GSResModule.
Windows HINSTANCE of an executable (*.exe or *.dll) module cast to GSResModule.

Remarks

Use this function to specify the resource module of a dialog newly created from code.

DG associates the currently active resource module to a dialog when it is being created by DGModalDialog or DGModelessInit. The application can affect which module will be associated with a dialog by calling this function before creating the dialog. DG uses the associated module whenever it wants to load a resource belonging to the dialog.

Dialog resources can be placed into more modules (see DGSetDialogResourceModules). Same type of resources can have identical ID in different modules, although resource IDs are unique within a module.

Requirements

Version: DG 2.0.0 or later
Header: DG.h
Import Library: DG.lib [WIN]
Module: DGLib [MAC], DG.dll [WIN]
