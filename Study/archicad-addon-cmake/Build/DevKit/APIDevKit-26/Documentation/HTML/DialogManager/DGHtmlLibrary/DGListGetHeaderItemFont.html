DGListGetHeaderItemFont

Retrieves the font style and size of a header item of a list box item.

    short DGListGetHeaderItemFont (
      short        dialId,
      short        item,
      short        headerItem
   );

Parameters

dialId
ID of the dialog or tab page. DG_TOP_MODAL specifies the topmost modal dialog.
 
item
1-based index of the dialog item.
 
headerItem
1-based index of the header item.

Return Values

Font size and style information that is bitwise OR (|) combination of the following size and style flags:

Item text facename and size:
Value Macintosh Windows
DG_IS_LARGE systemFont 12. Name and size are specified in DGInit.
DG_IS_SMALL applFont 10.
DG_IS_EXTRASMALL applFont 9.
Item text style:
Value Meaning
DG_IS_PLAIN Item text is plain.
DG_IS_BOLD Item text is bold.
DG_IS_ITALIC Item text is italic.
DG_IS_UNDERLINE Item text is underlined.

Remarks

The list box header is displayed at the top of the list box. The header can be divided into more header items. The header is used to display information about the list box columns and can be configured to enable the user to dynamically adjust the size and the order of the list box columns. The number of the header items can be retrieved by DGListGetHeaderItemCount. The font style and size of a header item can be set by DGListSetHeaderItemFont.

Use DG_IS_SIZEMASK and DG_IS_STYLEMASK to retrieve the size and style flags from the returned value.

With DG_IS_EXTRASMALL only the DG_IS_PLAIN style is allowed.

If the isSynchron flag is set to true on the list box header, the properties of the header items are synchronized with the corresponding tabulator fields by DG, otherwise they should be handled independently by the application. These properties are: the number and size of the header item, the font type and size, the alignment, the truncation of the text of the header item. If this flag is set, these properties are automatically updated when a property of a tabulator field is changed.
If the isSynchron flag is set on the list box header, the DGListSetHeaderItemFont function does not affect the header item properties.
The header flags can be set and retrieved by the DGListSetHeaderFlags and DGListGetHeaderFlags functions respectively.
