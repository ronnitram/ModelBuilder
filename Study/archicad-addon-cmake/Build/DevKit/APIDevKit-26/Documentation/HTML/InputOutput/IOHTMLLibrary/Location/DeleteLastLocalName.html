Location::DeleteLastLocalName


Deletes the last part (name) from the location's local address.

GSErrCode    DeleteLastLocalName (
    void
);

Return value

Value Meaning
NoError* OK
ErrMemoryFull* out of memory
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

This method deletes the last part (name) from the location's local address. Length of the local address is decreased by 1. Local address should not be empty before the invocation.


Available from version: 1.0
