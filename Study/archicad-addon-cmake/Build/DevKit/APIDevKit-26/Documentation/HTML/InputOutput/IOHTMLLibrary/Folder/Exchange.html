Folder::Exchange


Swaps two files contents.

virtual GSErrCode  Exchange (
    const Name& from,
    const Name& to
);

Parameters

from
First operand of the Excahnge method.
to
Second operand of the Excahnge method.

Return value

Value Meaning
NoError* OK
SourceNotFound* source entry to be exchanged is not found
DeviceIsFull* device is full
AccessDenied* operation is not allowed on this folder or on the entry
Error* unspecified error

* declared in Folder class scope
* declared in GS namespace, but accessible in the global scope

Description

This method exchanges two files contents. It means the contents are fully swapped, but attributes(access, comment, etc.) of the files remain.
