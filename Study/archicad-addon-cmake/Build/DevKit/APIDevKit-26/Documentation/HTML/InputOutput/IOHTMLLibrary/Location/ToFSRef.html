Location::ToFSRef


Returns native Macintosh representation of the location's target.

GSErrCode ToFSRef (
    FSRef* result
) const;

Parameters

result
On return: native Macintosh representation of the location's target.

Return value

Value Meaning
NoError* OK
ErrMemoryFull* out of memory
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

This method returns native Macintosh representation of the location's target in result. This method is used for only existing entries.
Note that there are locations for which there is no valid FSRef representation (e.g. local root is virtual on Macintosh). For such locations Error is returned.

 Windows notes:
  • This method doesn't work on Windows.
