FileSystem::GetResolvedType


Returns resolved type of a file system entry identified with its location.

GSErrCode  GetResolvedType (
    const Location& loc,
    EntryType*      result
) const;

Parameters

loc
Location of the file system entry of which resolved type should be determined.
result
On return: resolved type of the entry. Possible values are:
Value Meaning
File file
Folder folder
Network network (unimplemented)

Return value

Value Meaning
NoError* OK
TargetNotFound* entry is not found
AccessDenied* operation is not allowed on the entry
Error* unspecified error

* declared in FileSystem class scope
* ddeclared in GS namespace, but accessible in the global scope

Description

This method returns resolved type of a file system entry (file, folder, link) identified with its location. Resolved type of a link is type of the entry which link (or chain of links) refers to. Resolved type of files and folders remains file and folder respectively.

Note that determining type of a link's target can be slow because it may be offline (see the IsOnLine method). If it is not desirable then EstimateResolvedType method can be used.

 Windows notes:
  • If link extensions are restricted (see the RestrictLinkExtensions function) then this method will examine file system entries only with extension ".lnk" for the potential link resolvation.
