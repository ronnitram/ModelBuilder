Location::GetDifference


Returns difference between two locations as a relative location.

GSErrCode  GetDifference (
    const Location&   rightOp,
    RelativeLocation* result
) const;

Parameters

rightOp
Right (second) operand of the difference operation.
result
On return: result of the difference operation.

Return value

Value Meaning
NoError* OK
InvalidOperand* operands are incompatible for this operation
ErrMemoryFull* out of memory
Error* unspecified error

* declared in Location class scope
* declared in GS namespace, but accessible in the global scope

Description

This method returns difference between two locations as a relative location. If the difference between locations x and y is the relative location z, then result of appending z to y will be x. Difference operation can be performed only between locations with the same network address (otherwise InvalidOperand will be returned), because relative locations are relative to the locations' local address.

If the first operand of the difference operation is not ancestor of the second one (according to the IsAncestorOf method), then the resulting relative location will have ".." names (meaning parent folder) in the leading positions. If there are more equivalent results, then the shortest one is returned.
