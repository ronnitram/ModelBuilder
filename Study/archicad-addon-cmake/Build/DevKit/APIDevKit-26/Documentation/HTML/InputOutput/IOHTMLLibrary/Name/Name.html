Description

This class holds a name of a file system entry in Unicode representation. It is elementary logical building block of locations (Location, RelativeLocation). On the other hand Name holds a string of arbitrary length and content.

Although generally there is no any restriction related to the content of a name, there can be some at the point of use on the particular platform. For example when creating a file, it is not allowed on Windows for its name to contain some special characters like '|' or '*'. User can check whether is a name syntactically legal on the current platform (with IsLegal method) or on every platform supported by InputOutput module (with IsMultiplatform method). Syntactically legal means simply that a file system entry with such name may exist.
