File::File   (3)


File constructors.

explicit File (
    const Location& loc,
    OnNotFound      action = Fail
);

Parameters

loc
Location of the file in the file system.
action
Action to be performed if the file is not found.

Description

This constructor creates connection to the file identified with the Location loc. If loc refers to a link, then it is automatically resolved. If the file is not found, then the action parameter is examined. On the default value Fail status of the File instance becomes invalid. On the value Create it will try to create the file in the file system if its parent folder (obtained from loc) exists.

Result of the construction can be examined by a subsequent invocation of the GetStatus method, where return value NoError indicates successful construction.

Instances should not be created before the module initialization!
