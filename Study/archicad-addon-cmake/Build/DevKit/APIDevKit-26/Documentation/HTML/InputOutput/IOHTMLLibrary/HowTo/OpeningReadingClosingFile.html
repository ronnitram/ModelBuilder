Opening, reading and closing a file



The following sample code demonstrates the basic file management task: opening a file (File), reading data from it and closing the file.

To operate on an OS file, you have to obtain its location (address, path) in the file system. Class Location is responsible for holding and managing these locations. User can obtain or build a location in many ways (=> more information). Here we will assume that our test file named "Test.dat" is located beside the running application.

Note that error codes of operations in the sample code are not processed, only their possibility is shown.

To this sample work, InputOutput module must be initialized (=> more information).



#include "File.hpp"          // brings everything needed to use the File (e.g. Location)
#include "FileSystem.hpp"    // to query location of the running application

GSErrCode errorCode;    // holds error codes of operations

    // obtaining location of the file: application folder's location + name of the file

IO::Location fileLoc;    // Location instance
errorCode = IO::fileSystem.GetSpecialLocation (IO::FileSystem::ApplicationFolder, &fileLoc);
errorCode = fileLoc.AppendToLocal ("Test.dat");

    // creating a File instance which will be used to operate on the desired OS file

IO::File file (fileLoc);

    // using the file

errorCode = file.Open (IO::File::ReadMode);    // opening the file in read-only mode

char buffer[256];
errorCode = file.ReadBin (buffer, 16);         // reading 16 bytes from the file into the buffer

errorCode = file.Close ();                     // closing the file
