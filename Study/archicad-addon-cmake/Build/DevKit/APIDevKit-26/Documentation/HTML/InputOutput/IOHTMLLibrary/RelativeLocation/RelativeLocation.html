Description

RelativeLocation is relative address or relative path of entries (file, folder, link) in the file system. RelativeLocation is an arbitrary long sequence of names (see class Name), yielding a relative hierarchy (tree), which can be freely manipulated with several methods. Each name consists of arbitrary long sequence of characters.

RelativeLocation is not useable in isolation, at some point of use it must be appended to an absolute location represented by the class Location. RelativeLocation is relative to the local address of the Location (there is no support for relative network addresses). So a relative location can be manipulated, saved, restored and finally appended to the local address of an absolute location.

Although generally there is no any restriction related to the content of a relative location, there can be some at the point of use on the particular platform. For example when creating a file, it is not allowed on Windows for its location to contain some special characters like '|' or '*'. User can check whether is a relative location syntactically legal on the current platform (with IsLegal method) or on every platform supported by InputOutput module (with IsMultiplatform method). Syntactically legal means simply that a file system entry with such relative location may exist.

When appending a relative location to an absolute location, some special names like "." and ".." have special meaning and special processing. For example "/A/B/C" + "../../D" = "/A/D". For more information see the corresponding Location::AppendToLocal method.

Implementation of the RelativeLocation class is highly optimized both to space and speed. Copy constructor and assignment operator are fast and do not require additional heap space. Internally, relative locations are stored in a common shared tree. Similar relative locations require additional space only for different segments.


Note: in every method where const Name& is expected, C string can be also passed, because there is implicit conversion from char* to Name.
