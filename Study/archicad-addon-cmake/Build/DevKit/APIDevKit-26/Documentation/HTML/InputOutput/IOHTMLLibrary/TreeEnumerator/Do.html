TreeEnumerator::Do


Starts the recursive enumeration.

virtual GSErrCode  Do (
    void
);

Return value

Value Meaning
NoError* OK
AccessDenied* access to some entry was denied
Error* unspecified error

* declared in TreeEnumerator class scope
* declared in GS namespace, but accessible in the global scope

Description

This method starts the recursive enumeration. Callback methods (EntryFound, FileFound etc.) will be invoked according to the tree structure.
Enumeration can be stopped with the Stop method. Status of the enumeration can be queried with the GetStatus method.
