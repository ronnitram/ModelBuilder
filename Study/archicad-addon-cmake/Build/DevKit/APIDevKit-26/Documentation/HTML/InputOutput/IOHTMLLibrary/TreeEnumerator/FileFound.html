TreeEnumerator::FileFound


Callback method which is invoked for files encountered during the enumeration.

virtual bool  FileFound (
    const Name& fileName
);

Parameters

fileName
Name of the file.

Return value

Controls further processing of the file. Currently false should be returned (it is the default implementation), which disables further processing.

Description

This method is invoked for files encountered during the enumeration and is intended to be overridden by the user. It is invoked only if the return value from the previous EntryFound method is true. It receives name of the file. Return value controls further processing (classification) of the file. Default implementation does nothing and disables further processing.
