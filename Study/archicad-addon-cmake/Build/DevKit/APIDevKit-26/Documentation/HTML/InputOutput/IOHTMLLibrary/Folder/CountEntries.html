Folder::CountEntries


Returns number of file system entries in the folder.

virtual GSErrCode  CountEntries (
    USize* result
) const;

Parameters

result
On return: number of file system entries in the folder.

Return value

Value Meaning
NoError* OK
AccessDenied* operation is not allowed on this folder
Error* unspecified error

* declared in Folder class scope
* declared in GS namespace, but accessible in the global scope

Description

This method returns number of file system entries (files, folders and links) in the folder in result.
