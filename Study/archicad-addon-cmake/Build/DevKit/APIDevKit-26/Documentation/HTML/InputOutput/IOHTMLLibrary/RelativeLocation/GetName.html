RelativeLocation::GetName


Returns the specified part (name) of the relative location.

GSErrCode  GetName (
    UIndex idx,
    Name*  result
) const;

Parameters

idx
Index of the relative location's part (name) to be queried. Indexing begins from 0. Part with index 0 is the highest order name.
Index should be between 0 end length of the relative location - 1 (in names).
result
On return: the specified part (name) of the relative location.

Return value

Value Meaning
NoError* OK
ErrMemoryFull* out of memory
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

This method returns part (name) with index idx of the relative location in result.
