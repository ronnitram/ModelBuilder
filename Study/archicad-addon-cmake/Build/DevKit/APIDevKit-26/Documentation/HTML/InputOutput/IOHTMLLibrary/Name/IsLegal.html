Name::IsLegal


Returns whether is the name syntactically legal on the current platform.

bool  IsLegal (
    void
) const;

Return value

Value Meaning
true name is syntactically legal on the current platform
false name is syntactically illegal on the current platform

 

Description

This method returns whether is the name syntactically legal on the current platform. Syntactically legal means simply that a file system entry with such name may exist.
For example on Windows file names can't contain some special characters like '|' or '*'.
