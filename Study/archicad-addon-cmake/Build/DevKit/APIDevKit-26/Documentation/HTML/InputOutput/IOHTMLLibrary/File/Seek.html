File::Seek


Adjusts file content position.

virtual GSErrCode  Seek (
    SeekOrigin so,
    Int64       offset
);

Parameters

so
Origin for the offset parameter. Possible values are:
Value Meaning
Start beginning of the file content
Current current file content position
End end of the file content
offset
Seek offset. It is related to the seek origin (so). Unit of the offset is character (byte).

Return value

Value Meaning
NoError* OK
IllegalPosition* illegal file content position was specified
GS::OChannel::OutputIsFull output device is full
GS::OChannel::WriteError write error on the output device
Error* unspecified error

* declared in File class scope
* declared in GS namespace, but accessible in the global scope

Description

This method sets file content position to the current value of the seek origin (so) offsetted by offset. Resulting position should not be negative. It is not allowed to set the position beyond the end of the file content, but it can be set to the end of the file content. Position setting can imply output buffers to be flushed, so write error may occur. 

This operation is not allowed if the file content was not opened, or it was opened in appending mode. It is also disallowed "inside" output channel (GS::OChannel) block operations (that is when mark was put, or when offset is not reset).
