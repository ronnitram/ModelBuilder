File::GetDataLength  (2)


Returns length of the file content.

virtual GSErrCode  GetDataLength (
    UInt64* result
) const;

Parameters

result
On return: the file content length. Unit of the length is character (byte).

Return value

Value Meaning
NoError* OK
AccessDenied* operation is not allowed on this file
GS::OChannel::OutputIsFull output device is full
GS::OChannel::WriteError write error on the output device
Error* unspecified error

* declared in File class scope
* declared in GS namespace, but accessible in the global scope

Description

This method returns length of the file content in result. If the file was already opened in some of write modes, then internal buffers are first flushed, so write error may occur.
