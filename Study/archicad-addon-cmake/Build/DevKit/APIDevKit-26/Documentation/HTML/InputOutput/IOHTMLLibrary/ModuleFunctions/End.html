End


Uninitializes the InputOutput module.

Header: IOMain.hpp

long  End (
    void
);

Return value

Value Meaning
0 OK
-1 an error was occurred during the uninitialization

Description

After the last use InputOutput module must be uninitialized with this function. This must happen inside the main (). End should be invoked exactly the same times as Init, and only the last one will perform the actual task of uninitialization.
