Description

Folder is a file system entry that contains other file system entries (files, folders and links) referred to as its content or children. Number of child entries is limited only by OS. Every child entry has a name associated with it, which is unique within its parent folder (that is in its context).
Folder has few attributes to query and set including its creation date, modification date, amount of free space in it and other properties.

Local file system of every machine has a single root parent folder. (It can be created from the location initialized with the default constructor). On some systems this root folder is virtual and it contains volumes (logical drives). However to obtain mount points, user should use EnumerateMountPoints method of the class FileSystem, instead of enumerating the root folder.

Folder is responsible for its direct content (direct children). This content can be enumerated with the Enumerate method. However often recursive enumeration of a whole folder tree is needed. This task is supported by the class TreeEnumerator.
