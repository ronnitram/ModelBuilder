Description

MemoryIChannel is an input channel which reads data from a buffer in memory.

The source buffer is supplied by the user. Its ownership can be controlled and changed. MemoryIChannel will delete a buffer (for example at destruction time) only if it owns that one.

Default input protocol is BinIProtocol.
