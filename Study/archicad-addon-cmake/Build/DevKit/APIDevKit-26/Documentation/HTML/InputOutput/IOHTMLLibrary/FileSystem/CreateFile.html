FileSystem::CreateFile


Creates a new file in the file system.

GSErrCode  CreateFile (
    const Location& loc
);

Parameters

loc
Location of the new file.

Return value

Value Meaning
NoError* OK
TargetExists* an entry at the same location already exists
DeviceIsFull* device is full
AccessDenied* operation is not allowed
Error* unspecified error

* declared in FileSystem class scope
* declared in GS namespace, but accessible in the global scope

Description

This method creates a new file at the specified location in the file system. Creation and modification dates are set appropriately. Content of the new file will be empty.

CreateFile fails if the file system already contains an entry (file, folder, link) at the specified location, and on non-writeable folders.

 Windows notes:
  • This method fails on the root folder (because it is virtual and not writeable).
  • File has no file type and creator attributes.
 Macintosh notes:
  • This method fails on the root folder (because it is virtual and not writeable).
  • File type and file creator attributes are set to 0.
  • Script system code of the new file is set to smSystemScript.
