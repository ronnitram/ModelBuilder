TreeEnumerator::GetStatus


Returns current status of the enumeration.

GSErrCode  GetStatus (
    void
) const;

Return value

Current status of the enumeration.

Description

This method can be used inside of callback methods to query current status of the enumeration.
Enumeration will continue even if an error is encountered (for example access to a folder was denied). In this case code of the first error is stored and can be queried with this method. It will be also returned as result of the enumeration (as return value of the Do method).
