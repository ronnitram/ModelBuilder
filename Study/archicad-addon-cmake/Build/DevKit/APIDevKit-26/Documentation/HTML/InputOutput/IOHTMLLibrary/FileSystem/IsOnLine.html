FileSystem::IsOnLine


Returns whether is the location online.

bool  IsOnLine (
    const Location& loc
) const;

Parameters

loc
Location which should be examined for the direct accessibility.

Return value

Value Meaning
true location is online
false location is not online (offline)

Description

This method determines whether is the location online. If a location is online then it can be used quickly and without side-effects regardless of whether it refers to an existing file system entry or not. Otherwise operations can be slow or can imply user interactions etc. Online/offline property has effect to virtually all classes and methods where locations can occur.

Note that while offline property is inherited (so descendants of an offline location are also offline) online is not (so a descendant of an online location can be either online or offline).

An offline location may become online due to its usage (for example through login dialog).
