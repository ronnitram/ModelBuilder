FileSystem::CreateFolderTree   (2)


Creates nested folder structure (degenerate folder tree).

GSErrCode  CreateFolderTree (
    const Location& tree
);

Parameters

tree
Location which local address represents nested folder structure (as sequence of folder names) to be created.

Return value

Value Meaning
NoError* OK
TargetNotFound* root of the tree (machine) is not found
NotAFolder* an entry other than folder is encountered during the tree creation
DeviceIsFull* device is full
AccessDenied* operation was not allowed at some point
Error* unspecified error

* declared in FileSystem class scope
* declared in GS namespace, but accessible in the global scope

Description

Based on the local address of the given location as sequence of folder names, this method creates the corresponding nested folder structure. If front (prefix) of the tree already exists then this method creates only the missing parts (suffix).


Available from version: 1.0

    const RelativeLocation& tree

Parameters

root
Location referring to the folder from (in) which nested folder structure should be created.

Relative location representing nested folder structure (as sequence of folder names) to be created.

Return value

Value Meaning

DeviceIsFull* device is full

* declared in FileSystem class scope
* ddeclared in GS namespace, but accessible in the global scope

Description

Based on the given relative location as sequence of folder names, this method creates the corresponding nested folder structure starting from (in) the given root folder. If front (prefix) of the tree already exists then this method creates only the missing parts (suffix).
