Init


Initializes the InputOutput module.

Header: IOMain.hpp

long  Init (
    void
);

Return value

Value Meaning
0 OK
-1 an error was occurred during the initialization, module can not be used

Description

Before the first use InputOutput module must be initialized with this function. This must happen inside the main (). It can be invoked arbitrary times regardless of result of the previous initializations.

Instances of classes Location and RelativeLocation created with the default constructor, furthermore instances of classes Name and protocol implementations can be used outside ("before") the main (), for example as static global variables. (Note that their constructor will be executed before the main (), that is before the module initialization!)

After the last use InputOutput module must be uninitialized. This can be done with the End function, which should be invoked exactly the same times as Init.
