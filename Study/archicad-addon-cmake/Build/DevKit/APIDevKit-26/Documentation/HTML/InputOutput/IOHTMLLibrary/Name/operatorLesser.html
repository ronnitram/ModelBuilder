Name::operator<


Returns whether the name is lesser than the right operand.

bool operator< (
    const Name& rightOp
) const;

Parameters

rightOp
Right operand of the < operation.

Return value

Value Meaning
true this name is lesser than the right operand
false this name is not lesser than the right operand

Description

This method returns whether this name is lesser than rightOp. The operation is platform dependent, because they are treated and compared as file system entry names, rather then plain strings. For example such platform dependent property is the case sensitivity-insensitivity of names.
