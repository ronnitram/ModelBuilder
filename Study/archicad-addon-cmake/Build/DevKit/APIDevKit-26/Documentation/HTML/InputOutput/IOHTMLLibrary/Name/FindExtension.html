Name::FindExtension


Returns pointer to the C string representation of the name's extension part.

const char*  FindExtension (
    void
) const;

Return value

Pointer to the C string representation of the name's extension part, or NULL pointer if no extension was found. This pointer points into the object, so it should not be permanently stored and it may become invalid after a subsequent invocation of a Name method on this object.

Description

This method returns pointer to the C string representation of the name's extension part. Extension is defined as suffix of the name after the last '.' (dot) character (not including the '.' character itself). If the name doesn't contain any '.' character, then it has no extension (NULL pointer will be returned).
