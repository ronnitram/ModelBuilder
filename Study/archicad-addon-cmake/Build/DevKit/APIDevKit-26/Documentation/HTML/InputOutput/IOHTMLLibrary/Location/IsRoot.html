Location::IsRoot


Returns whether is the local address of the location empty.

bool  IsRoot (
    void
) const;

Return value

Value Meaning
true local address is empty
false local address is not empty

 

Description

This method returns whether is the local address of the location empty. Local address is empty if its length in names (constituent parts) is 0. Location with empty local address refers to the root of the local file system of the machine determined by the network address. Note that this root is virtual on some platforms, e.g. on Windows and Macintosh it contains volumes (logical drives).
