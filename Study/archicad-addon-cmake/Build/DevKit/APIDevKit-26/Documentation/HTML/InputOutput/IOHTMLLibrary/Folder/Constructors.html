Folder::Folder   (3)


Folder constructors.

explicit Folder (
    const Location& loc,
    OnNotFound      action = Fail
);

Parameters

loc
Location of the folder in the file system.
action
Action to be performed if the folder is not found.

Description

This constructor creates connection to the folder identified with the Location loc. If loc refers to a link, then it is automatically resolved. If the folder is not found, then the action parameter is examined. On the default value Fail status of the Folder instance becomes invalid. On the value Create it will try to create the folder in the file system if its parent folder (obtained from loc) exists.

Result of the construction can be examined by a subsequent invocation of the GetStatus method, where return value NoError indicates successful construction.

Instances should not be created before the module initialization!
