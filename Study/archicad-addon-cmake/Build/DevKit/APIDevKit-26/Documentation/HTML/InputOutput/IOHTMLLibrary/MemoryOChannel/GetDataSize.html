MemoryOChannel::GetDataSize


Returns size of the data sent to this output channel in characters.

USize  GetDataSize (
    void
) const;

Return value

Size of the data sent to this output channel in characters.

Description

This method returns size of the data sent to this output channel in characters. This size is usually greater then size of the data in the destination buffer due to the internal buffering. If this causes a problem, Flush method should be invoked to flush internal buffers. Note that GetDestination method performs this flushing. See OChannelAdapter base class about buffering details.
