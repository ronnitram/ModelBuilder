File::New


Creates a valid new File instance on the heap.

static GSErrCode  New (
    File** result,
    par1,
    par2,
    ...,
    parn
);

Parameters

result
On return: if there was no error then pointer to a valid new File instance allocated on the heap, otherwise NULL.
par1 par2 ... parn
Parameters according to the appropriate constructor.

Return value

Value Meaning
NoError* OK
ErrMemoryFull* out of memory
other error raised inside the constructor

* declared in GS namespace, but accessible in the global scope

Description

For each File constructor there is a corresponding (static) New method which creates valid new File instances on the heap. It returns a new instance only if the memory allocation was successful and state status of the new instance was valid (that is only if the new instance is usable). Instance should be deleted with the delete operator when it is no longer needed.

Note that this method is made for convenience because no exception handling is used in the InputOutput module.
