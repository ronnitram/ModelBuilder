Folder::GetFreeSpace


Returns amount of free space in the folder.

virtual GSErrCode  GetFreeSpace (
    UInt64* result
) const;

Parameters

result
On return: the free space in the folder. Unit of the free space is character (byte).

Return value

Value Meaning
NoError* OK
AccessDenied* operation is not allowed on this folder
Error* unspecified error

* declared in Folder class scope
* declared in GS namespace, but accessible in the global scope

Description

This method returns amount of free space in the folder in result.

Generally user should not assume any relation between amounts of free space in different folders. That is, consuming free space in one folder may or may not affect amount of free space in other folders. Folders belonging to the same mount point usually share common free space, that is, consuming free space in one folder will usually also consume free space of other folders belonging to the same mount point (user quotas may complicate the problem). However, it can be assumed that a parent folder has at least as much of free space as its child with the largest amount of free space has.
