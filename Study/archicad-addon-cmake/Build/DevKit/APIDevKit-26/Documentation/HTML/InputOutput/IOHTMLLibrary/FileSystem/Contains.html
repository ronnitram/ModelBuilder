FileSystem::Contains


Returns whether file system contains an entry on a given location.

GSErrCode  Contains (
    const Location& loc,
    bool*           result
) const;

Parameters

loc
Location which should be examined for an existing file system entry.
result
On return:
Value Meaning
true entry exists
false no entry was found

Return value

Value Meaning
NoError* OK
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

This method returns in result whether exists a file system entry on the location loc.
