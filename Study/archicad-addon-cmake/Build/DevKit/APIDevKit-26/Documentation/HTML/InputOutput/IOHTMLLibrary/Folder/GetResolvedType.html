Folder::GetResolvedType


Returns resolved type of the entry in the folder specified with its name.

virtual GSErrCode  GetResolvedType (
    const Name& name,
    EntryType*  result
) const;

Parameters

name
Name of the entry in the folder which resolved type should be determined.
result
On return: resolved type of the entry. Possible values are:
Value Meaning
FileEntry file
FolderEntry folder

Return value

Value Meaning
NoError* OK
TargetNotFound* entry is not found
AccessDenied* operation is not allowed on this folder
Error* unspecified error

* declared in Folder class scope
* declared in GS namespace, but accessible in the global scope

Description

This method returns resolved type of the file system entry (file, folder, link) in the folder identified with its name. Resolved type of a link is type of the entry which link (or chain of links) refers to. Resolved type of files and folders remains file and folder respectively.

 Windows notes:
  • If link extensions are restricted (see the RestrictLinkExtensions function) then this method will examine file system entries only with extension ".lnk" for potential link resolvation.
