File::Close


Closes access to the content of the file.

virtual GSErrCode  Close (
    void
);

Return value

Value Meaning
NoError* OK
GS::OChannel::OutputIsFull output device is full
GS::OChannel::WriteError write error on the output device
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

If the file content was previously opened by the Open method, then this method closes access to it. Internal (and OS) buffers are flushed to the output device. Anyway, if the file content was left opened, it will be automatically closed at the destruction time. After this method was invoked, file content is treated closed without respect to possible errors.

If the file content was not opened (or was already closed), then this method does nothing.
