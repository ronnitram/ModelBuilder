File::IsWriteable


Returns if the file is writeable at this moment.

virtual bool  IsWriteable (
    void
) const;

Return value

Value Meaning
true file is writeable
false file is not writeable

Description

This method returns whether is the file writeable at this moment. If a file is not writeable, then it can not be opened in modes with write semantics. It returns true if this object is already opened in a mode with write semantics.
