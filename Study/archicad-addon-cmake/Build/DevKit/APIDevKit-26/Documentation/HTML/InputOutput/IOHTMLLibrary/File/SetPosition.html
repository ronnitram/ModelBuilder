File::SetPosition


Sets file content position.

virtual GSErrCode SetPosition ( 
    UInt64 result
);

Parameters

result 
The new file content position. Unit of the position is character (byte).

Return value

Value Meaning
NoError* OK
IllegalPosition* illegal file content position was specified
GS::OChannel::OutputIsFull output device is full
GS::OChannel::WriteError write error on the output device
Error* unspecified error

* declared in File class scope
* declared in GS namespace, but accessible in the global scope

Description

This method sets file content position to the value determined by result. It is not allowed to set the position beyond the end of the file content, but it can be set to the end of the file content. Position setting can imply output buffers to be flushed, so write error may occur.
Note that this method is equivalent to Seek (IO::File::Start, offset);

This operation is not allowed if the file content was not opened, or it was opened in appending mode. It is also disallowed "inside" output channel (GS::OChannel) block operations (that is when mark was put, or when offset was not reset).
