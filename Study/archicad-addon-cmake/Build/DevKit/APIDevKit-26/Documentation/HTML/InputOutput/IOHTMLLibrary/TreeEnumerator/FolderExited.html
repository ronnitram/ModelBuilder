TreeEnumerator::FolderExited


Callback method which is invoked when a subfolder is exited.

virtual void  FolderExited (
    const Name&   folderName,
    const Folder& current
);

Parameters

folderName
Name of the subfolder that has been processed.
current
Subfolder that has been processed.

Description

This method is invoked when a subfolder is exited (after its enumeration) and is intended to be overridden by the user. It receives name of the subfolder and the subfolder object itself. Default implementation does nothing.

Note that this method is not invoked for the enumeration root (that is after the whole enumeration), only for subfolders.
