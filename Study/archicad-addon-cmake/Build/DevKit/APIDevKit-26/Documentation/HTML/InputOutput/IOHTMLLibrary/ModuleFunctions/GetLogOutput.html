GetLogOutput


Returns output of the log support.

Header: IOMain.hpp

GS::OChannel*  GetLogOutput (
    void
);

Return value

Output channel where log messages are sent to. NULL indicates that no output was set, that is logging is disabled.

Description

This method returns the current log output. Results of some file and folder operations and few other failures are logged to this output.
