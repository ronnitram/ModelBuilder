Folder::PrintContent


Writes list of names of all entries in the folder to the specified output channel.

virtual GSErrCode  PrintContent (
    GS::OChannel& oc
) const;

Parameters

oc
Output channel where list of names should be written to.

Return value

Value Meaning
NoError* OK
OChannel::OutputIsFull* output device is full
OChannel::WriteError* write error on the output device
AccessDenied* operation is not allowed on this folder
Error* unspecified error

* declared in Folder class scope
* declared in GS namespace, but accessible in the global scope

Description

This method writes list of names of all entries in the folder to the output channel oc. Names are closed with trailing '\0' character and are separated from each other with '\n' character.
