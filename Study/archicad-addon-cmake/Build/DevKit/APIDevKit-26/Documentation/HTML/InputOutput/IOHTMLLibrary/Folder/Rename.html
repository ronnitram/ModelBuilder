Folder::Rename


Renames the specified entry in the folder.

virtual GSErrCode  Rename (
    const Name& from,
    const Name& to
);

Parameters

from
Original name of the entry (in the folder) to be renamed.
to
New name of the entry (in the folder).

Return value

Value Meaning
NoError* OK
SourceNotFound* original (source) entry is not found
TargetExists* an entry with the new (target) name already exists in the folder
AccessDenied* operation is not allowed on this folder or on the entry
Error* unspecified error

* declared in Folder class scope
* declared in GS namespace, but accessible in the global scope

Description

This method renames the file system entry (file, folder, link) with original name from to the new name to in the folder.

Rename fails if the folder already contains an entry (file, folder, link) with the name to, and on non-writeable folders.

 Windows notes:
  • This method fails on the root folder (because it is virtual and not writeable).
 Macintosh notes:
  • This method specifically works for the root folder (although it is treated virtual and not writeable), so it is possible to rename volumes.
