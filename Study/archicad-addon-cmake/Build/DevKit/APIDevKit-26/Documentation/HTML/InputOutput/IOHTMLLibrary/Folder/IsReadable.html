Folder::IsReadable


Returns whether is the folder readable at this moment.

virtual bool  IsReadable (
    void
) const;

Return value

Value Meaning
true folder is readable
false folder is not readable

Description

This method returns whether is the folder readable at this moment. If a folder is not readable, then operations with read-like semantics (content query, attribute query) will surely fail, otherwise they may succeed.

 Windows notes:
  • Root folder is readable. (Readability of other folders is currently determined by trying to enumerate the given folder).
 Macintosh notes:
  • Root folder is readable.

Available from version: 1.0
