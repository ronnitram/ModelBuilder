SetLogOutput


Sets output of the log support.

Header: IOMain.hpp

void  SetLogOutput (
    GS::OChannel* lo
);

Parameters

lo
Output channel where log messages will be sent to. It can be NULL, which disables logging.

Description

There is a log support in the InputOutput module. Results of some file and folder operations and few other failures are logged to the current log output. This method sets the log output.
