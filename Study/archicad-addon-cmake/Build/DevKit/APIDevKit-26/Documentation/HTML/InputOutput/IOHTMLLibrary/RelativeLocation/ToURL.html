RelativeLocation::ToURL


Returns Unicode UTF-16 string representation of the relative location.

GSErrCode  ToURL (
    GS::UniString* result
) const;

Parameters

result
On return: the specified part (name) of the relative location.

Return value

Value Meaning
NoError* OK
IllegalAddress* illegal address was passed
ErrMemoryFull* out of memory
Error* unspecified error

* declared in RelativeLocation class scope

* declared in GS namespace, but accessible in the global scope

Description

This method returns part (name) with index idx of the relative location in result.
