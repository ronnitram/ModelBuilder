Name::operator[]


Returns the specified character of the name.

GS::UniChar operator[] (
     UIndex idx
) const;

Parameters

idx
Index of the desired character of the name. Indexing begins from 0. Operator with index 0 returns the first character of the name.
Index should be between 0 end length of the name-1.

Return value

Specified part (Unicode UTF-16 char) of the name.

Description

This method returns character (Unicode UTF-16 char) with index idx of the name.
