ZipOChannel::ZipOChannel


ZipOChannel constructor.

ZipOChannel (
   OChannel& output, 
   CompressionLevel compressionLevel = DefaultCompression
);

Parameters

output
Determines the output channel, into where the compressed data is written.
compressionLevel
Compression level determines the kind of compression, between the quickest compression and the best compression. Default value 6 specifies average compression.
Value Meaning
NoCompression There's no compression (the input data is simply copied a block at a time).
BestSpeed It's the quickest compression
BestCompression It's the best compression
DefaultCompression It's average compression
(This is the default case.)

Description

This constructor initializes the given instance to use the target output channel supplied in output . Until using the ZipOChannel the output channel is not allowed to be used. CompressionLevel determines the goodness of compression. Its default value is the average compression (DefaultCompression = 6), and the value must be between 0 and 9.

Instances should not be created before the module initialization!
