RelativeLocation::IsRelativeURL   (2)


Returns whether the URL is relative.

static bool IsRelativeURL (
    const char* targetPath
);

Parameters

targetPath
Current platform's text representation (as C string) of the URL.

Return value

Value Meaning
true the URL is relative
false the URL is not relative

Description

This (static) method returns whether the URL is relative. It returns true if the URL specified by targetPath begins with "url:rel:" or "rel:".
