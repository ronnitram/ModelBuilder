Location::ToDisplayText   (2)


Returns current platform's display text representation of the location's target.

GSErrCode ToDisplayText (
    Path* result
) const;

Parameters

result
On return: current platform's display text representation of the location's target. Type Path is a simple string storage class declared in the IO namespace. From the user's point of view its most important properties are that it has default constructor and is automatically (implicitly) converted to C string when needed. 

Return value

Value Meaning
NoError* OK
ErrMemoryFull* out of memory
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

This method returns current platform's display text representation of the location's target in result. The resulting text is for displaying purposes only, in the form accustomed on the actual platform. The original location usually can not be fully restored from it.

 Windows notes:
  • There is no accustomed text representation for the locations with network address containing more than one name, so the network names except the last one will be omitted.
 Macintosh notes:
  • There is no accustomed text representation for the locations with non-empty network address, so the network names will be omitted.

Available from version: 1.0
