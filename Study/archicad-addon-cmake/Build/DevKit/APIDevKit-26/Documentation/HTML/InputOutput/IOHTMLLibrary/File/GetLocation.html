File::GetLocation


Returns location of the file.

virtual void  GetLocation (
    Location* result
) const;

Parameters

result
On return: location of the file.

Description

This method returns location of the file in result. It is not necessarily equal to the location passed in constructor, because links are automatically resolved at creation time and this method returns the final target location which the file was initialized for.
