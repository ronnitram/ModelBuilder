FileSystem::Rename


Renames the specified entry in the file system.

GSErrCode  Rename (
    const Location& loc,
    const Name&     to
);

Parameters

loc
Location of the entry to be renamed.
to
New name of the entry.

Return value

Value Meaning
NoError* OK
SourceNotFound* original (source) entry is not found
TargetExists* an entry with the new (target) name already exists in the parent folder
AccessDenied* operation is not allowed in the parent folder or on the entry
Error* unspecified error

* declared in FileSystem class scope
* declared in GS namespace, but accessible in the global scope

Description

This method renames the file system entry (file, folder, link) identified with its lcation to the given new name.

Rename fails if the parent folder already contains an entry (file, folder, link) with the given new name, and in non-writeable parent folders.

 Windows notes:
  • This method fails in the root folder (because it is virtual and not writeable).
 Macintosh notes:
  • This method specifically works in the root folder (although it is treated virtual and not writeable), so it is possible to rename volumes.
