Location::ToPath   (2)


Returns current platform's text representation of the location's target.

GSErrCode ToPath (
    Path* result
) const;

Parameters

result
On return: current platform's text representation of the location's target. Type Path is a Unicode UTF-8 string storage class declared in the IO namespace. From the user's point of view its most important properties are that it has default constructor and is automatically (implicitly) converted to C string when needed. 

Return value

Value Meaning
NoError* OK
ErrMemoryFull* out of memory
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

This method returns current platform's UTF-8 text representation of the location's target in result (if the current platform has appropriate representation).

 Windows notes:
  • Windows has one level network model (there are no subnetworks), therefore only the last network name is used for the text representation.
 Macintosh notes:
  • There is no text representation for the locations with non-empty network address.

Available from version: 1.0
