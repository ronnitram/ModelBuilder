Description

TextOProtocol is a text based output protocol which decomposes simple C++ types by writing their text representation to the data destination (given output channel). TextOProtocol instances can be selected into arbitrary number of output channels.

There is a default instance named textOProtocol declared in the IO namespace.

The following table summarizes output formats for simple types:

Type Format
char %c
unsigned char %c
bool true / false
short %hd
unsigned short %hu
int %d
unsigned int %u
long %ld
unsigned long %lu
float %f
double %.16g
long double %.20lg
const void* 0x%08X / NULL
const char* %s
