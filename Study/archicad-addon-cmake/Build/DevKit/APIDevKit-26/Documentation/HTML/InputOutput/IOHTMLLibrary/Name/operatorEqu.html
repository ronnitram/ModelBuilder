Name::operator==   (3)


Returns whether two names are equal.

bool  operator== (
    const Name& rightOp
) const;

Parameters

rightOp
Right operand of the == operation.

Return value

Value Meaning
true names are equal
false names are not equal

Description

This method returns whether this name is equal with rightOp. The equality of two names is platform dependent, because they are treated and compared as file system entry names, rather then plain strings. For example such platform dependent property is the case sensitivity-insensitivity of names (that is whether are the names "abcde" and "ABCDE" equal).


Available from version: 1.0
