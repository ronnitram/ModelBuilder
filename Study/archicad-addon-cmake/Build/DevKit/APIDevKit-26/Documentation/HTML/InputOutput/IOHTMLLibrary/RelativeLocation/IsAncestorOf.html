RelativeLocation::IsAncestorOf


Examines whether is ancestor-descendant relationship between two relative locations.

bool  IsAncestorOf (
    const RelativeLocation& rightOp
) const;

Parameters

rightOp
Right operand of the "is ancestor of" operation.

Return value

Value Meaning
true this relative location is ancestor of rightOp
false this relative location is not ancestor of rightOp

 

Description

This method returns whether is this relative location ancestor of rightOp.

A relative location X is ancestor of a relative location Y if X is proper prefix of Y when sequence of names from both of the relative locations are considered. Proper prefix means that length of Y's sequence of names must be greater then length of X's sequence of names.
