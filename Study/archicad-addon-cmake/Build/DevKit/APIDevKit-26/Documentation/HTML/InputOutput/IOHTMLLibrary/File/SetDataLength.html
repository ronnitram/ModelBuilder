File::SetDataLength


Sets length of the file content.

virtual GSErrCode SetDataLength ( 
    USize newLength
);

Parameters

newLength
The new file content length. Unit of the length is character (byte).
 

Return value

Value Meaning
NoError* OK
AccessDenied* operation is not allowed on this file
GS::OChannel::OutputIsFull output device is full
GS::OChannel::WriteError write error on the output device
Error* unspecified error

* declared in File class scope
* declared in GS namespace, but accessible in the global scope

Description

This method sets length of the file content to the value determined by newLength. As result, file content may be truncated or extended. If it is extended, then content of the appended part is undefined.
If the file was already opened in some of write modes, then file content position will be set to the new end of the file content. In this case internal buffers are first flushed, so write error may occur.

This operation is not allowed if the file was already opened in read only mode, because this operation has write like semantics on the file content.
