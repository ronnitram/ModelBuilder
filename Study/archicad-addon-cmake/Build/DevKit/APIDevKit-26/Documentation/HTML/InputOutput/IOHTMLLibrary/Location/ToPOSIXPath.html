Location::ToPOSIXPath   (2)


Returns standard UNIX representation of the location's target.

GSErrCode ToPOSIXPath (
    Path* result
) const;

Parameters

result
On return: standard UNIX text representation of the location's target (e.g.: Work/Documents/Hello.txt). Type Path is a Unicode UTF-8 string storage class declared in the IO namespace. From the user's point of view its most important properties are that it has default constructor and is automatically (implicitly) converted to C string when needed.

Return value

Value Meaning
NoError* OK
ISRootError there is no POSIXPath representation for the root locations
ErrMemoryFull* out of memory
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

This method returns standard UNIX UTF-8 text representation of the location's target in result (if the current platform has appropriate representation).

 Macintosh notes:
  • There is no text representation for the locations with non-empty network address.

Available from version: 1.0
