Folder::Delete


Deletes the specified entry from the folder.

virtual GSErrCode  Delete (
    const Name& name,
    DataSafety  safety = Normal
);

Parameters

name
Name of the entry (in the folder) to be deleted.
safety
Data safety level controlling scope of the operation.

Return value

Value Meaning
NoError* OK
TargetNotFound* entry is not found
AccessDenied* operation is not allowed on this folder or on the entry
Error* unspecified error

* declared in Folder class scope
* declared in GS namespace, but accessible in the global scope

Description

This method deletes the file system entry (file, folder, link) with name specified in name from the folder. Delete operation is applied to the whole entry (together with its internal content), meaning that delete operation is inherently recursive.

Delete fails if the folder is not writeable.

Delete fails on write-protected entries (read-only entries on Windows and locked entries on Macintosh) unless IgnoreWriteProtection value is passed in the safety parameter.

 Windows notes:
  • This method fails on the root folder (because it is virtual and not writeable).
 Macintosh notes:
  • This method fails on the root folder (because it is virtual and not writeable).
