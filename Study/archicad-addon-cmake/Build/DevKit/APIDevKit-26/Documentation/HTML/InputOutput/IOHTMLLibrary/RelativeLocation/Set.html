RelativeLocation::Set   (2)


Sets relative target of the relative location.

GSErrCode  Set (
    const char* targetPath
);

Parameters

targetPath
Text representation (as C string) of the relative file system address (relative path). It can be current platform's text representation or URL representation. URL representation uses the relative URL protocol with the following syntax:

    "rel://""/" localAddress

        localAddress   = (localName ("/" localName)*)?

'*' means 0 or more, '?' means 0 or 1 (optional) and '|' means or. localName can contain any character but '/'.
For example: "rel:///Share/Folder/File.ext".

Return value

Value Meaning
NoError* OK
IllegalAddress* illegal address was passed
ErrMemoryFull* out of memory
Error* unspecified error

* declared in RelativeLocation class scope
* declared in GS namespace, but accessible in the global scope

Description

This method sets the given instance to refer to the relative target specified by targetPath.

Set works properly even if the state of the instance is invalid (it will reset both content and internal status of the instance).


Available from version: 1.0
