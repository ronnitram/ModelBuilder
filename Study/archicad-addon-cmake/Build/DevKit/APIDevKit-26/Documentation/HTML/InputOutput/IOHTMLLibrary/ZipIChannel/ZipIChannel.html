Description

ZipIChannel is an input channel which reads compressed data from the source input channel and decompresses it.

The source input channel is supplied by the user. Until the ZipICahnnel is opening, the source input channel is not allowed to be used.

ZipIChannel will be closed at destruction time.

ZipIChannel uses the protocol of the source input channel as default.
