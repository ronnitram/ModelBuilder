Description

BinIProtocol is a binary input protocol which composes simple C++ types by reading sizeof (type) characters (bytes) from the data source (given input channel) into the particular type. C strings are composed from the 'length' + 'characters' form for the efficiency reasons.

BinIProtocol instances can be selected into arbitrary number of input channels.

There is a default instance named binIProtocol declared in the IO namespace.
