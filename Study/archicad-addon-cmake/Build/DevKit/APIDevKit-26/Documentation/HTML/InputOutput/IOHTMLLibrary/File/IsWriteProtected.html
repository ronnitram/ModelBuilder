File::IsWriteProtected


Returns whether is the file write-protected.

virtual GSErrCode  IsWriteProtected (
    bool* result
) const;

Parameters

result
On return: true if the file is write protected, otherwise false.

Return value

Value Meaning
NoError* OK
AccessDenied* operation is not allowed on this file
Error* unspecified error

* declared in File class scope
* declared in GS namespace, but accessible in the global scope

Description

This method returns whether is the file write-protected. Write-protected files can not be written, deleted and moved.

 Windows notes:
  • Write-protection is implemented with the 'read-only' file attribute.
 Macintosh notes:
  • Write-protection is implemented with the 'lock' file attribute.
