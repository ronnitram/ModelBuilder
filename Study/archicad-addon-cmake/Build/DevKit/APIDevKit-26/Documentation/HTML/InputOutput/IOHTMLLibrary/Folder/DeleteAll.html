Folder::DeleteAll


Deletes all entries in the folder.

virtual GSErrCode  DeleteAll (
    DataSafety safety = Normal
);

Parameters

safety
Data safety level controlling scope of the operation.

Return value

Value Meaning
NoError* OK
AccessDenied* operation is not allowed on this folder or on any of its entries
Error* unspecified error

* declared in Folder class scope
* declared in GS namespace, but accessible in the global scope

Description

This method deletes all entries in the folder (so it will be empty). Delete operation is applied to whole entries (together with their internal content), meaning that delete operation is inherently recursive.

DeleteAll fails if the folder is not writeable.

DeleteAll fails on write-protected entries (read-only entries on Windows and locked entries on Macintosh) unless IgnoreWriteProtection value is passed in the safety parameter.

 Windows notes:
  • This method fails on the root folder (because it is virtual and not writeable).
 Macintosh notes:
  • This method fails on the root folder (because it is virtual and not writeable).
