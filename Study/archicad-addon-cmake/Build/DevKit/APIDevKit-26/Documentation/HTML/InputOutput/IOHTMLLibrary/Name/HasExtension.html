Name::HasExtension


Returns whether file has extension.

bool  HasExtension (
    void
) const;

Return value

Value Meaning
true file has extension
false file doesn't have extension

Description

This method returns whether file has extension. The method returns true whether file has a '.'(dot) after the base file name. It follows that the method will return true, if the file has got a '.'(dot), although the extension itself is an empty string.
