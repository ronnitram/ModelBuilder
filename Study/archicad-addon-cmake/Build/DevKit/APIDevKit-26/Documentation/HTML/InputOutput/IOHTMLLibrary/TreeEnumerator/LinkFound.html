TreeEnumerator::LinkFound


Callback method which is invoked for links encountered during the enumeration.

virtual bool  LinkFound (
    const Name& linkName
);

Parameters

linkName
Name of the link.

Return value

Controls further processing of the link. Return value true (which is the default implementation) enables further processing. In this case type of the link's final target is determined and either FileFound or FolderFound method is invoked depending on this type (dead links are skipped). Name of the link (linkName) is passed to these methods (and not name of the link's target), that is link is presented as file or folder. Return value false disables further processing of the link and next entry in the parent folder will be processed.

Description

This method is invoked for links encountered during the enumeration and is intended to be overridden by the user. It is invoked only if the return value from the previous EntryFound method is true. It receives name of the link. Return value controls further processing of the link. Default implementation does nothing and enables further processing.
