RelativeLocation::IsMultiplatform


Returns whether is the relative location syntactically legal on every platform supported by InputOutput module.

bool  IsMultiPlatform (
    void
) const;

Return value

Value Meaning
true relative location is syntactically legal on every platform supported by InputOutput module
false relative location is not syntactically legal on every platform supported by InputOutput module

 

Description

This method returns whether is the relative location syntactically legal on every platform supported by InputOutput module.

For example file names on Windows can't contain some special characters like '|' or '*'.
