File::GetPosition  (2)


Returns file content position.

virtual GSErrCode  GetPosition (
    UInt64* result
) const;

Parameters

result
On return: the file content position. Unit of the position is character (byte).

Return value

Value Meaning
NoError* OK
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

This method returns current value of the file content position in result.

This operation is not allowed if the file content was not opened, or it was opened in appending mode.
