Name::GetStatus


Returns state status of the Name instance.

GSErrCode    GetStatus (
    void
) const;

Return value

Value Meaning
NoError* state is OK
ErrMemoryFull* out of memory
Error* state is invalid (unspecified error)

* declared in GS namespace, but accessible in the global scope

Description

This method returns state status of the Name instance. It can be used to obtain result of the construction (because no exception handling is used). State of an instance may become invalid in the constructor, or later, due to content manipulation (e.g. memory exhaustion may occur). Object is treated invalid if its status is different from NoError. Only methods with GSErrCode as return value can be invoked on an invalid Name object, in such cases the method does nothing and return value is the error status, furthermore the Clear method, which clears content of the Name instance (and internal status), so it may become valid again.
