File::GetTypeInfo


Returns file type information attributes.

virtual GSErrCode  GetTypeInfo (
    unsigned long* type,
    unsigned long* creator
) const;

Parameters

type
On return: file type attribute (0 on Windows).
creator
On return: file creator attribute (0 on Windows).

Return value

Value Meaning
NoError* OK
AccessDenied* operation is not allowed on this file
Error* unspecified error

* declared in File class scope
* declared in GS namespace, but accessible in the global scope

Description

This method returns file type information attributes: file type in type and file creator in creator.

 Windows notes:
  • This method returns 0 both for file type and file creator, because Windows has no such file attributes.
