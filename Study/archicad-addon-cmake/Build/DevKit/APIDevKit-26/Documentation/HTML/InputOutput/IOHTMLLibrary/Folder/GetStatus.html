Folder::GetStatus


Returns state status of the Folder instance.

GSErrCode    GetStatus (
    void
) const;

Return value

Value Meaning
NoError* state is OK
FolderNotFound* folder is not found
NotAFolder* not a folder was specified
AccessDenied* access to folder is denied
CreationError* creation of the folder is failed
Error* state is invalid

* declared in Folder class scope
* declared in GS namespace, but accessible in the global scope

Description

This method returns state status of the Folder instance. It can be used to obtain result of the construction (because no exception handling is used). In case of the Folder class, state status is set to the result of the construction, and it never changes later. Object is treated invalid if its status is different from NoError. Every method can be invoked on an invalid Folder object, in such cases method does nothing and return value is the error status.
