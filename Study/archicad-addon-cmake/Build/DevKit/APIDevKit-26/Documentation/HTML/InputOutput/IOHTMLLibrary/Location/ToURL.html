Location::ToURL   (2)


Returns URL representation of the location's target.

GSErrCode ToURL (
    URL* result
) const;

Parameters

result
On return: URL representation of the location's target. Type URL is a simple string storage class declared in the IO namespace. From the user's point of view its most important properties are that it has default constructor and is automatically (implicitly) converted to C string when needed. 

Return value

Value Meaning
NoError* OK
ErrMemoryFull* out of memory
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

This method returns URL representation of the location's target in result.

Note: currently the nonexisting "lan:" protocol (LAN = Local Area Network) is used in the URL representation, to indicate that the InputOutput module can handle local networks only (so for example although arbitrary location can be built, files across internet can not be accessed).


Available from version: 1.0
