FileSystem::ResolveLocation   (2)


Returns target location of a link.

GSErrCode  ResolveLocation (
    const Location& source,
    Location*       target
) const;

Parameters

source
Location of the link to be resolved.
target
On return: target location of the link.

Return value

Value Meaning
NoError* OK
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

Link is a file system entry which refers to an another file system entry (more precisely it refers to a location). As every file system entry, link is also identified by its unique location. The location link is referring to is called its target.

This method determines target location of a link (resolvation). It returns the final target if chain of links is encountered (that is, when a link refers to an another link and so on). If the entry specified by source is not a link, then target will be set to source (this is a special case when chain of links consists from 0 links).

Note that there is no separate Link class in the InputOutput module.

 Windows notes:
  • If link extensions are restricted (see the RestrictLinkExtensions function) then this method will examine file system entries only with extension ".lnk" for potential link resolvation.

Available from version: 1.0
