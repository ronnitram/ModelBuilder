TreeEnumerator::EntryFound


Callback method which is invoked for each entry encountered during the enumeration.

virtual bool  EntryFound (
    const Name& entryName,
    bool        isComposite
);

Parameters

entryName
Name of the current entry.
isComposite
True if the particular entry is composite (e.g. folder), otherwise false.

Return value

Controls further processing of the entry. With the value true (which is the default implementation) either FileFound, FolderFound or LinkFound method will be invoked depending on the type of the entry, otherwise next entry will be processed.

Description

This method is invoked for each file system entry (file, folder, link) encountered during the enumeration and is intended to be overridden by the user. It receives name of the entry and some kind of type information. Return value controls further processing (classification) of the entry. Default implementation does nothing and enables further processing.
