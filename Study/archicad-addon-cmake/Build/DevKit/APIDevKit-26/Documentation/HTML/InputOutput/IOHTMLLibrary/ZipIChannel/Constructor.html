ZipIChannel::ZipIChannel


ZipIChannel constructor.

ZipIChannel (
   IChannel& input
);

Parameters

input
Determines the source input channel, from where the compressed data is get.

Description

This constructor initializes the given instance to use the compressed data source in input. Until the ZipICahnnel is opening, the source input channel is not allowed to be used.

Instances should not be created before the module initialization!
