Folder::GenerateUnusedName   (3)


Generates a new name which is not used within the folder.

virtual GSErrCode GenerateUnusedName (
    Name* result
) const;

Parameters

result
On return: a new name which is not used within the folder.

Return value

Value Meaning
NoError* OK
AccessDenied* operation is not allowed on this folder
Error* unspecified error

* declared in Folder class scope
* declared in GS namespace, but accessible in the global scope

Description

This method generates a new name in result which is not used as name of a child entry within the folder (so it is currently unique).

For example, this name can be used as a temporary file name.


Available from version: 1.0
