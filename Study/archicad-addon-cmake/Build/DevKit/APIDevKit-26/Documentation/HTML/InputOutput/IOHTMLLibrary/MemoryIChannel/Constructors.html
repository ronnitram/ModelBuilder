MemoryIChannel constructors.

Description

This constructor initializes the given instance to use an empty data source. Reading from a such MemoryIChannel instance causes GS::IChannel::EndOfInput to be returned.

Instances should not be created before the module initialization!


Available from version: 1.0



MemoryIChannel (
    const char*   sourceBuffer,
    USize         sourceDataLength,
    OwnershipType ot = OriginalOwnership
);

Parameters

sourceBuffer
Source buffer containing the source data.
sourceDataLength
Size of the data in the source buffer (sourceBuffer) in characters.
ot
Determines the ownership of the source buffer. Possible values are:
Value Meaning
OriginalOwnership Ownership of the buffer is not changed, it still belongs to the user. User is responsible for the deleting of the buffer.
(This is the default case.)
PassOwnership Ownership of the buffer is passed to the MemoryIChannel instance. It is responsible for the deleting of the buffer.

Buffer should be allocated with the BMpAll function!

Description

This constructor initializes the given instance to read the data from the source buffer specified by sourceBuffer. Size of the data is specified by sourceDataLength. Trying to read beyond the end of the data causes GS::IChannel::EndOfInput to be returned from read methods.

Ownership of the source buffer can be controlled with the ot parameter. Default value OriginalOwnership specifies that the buffer belongs to the user, it will be never deleted by the particular instance. Value PassOwnership specifies that the buffer belongs to the particular instance, it will deleted when needed (for example at destruction time). In order to assure correct deleting, such buffers should be allocated with the BMpAll function by the user!

Instances should not be created before the module initialization!
