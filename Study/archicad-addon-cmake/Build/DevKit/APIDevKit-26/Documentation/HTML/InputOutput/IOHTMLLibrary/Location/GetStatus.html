Location::GetStatus


Returns state status of the Location instance.

GSErrCode    GetStatus (
    void
) const;

Return value

Value Meaning
NoError* state is OK
IllegalAddress* illegal address was passed
ErrMemoryFull* out of memory
Error* state is invalid (unspecified error)

* declared in Location class scope
* declared in GS namespace, but accessible in the global scope

Description

This method returns state status of the Location instance. It can be used to obtain result of the construction (because no exception handling is used). State of an instance may become invalid in the constructor, or later, due to content manipulation (e.g. memory exhaustion may occur or illegal address may be passed). Object is treated invalid if its status is different from NoError. Almost every method can be invoked on an invalid Location object, in such cases method does nothing and return value is the error status. Exceptions are methods that have normal return value only, so that they can not indicate error (e.g. GetLocalLength method). Note that Set and Clear methods will reset content (and internal status) of the Location instance, so it may become valid again.
