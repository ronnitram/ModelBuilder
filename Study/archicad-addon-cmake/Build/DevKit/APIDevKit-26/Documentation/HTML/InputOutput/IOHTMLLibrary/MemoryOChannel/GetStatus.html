MemoryOChannel::GetStatus


Returns state status of the MemoryOChannel instance.

GSErrCode    GetStatus (
    void
) const;

Return value

Value Meaning
NoError* state is OK
ErrMemoryFull* out of memory
Error* state is invalid (unspecified error)

* declared in GS namespace, but accessible in the global scope

Description

This method returns state status of the MemoryOChannel instance. It can be used to obtain result of the construction (because no exception handling is used). State of an instance may become invalid in the constructor, or later, due to content manipulation (e.g. memory exhaustion may occur). Object is treated invalid if its status is different from NoError. In case of the memory exhaustion SetDestination method may correct object's state.
