TreeEnumerator::FolderEntered


Callback method which is invoked when a new subfolder is entered.

virtual void  FolderEntered (
    const Name&   folderName,
    const Folder& current
);

Parameters

folderName
Name of the subfolder that will be processed.
current
Subfolder that will be processed.

Description

This method is invoked when a new subfolder is entered (before its enumeration) and is intended to be overridden by the user. It is invoked only if the return value from the previous FolderFound method is true. It receives name of the subfolder and the subfolder object itself. Default implementation does nothing.

Note that this method is not invoked for the enumeration root (that is before the whole enumeration), only for subfolders.
