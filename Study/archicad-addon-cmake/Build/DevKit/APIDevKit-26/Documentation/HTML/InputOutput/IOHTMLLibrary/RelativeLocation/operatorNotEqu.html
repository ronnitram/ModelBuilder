RelativeLocation::operator!=


Returns whether are two relative locations different.

bool  operator!= (
    const RelativeLocation& rightOp
) const;

Parameters

rightOp
Right operand of the != operation.

Return value

Value Meaning
true relative locations are different
false relative locations are not different (they are equal)

 

Description

This method returns whether is this relative location different from rightOp.
Two relative locations are equal if they have the same length (in names) and the corresponding parts (names) are equal according to the operator== method of the class Name.
