File::SetTypeInfo


Sets file type information attributes.

virtual GSErrCode  SetTypeInfo (
    unsigned long newType,
    unsigned long newCreator
);

Parameters

newType
New value of the file type attribute.
newCreator
New value of the file creator attribute.

Return value

Value Meaning
NoError* OK
AccessDenied* operation is not allowed on this file
Error* unspecified error

* declared in File class scope
* declared in GS namespace, but accessible in the global scope

Description

This method sets file type information attributes: file type to newType and file creator to newCreator.

 Windows notes:
  • This method has no effect (and returns NoError), because Windows has no such file attributes.

Available from version: 1.0
