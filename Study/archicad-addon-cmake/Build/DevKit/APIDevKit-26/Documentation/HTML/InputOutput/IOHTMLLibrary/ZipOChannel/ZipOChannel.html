Description

ZipOChannel is an output channel which writes compressed data into the output channel supplied by the user..

CompressionLevel determines the compression level. It must be DefaultCompression (Default value = 6), or between 0 and 9: 1 gives best speed, 9 gives best compression, 0 gives no compression at all.

Until using the ZipOChannel the output channel is not allowed to be used. The destructor will close the ZipOChannel.

ZipOChannel uses the protocol of the destination output channel as default.
