RelativeLocation::Append   (2)


Appends new part(s) (names) to the relative location.

GSErrCode    Append (
    const Name& name
);

Parameters

name
New part (name) to be appended to the relative location. C string (char*) can be also passed instead of Name.

Return value

Value Meaning
NoError* OK
ErrMemoryFull* out of memory
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

This method appends name to the relative location (as last part). Length of the relative location is increased by 1.


Available from version: 1.0



GSErrCode    Append (
    const RelativeLocation& relLoc
);

Parameters

relLoc
Relative location which content (sequence of names) should be appended to this relative location.

Return value

Value Meaning
NoError* OK
ErrMemoryFull* out of memory
Error* unspecified error

* declared in GS namespace, but accessible in the global scope

Description

This method appends content of relLoc (sequence of names) to this relative location (as trailing parts). Length of this relative location is increased by length of relLoc.

Note that special names like "." and ".." are not considered, they are treated as ordinary names.


Available from version: 1.0
