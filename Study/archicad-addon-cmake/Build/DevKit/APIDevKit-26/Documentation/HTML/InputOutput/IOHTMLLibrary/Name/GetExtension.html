Name::GetExtension


Returns extension part of the name.

GS::UniString  GetExtension (
    void
) const;

Return value

Unicode UTF-16 string representation of the extension.

Description

This method returns extension part of the name. Extension part is suffix of the name after the last '.' (dot) character (not including the '.' character itself). If there is no '.' character in the name then the extension part is empty.
