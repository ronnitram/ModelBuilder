Body::GetColorIndex


Get the color index of the body.


Int32  GetColorIndex (
    void
);

Return value

The color index of the body.

Description

Get the own color index of this body. Each edge has its own color index in the body, but new edges can be created by cutting. These new edges have to use this color index.
