Light::GetRadius


Get the radius parameter of the spot light.


double  GetRadius (
    void
) const;

Return value

The radius parameter of the spot light.

Description

This method returns the radius parameter of the light. This value is valid, if the light type is SpotLight or DirectionLight. The spot and directional lights are projected lights from a circle source in the Archicad. This parameter is the radius of this circle.
