ParameterList::GetParameter


Access a parameter.


bool  GetParameter (
    Int32  index,
    Parameter*   parameter
);

Parameters

index
The index of the parameter. The index range is [1..n] where n = GetParameterNum ()
parameter
A pointer to the result parameter.

Return value

True, if the parameter already exists in the list.

Description

This method gets a reference to a parameter from the list and puts it into parameter. If the index is out of range, then it throws an exception.

Available from version: 9
