ArrayParameter::GetArrayItem


Access an item from a row of an array parameter.


void  GetArrayItem (
    Int32            index,
    ArrayParameter*  arrayItem
);

Parameters

index1
The column index.
arrayItem
Pointer to the result array item.

Description

This method gets a reference to an item from a row of an array parameter into arrayItem. This method can be used only for an item, which represents a row of an array.
