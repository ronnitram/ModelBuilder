BaseElem::GetLight


Get a light associated with a base element.


void  GetLight (
    Int32   lightIndex,
    Light*  light
) const;

Parameters

lightIndex
The index of the light.

light
A pointer to the result light.

Description

This method gets a reference to a light associated with the base element and puts it into light. If the lightIndex is out of range, then it throws an exception.
