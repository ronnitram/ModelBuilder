Texture::PixelType

Pixel types stored in a texture pixel map.


Values

ARGBPixelType
Corresponds to an ARGBPixel.
Grayscale8PixelType
Corresponds to a Grayscale8Pixel.
BlackAndWhitePixelType
Corresponds to a BlackAndWhitePixel.

Detailed description

This enum describes the possible pixel types stored in a texture pixel map.
