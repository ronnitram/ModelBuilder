ConvexPolygon::GetVertexIndex


Calculate body relative vertex index.


Int32  GetVertexIndex (
    Int32   vertexIndex
);

Parameters

vertexIndex
Index of the vertex. The index range is [1..n] where n = GetVertexNum ().

Return value

The body relative index.

Description

The convex polygon stores the reference of a vertex instead of vertex data. This method calculates the body relative vertex index from polygon relative vertex index. This index can be used to get the vertex from the body which contains this polygon. If vertexIndex is out of range, then it throws an exception.
Here is an example how to enumerate convex polygons and their vertexes:


// body: The body, which is enumerated.

for (Int32 polygonIndex = 1; polygonIndex <= body.GetPolygonNum (); polygonIndex++) {
    ModelerAPI::Polygon polygon;
    body.GetPolygon (polygonIndex, &polygon);
    // Begin of polygon.
    Int32 vertexIndex = 1;
    for (Int32 convexPolygonIndex = 1 ; convexPolygonIndex <= polygon.GetConvexPolygonNum (); convexPolygonIndex++) {
        ModelerAPI::ConvexPolygon convexPolygon;
        polygon.GetConvexPolygon (convexPolygonIndex, &convexPolygon);
        // Begin of convex polygon.
        for (Int32 vertexIndex = 1 ; vertexIndex <= convexPolygon.GetVertexNum (); vertexIndex++) {
            Int32 bodyVertexIndex = convexPolygon.GetVertexIndex (vertexIndex);
            ModelerAPI::Vertex vertex;
            body.GetVertex (bodyVertexIndex, &vertex);
            // vertex processing
        }
        // End of convex polygon.
    }
    // End of polygon.
}
