Material::GetTransparencyAttenuation


Get the transparency attenuation.


double  GetTransparencyAttenuation (
    void
);

Return value

The transparency attenuation.

Description

This method returns the transparency attenuation of the material. This value controls the falloff of the transparency ratio (decrease of intensity depending on the angle of incidence).
The transparency ratio in a given point can be calculated using the next algorithm:


// material:               The material of the drawable polygon.
// normal:                 The normal vector of the surface in the given point.
// ViewDirection:          The direction of the view point from the given point.
// ScalarProduct:          The scalar product of two vector.
// transparency:           The result transparency ratio.

transparency = 0.0;
if (material.GetTransparency () > 0.0) {
    transparency = material.GetTransparency ();
    double transparencyAttenuation = material.GetTransparencyAttenuation ();
    if (transparencyAttenuation != 0.0) {
        transparency *= exp (transparencyAttenuation * log(ScalarProduct (normal, viewDirection));
    }
}
