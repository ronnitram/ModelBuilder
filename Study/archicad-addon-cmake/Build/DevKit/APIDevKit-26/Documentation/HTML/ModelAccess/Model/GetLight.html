Model::GetLight


Get a light attribute from the model.


void  GetLight (
    Int32   lightIndex,
    Light*  light
) const;

Parameters

lightIndex
The index of the special light. The index range is [1..3].

light
A pointer to the result light.

Description

This method gets a reference to a light attribute of the model and puts it into light. If the lightIndex is out of range, then it throws an exception.
