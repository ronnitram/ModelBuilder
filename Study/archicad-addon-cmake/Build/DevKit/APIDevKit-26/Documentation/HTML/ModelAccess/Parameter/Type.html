Parameter::Type


Type of the parameter.


enum  Type (
    UndefinedType,
    NumericType,
    StringType,
    NumericArrayType,
    StringArrayType
);

Values

UndefinedType
Parameter type is undefined.
NumericType
Parameter contains numeric value.
StringType
Parameter contains string value.
NumericArrayType
Parameter contains numeric array.
StringArrayType
Parameter contains string array.

Description

This type describes the type of the parameter.
