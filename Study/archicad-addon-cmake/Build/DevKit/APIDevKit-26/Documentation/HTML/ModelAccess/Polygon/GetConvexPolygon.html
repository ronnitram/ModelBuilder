Polygon::GetConvexPolygon


Get a convex polygon from the convex decomposition of the polygon.


void  GetConvexPolygon (
    Int32           polygonIndex,
    ConvexPolygon*  polygon
);

Parameters

polygonIndex
Index of the polygon. The index range is [1..n] where n = GetConvexPolygonNum ().
polygon
A pointer to the result convex polygon.

Description

This method gets a reference to a convex polygon from the convex decomposition of the polygon and puts it into polygon. If polygonIndex is out of range, then it throws an exception.
