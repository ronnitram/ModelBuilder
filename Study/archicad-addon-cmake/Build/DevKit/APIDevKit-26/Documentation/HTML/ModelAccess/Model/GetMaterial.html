Model::GetMaterial


Get a material attribute from the model.


void  GetMaterial (
    Int32      materialIndex,
    material*  material
) const;

Parameters

materialIndex
The index of the material. The index range is [1..n] where n = GetMaterialNum ().
color
A pointer to the result material.

Description

This method gets a reference to a material attribute of the model and puts it into material. If the materialIndex is out of range, then it throws an exception.
