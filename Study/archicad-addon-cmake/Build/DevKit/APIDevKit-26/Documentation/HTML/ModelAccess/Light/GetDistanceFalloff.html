Light::GetDistanceFalloff


Get the light distance dependent falloff intensity of spot light.


double  GetDistanceFalloff (
    void
) const;

Return value

The light distance dependent falloff intensity.

Description

This method returns the distance dependent falloff intensity (decrease of intensity depending on the distance). This parameter is valid if the light type is DirectionalLight, PointLight or SpotLight
Here is an example how to calculate the light color for a given point:


// light:    The light object.
// distance: The distance of the given point from the light source.
// color:    The light color in the given point.

color.red   = 0.0;
color.green = 0.0;
color.blue  = 0.0;
if (distance >= light.GetMinDistance () && distance <= light.GetMaxDistance ()) {
    double attenuation = 1.0 / (1.0 + (distance - light.GetMinDistance ()) * light.GetDistanceFalloff ());

    color.red   = light.GetColor ().red   * attenuation;
    color.green = light.GetColor ().green * attenuation;
    color.blue  = light.GetColor ().blue  * attenuation;
}
