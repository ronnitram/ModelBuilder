Body::GetPolygon


Get a reference to a polygon of the body.


void  GetPolygon (
    Int32     polygonIndex,
    Polygon*  polygon
);

Parameters

polygonIndex
The index of the polygon. The index range is [1..n] where n = GetPolygonNum ().
polygon
A pointer to the result polygon.

Description

This method gets a reference to a polygon of the body and puts it into polygon. If the polygonIndex is out of range, then it throws an exception.
