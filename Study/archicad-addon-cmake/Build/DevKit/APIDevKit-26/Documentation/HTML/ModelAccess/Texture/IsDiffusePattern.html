Texture::IsDiffusePattern


Returns whether or not the alpha channel contains diffuse reflection ratio information.


bool  IsDiffusePattern (
    void
);

Return value

True, if the alpha channel contains diffuse reflection ratio information.

Description

This method returns the alpha channel usage information. If it returns true, then the alpha channel information is modifying the diffuse reflection ratio of the material.
Here is an example how to calculate the diffuse reflection ratio in a given point:


// material: Material of the actual polygon.
// texture:  Texture of the actual polygon.

double diffuseReflection = material.GetDiffuseReflection ();
if (texture.HasAlphaChannel () && texture.IsDiffusePattern ()) {
    ModelerAPI::Color color;
    double alpha;
    texture.GetPixel (x, y, &color, &alpha);
    diffuseReflection = diffuseReflection * (1.0 - alpha) + alpha;
}
