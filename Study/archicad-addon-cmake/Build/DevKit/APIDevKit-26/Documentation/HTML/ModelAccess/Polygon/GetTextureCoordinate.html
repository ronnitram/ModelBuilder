Polygon::GetTextureCoordinate


Get texture coordinate for a given point of the polygon.


void  GetTextureCoordinate (
    const Vertex*       position,
    TextureCoordinate*  textureCoordinate
) const;

Parameters

position
Position of the point.
textureCoordinate
A pointer to the result textureCoordinate.

Description

This method calculates the texture coordinate of a polygon point and puts it into textureCoordinate. This coordinate can be used to read a pixel from the texture pixel map.
