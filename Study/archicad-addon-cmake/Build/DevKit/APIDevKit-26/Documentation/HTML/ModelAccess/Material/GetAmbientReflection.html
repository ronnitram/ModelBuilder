Material::GetAmbientReflection


Get the ambient reflection ratio.


double  GetAmbientReflection (
    void
);

Return value

The ambient reflection ratio.

Description

This method returns the ambient reflection ratio of the material.
The reflected ambient color can be calculated using the next algorithm:


// material:     The material of the drawable polygon.
// ambientColor: The color of ambien light.
// color:        The result color.

ModelerAPI::Color surfaceColor = material.GetSurfaceColor ();
double ambientReflection = material.GetAmbientReflection ();

color.red   = surfaceColor.red   * ambientReflection * ambientColor.red;
color.green = surfaceColor.green * ambientReflection * ambientColor.green;
color.blue  = surfaceColor.blue  * ambientReflection * ambientColor.blue;
