Material::Compare


Compare function.


CompareResult  Compare (
    const Material& material
) const;

Parameters

material
The other material.

Return value

The result of the comparison.

Description

This method compares this material to the material parameter. It can be used to make partial conversion. It helps to find the unchanged materials of the model. The materials are sorted partially. If the two material objects are connected to different material attributes, then the result will be the comparison of the names of the two materials (Less or Greather). If the two material objects are connected to the same material attributes, then the result is Equal or NotEqual, depending on if the material is modified or not.
