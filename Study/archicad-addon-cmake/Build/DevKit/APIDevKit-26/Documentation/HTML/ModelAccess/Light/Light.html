Detailed description

This object is reference to a light of the model. Lights contain some predeclared parameters. These parameters exist in all materials, but the used parameter set depends on the type of the light. Add-ons can extend this parameter set by own parameters. These parameters are the extra parameters, and they are stored in the parameter list.
