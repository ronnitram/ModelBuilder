Body::GetVector


Get a vector from the body.


void  GetVector (
    Int32    vectorIndex,
    Vector*  vector
);

Parameters

vectorIndex
The absolute value of this parameter is the index of the vector. The index range is [1..n] where n = GetVectorNum (). If this parameter is negative, then the vector will be stored with opposite direction.
edge
A pointer to the result vector.

Description

This method reads a vector from the body and puts it into vector. If the vectorIndex is negative, then the result vector direction will be changed. If the absolute value of the vectorIndex is out of range, then it throws an exception.
