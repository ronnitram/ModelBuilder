Texture::IsMirroredInY


Returns whether or not the texture mapping contains mirroring effect along the y coordinate.


bool  IsMirroredInY (
    void
) const;

Return value

True, if the texture mapping contains mirroring effect along the y coordinate.

Description

This method returns texture mapping information. If it returns true, then the texture mapping contains mirroring effect along the y coordinate. If the y size of the polygon is greater than the texture y size, then the texture will be drawn repeatedly. Every second texture section must be mirrored along the y coordinate before it is drawn.

Available from version: 8
