Element::GetTransformation


Get the element transformation matrix.


const Transformation*  GetTransformation (
    void
) const;

Return value

Pointer to the element transformation matrix.

Description

This method returns the element transformation matrix (local to world). The matrix size is 4x3 and stored as a two-dimensional array. The first index is the row index, the second one is the column index. Use this formula in transformation calculation: v2 = M * v1.
