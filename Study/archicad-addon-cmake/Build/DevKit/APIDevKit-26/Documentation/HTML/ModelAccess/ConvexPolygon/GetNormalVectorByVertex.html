ConvexPolygon::GetNormalVectorByVertex


Get normal vector index of the given vertex.


ModelerAPI::Vector  GetNormalVectorByVertex (
    Int32   vertexIndex
) const;

Parameters

vertexIndex
Index of the vertex. The index range is [1..n] where n = GetVertexNum ().

Return value

The normal vector of given vertex.

Description

Here is an example how to enumerate convex polygons and their vertexes with normal vectors:


// body: The body, which is enumerated.

for (Int32 polygonIndex = 1; polygonIndex <= body.GetPolygonNum (); polygonIndex++) {
    ModelerAPI::Polygon polygon;
    body.GetPolygon (polygonIndex, &polygon);
    ModelerAPI::Vector polygonNormal;
    body.GetVector (polygon.GetNormalVectorIndex (), &polygonNormal);
    // Begin of polygon.
    Int32 vertexIndex = 1;
    for (Int32 convexPolygonIndex = 1 ; convexPolygonIndex <= polygon.GetConvexPolygonNum (); convexPolygonIndex++) {
        ModelerAPI::ConvexPolygon convexPolygon;
        polygon.GetConvexPolygon (convexPolygonIndex, &convexPolygon);
        // Begin of convex polygon.
        for (Int32 vertexIndex = 1 ; vertexIndex <= convexPolygon.GetVertexNum (); vertexIndex++) {
            ModelerAPI::Vector vertexNormal = convexPolygon.GetNormalVectorIndex (vertexIndex);
            // vertex processing
        }
        // End of convex polygon.
    }
    // End of polygon.
}
