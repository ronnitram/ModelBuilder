Parameter::GetArrayItem


Access an item of an array parameter.


void  GetArrayItem (
    Int32             index1,
    Int32             index2,
    ArrayParameter*  arrayItem
);

Parameters

index1
The first array index.
index2
The second array index.
arrayItem
Pointer to the result array item.

Description

This method returns an item of an array parameter. If this array has only one dimension, then the index2 parameter isn't used. If this array has two dimensions and the index2 value is zero, then the result will be a row of the array. This method can be used only for array parameters.
