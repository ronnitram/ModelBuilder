Material::Type


Type of the material.


enum  Type (
    General,
    Simple,
    Matte,
    Metal,
    Plastic,
    Glass,
    Glowing,
    Constant
);

Values

General
General type.
Simple
General type with reduced parameter set.
Matte
Matte material.
Metal
Metal material.
Plastic
Plastic material.
Glass
Glass material.
Glowing
Glowing material.
Constant
Light independent material. This is a special material type. In this case, the surface color does not depend on the light condition. The result color is equal to the surface color of the material.

Description

This type describes the main type of the material. The Constant type has special meaning (light independent material). All other type names are just for information because they don't affect the validity of the parameters.
