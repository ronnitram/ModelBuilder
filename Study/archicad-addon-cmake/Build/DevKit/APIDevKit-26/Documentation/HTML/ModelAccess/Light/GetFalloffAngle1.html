Light::GetFalloffAngle1


Get the first angle parameter for angle dependent falloff.


double  GetFalloffAngle1 (
    void
) const;

Return value

The first angle parameter for angle dependent falloff.

Description

This method returns the angle of the cone within which the light has its maximum intensity. This value is valid if the light type is SpotLight.
Here is an example how to calculate the cone angle depended light color:


// light: The light object.
// angle: The cone angle.
// color: The light color.

color.red   = 0.0;
color.green = 0.0;
color.blue  = 0.0;

if (angle <= light.GetFalloffAngle2 ()) {
    double attenuation = 1.0;
    if (light.GetAngleFalloff () > 0.0) {
        attenuation = exp (light.GetAngleFalloff () * log (cos (angle)));
        if (angle > light.GetFalloffAngle1 ()) {
            attenuation *= (light.GetFalloffAngle2 () - angle) / (light.GetFalloffAngle2 () - light.GetFalloffAngle1 ());
        }
    }

    color.red   = light.GetColor ().red   * attenuation;
    color.green = light.GetColor ().green * attenuation;
    color.blue  = light.GetColor ().blue  * attenuation;
}
