Material::GetEmissionAttenuation


Get the emission attenuation.


double  GetEmissionAttenuation (
    void
);

Return value

The emission attenuation.

Description

This method returns the emission attenuation of the material. This value controls the falloff of the emission (decrease of intensity depending on the distance).
Here is an example, how to use this value:


// minDistance: The minimun distance
// maxDistance: The maximum distance
// distance:    The distance of the surface in the given point.
// color:       The result emission light

double attenuation = 1.0 / (1.0 + (distance - minDistance) / (maxDistance - minDistance) * material.GetEmissionAttenuation ());
ModelerAPI::Color emissionColor = material.GetEmissionColor ();

color.red   = attenuation * emissionColor.red;
color.green = attenuation * emissionColor.green;
color.blue  = attenuation * emissionColor.blue;
