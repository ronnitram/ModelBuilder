CompareResult type


Result of comparing.


enum  CompareResult (
    Equal     = 1,
    Less      = 2,
    Greater   = 3,
    NotEqual  = 4
);

Values

Equal
The two data are equal.
Less
The first data is less then the second.
Greater
The first data is greater then the second.
NotEqual
The two data are not equal.

Description

This type can store a compare result. If data can not be sorted, then Equal and NotEqual are the valid values. If data can be sorted completely, then Equal, Less and Greater are the valid values. If data can be sorted partially, all values are valid.
