Texture::GetPixelMapCheckSum


Get checksum from pixel map.


void  GetPixelMapCheckSum (
    char*  checkSum,
    Int32  strLen
) const;

Parameters

checkSum
Pointer to a buffer to receive checksum.
strLen
The size of the buffer.

Description

Deprecated! This method retrieves an MD5 checksum from the pixel map. Only strLen - 1 bytes of the checksum will be written to the buffer to prevent overflow.
