Material::GetExtraParameters


Get the extra parameters of the material.


void  GetExtraParameters (
    ParameterList*  parameters
);

Parameters

parameters
Pointer to the result parameter list.

Description

Add-ons can extend the base parameter set with own parameters. This method reads the extra parameters and puts it into parameters.
