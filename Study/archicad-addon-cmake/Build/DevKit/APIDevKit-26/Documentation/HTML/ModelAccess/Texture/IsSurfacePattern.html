Texture::IsSurfacePattern


Returns whether or not alpha channel contains color mixing information.


bool  IsSurfacePattern (
    void
) const;

Return value

True, if alpha channel contains color mixing information.

Description

This method returns the alpha channel usage information. If it returns true, then the alpha channel information controls the mixing the texture color and the surface color of the material. The 0.0 alpha value means, that the material surface color is used. The 1.0 value means, that the texture color is used.
Here is an example how to calculate the surface color in a given point:


// material: Material of the actual polygon.
// texture:  Texture of the actual polygon.

ModelerAPI::Color surfaceColor;
double alpha;
texture.GetPixel (x, y, &surfaceColor, &alpha);
if (texture.HasAlphaChannel () && texture.IsSurfacePattern ()) {
    ModelerAPI::Color materialSurfaceColor = material.GetSurfaceColor ();

    surfaceColor.red   = surfaceColor.red   * alpha + materialSurfaceColor.red   * (1.0 - alpha);
    surfaceColor.green = surfaceColor.green * alpha + materialSurfaceColor.green * (1.0 - alpha);
    surfaceColor.blue  = surfaceColor.blue  * alpha + materialSurfaceColor.blue  * (1.0 - alpha);
}
