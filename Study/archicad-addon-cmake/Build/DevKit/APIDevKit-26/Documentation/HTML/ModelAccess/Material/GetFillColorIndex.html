Material::GetFillColorIndex


Get the index of the color of the associated fill.


Int32  GetFillColorIndex (
    void
) const;

Return value

The index of the color of the associated fill or zero.

Description

This method returns the index of the color of the associated fill. If this material doesn't have associated fill, then the returned value will be zero.
