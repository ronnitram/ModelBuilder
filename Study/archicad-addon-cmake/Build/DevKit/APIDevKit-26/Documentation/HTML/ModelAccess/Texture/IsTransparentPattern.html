Texture::IsTransparentPattern


Returns whether or not alpha channel contains transparency information.


bool  IsTransparentPattern (
    void
) const;

Return value

True, if alpha channel contains transparency information.

Description

This method returns the alpha channel usage information. If it returns true, then the alpha channel information is modifying the transparency ratio of the material. The alpha value contains the inverse of the transparency ratio, so the 0.0 alpha value represents the 1.0 transparency ratio and the 1.0 alpha value represents the 0.0 transparency ratio. In this case, the inverse transparency ratio of a point will be the product of the inverse material transparency ratio and the inverse value defined by alpha value.
Here is an example how to calculate the transparency value in a given point:


// material: Material of the actual polygon.
// texture:  Texture of the actual polygon.

double transparency = material.GetTransparency ();
if (texture.HasAlphaChannel () && texture.IsTransparentPattern ()) {
    ModelerAPI::Color color;
    double alpha;
    texture.GetPixel (x, y, &color, &alpha);
    transparency = 1.0 - (1.0 - transparency) * alpha;
}
