Polygon::GetVertexIndex


Calculate body relative edge index.


Int32  GetVertexIndex (
    Int32   vertexIndex
);

Parameters

vertexIndex
Index of the vertex. The index range is [1..n] where n = GetEdgeNum () (because the number of the edges and the number of the vertexes are equal).

Return value

The body relative index, or zero value, if it is a contour separator.

Description

The polygon stores reference of a vertex instead of vertex data itself. This method calculates the body relative vertex index from polygon relative vertex index. The returned zero value shows that this is not a valid vertex, but this is a contour separator. This index (if it is not zero) can be used to get the vertex from the body which contains this polygon. If vertexIndex is out of range, then it throws an exception.
Here is an example how to enumerate polygons and their vertexes:


// body: The body, which is enumerated.

for (Int32 polygonIndex = 1; polygonIndex <= body.GetPolygonNum (); polygonIndex++) {
    ModelerAPI::Polygon polygon;
    body.GetPolygon (polygonIndex, &polygon);
    // Begin of polygon.
    Int32 vertexIndex = 1;
    while (vertexIndex <= polygon.GetEdgeNum ()) {
        // Begin of contour.
        for ( ; vertexIndex <= polygon.GetEdgeNum (); vertexIndex++) {
            Int32 bodyVertexIndex = polygon.GetVertexIndex (vertexIndex);
            if (bodyVertexIndex == 0) {
                break;
            }
            ModelerAPI::Vertex vertex;
            body.GetVertex (bodyVertexIndex, &vertex);
            // vertex processing
        }
        // End of contour.
        vertexIndex++;
    }
    // End of polygon.
}
