Texture::operator>=;


Returns whether or not this texture is greater than or equal to the right operand.


bool  operator>=; (
    const Texture& texture
) const;

Parameters

element
Right operand of the >= operation.

Return value

True if this texture is greater than or equal to the right operand.

Description

This method compare this texture to the texture parameter. It can be used to make partial conversion. It helps to find the unchanged textures of the model.
