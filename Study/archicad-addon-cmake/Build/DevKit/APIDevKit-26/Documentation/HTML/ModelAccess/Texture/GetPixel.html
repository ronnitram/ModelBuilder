Texture::GetPixel


Get pixel data from a pixel map.


void  GetPixel (
    Int32     x,
    Int32     y,
    ARGBPixel*   pixel
) const;

Parameters

x
The x coordinate of the pixel in texture map space. The value range is [0..n] where n = GetPixelMapXSize () - 1.
y
The y coordinate of the pixel in texture map space. The value range is [0..n] where n = GetPixelMapYSize () - 1.
pixel
Pointer to the resulting pixel.

Description

This method reads a pixel value from pixel map using a pixel map coordinate.

Available from version: 9

void  GetPixel (
    const TextureCoordinate*  textureCoordinate,
    ARGBPixel*   pixel
) const;

Parameters

textureCoordinate
Normalized texture coordinate in texture space. The u and v coordinate range is [0.0..1.0).
pixel
Pointer to the resulting pixel.

Description

This method reads a pixel value from a pixel map using normalized texture coordinates. The normalized texture coordinates can be calculated from the polygon point texture coordinate. The GetTextureCoordinate method of the polygon returns the polygon point texture coordinate. This coordinate transformation has to contain the mirroring, randomly shifting, rotating effect and the normalization.

Available from version: 9
