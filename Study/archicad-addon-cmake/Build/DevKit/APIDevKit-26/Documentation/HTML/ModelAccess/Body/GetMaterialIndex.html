Body::GetMaterialIndex


Get the material index of the body.


Int32  GetMaterialIndex (
    void
);

Return value

The material index of the body.

Description

Get the own material index of this body. Each polygon has own material index in the body, but new polygons can be created by cutting. These new polygons have to use this material index.
