ParameterList::DeleteParameter


Delete a parameter.


void  DeleteParameter (
    Int32  index
);

Parameters

index
The index of the parameter. The index range is [1..n] where n = GetParameterNum ()

Description

This method deletes a parameter from the list. If the index is out of range, then it throws an exception.

Available from version: 9
