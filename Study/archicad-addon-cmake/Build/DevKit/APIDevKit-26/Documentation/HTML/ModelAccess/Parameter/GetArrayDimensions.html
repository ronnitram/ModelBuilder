Parameter::GetArrayDimensions


Get the dimensions of an array parameter.


void  GetArrayDimensions (
    Int32*  dimension1,
    Int32*  dimension2
) const;

Parameters

dimension1
Pointer to the result first dimension of the array.
dimension2
Pointer to the result second dimension of the array.

Description

This method gets the dimensions of an array parameter and puts it into dimension1 and dimension2. If this array has only one dimension, the the dimension2 will be set to zero. This method can be used only for array parameters.
