Polygon::GetNormalVectorIndex


Get normal vector index.


Int32  GetNormalVectorIndex (
    void
) const;

Return value

The signed normal vector index.

Description

The polygon stores reference of normal vectors instead of normal vector data itself. This method returns the signed normal vector index of the polygon. This index can be used to get the vector from the body which contains this polygon.
