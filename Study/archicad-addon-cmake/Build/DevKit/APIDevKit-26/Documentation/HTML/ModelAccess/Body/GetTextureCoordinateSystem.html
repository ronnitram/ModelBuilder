Body::GetTextureCoordinateSystem


Get the texture coordinate system of the body.


TextureCoordinateSystem*  GetTextureCoordinateSystem (
    void
);

Return value

Pointer of the texture coordinate system or NULL pointer, if it isn't exist (cusom mode).

Description

Get the own texture coordinate system of this body. This coordinate system describes the texture wrapping mode. If the wrapping mode is custom, than the returned value is a NULL pointer.
