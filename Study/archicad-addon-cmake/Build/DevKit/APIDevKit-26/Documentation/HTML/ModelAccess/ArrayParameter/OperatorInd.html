ArrayParameter::operator[]


Access an item from a row of an array parameter.


ArrayParameter  operator[] (
    Int32 index
);

Parameters

index
The column index.

Return value

The result array item.

Description

This method returns an item from a row of the array parameter. This method can be used only for items, that represent a row of an array.
