Material::operator==


Returns whether or not the two materials are equal. .


bool  operator== (
    const Material& material
) const;

Parameters

material
Right operand of the == operation.

Return value

True, if the two materials are equal.

Description

This method compares this material to the material parameter. It can be used to make partial conversion. It helps to find the unchanged materials of the model. Two material objects are equal, if they are generated from the same material attribute, and contain the same parameters.
