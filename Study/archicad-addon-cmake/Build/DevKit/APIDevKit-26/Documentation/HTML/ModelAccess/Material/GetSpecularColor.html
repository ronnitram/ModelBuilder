Material::GetSpecularColor


Get color for specular reflection.


Color  GetSpecularColor (
    void
);

Return value

The specular color.

Description

This method returns the specular reflection color of the material. It can be used in specular reflection calculation.
The reflected specular color in a given point can be calculated using the next algorithm:


// material:               The material of the drawable polygon.
// lightColor:             The color of the light.
// normal:                 The normal vector of the surface in the given point.
// lightDirection:         The direction of the light source from the given point.
// mirorredLightDirection: The direction of the reflected light rays in the given point.
// ViewDirection:          The direction of the view point from the given point.
// ScalarProduct:          The scalar product of two vector.
// color:                  The result color.

color.red   = 0.0;
color.green = 0.0;
color.blue  = 0.0;

if (ScalarProduct (normal, lightDirection) > 0.0) {
    ModelerAPI::Color specularColor = material.GetSpecularColor ();
    double reflectionRatio = material.GetSpecularReflection ();
    double shining = material.GetShining ();
    double specularReflection = reflectionRatio * exp (shining * log(ScalarProduct (mirorredLightDirection, viewDirection));

    color.red   = surfaceColor.red   * specularReflection * lightColor.red;
    color.green = surfaceColor.green * specularReflection * lightColor.green;
    color.blue  = surfaceColor.blue  * specularReflection * lightColor.blue;
}
