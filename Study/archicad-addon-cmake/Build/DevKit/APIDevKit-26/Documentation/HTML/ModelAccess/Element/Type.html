Element::Type


Type of the element.


enum  Type (
    UndefinedElement,
	WallElement,
	SlabElement,
	RoofElement,
	CurtainWallElement,
	CWFrameElement,
	CWPanelElement,
	CWJunctionElement,
	CWAccessoryElement,
	CWSegmentElement,
	ShellElement,
	SkylightElement,
	FreeshapeElement,
	DoorElement,
	WindowElement,
	ObjectElement,
	LightElement,
	ColumnElement,
	MeshElement,
	BeamElement,
	RoomElement,
	OtherElement
);

Values

UndefinedElement
Element type information is not available.
WallElement
The element is a wall.
SlabElement
The element is a slab.
RoofElement
The element is a roof.
CurtainWallElement
The element is a curtain wall.
CWFrameElement
The element is a curtain wall frame.
CWPanelElement
The element is a curtain wall panel.
CWJunctionElement
The element is a curtain wall junction.
CWAccessoryElement
The element is a curtain wall accessory.
ShellElement
The element is a shell.
SkylightElement
The element is a skylight.
FreeshapeElement
The element is a freeshape.
DoorElement
The element is a door.
WindowElement
The element is a window.
ObjectElement
The element is an object.
LightElement
The element is a Light.
ColumnElement
The element is a column.
MeshElement
The element is a mesh.
BeamElement
The element is a beam.
RoomElement
The element is a room.
OtherElement
The element is none of the above types.

Description

This type describes the type of the 2d element which the model element was generated from. The type names are just for information because they don't affect the validity of the parameters.
