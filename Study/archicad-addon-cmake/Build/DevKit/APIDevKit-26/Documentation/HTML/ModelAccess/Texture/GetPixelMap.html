Texture::GetPixelMap


Get the whole pixel map.


void  GetPixelMap (
    ARGBPixel*  pixelMap
) const;

Parameters

pixelMap
Pointer to the result pixel map buffer.

Description

This method reads the whole pixel map and puts it into the pixelMap buffer. The pixel map size can be obtained using the GetPixelMapSize method. The buffer size can be obtained using the GetPixelMapBufferSize method. If the pixel map's pixel type is not ARGBPixelType then this function will throw an exception.

Available from version: 8
