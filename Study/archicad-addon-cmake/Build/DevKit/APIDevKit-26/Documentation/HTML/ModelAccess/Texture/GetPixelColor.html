Texture::GetPixelColor


Get pixel data from pixel map.


void  GetPixelColor (
    Int32     x,
    Int32     y,
    Color*   color,
    double*  alpha = NULL
) const;

Parameters

x
The x coordinate of the pixel in texture map space. The value range is [0..n] where n = GetPixelMapXSize () - 1.
y
The y coordinate of the pixel in texture map space. The value range is [0..n] where n = GetPixelMapYSize () - 1.
color
Pointer to the resulting color.
alpha
Pointer to the resulting alpha value. It may be a NULL pointer. The default value is NULL.

Description

This method read pixel color and alpha value from pixel map using pixel map coordinate.

Available from version: 8



void  GetPixelColor (
    const TextureCoordinate*  textureCoordinate,
    Color*                    color,
    double*                   alpha = NULL
) const;

Parameters

textureCoordinate
Normalized texture coordinate in texture space. The u and v coordinate range is [0.0..1.0).
color
Pointer to the resulting color.
alpha
Pointer to the resulting alpha value. It may be a NULL pointer. The default value is NULL.

Description

This method reads pixel color and alpha value from a pixel map using normalized texture coordinates. The normalized texture coordinates can be calculated from the polygon point texture coordinate. The GetTextureCoordinate method of the polygon returns the polygon point texture coordinate. This coordinate transformation has to contain the mirroring, randomly shifting, rotating effect and the normalization.

Available from version: 8
