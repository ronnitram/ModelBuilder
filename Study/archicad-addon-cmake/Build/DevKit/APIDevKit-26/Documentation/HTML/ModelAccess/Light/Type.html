Light::Type


Type of the light.


enum  Type (
    DirectionLight,
    SpotLight,
    PointLight,
    SunLight,
    EyeLight,
    AmbientLight,
    CameraLight
);

Values

DirectionLight
This light source emits paralell rays from a circle source.
SpotLight
Direction dependent light source.
PointLight
Direction independent light source.
SunLight
This light source emits paralell rays from infinity distance.
EyeLight, CameraLight
Point light on the camera position.
AmbientLight
Ambient light.

Description

This type describes the type of the light. This type controls which other parameters of the light contain relevant data.
