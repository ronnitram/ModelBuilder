Parameter::SetType


Set the type of the parameter.


void  SetType (
    Type  t
);

Parameters

t
The new type of the parameter.

Description

This method sets the type of the parameter to t. When the type is changed, the old value is lost. This method has to be called befor setting the value, if actual type and the new type of the value are different.

Available from version: 8
