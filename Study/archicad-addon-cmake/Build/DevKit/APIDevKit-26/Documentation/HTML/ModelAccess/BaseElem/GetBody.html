BaseElem::GetBody


Get a body from the base element.


void  GetBody (
    Int32  bodyIndex,
    Body*  body
);

Parameters

bodyIndex
The index of the body. The index range is [1..n] where n = GetBodyNum ().
body
A pointer to the result body.

Description

This method gets a reference to a body of the base element and puts it to the body. If the bodyIndex is out of range, then it throws an exception.
