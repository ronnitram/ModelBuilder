Light::GetAngleFalloff


Get the light angle dependent falloff intensity of spot light.


double  GetAngleFalloff (
    void
) const;

Return value

The light angle dependent falloff intensity.

Description

This method returns the light angle dependent falloff intensity (decrease of intensity depending on the cone angle). This value is valid if the light type is SpotLight.
Here is an example how to calculate the cone angle dependent light color:


// light: The light object.
// angle: The cone angle.
// color: The light color.

color.red   = 0.0;
color.green = 0.0;
color.blue  = 0.0;

if (angle <= light.GetFalloffAngle2 ()) {
    double attenuation = 1.0;
    if (light.GetAngleFalloff () > 0.0) {
        attenuation = exp (light.GetAngleFalloff () * log (cos (angle)));
        if (angle > light.GetFalloffAngle1 ()) {
            attenuation *= (light.GetFalloffAngle2 () - angle) / (light.GetFalloffAngle2 () - light.GetFalloffAngle1 ());
        }
    }

    color.red   = light.GetColor ().red   * attenuation;
    color.green = light.GetColor ().green * attenuation;
    color.blue  = light.GetColor ().blue  * attenuation;
}
