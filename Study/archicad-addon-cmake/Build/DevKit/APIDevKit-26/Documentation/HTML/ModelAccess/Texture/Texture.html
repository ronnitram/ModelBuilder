Detailed description

This object is reference to a texture attibute. The texture has two size parameters. One of them is the size of the texture pixel map, the other is the drawing size in the model space. The pixels of the pixel map can be read using two kind of methods. The first method is, that the whole texture pixel map can be read to a buffer. The second one is, that an individual pixel can be read using pixel map or texture coordinates. The texture coordinates show, how to draw the texture to a surface.
