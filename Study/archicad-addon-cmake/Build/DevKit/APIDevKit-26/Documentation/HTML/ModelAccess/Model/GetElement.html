Model::GetElement


Get a model element.


void  GetElement (
    Int32     elementIndex,
    Element*  element
) const;

Parameters

elementIndex
The index of the element. The index range is [1..n] where n = GetElementNum ().
element
A pointer to the result model element.

Description

This method gets a reference to an element of the model and puts it into element. If the elementIndex is out of range, then it throws an exception.
