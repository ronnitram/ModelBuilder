Model::GetTexture


Get a texture attribute from the model.


void  GetTexture (
    Int32     textureIndex,
    Texture*  textutre
) const;

Parameters

textureIndex
The index of the texture. The index range is [1..n] where n = GetTextureNum ().
textutre
A pointer to the result texture.

Description

This method gets a reference to a texture attribute of the model and puts it into textutre. If the textureIndex is out of range, then it throws an exception.

Available from version: 8
