Model::GetColor


Get a color from the color table.


void  GetColor (
    Int32   colorIndex,
    Color*  color
) const;

Parameters

colorIndex
The index of the color. The index range is [1..n] where n = GetColorNum ().
color
A pointer to the result color.

Description

This method reads a color from the model color table and puts it into color. If the colorIndex is out of range, then it throws an exception.
