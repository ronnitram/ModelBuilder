ArrayParameter::GetStringValue


Converts a string array item to C string.


void  GetStringValue (
    char*  string,
    Int32  bufferLength
) const;

Parameters

string
Pointer to the result string.
bufferLength
Length of the result string buffer.

Description

This method converts a string array item to C string. This method can be used only for string array items.
